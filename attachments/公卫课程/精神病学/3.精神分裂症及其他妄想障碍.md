# 1.概述
**定义**：一组（种）**病因未明**的精神病，多起病于**青壮年**，病程多迁延。常有**感知、思维、情感、行为**等多方面的障碍和精神活动的不协调。一般无意识障碍，是最常见的重性精神疾病之一

**流行病学**
- 患病率：不同国家、地区差异较大
	- 传统的观点：成年人终身患病率大约为1%
	- WHO（access 2025）-精神分裂症影响全球约2400万人，即每300人中就有一人（0.32%）。在成年人中，这一比例为1/222人（0.45%）
	- 全国流行病学调查：终身患病率0.6%（黄悦勤，2019）
- 起病年龄：多起病于**青壮年**，**男早于女**
- 性别差异：**男性更为易感**
	- 全国流行病学调查：大致相等 男：女=6：5（黄悦勤，2019）

**病因及发病机制**
- 遗传因素（最重要且最有证据的因素）
	- 遗传度80%
	- 患病风险随血缘亲近程度逐渐增大
	- 精分患者一级亲属患病率1.4%~16.2% VS 健康对照一级亲属患病率 0.2%~1.1%
	- 单卵双生子同病率显著高于异卵双生子4~6倍
	- 寄养研究发现，母亲患有精神分裂症的子女交给正常家庭抚养，与母亲正常的子女寄养与正常家庭，前者精神分裂症患病率明显高于后者
	- 多基因遗传
- 中枢神经递质功能异常——多巴胺(DA)
	- 多巴胺（DA）功能异常假说
	- 谷氨酸功能异常（不足）假说
	- 5-HT功能异常假说
	- γ-氨基丁酸（GABA）功能降低假说
- 脑结构异常改变
	- 侧脑室扩大约25%
	- 脑体积减少约2%
		- 灰质＞白质
		- 额叶、颞叶、海马为著
- 脑功能改变
	- 额叶功能异常（减退、过度激活）
- 环境因素
	- 孕期异常因素
	- 孕期烟酒、毒物、毒品接触等
	- 产科并发症
- 社会心理因素
	- 应激等负性生活事件
	- 职业、阶层
	- 移民
	- 人格因素

# 2.临床表现
- 可以出现症状学中绝大多数的症状：感知思维、情感、行为障碍的各种表现
- 症状多样、复杂，不同的阶段、不同类型表现差别大
- 分裂的本质——精神活动的不协调
	- 精神活动与环境不协调：脱离现实
	- 精神活动的各方面（知、情、意）不协调：统一性和完整性受损

==**症状归类——三分法**==
- 阳性症状
	- 幻觉
		- 言语性幻听：评论性、议论性、命令性、思维鸣响
		- 幻觉行为
			- 长时间倾听、对话
			- 因为幻听内容而产生的情绪变化——愤怒、大骂、大笑、沾沾自喜等
			- 沉浸状态、发呆、喃喃自语、自语自笑
			- 寻找幻听
			- 命令性幻听影响——不合时宜的行为、冲动、攻击、自伤、自杀
	- 妄想
		- **被害妄想、关系妄想**、嫉妒妄想、钟情妄想、非血统妄想、其他
		- 情绪影响：恐惧、愤怒、喜悦
		- 行为影响：冲动、回避、拒食
	- 被动体验
		- 思维插入、思维被撤走及思维广播
		- 被洞悉感
		- 被控制感
		- 强制性思维
	- 思维紊乱
		- 多为思维形式障碍（思维联想过程缺乏连贯性和逻辑性）
		- 思维松弛——成句子，逻辑松散，费解，无法深入，不知所云，说不到点子上
		- 思维破裂——不成句子，“语词杂拌”
		- 逻辑倒错性思维——推理荒谬
		- 病理性象征性思维——普通的词句、动作等赋予他人无法理解的意思
		- 语词新作：无关概念、字词的拼凑，赋予意义
		- 诡辩证、内向性思维、矛盾思维
	- 紧张症
		- 紧张性木僵和紧张性兴奋交替出现
		- 紧张性木僵：亚木僵、蜡样屈曲、空气枕头
		- 紧张性兴奋：不可理解的冲动行为，行为无目的性
	- 精神运动兴奋与抑制
		- 不协调的精神运动性兴奋：思维紊乱、情感紊乱、行为紊乱
- 阴性症状
	- 思维贫乏
	- 情感平淡/淡漠
	- 意志缺乏
- 认知症状
	- 精神分裂症的特质性改变，涉及多个认知领域，核心又持久，与阴性症状阳性症状密切联系
	- 注意障碍、记忆障碍、工作记忆损害、执行功能障碍、抽象思维障碍、信息整合功能障碍、运动协调功能障碍
- 其他精神症状
	- 自知力障碍：不完整或缺失
	- 人格缺陷
	- 强迫症状、焦虑、抑郁
	- 生物学症状：睡眠障碍、性功能障碍等

**常见综合征**
- 幻觉妄想综合征：一般先出现幻听或其他类型幻觉，然后相继出现相关妄想。幻觉与妄想间相互联系、影响
- 紧张综合征：动作抑制；活动过多；违拗；怪异的随意动作；模仿语言或模仿动作
- 精神自动综合征：假性幻觉、被动体验、继发妄想；互相交织、互相影响

**病程特点——间断发作或持续性**
- 起病情况
	- 70%以上慢性起病：从可疑早期症状到明显精神异常>3个月
	- 部分亚急性起病：从可疑早期症状到明显精神异常2周—3个月
	- 少数急性起病：从正常到明显精神异常<2周
- 70%以上病情波动迁延/多次复发

**前驱症状**
- 大约73%的患者存在前驱症状
- 前驱症状持续一周到几年不等，中位持续时间似乎略低于12个月
- 前驱期表现：从无法形容的内心变化，到新兴趣的发展（例如，在哲学和精神问题上），到愤怒、易怒、焦虑和抑郁，再到社会退缩和角色功能退化。可能有持续几天的短暂、有限的间歇性精神病症状（也称为BLIPS）的发生

**病程特点**
- 进展性、慢性迁延的
	- 精神分裂症的进展性表现在，首发精神分裂症患者反复发作后，病情不断恶化加重，脑组织丢失和治疗抵抗
- 高复发性、高度不依从性
	- 高复发率：首次发作者5年内复发率超过80%
	- 高度不依从：存在不依从性的患者超过80%
	- 中断药物治疗的患者复发风险，是持续治疗患者的5倍

**疾病预后**：五种可能的结局
- 完全持久地恢复正常——少数
- 多次复发但缓解期基本正常——可实现的目标
- 部分缓解，生活自理——半数以上的临床现实
- 维持慢性状态——临床应避免的情况
- 完全步入衰退——少数

**预后影响因素**
- 起病因素：起病诱因，起病急缓，起病年龄
- 临床表现：症状，病程，复发次数
- 治疗方面：未治时间，药物敏感性，维持治疗
- 其他：家族史，病前性格，家庭和社会支持

# 3.诊断与鉴别诊断
**常用量表**
- 诊断性量表：简明国际神经精神障碍访谈（MINI）及DSM-5障碍定式临床检查（SCID）
- 评估疾病严重程度及临床症状：临床总体印象量表（CGI）、阴性与阳性症状量表（PANSS）、简明精神病评定量表（BPRS）
- 评估药物不良反应：药物不良反应量表（TESS）

**临床分型**
- 偏执型（妄想型）：幻觉妄想为主，最常见，预后较好
- 青春型（瓦解型）：“精神活动的全面瓦解”即：青春性兴奋（不协调性精神运动 性兴奋）
- 紧张型：紧张症综合征，预后最好
- 单纯型：思维贫乏、情感淡漠、意志缺乏，预后最差
- 未分化型（未定型）：症状多样难归类
- 精神分裂症后抑郁：1年内曾确诊为精神分裂症，目前以典型的抑郁为主要表现

**鉴别诊断**
- 抑郁发作
- 躁狂发作
- 分裂情感性精神障碍
- 强迫症
- 创伤后应激障碍
- 妄想性障碍
- 人格障碍

# 4.治疗
**方法**：药物治疗为主的综合措施
- 药物治疗
	- 第一代抗精神病药物
	- 第二代抗精神病药物
- 物理治疗
	- MECT（无抽搐电休克治疗）
	- rTMS（重复经颅磁刺激治疗）
- 社会心理干预
	- 支持性心理治疗
	- CBT
- 康复治疗
	- 医学康复
	- 心理康复
	- 社会康复
	- 职业康复

**抗精神病药的治疗作用**
- 缓解阳性症状
- 改善阴性症状
- 改善至少不损伤认知功能
- 长期治疗防止复发

**抗精神病药物分类**
- 第一代抗精神病药物FGAs（传统药物）
	- D2受体强拮抗剂（神经阻滞剂）
	- 氟哌啶醇、氯丙嗪
	- 仅对阳性症状有效，对阴性症状作用无效（甚至加重）
	- 不能有效改善认知缺陷（甚至加重）
	- **锥体外系副反应（EPS）总体较重; 迟发性运动障碍（TD）风险高**
	- 便宜（月治疗费用在几块~几十块不等）
- 第二代抗精神病药物SGAs（非典型药物）
	- D2受体部分激动作用、5-HT受体部分激动剂
	- 利培酮、奥氮平
	- 不仅对阳性症状有效，能改善阴性症状
	- 可改善认知缺陷
	- **锥体外系副反应（EPS）总体较轻; 迟发性运动障碍（TD）风险低**
	- 贵（原研药物月费用几百~几千；国产几十~几百）

**抗精神病药物的选择**
- 抗精神病药物的选择取决于药物的不良反应、患者偏好、共患的躯体疾病以及与其他药物的潜在相互作用
- 各种抗精神病药物（除氯氮平外）之间疗效差异证据不足，不良反应差异显著，抗精神病药物选择时重点考虑不良反应差异(1A)。首发精神分裂症患者倾向于推荐除氯氮平外的第二代抗精神病药物(1B)
- 复发患者重点参考既往用药情况

==**抗精神病药物常见不良反应**==
- 锥体外系综合征(Extrapyramidal Symptoms, EPS)
	- 急性EPS
		- 急性肌张力障碍
		- 静坐不能
		- 帕金森综合征
	- 迟发性运动障碍TD
- 代谢综合征(Metabolic syndrome, MS)
	- 四高：血压、血糖、血脂、体重高
- 高泌乳素血症(Hyperprolactinemia)
	- 泌乳、闭经、性功能障碍
	- 骨质疏松、乳癌
- 镇静

**罕见不良反应**
- 恶性综合征(neuroleptic melignant syndrome, NMS)：NMS四联症
	- 肌张力障碍
	- 高热
	- 意识障碍
	- 自主神经系统症状

**药物治疗原则**
- 一旦确诊，即开始药物治疗
- 个体化选择用药，平衡疗效及不良反应
- 优先选用既往有效药物
- 依从性差者考虑长效针剂
- 小剂量起始，根据病情和治疗场所调整滴定速度
- 尽量单一用药
- 足量足疗程治疗
- 长期维持治疗

**疗效判断**
- 症状全部消失
- 自知力完全恢复
- 社会功能复原
- 其中，药物的疗效占70%左右，其他依靠心理治疗、康复训练，家庭和社会支持

**心理治疗**
- 改善症状
- 恢复自知力
- 增加并保持治疗依从性
- 矫正性格缺陷
- 认知行为调整
- 应对歧视偏见

**康复治疗**
- 医学康复：精神疾病的治疗，防复发，健康维护
- 生活技能：独立生活，尽可能减少他人的照顾
- 休闲娱乐：在不妨碍他人的情况下，愉快地活着
- 心理康复：应对压力，积极乐观
- 社会康复：有机会，有能力
- 职业康复：工作+学习
