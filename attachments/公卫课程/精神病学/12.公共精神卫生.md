# 1.公共精神卫生的概念与起始
**公共卫生的概念**
- 一门通过有组织的社区活动来改善环境、预防疾病、延长寿命和促进心理和躯体健康，并能发挥个人更大潜能的科学
- 包括环境卫生、控制传染病、进行个体健康教育、组织医护人员对疾病进行早期诊断和治疗，发展社会体制，保障公民都享有足以维持健康的生活水平和应有的健康与寿命
- WHO于1952年采纳了这一定义
- 中国：组织社会共同努力，改善环境卫生条件，控制传染病和其他疾病流行，培养良好的卫生习惯和文明的生活方式，提供医疗卫生服务，达到预防疾病，促进人民身体健康的目的
- 特点：成本低、效果好，但社会效益回报周期相对较长

==**预防疾病与健康促进**==
- 预防疾病：3个层面，三级预防
- 健康促进：3个层面
    - 一级促进：针对整个人群健康和福祉
    - 二级促进：针对高危人群采用特定方法
    - 三级促进：针对已病人群康复和复发预防
- 关注健康和福祉的决定性因素而非疾病本身，可以发生在个体、社区和体制等不同层面
- 目标是改善个体福祉，促进更健康和更可持续发展的社区，改善环境以促进健康，实现政策和法律等体制改变，进而提高服务的可及性和公平性

**公共卫生与医疗服务的区别**

| 公共卫生              | 医疗服务         |
| ----------------- | ------------ |
| 关注人群的健康           | 关注个体的健康      |
| 关注服务的公平性、可及性、社会效益 | 关注诊断准确性、治疗效果 |

**公共精神卫生的起始**
- 传染病：控制传染源，切断传播途径
- 麻风病院（隔离）——精神病院
- 精神科量表、诊断系统（ICD&DSM）
- 2000年，WHO：“没有精神健康就没有健康”；“健康不仅仅是没有疾病或虚弱，而且还应有身体的、精神的良好状态和社会适应能力”

**中国公共精神卫生的起始**
- 1998年，卫生部医政司——疾病控制司
- 2000年，开始起草《中国精神卫生工作规划（2002-2010年）》
- 2002年，卫生部成立中国CDC精神卫生中心，挂靠北大六院，承担国家级公共精神卫生工作职能
- 2003年，SARS事件，财政部投入37亿启动中央转移支付地方公共卫生项目，共立项200多个
- 2004年8月，北大医院幼儿园事件
- 2004年底，686项目立项（重性精神疾病监管治疗项目）
- 2006年5月，卫生部在疾控局内设立精神卫生处

# 2.公共精神卫生服务的内容与需求
## 2.1公共精神卫生服务内容
### 2.1.1精神疾病预防
==**精神疾病预防**==
- 一级预防（病因预防）
    - 针对广大人群的一般性干预
    - 针对某种疾病高风险人群（例如精神分裂症患者的子女）的选择性干预
    - 针对有体征或症状可能发展为某一精神疾病，但当前并不符合该疾病诊断人群的指征性干预
- 二级预防：早发现、早诊断、早治疗
- 三级预防：防止伤残、促进功能恢复和预防复发
- 消除耻感和社会歧视：公共精神卫生的一个核心内容

**一级干预的效果**
- 采用教育、心理治疗、药物治疗和生活方式及营养改善的综合预防性干预措施，可降低抑郁症发病率（下降约25%）
- 在学校开展情感恢复力、自信和认知问题解决技能的干预，能预防焦虑障碍的发生
- 改善儿童、父母、同龄人和老师的社会能力，促进亲社会行为的一般性干预措施能预防年轻人的品行障碍
- 物质滥用障碍可通过一般性政策得以预防，通过加税、禁止所有形式的直接和间接广告、全面限制等措施降低酒精和毒品的可及性
- 对精神疾病患者子女开展选择性干预，同时对患病父母进行相关培训，能使子代患精神疾病的风险下降40%
- 一般性和选择性干预在精神病性障碍预防中的作用仍在进一步探索
- 指征性干预和尽早识别可以缩短未治期，一定程度上减轻疾病负担

**疾病预防：预防自杀**
- 普遍性干预方法
    - 培训和教育
    - 要求媒体正确报道自杀新闻并提供心理帮助联系方式
    - 限制毒药和枪支等高致命性自杀工具的获得
- 选择性干预方法
    - 针对有自伤史、自杀行为史、严重自杀意念以及重性精神疾病患者等高危人群开展

**流行病学调查与长期纵向研究**
- 积极开展精神障碍流行病学调查和长期纵向研究，建立全民信息库
- 获得人群精神疾病患病水平、疾病危险因素和保护因素的信息，才能设计和提供更有针对性的干预
- 需要关注
    - 精神疾病共病慢性躯体疾病
    - 早年生活经历，如营养不良、生产创伤、母亲抑郁等
    - 贫困、教育程度低、失业、高负债、社会隔离和重大生活事件等

### 2.1.2精神健康促进
==**精神健康促进**==
- 生的健康
    - 给新生儿和幼儿一个良好的人生开端
    - 生产前、中和后对父母心理和躯体方面的支持
    - 例如，减少母亲吸烟与降低新生儿行为问题和ADHD相关
- 长的健康
    - 帮助儿童形成良好的心理状态
    - 学前和早期心理社会干预能改善儿童认知技能和阅读能力，提高学业，预防情绪或品行障碍的发生
    - 学校精神健康促进项目能降低抑郁、焦虑、攻击、强迫和反社会行为，预防和降低早期性行为、酒药滥用和暴力行为等
- 活的健康
    - 通过改善住房、减少贫困、加强理财能力、丰富社会活动、保持锻炼、促进邻里和睦、积极心理干预等多个方面实现
- 工作健康
    - 针对职场人群，目的是获得自我价值感和效力感
    - 干预管理者，改善其精神健康意识和应激风险管理技能
    - 干预环境，改善工作条件、调整工作量、更灵活的工作时间
    - 干预员工，提供交流与培训，减少因病缺勤，降低焦虑抑郁发生
- 老的健康
    - 旨在提高老年人的社会接触，预防社会隔离

## 2.2公共精神卫生服务需求
**精神卫生服务的总体需求**
- 人群患病率
- 疾病负担（致残率，病死率等）
- 不同阶段不同时期有所变化
	- 老龄化、物质滥用、性取向的社会接纳度

**不同人群的需求不同**
- 患者
- 家属
- 服务提供者
- 大众
- 重点人群：儿童、老年、妇女、职场人群、流动人口

# 4.精神卫生服务现状与发展
## 4.1WHO推荐的精神卫生服务系统
**五级滤过模式**
- 第一级：整个人群
- 第二级：全部精神问题人群
- 第三级：被确诊的精神疾病患者
- 第四级：在精神卫生服务机构就诊的人群
- 第五级：在精神科住院治疗的人群

## 4.2部分国家的精神卫生服务
以社区为基础的精神卫生体系是国际精神卫生服务模式的主流和发展趋势
**美国**
- 发源地；去机构化运动；替代性的精神卫生系统
- 全美精神科床位：50年代56万张，1982年22万张，2010年6.3万张（2张/万人口）
- 60年代，普遍建立社区精神卫生中心CMHC
- 1977年，NIMH策划实施Community Support Program

**意大利**
- 采取社区为中心精神卫生服务模式的典型国家
- 1978年，180法案，禁止新收非强制治疗患者入院，逐渐关闭所有76所公立精神专科医院；在综合医院设不超过15张病床的精神科提供急性住院治疗服务，平均住院日7-14天
- 人力资源较丰富，每十万人精神科医生6.4名，精神科护士17名
- 建立大量社区精神卫生中心，平均每8-10万人设一个
- 社区服务“奢侈”，患者出住宿/日间康复机构难——“家”

**澳大利亚**
- WHO在西太平洋地区推荐的模式
- 吸取美国经验教训，逐步调整关闭精神病院、残疾人院，缩小护理院和养老院，大力发展社区服务和支持，将精神卫生服务分派到各地综合医院精神科牵头负责
- 以社区为基础划分服务区（大约20万人口）
    - 区域精神卫生中心
    - 社区卫生中心
    - 全科医生诊所
    - 综合医院精神科
    - 住宿机构和NGO服务
- 全方位照护；多学科团队；针对不同人群的多种类型项目

**日本**
- 以住院治疗为主提供精神卫生服务的代表国家
- 精神科床位数量居世界之首：2010年，日本精神科床位27张/10万人口，大多数为慢性住院治疗床位，平均住院日301天，67%的患者平均住院超过1年
- 1974年开始日间治疗服务，目前有职业康复机构、福利工厂、社区支持中心、中途宿舍等

**国际精神卫生服务发展趋势**
- 医院社区一体化连续性服务，全社会参与
- 区域化管理，全方位服务，多渠道筹资
- 急性期短期住院治疗
- 社区管理、随访：分级管理
- 多学科治疗/支持团队
    - 精神卫生从业人员
    - 生活、就业、司法等政府部门人员
    - 非政府组织

## 4.3中国精神卫生服务发展
**中国精神卫生服务发展：4个重点项目**
- 2004年：严重精神障碍管理治疗项目
	- 医院社区整合、连续的服务
- 2009年：国家基本公共卫生服务项目
	- 从部分地区试点到覆盖全国
- 2015年：精神卫生综合管理试点项目
    - 建立多部门协作工作机制，整合资源
- 2018年：社会心理服务体系建设试点项目
    - 从疾病到健康，从患者到公众

**国家基本公共卫生服务**
- 2009年启动
- 针对对居民健康影响大、具有普遍性和严重性的主要公共卫生问题
- 实施：乡镇卫生院、村卫生室和社区卫生服务中心（站）
- 服务内容由国家为城乡居民免费提供
- 经费由政府承担
- 人均基本公卫服务经费从2011年的25元提高到2020年74元

**国家基本公共卫生服务项目内容：14项**
- 城乡居民健康档案管理、健康教育、预防接种、0～6岁儿童健康管理、孕产妇健康管理、老年人健康管理
- 慢性病患者健康管理（高血压、糖尿病）、严重精神障碍患者管理、肺结核患者健康管理
- 中医药健康管理
- 传染病及突发公共卫生事件报告和处理服务
- 卫生计生监督协管

**全国精神卫生综合管理试点**
- 国家6部委和组织发文（国卫疾控发〔2015〕57号）
	- 卫生计生、综治、公安、民政、人社、残联
	- 《关于开展全国精神卫生综合管理试点工作的通知》
- 三年连续工作：2015－2017

**全国综合管理试点城市（40个）要求**
- 健全市县两级综合管理机制，70%乡镇建立综合管理小组
- 患者管理率75%、治疗率65%，个案服务率90%，提高发现登记
- 乡镇级康复机构覆盖率60%（东部）、30%（中西部）
- 贫困患者自付比例不超过10%
- 乡镇街道设专兼职精防人员
- 知识知晓率城市80%、农村70%
- 社会组织和志愿者
- 自选动作
    - 提高报销比例
    - 医疗和康复全程服务
    - 鼓励政策和制度创新
    - 加强心理健康服务等

**综合与创新**
- 综合
    - 多部门工作的综合：落实在基层
    - 救治救助：综合多渠道资金
    - 管理服务流程：筛查、诊断、治疗、康复
    - 信息交换
- 创新
    - 降低患者自付费用，一站式服务
    - 建立社会康复体系网络
    - 工作方式、方法
    - 加强能力建设
    - 流动人口管理

## 4.4社会心理服务体系建设试点
**工作指标**
- 社区
    - 村（社区）设心理咨询室/社会工作室，建成率≥80%
- 学校
    - 高等院校普遍设心理健康教育与咨询中心（室），健全队伍
    - 中小学设心理辅导室，配专/兼职教师，创特色学校
- 机关企事业
    - 党政机关和厂矿、企事业单位、新经济组织等，设心理健康辅导室/购买服务
- 机构
    - 100%专科医院设心理门诊
    - 40%≥二级综合医院设心理门诊
    - 培育社会心理服务专业机构
    - 建24小时公益心理援助平台
    - 建心理危机干预队伍

**重点任务**
- 基层心理服务平台搭建
- 教育系统心理服务网络完善
- 机关和企事业单位心理服务网络健全
- 社会心理服务机构规范发展
- 医疗机构心理健康服务能力提升
- 心理援助服务平台建立健全
- 心理健康科普宣传网络健全
- 严重精神障碍患者服务工作机制完善

**加强心理服务人才队伍建设**
- 培育心理咨询人员队伍
- 发展心理健康领域社会工作专业队伍
- 健全社会心理服务行业组织，加强管理
- 组建心理健康服务志愿者队伍
- 发展医疗机构心理健康服务队伍
