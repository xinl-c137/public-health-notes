# 1.概述
==**性传播疾病(Sexually transmitted disease, STD)**==
- 由多种不同类型的病原体引起
- 以性行为接触或类似性行为接触为主要传播途径
- 可引起泌尿生殖器官及其附属淋巴系统病变的一类疾病，也可导致全身主要器官的病变

**性传播感染(Sexually Transmitted Infection, STI)**
- 通过人与人之间的性接触而传播的感染，包括有临床症状的疾病和无症状感染

**现状**
- 无症状感染者是极易被忽视的传染源，导致STI更加广泛的传播
- 传统传染病监测报告体系基于STD病例报告，是STI流行状态的主要资料来源
- 无症状感染难以识别，有症状感染管理是最为经济有效的STI管理模式

四种可治愈的性传播感染：滴虫病、淋病、衣原体、梅毒

**病原学**
- 30+种病原体，包括细菌、病毒、寄生虫等
- 随着STD概念的外延，范围扩大到“能通过性接触传播的病原体所引起的疾病”
	- 病毒性肝炎
	- 细菌性和阿米巴性痢疾
	- 生殖道支原体
	- 滴虫病
	- 细菌性阴道炎
	- 生殖器疱疹
- 常见性传播疾病及其病原体
	- 细菌性感染：**淋病、衣原体感染**、性病性淋病肉芽肿、**梅毒**、软下疳、腹股沟肉芽肿
	- 病毒性感染：**艾滋病、生殖器疱疹、生殖器疣、病毒性肝炎**、巨细胞病毒感染、传染性软疣、卡波西肉瘤
	- 原虫感染：**滴虫病**
	- 真菌感染：念珠菌病
	- 寄生虫侵袭：阴虱病、疥疮
- 8种常见的STIs：淋病、衣原体感染、梅毒、艾滋病、生殖器疱疹、生殖器疣、病毒性肝炎、滴虫病
- 相关法律法规
	- 1970s：4种经典性病：梅毒、淋病、软下疳、性病淋巴肉芽肿
	- 1991《性病防治管理办法》8种：增加艾滋病、非淋菌性尿道炎、尖锐湿疣、生殖器疱疹
	- 2013《全国性病监测方案（试行）》：5种重点监测和防治
		- 《传染病防治法》乙类传染病：淋病、梅毒
		- 《性病防治管理办法》：生殖道衣原体感染、尖锐湿疣、生殖器疱疹
	- 《艾滋病防治条例》：艾滋病

**危害**
- 劳动力人口急剧减少，人口比例失衡，人均寿命降低
	- 全球15-49岁人群2008每年新发4.98亿可治愈的STD
	- 主要分布在南亚和东南亚、撒哈拉以南的非洲地区、拉丁美洲和加勒比海地区
- 直接影响经济发展
- 加重国家财政负担
- 许多性传播疾病感染后没有明显的临床症状，容易造成治疗延误，导致严重的并发症
	- 宫颈癌
	- 盆腔炎、异位妊娠、不孕症等
	- 艾滋病风险：梅毒、淋病和单纯疱疹病毒
	- 生理、心理和社会后果严重影响生活质量
	- 胎儿和新生儿死亡：结膜炎、角膜瘢痕、失明，死胎，低出生体重，肺炎，先天梅毒

**流行概况**
- 流行趋势
	- 全球：WHO：1990年2.5亿新发病例，1995年3.3亿，2005年4.5亿
	- 我国
		- 新中国成立前：以梅毒和淋病为主，1000万病人，患病率高
		- 新中国成立后：采取系列措施，发病率下降，1964年基本消灭性病
		- 1977年：疫情上升，地区扩大，发病人口增多；从沿海到内地，城市到农村，社会向家庭蔓延
		- 2001年疫情开始下降，较2000年下降7.38%；梅毒发病例数上升，淋病报告病例持续下降
- 地区分布
	- 分布差异大，南亚和东南亚最为严重，每天≥40万新病例
	- 发展中国家>发达国家
	- 发展中国家育龄妇女疾病、死亡和健康生命损失的第二大原因，进一步造成胎儿和新生儿死亡
	- 我国地区分布的特点
		- 沿海＞内地
		- 经济发达地区＞经济落后地区
		- 城市＞农村
		- 发病率高的地区（城市）：珠江三角洲、长江三角洲、闽江地区、东北三省、京津地区、重庆地区
		- 流动人口导致农村STD的病例报告数增加
		- 边远地区的少数民族
		- 注意各地区在STD发现、登记和报告方面的系统完整性和监测能力的差别
- 人群分布
	- 年龄分布：性活跃人群
		- 美国15-24岁约占新发病例的50%
		- 我国20-39岁年龄组占67%
	- 性别
		- 报告数和发病率均男性＞女性，但女性增长速度快，病例的性别比逐渐缩小
		- 2008年监测结果
			- 男女性发病率接近，1.14:1，与女性主动就诊增加、诊断水平提高、或者女性发病人数增加
			- 首位：梅毒，第二位（男性淋病，女性衣原体感染）

# 2.流行过程
**传染源**
- 无论有无症状，只要体内有性传播疾病病原体生长繁殖并能通过直接性接触或间接接触感染他人者都是STD的传染源
	- 无症状、临床表现不典型
	- 社会舆论、心理压力
	- 非正规治疗，导致病情迁延、感染加剧
- 传染源的高危人群
	- 性乱者：最重要的传染源
		- 商业性行为者：卖淫、嫖娼者
		- 男男性行为者MSM：多性伴的双性性行为者
		- 特殊职业和流动人群
	- 吸毒人群：静脉注射
	- 特殊人群
		- STD患者的性伴和配偶
		- 收容审查、看守和劳教劳改人员等
		- 献血者和输血者：AIDS、HBV、HCV等

**传播途径**
- 性行为传播：主要
	- 直接性接触：异性、同性及双重性接触
	- 接吻、触摸等
- 非性行为的直接接触传播：破损皮肤
- 医源性传播和血液传播
	- 污染的器械
	- 污染的生物制品或药品
- 垂直传播：梅毒、淋病、AIDS、HBV、衣原体感染等
- 日常生活接触传播

**易感人群主要特点**
- 普遍易感，没有年龄、性别差异
- 无先天免疫和后天获得性免疫，可出现反复感染、迁延不愈、反复发作和多重感染
- 控制其他种类STD，有助于降低普通人群中的HIV感染率

==**影响流行过程的因素**==
- 生物学因素
	- 病原体种类繁多，临床表现不同
	- 多为无症状感染或感染后临床表现不典型，造成漏诊、漏治，增大了流行范围
		- 淋病：50%~80%为轻型和不典型
		- 衣原体感染：70%女性无明显症状
		- 耐药菌株的出现
	- 人群缺乏免疫力，普遍易感，可发生反复感染、间歇性排菌甚至多重感染
	- 无有效的人工免疫方法，化学预防尚未普及
- 社会学因素
	- 性活跃人群的人口迁徙、流动
	- 健康教育不充分
	- 吸毒、贩毒
	- 商业性行为
	- 性病医疗市场混乱
	- 性行为因素：同性、异性、多性伴

# 3.艾滋病
**概述**
- 获得性免疫缺陷综合征Acquired Immune Deficiency Syndrome, AIDS
	- 由人类免疫缺陷病毒（human immunodeficiency virus, HIV）感染引起的以**T细胞免疫功能缺陷**为主的、病死率极高的慢性传染病
- HIV本身不直接致病，但导致人体失去抵抗能力而感染其他导致死亡

**病原学特征**
- 逆转录病毒科慢病毒属
- 包膜：糖蛋白刺突三聚体（gp120+gp41）、内膜（p17）
- 核心
	- 两条相同的单股RNA链
	- 核心结构蛋白
	- 酶：逆转录酶、整合酶、蛋白酶、RNA酶
- HIV基因
	- 9.2~9.7 kb
	- 3个结构基因（gag，pol，env），编码基质蛋白质、衣壳蛋白、核酸结合蛋白、蛋白酶、逆转录酶、表面亚单位蛋白和一小部分跨膜蛋白
	- 6个调节基因（tat，rev，nef，vif，vpu/vpx，vpr）
	- 5’和3’端长末端序列
- 型别和变异
	- HIV-1：引起全球流行
	- HIV-2：在西非呈地域性流行
	- 逆转录酶转录时高度错配，变异频繁，以env最为高

==**艾滋病的分期**==
- 急性感染期
	- 急性反应：表现为发热、皮疹、淋巴结肿大、乏力
	- ==**窗口期**==
		- 在感染开始的一段时间内，虽然感染者体内有HIV存在并具感染性但血清中尚不能检测到HIV抗体
		- 大约持续**2周~3个月**
		- 此时怀疑有HIV感染者，如果呈阴性，需要等2周~3个月以后复查
- 无症状感染期（潜伏期）
	- 没有任何临床症状，具有传染性
	- 2~10年，最长15年，这段时期是重要的传染源
- 艾滋病前期
	- 开始出现与AIDS有关的症状和体征
	- 出现细胞免疫缺陷
- 典型的艾滋病期
	- 严重的细胞免疫缺陷
	- 致命性机会性感染、出现综合病症导致死亡

==**危害**==
- 影响人口质量和人口结构：中青年、女性
- 贫穷和不平等
	- 经济负担：穷人、受教育程度低、非职业技术人群、女性
- 危害家庭
- 偏见和歧视
- 卫生服务
- 社会经济和政治

**传染源**：HIV感染者和艾滋病病人
- 急性感染期：传播能力最强
- 无症状感染期：传播能力较弱，但此时期长，体内病毒载量变化大，感染者活动自如，传播危险性仍然很大
- 艾滋病病人：体内病毒载量高，传播能力强，但传染源的行为活动能力相对较弱，传播能力相对降低
- 药物对传播能力的影响：与病毒载量有关

==**传播途径**==
- HIV主要存在于体液中
	- 例如，感染者的血液、精液、阴道分泌物、乳汁等
- 传播途径
	- 同性及异性的性接触
	- 医源性传播和血液感染
		- 静脉输入被感染的血液
		- 血液成分或血液制品
		- 静脉注射毒品
	- 母婴垂直传播：妊娠、分娩和哺乳

**人群易感性**
- 人群对HIV普遍易感
- 感染后，可产生一定的免疫力，但这种免疫力不能将HIV清除

**流行概况**
- 全球
	- 2009年WHO/UNAIDS估计
		- HIV感染者3300万，其中成年人3080万，女性占51.6%
		- 新发感染：260万
		- 死亡：180万
	- 2011年底：3400万
	- 目前疫情得到遏制并开始逆转，每年新发病例减少，只有东欧和中亚地区是目前唯一保持增长的地区
	- 地区差异明显：撒哈拉以南的非洲地区占2/3，亚洲及太平洋地区（除泰国）的感染率较低，印度是亚洲HIV感染人数最多的国家
- 中国
	- 1985年：发现首例艾滋病病例
	- 1989年：发现经性途径感染HIV的国内感染者
	- 1989年：从注射毒品者中发现HIV感染者
	- 1995年：发现经母婴途径传播的HIV感染
	- 2009年：HIV感染和艾滋病患者74万（后者10.5万人），感染率0.057%
	- 2011年底，UNAIDS/WHO 《2011中国艾滋病疫情估计》
		- HIV感染和AIDS患者：78万人（后者15.4万人）
		- 女性：28.6%
		- 新发：4.8万人；死亡：2.8万人
		- 感染率：0.058%
- 流行的三个阶段
	- 第一阶段（1985-1988）：输入散发期
		- 病例高度分散，境外输入性病例
		- 集中在沿海地区，7个省有病例报告
		- 除4例血友病病人为国内感染者外，其它均外国人或海外华人（15/19）
	- 第二阶段（1989-1994）：扩散（局部流行）期
		- HIV/AIDS扩散到21个省
		- 经性传播途径：沿海和大城市
		- 吸毒人群：云南德宏，瑞丽市
		- 伴有输血和单采血浆
	- 第三阶段（1995至今）：广泛流行期
		- 静脉注射吸毒：云南、广西、新疆、四川等地区，快速扩散到大部分地区
		- 非法采供血人群：中部省份，经人口流动和第二代转播
		- 性乱人群：东南沿海地区和部分中心城市
		- 开始由高危人群向一般人群扩散，HIV感染者开始进入发病期
		- 31个省报告发现HIV/AIDS
		- 已在28个省报告在注射毒品者中发现HIV/AIDS
		- 大部分省均报告发现经性途径传播的HIV/AIDS
		- 母婴传播病例已经发现
- 现阶段的流行特征
	- 疫情保持低流行态势，部分地区和高危人群疫情严重
		- 截至2013年底，43.7万人，0.33‰
			- 艾滋病病毒感染者26.3万人
			- 艾滋病病人17.4万人
		- 超过1w人省份：云南、广西、四川、河南、新疆、广东、重庆、湖南和贵州
		- 高危人群艾滋病病毒抗体阳性率较高，注射吸毒人群为6.3%，男性同性性行为人群为7.3%
	- 报告存活HIV和AIDS持续增加，艾滋病人在不同人群中的流行呈现较大差异
	- 艾滋病病人数明显增加，全死因死亡人数增加
		- 随着病程进展，存活的艾滋病病毒感染者和病人进展到临床期病人的比例增加
		- 随着存活者的年龄自然增长和每年新发现的晚期高年龄组病人比例增加，使得存活的高年龄组病人所占比例升高
	- 性传播持续成为主要传播途径，男性同性传播增加明显
	- 15-24岁青年学生、50岁以上老人、低档暗娼

# 4.预防策略与措施
**预防策略**
- 全民健康覆盖
	- 提供更广泛、更优质和更可及的基本卫生干预措施和服务
	- 公平、合理、降低成本
- 完整连续的STDs防治服务：从预防、诊治到随访
- 公共卫生方针：借助立法、监管和政策改革

==**预防措施**==
- 针对传染源的措施
	- 积极发现病人、管理病人
		- 发现病例筛查措施
			- 自愿咨询和检测（VCT）
			- 医疗机构医务人员主动提供HIV检测咨询（PITC）
			- 必要时采用强制检查
		- 病人的管理：积极规范治疗
			- 密切随访观察，提供医学、心理咨询，采取综合防治措施
			- 建立良好的医患关系
			- HAART，高活性抗病毒治疗
			- “四免一关怀”缓解患者的经济压力
	- 流行病学治疗和规范化的STD诊疗
		- 流行病学治疗
			- 基于感染危险性而不是基于诊断的治疗，如果感染的危险性很高，即使是其性伴被感染的情况尚未证实，也需要接受必要的治疗
			- 例如：性交易者等高危人群、与STD患者有接触史者、HIV/AIDS的密切接触者
		- 规范化的STD诊疗：减少STD的耐药问题
- 针对传播途径的措施
	- 改变不安全性行为，预防和控制经性接触传播的STD和AIDS
		- 健康教育和健康促进活动
		- 例如：同伴教育、流动宣传、电话咨询等
	- 切断医源性感染、经血传播和日常生活接触传播
	- 防止母婴传播
- 针对易感人群的措施
	- 加强宣传教育、提高群众自我保护意识
		- 性病健康教育：最为经济有效的方法
		- 在青少年中开展早期性教育、普及性卫生知识
	- 预防接种
		- 疫苗研究和应用困难
		- HPV疫苗预防宫颈癌开始在各国陆续应用
- 监测
	- ==目的和意义==
		- 及时了解STD的流行动态，了解其传染的来源，调查各方面的影响因素，考核防治效果，为制定防制措施提供依据
		- 及早发现、彻底治疗并追踪检查其性伴
	- STD监测范围：淋病、梅毒、生殖道衣原体感染、尖锐湿疣、生殖器疱疹
	- 监测点及监测系统的完善
		- STD监测
			- 法定传染病报告系统
			- STD哨点监测系统（5种）
