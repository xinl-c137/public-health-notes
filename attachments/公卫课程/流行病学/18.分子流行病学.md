# 1.概述
**分子流行病学产生背景**
- 疾病防治中出现的新问题
	- 传染性疾病：耐药性病原体；病原体变异；新发传染病
	- 慢性非传染病：病因学研究中出现存在遗传的异质性、基因多效性、拟表型、基因-基因，基因环境交互作用……
	- 人群易感性或治疗反应：生物个体间的遗传和环境不同，对于疾病的易感性或者治疗的反应存在个体差异
	- 机制验证需求：基础医学提出的微观发病机制与假设，亟需在人群层面进行验证与应用，而传统流行病学手段难以在人群中进行验证
- 分子生物学理论和技术的迅速发展
	- 理论突破：比如DNA双螺旋结构的发现、遗传中心法则的确立、RNA逆转录、表观遗传修饰、微小RNA理论的提出，深化了人类对生命本质的理解
	- 技术革新：高通量测序技术、一代测序、二代测序、生物芯片、质谱技术等 的快速迭代，让大规模、低成本地获取个体基因与蛋白等信息成为可能

**分子流行病学的发展历程**
- 国外分子流行病学概念的演变
	- 1972年，提出“流感的分子流行病学”的名词
	- 1977年，分子流行病学是“应用精细技术进行生物材料的流行病学研究”
	- 1982年，提出“癌症分子流行病学”
	- 1986年定义：分子流行病学的核心在于“把先进的实验室方法和分析流行病学结合起来，从而查明环境和（或）宿主病因”
	- 1993年定义：分子流行病学是“在流行病学中应用生物标志或生物学测量”
	- 1996年定义：分子流行病学狭义上讲是“测量作为暴露和效应的生物标志——信息大分子：DNA、RNA和蛋白质”
- 我国分子流行病学的发展
	- 20世纪80年代初开始进行分子流行病学研究，当时仅限于传染病，如对轮状病毒腹泻、大肠杆菌腹泻等的研究
	- 90年代中后期，分子流行病学研究开始得到了长足的发展，逐渐推广到流行病学的各个领域
	- 1997年第四届中华流行病学会委员会首次设立分子流行病学学组，次年召开了第一次全国分子流行病学学术会议

==**分子流行病学的定义**==
- **分子流行病学 (molecular epidemiology)** 是传统流行病学与新兴的**生物学技术**相结合而产生的一个新分支
- 从**分子水平**阐明疾病发生、发展的规律及其影响因素，有效解决传统流行病学遇到的一系列困难，使疾病的预防及干预手段更加先进、有效
- 分子流行病学是阐明**人群和生物群体中医学相关生物标志**的分布及其与疾病/健康的关系和影响因素，并研究防治疾病、促进健康的策略与措施的科学
- **生物标志(biological markers或biomarkers)** 主要指从暴露到疾病这个连续过程中可测量的、能反映功能或结构变化的细胞、亚细胞、分子水平的物质

# 2.生物标志
**暴露标志**：与疾病或健康状态有关的暴露因素的生物标志。包括外暴露标志、内暴露标志和生物有效剂量标志
**效应标志**：宿主暴露后产生功能性或结构性变化，并进一步引起疾病亚临床阶段和疾病发生过程的生物标志。包括早期生物效应标志、结构和（或）功能改变标志、临床疾病标志等
**易感性标志**：指宿主对疾病发生、发展易感程度的生物标志

## 2.1暴露标志
==**外暴露标志External exposure marker**==
- 是指暴露因素进入机体之前的标志和剂量
- 如病毒、细菌、生物毒素、吸烟烟雾浓度、环境物质（有毒元素、化学物质、大气污染）、饮食因素等

**内暴露标志**
- 指被宿主吸收的外源性暴露物质的量
- 包括对化学毒物、饮食中的营养素和可能致癌物以及微生物感染等多种指标的测定，如血或尿液中的尼古丁、烟草代谢产物可替宁、血液中的营养素、血清或脂肪组织中的杀虫剂DDT等

**生物有效剂量标志**
- 指经吸收、代谢活化、转运，最终与靶组织细胞内DNA或蛋白质相互作用的外源性物质或其反应产物的含量
- 包括DNA 加合物、蛋白质加合物、DNA 蛋白交联物等

| 生物学标志        | 环境暴露物质     | 生物材料 |
| ------------ | ---------- | ---- |
| DNA 加合物      | 苯并（a）芘、亚硝胺 | 白细胞  |
| 蛋白质加合物（血红蛋白） | 环氧乙烯       | 红细胞  |

**暴露组学**
- 暴露组是生命全程中环境影响和相关生物反应的累积测量，包括来自环境、饮 食、行为和内生过程的暴露
- 暴露组学（exposomics）被定义为研究暴露组以及暴露组对人类疾病过程的影响。它关注个体一生中所有暴露的测量以及这些暴露如何与疾病建立联系
- 暴露组学的特点
	- 利用生物标志物确定暴露、暴露的影响、疾病的发展过程和敏感因素
	- 新技术的应用产生海量数据
	- 利用数据挖掘技术发现暴露、暴露影响、其他因素（如基因）与疾病之间的统计学关系

## 2.2效应标志
**早期生物效应标志**
- 指结合到靶组织上的外源性物质的持续作用，引起组织细胞的生物改变，从而产生疾病前期的生物标志
- 包括细胞毒性反应，染色体畸变，DNA、RNA和蛋白表达，以及细胞功能的一些早期变化（如DNA修复以及免疫功能的改变）等

**结构和（或）功能改变病理标志**
- 指形态学或功能学的改变，是在暴露-疾病连续带中更接近观察终点，即疾病发生的标志物
- 结构和（或）功能改变标志通常来自于靶器官的组织

**临床疾病生物标志**
- 指疾病发生后特有的分子生物标志
- 具有一定辅助诊断价值的标志物，如血清甲胎蛋白（AFP）、癌胚抗原（CEA）、前列腺特异性抗原（PSA）等肿瘤标志物、血糖，血脂、血生化等指标

**表型组学**
- 表型组（phenome）：某一生物的全部性状特征，是生物体从胚胎发育到出生、成长、衰老乃至死亡过程中，形态特征、功能、行为、分子组成规律等所有生物、物理和化学特征的集合
- 表型组学（phenomics）：在基因组水平上系统研究某一生物或细胞在各种不同环境条件下所有表型（从基因、蛋白质到代谢）的学科
- 主要研究内容：对某一生物的物理和化学性状随着基因突变及环境改变而变化的规律
- 特点：表型组学是多维度的，并且在不同的时间和空间环境中变化万千。广义上讲，表型包括表观基因组学、转录组学、蛋白组学、代谢组学和许多其他和生物化学、细胞过程相关的定量测量的“组学”数据。作为生物科学的两个基本的重要分支，基因组学和表型组学位于“多组学”家族的两端。当前生物学的一 个中心目标是在基因和表型之间建立完整的功能联系，即所谓的基因型-表型图谱（genotype-phenotype map）

## 2.3易感性标志
- 非遗传因素如年龄、健康状况等都可能影响个体对疾病的易感性
- 遗传易感性生物标志是机体稳定存在的遗传性可测量指标。可以是基因、RNA、甲基化、也可以是蛋白、代谢产物等功能学指标

## 2.4多组学
- 基因组（genomics）：生物体的全部基因组，基因组中的变异分为单核苷酸变异（simple nucleotide variation, SNV）和结构变异（structural variation, SV）
- 表观遗传组（epigenetics）：在不改变DNA序列情况下，基因表达的广泛变化，包括DNA甲基化水平的变化、组蛋白修饰、染色质重塑、非编码RNA（non-coding RNA, ncRNA）功能的变化等
- 转录组（transcriptomics）：细胞中基因转录所得产物RNA的集合，由占1-4%的信使RNA（mRNA）和占>95%的非编码RNA（ncRNA）组成
- 蛋白质组（proteomics）：在生物体、系统或生物学环境中产生的蛋白质，人类能够被表达和加工为蛋白质的基因约有2万个，人类蛋白质组计划已收集的质谱数据约涵盖这2万种蛋白质的90％
- 代谢组（metabolomics）：对机体中小分子成分的系统研究，通常涉及数百至数千种代谢物的测量，既代表基因组的下游输出，又代表环境的上游输入
- 微生物群系（microbiome）：在一个合理定义的栖息地中，具有独特理化性质的微生物群落，包括微生物及其周围“活动场所”，不仅限于微生物本身，还包含其功能、代谢产物及与周围环境的关系

# 3.主要研究方法
## 3.1研究设计与分析
**主要的研究方法**
- 病例对照研究（case-control study）
- 病例-病例研究（case-case study）
- 病例-队列研究（case-cohortstudy）
- 巢式病例对照研究（nested case-control study）
- 队列研究（cohortstudy）
- 基于家系的研究（family-based study）

**研究设计与实施**
- 研究目的
- 调查方法与样本
- 测量指标
- 测量方法
- 结果与分析
- 质量控制
- 注意事项

**资料处理与分析**
- 传统流行病学指标分析：疾病率、暴露率、比值比OR和 相对危险度RR....
- 常规统计分析方法：基因多态性与疾病关联研究分析，通过分析目标等位基因和基因型频率在病例组和对照组中分布的差异，判断基因多态性是否与疾病风险存在关联
- 基因组学统计分析方法

## 3.2生物标本的采集和储存
**生物标本**：病原生物标本、血液（血清、血浆、白细胞、红细胞）标本、组织标本、其他生物标本（如唾液、胃液、尿液、头发等）
**要求**：在采集和储存过程中不能受到“污染”，所有的生物标本都应有详细的背景材料和鉴别标识
**生物样本库**：通常将储存有一种类型或多种类型生物标本，并能保持它们的生物活性以供研究之用的系统称为生物样本库（biological specimen bank, BSB），如血清库、组织库、病原生物库等
**大型人群队列的生物样本库**
- 中国慢性病前瞻性研究（CKB队列）
- 英国生物银行（UKB队列）
- 中国国家出生队列（China National Birth Cohort）

## 3.3生物标志的选择和检测
==**生物标志的选择**==
- 生物标志选择的原则
	- 生物标志应特异、稳定
	- 标本采集、储存方便
	- 检测方法比较简单、实用，而且操作规范，便于与同类研究结果比较
	- 检测方法灵敏度和特异度高
- 测量指标的原则
	- 暴露指标：最好能代表接触剂量或生物作用剂量
	- 效应指标：早期生物效应标志、结构和功能病理改变、临床诊断生物标志
	- 易感性指标
		- 传染病：一般选择抗体水平等作为易感性指标
		- 慢性非传染病一般选择多组学指标

**生物标志的检测**
- 生物标志的特性
	- 分子特性
	- 时相特性
	- 个体内变异
	- 个体间变异
	- 群体间变异
	- 储存变异
- 生物标志物的检测方法
	- 核酸研究技术（DNA & RNA）
		- 核酸体外扩增
		- 核酸酶切电泳图谱
		- DNA测序
		- 核酸质谱
	- 代谢组学技术
		- 基于质谱的方法来检测目标小分子丰度，包括靶向和非靶向两种策略
	- 酶学技术
		- 包括定性和定量检测，要求一定的条件多位点酶电泳法
	- 蛋白组学技术
		- 凝胶电泳、蛋白质转印杂交、色谱分析、蛋白质测序等
	- 其他技术：免疫学技术、生物芯片技术等


**实验中的质量控制**
- 一般实验质量控制
	- 标本采集和储存
	- 试剂和材料
	- 仪器
	- 实验方法
	- 操作规范
- 设立对照
	- 标准对照
	- 空白对照
	- 重复对照
- 重复试验
	- 实验室内重复试验
	- 实验室间重复试验

# 4.应用与展望
**传染病的预防与控制**
- 病原体的分离和检测
- 病原生物进化变异规律研究
- 传染源追踪
- 确定传播途径
- 传染病防制措施及其效果评价

==**慢性非传染性疾病的预防与控制**==
- 探索疾病的**病因及发病机制**
- 评估个体**易感性**和**确定高危人群**
- 慢性病防治措施的制定及其效果评价
- 辅助疾病早期诊断及临床个体化治疗

**展望**
- 大规模队列研究的质量控制
- 临床转化应用研究
- 基因—环境交互作用研究
- 大数据和精准医学
