# 1.定义
**国际劳工组织(international labor organization)对尘肺病定义**：尘肺是粉尘在肺内的蓄积和组织对粉尘存在的反应
**肺尘埃沉着病(pneumoconiosis, 简称尘肺)**：是指在职业活动中长期吸入生产性粉尘而引起的以肺组织弥漫性纤维化为主的全身性疾病
煤工尘肺、矽肺是最主要的尘肺病

- 按病因分类

	* 石棉肺 (asbestosis)
	* 滑石尘肺 (talcosis or talc pneumoconiosis)
	* 铝尘肺 (aluminosis)
	* 云母尘肺 (mica pneumoconiosis)
	* 水泥尘肺 (cement pneumoconiosis)
	* 石墨尘肺 (graphite pneumoconiosis)

	* 铸工尘肺 (foundry workers' pneumoconiosis)
	* 电焊工尘肺 (welders' pneumoconiosis)
	* 煤工尘肺 (coal pneumoconiosis)
* 根据《尘肺病诊断标准》和《尘肺病理诊断标准》可以诊断的其他尘肺

# 3.发病机制
**病因**：吸入**生产性矿物性粉尘**，如游离二氧化硅粉尘、硅酸盐粉尘、石棉粉尘、煤尘、石墨粉尘
**粉尘定义**：以气溶胶状态或以烟雾状态存在的能较长时间漂浮于空气中的固体微粒。人类的生产活动和生活活动及自然界的风化、腐蚀、气体流动均可产生粉尘。国际标准化组织规定，粒径小于75μm的固体悬浮物定义为粉尘
**生产性粉尘**是专指在人类的生产活动中产生的能够较长时间漂浮于生产环境中的固体微粒

**生产性粉尘分类**
- 无机粉尘一矿物性粉尘
	- 矽尘：一般指含游离SiO2 10%以上的矿物粉尘
	- 硅酸盐尘：含有Al、Mg、Ca、Fe等金属元素的硅酸盐，如：石棉、水泥、滑石、云母、高岭土
	- 含炭粉尘：煤尘、石墨、炭黑等
	- 金属粉尘：铝、钡、铁、锡等金属
	- 人工无机粉尘：玻璃纤维、金刚砂、矿物棉等
- 有机粉尘一植物性、动物性和人工有机粉尘
- 混合性粉尘

**生产性粉尘的来源**
- 根据生产方式  
	- 固体物质的机械加工：粉碎、钻孔、研磨、切割等
	- 固体物质的不完全燃烧：燃煤产生的煤烟雾
	- 固体加热时蒸汽的凝结：金属冶炼、铸造、电焊
	- 固体粉末的包装、搬运、混合、过筛等
- 产生粉尘的行业
	- 矿山开采业：金属矿山和非金属矿山的开采
	- 机械加工业：铸造业
	- 冶炼业：炼铁、炼钢等的矿石的粉碎、烧结、选矿
	- 筑路业：铁路、公路修建中的隧道开凿及铺路
	- 水电业：水电行业中的隧道开凿及运输

**生产性粉尘对健康的影响**
- 呼吸系统的防御反应：保护反应与病理反应的平衡
- 98%左右的进入呼吸道的尘粒可通过清除机制排出体外，滞留于肺内的尘粒只是吸入粉尘总量的2-3%

**生产性粉尘的致病作用**
- 刺激作用：机械或化学性的刺激致呼吸道粘膜的损伤
- 致癌作用
- 粉尘沉着症

**尘肺病的发病机制**
- 粉尘的理化性质
- 刺激肺内发生炎症反应：肺泡巨噬细胞反应和细胞因子释放
- 机体的免疫反应
- 氧化应激反应

# 4.影响因素
==**影响发病的因素**==
- **空气中的粉尘浓度**：特别游离SiO<sub>2</sub>含量的高低
- **粉尘的性质（种类）**：游离二氧化硅的纤维化作用最强，其次为石棉和滑石
- **空气中粉尘的分散度（颗粒的大小）**：分散度越高致病性越强
- **粉尘颗粒是否新鲜及表面粗糙程度**
- **接触粉尘的时间**
- **机体一般状况**：特别是上呼吸道和肺部情况
- **防尘措施**

**尘粒表面的生物活性**
- 尘粒的表面粗造程度与尘粒间粘附能力密切相关，尘粒表面越粗糙，产生炎症刺激和纤维化的能力越强
- 亲水性尘粒表面含有较多的硅烷醇基(-SiOH)，能与金属等阳离子（铝、镁、铁、钛等）金属络合，从而改变尘粒的离子状态，增强产生自由基能力
- 不同的粉尘表面位点或污染物可影响粉尘的氧化活性
- 新鲜粉尘表面较之陈旧粉尘表面的生物学活性和致纤维化能力更强
- 流行病学资料证实，磁厂粘土覆盖于尘粒表面，降低了尘粒的表面活性，因而滋厂工人的尘肺发病率低于其他粉尘作业工人。不同煤矿矿尘肺发病率高低的差异，除与岩石中游离SiO<sub>2</sub>含量有关外，也与煤中粘土含量有关

# 5.病理
**基本病理改变**
- 巨噬细胞性肺泡炎
- 尘细胞肉芽肿
- 尘性纤维化

==**病理类型**==：病变主要位于肺间质，主要病理改变为肺纤维化
- 结节型
- 尘斑型
- 弥漫性纤维化型

# 6.临床表现
**症状**
- 早期症状常不明显，甚至可无自觉症状
- 后期才因肺循环和换气功能障碍出现胸闷、乏力、劳力性呼吸困难等缺氧表现
- 最常见的症状为气短，呈进行性。咳嗽、咳痰、胸闷、胸痛等较常见
- 症状出现的早晚和程度与尘肺病的种类、病变范围和有无并发症有密切关系
- 症状的轻重程度与X线表现不一定平行，但与肺功能大致平行

**体征**
- 早期尘肺病人一般无体征，阳性体征主要是并发症所致
- 肺气肿体征
- 肺心病体征

==**X线表现**==
- 小阴影：在X射线胸片上，肺野内直径或宽度不超过10mm的阴影
	- 根据形态的不同，又分为
		- 圆形阴影(直径,p型<1.5mm, q型1.5-3mm,r型>3mm)
		- 不规则阴影(宽度,s型<1.5mm, t型1.5-3mm,u型>3mm) 
- 大阴影 ：直径和宽度大于10mm以上的阴影
- 肺门：增大、致密、蛋壳样钙化
- **胸膜：肥厚、胸膜斑、钙化**
- 密集度(profusion)：一定范围内小阴影的数量。可分为四大级(0123)十二小级(0/-、0/0、0/1，1/0、1/1、1/2， 2/1、2/2、2/3，3/2、3/3、3/+)
	- 四大级分级：密集度可简单地划分为四级
		- 0级：无小阴影或甚少，不足1级的下限
		- 1级：有一定量的小阴影
		- 2级：有多量的小阴影
		- 3级：有很多量的小阴影
	- 十二小级分级：小阴影密集度是一个连续的由少到多的渐变过程，为客观地反映这种改变，在四大级的基础上再把每级划分为三小级
		- 0/-，0/0，0/1为0级
		- 1/0，1/1，1/2为1级
		- 2/1，2/2，2/3为2级
		- 3/2，3/3，3/+为3级
		- 目的在于提供更多的信息，更细致地反映病变情况，进行流行病学研究和医学监护

**小阴影聚集small opacity aggregation**：在X射线胸片上，肺野内出现局部小阴影明显增多聚集成簇的状态，但尚未形成大阴影

==**胸膜斑pleural plague**==：在X射线胸片上，肺野内除**肺尖部和肋膈角**区以外出现的厚度**大于5mm**的局限性胸膜增厚，或局限性钙化胸膜斑块。一般由于长期接触**石棉粉尘**而引起

**肺区zone of lung**：在X射线胸片上，将肺尖至隔顶的垂直距离等分为二，用等分点的水平线将左右肺野各分为上、中、下三个肺区，左右共6个肺区

**总体密集度判定**：总体密集度是指全肺内密集度最高肺区的密集度，是在对小阴影密集度分肺区判定的基础上对全肺小阴影密集度的一个总体判定，以4大级分级表示

**分布范围判定**：小阴影分布范围是指出现有密集度1级及以上小阴影的肺区数

**其他化验检查**
- 常规检查：血常规、痰菌培养、痰结核菌
- 肺功能检查：有重要意义，评定病情、劳动能力评定，关键是质量控制
	- 肺通气
		- FVC：尽力吸气后，再尽力尽快呼出的最大气体量，称之为用力肺活量(forced expiratory volume, FEV)
		- FEV1：第1s内的用力呼气量
		- FEV1/FVC：一秒率，第一秒内的用力呼气量占用力肺活量的百分比
		- 三者可用于鉴别限制性（如肺纤维化、气胸等）和阻塞性（如支气管哮喘，慢性支气管炎等）肺疾病
	- 肺活量
		- SVC：缓慢肺活量(slow vital capacity)，尽力深吸气后，所能呼出的肺内气体总量，略大于FVC，难以充分反映肺组织的的弹性状态和气道通畅程度
		- IC：深吸气量(inspiratory capacity)，平挣呼气末作深吸气所能吸入的最大气量，主要影响因素为吸气肌功能，其次包括胸廓与肺活动度，肺组织的弹性回缩力
		- ERV：补呼气量(expiratory reserve volume)，平静呼气末再用力呼气时所呼出的容积
		- FRC：功能残气量(functional residual capacity)，平静呼气末仍留存肺内的气量
		- RV：残气量(residual volume)，最大呼气末仍带留在肺内的气体量
		- TLC：肺总量(total lung capacity, TLC)，深吸气后肺内所含全部气量
	- 弥散功能
		- DL：肺弥散量(diffusing capacity)，肺泡膜气体两侧气体分压差为1.00mmHg时，每分钟所能透过的气体量
		- DL/VA：比弥散量，即肺弥散量比上肺泡通气量，从而排除肺容积、吸入的气体量对肺弥散功能的影响
		- 其升高意义不大，降低主要见于肺弥散面积减少或肺泡毛细血管膜增厚
	- 质量控制
		- 必须实际测量身高、体重
		- 一周内患感冒应更改测定时间
		- 测定半小时内不能吸烟
		- 各项测定指标重复测定2-3次，每次误差应<5%
		- 测定结果不可靠或病情不允许者可改做血气分析
		- 肺功能仪应定期进行校正
- 血气分析：客观
	- 用血气分析仪，通过测定人体血液的H+浓度和溶解在血液中的气体（主要指CO<sub>2</sub>、O<sub>2</sub>)，来了解人体呼吸功能与酸碱平衡状态的一种手段，它能直接反映肺换气功能及其酸碱平衡状态
	- 标本：动脉血
	- 适用于：低氧血症和呼吸衰竭的诊新；呼吸困难的鉴别诊断；昏迷的鉴别诊断；手术适应证的选择；呼吸机的应用、调节、撤机；呼吸治疗的观察；酸碱失衡的诊断等
- 生化、免疫学指标：研究很多，实用很少

# 7.诊断
**诊断原则**
- 根据可靠的**生产性矿物性粉尘**接触史，以技术质量合格的X射线高千伏或数字化摄影(DR)后前位胸片表现为主要依据，结合工作场所职业卫生学、尘肺流行病学调查资料和职业健康监护资料，参考临床表现和实验室检查，排除其他类似肺部疾病后，对照尘肺病诊断标准片，方可诊断
- 劳动者临床表现和实验室检查符合尘肺病的特征，没有证据否定其与接触粉尘之间必然联系的，应当诊断为尘肺病

==**尘肺病诊断标准GBZ 70-2015（基本必考）**==
- 尘肺壹期：有下列表现之一者
	- **有总体密集度1级的小阴影，分布范围至少达到2个肺区**
	- 接触石棉粉尘，有总体密集度1级的小阴影，分布范围只有 1个肺区，同时出现胸膜斑
	- 接触石棉粉尘，小阴影总体密集度为0，但至少有两个肺区小阴影密集度为0/1，同时出现胸膜斑
- 尘肺贰期：有下列表现之一者
	- **有总体密集度2级的小阴影，分布范围超过4个肺区**
	- **有总体密集度3级的小阴影，分布范围达到4个肺区**
	- 接触石棉粉尘，有总体密集度1级的小阴影，分布范围超过4个肺区，同时出现胸膜斑并已累及部分心缘或隔面
	- 接触石棉粉尘，有总体密集度2级的小阴影，分布范围达到4个肺区，同时出现胸膜斑并已累及部分心缘或隔面
- 尘肺叁期：有下列表现之一者：
	- **有大阴影出现，其长径不小于20mm，短径大于10mm**
	- **有总体密集度3级的小阴影，分布范围超过4个肺区并有小阴影聚集**
	- **有总体密集度3级的小阴影，分布范围超过4个肺区并有大阴影**
	- 接触石棉粉尘，有总体密集度3级的小阴影，分布范围超过4个肺区，同时单个或两侧多个胸膜斑长度之和超过单侧胸壁长度的二分之一或累及心缘使其部分显示蓬乱

**诊断**
- 诊断必须公正、合法。
- 粉尘职业接触史不可缺少
- 胸部X线是诊断关键
- 鉴别诊断应该想到
- 肺功能测定也很重要

# 8.鉴别诊断
- 尘肺病的X射线胸片改变具有一定的特征性，但不具有特异性，许多其他肺部弥漫性疾病的X线胸片表现可与尘肺相似，需要进行鉴别诊断
- ==**常见的鉴别诊断疾病有肺结核、肺癌、特发性肺纤维化(IPF)、结节病、过敏性肺炎、肺含铁血黄素沉着症、肺泡微结石症、组织胞浆菌病等**==
- 职业史，尘肺病必须有明确的矿物性粉尘接尘史，没有粉尘接触史，则不会患尘肺病
- 尘肺病典型的X线胸片特征改变是胸片出现圆形或不规则小阴影，随着病变的进展，小阴影可逐渐由少到多，密集度逐渐增高，继而可出现小阴影聚集或形成大阴影，小阴影聚集或大阴影一般发生在肺野的上部，典型者双侧可呈对称性改变
- 尘肺病早期多无明确的临床表现，而其他需要鉴别的疾病多有特征性的临床表现和病程
- 诊断性治疗的结果不同，如肺结核经过一段时间治疗后，X射线胸片病变会好转吸收等

# 9.尘肺并发结核
**尘肺易并发结核原因**
- 粉尘对呼吸道粘膜的慢性刺激，损伤粘膜
- 石英可增强结核菌的毒力
- SiO2降低机体对结核菌的细胞免疫力
- 肺的淋巴管破坏，淋巴循环障碍，有利于结核菌的生长和繁殖
- 广泛肺间质纤维化，肺内微循环受损，血供障碍

**尘肺结核的早期诊断**
- 临床出现结核毒血症症状，特别是长期低烧、咯血
- 血沉明显增快，肺部肿瘤已排除
- 痰查结核菌多次阳性
- 动态胸片观察到肺尖和肺中、上野的结节影增加，结节的形态呈多形性 
- 出现单侧非血性胸水    
- 不明原因的肺门淋巴结肿大

**尘肺结核治疗**
治疗原则是：早期、联合、规则、足量、全程

# 10.肺源性心脏病
**定义**：是指由支气管-肺组织、胸廓或肺血管病变致肺血管阻力增加，产生肺动脉高压，继而右心室结构或（和）功能改变的疾病

慢性肺源性心脏病是由肺组织、肺血管或胸廓的慢性病变引起肺组织结构和（或）功能异常，产生肺血管阻力增加，肺动脉压力增高，使右心室扩张或（(和）肥厚，伴或不伴右心功能衰竭的心脏病，并排除先天性心脏病和左心病变引起者

# 11.治疗
**治疗**
- 迄今国内外均无根治肺纤维化的有效方法
- 综合治疗及康复治疗的方法，以期达到减轻病人症状、改善生活和生命质量、延长寿命的目的
- **==治疗原则==**
	- ①去除病因
	- ②抑制肺纤维化的发生和发展
	- ③改善肺功能，增强机体抵抗力
	- ④防治并发症

**健康管理**
- 职业病登记报告
	- 按照国家法律法规的相关规定，将确诊尘肺病的患者登记在册并向卫生行政部门和有关部门进行职业病报告，以便纳入尘肺病健康管理体系，掌握患者的相关信息，随时了解病情，安排职业健康监护和必要的追踪
- 脱离粉尘作业
	- 尘肺病一经诊断，患者即应脱离原粉尘作业岗位，并不得再重新从事其他接触粉尘的作业
- 参加健康监护
	- 尘肺病是慢性进展性疾病，根据国家《职业健康监护技术规范》的规定，用人单位应当安排尘肺病患者参加定期健康检查。
- 自我管理
	- 戒烟，包括避免二手烟吸入，避免生活性粉尘接触，加强营养和养成健康良好的生活习惯
	- 煤仍是日常生活的主要能源，在其使用和燃烧过程中所产生的煤尘和煤灰尘均会促进肺纤维化的进展，有条件者应进行生活能源改造，使用清洁能源
	- 病情严重的尘肺病患者，或因合并肺结核或反复发生肺部感染者，常伴有营养不良。有证据表明，营养不良患者通过营养补充可以明显延长6min步行试验距离，增强呼吸肌力，改善健康状况
	- 加强运动锻炼，包括耐力训练、呼吸肌训练等，能促进肌肉细胞代谢，有利于提高免疫力，增强机体抵抗力

**综合治疗**
- 对症治疗
- 合理氧疗
- 并发症和合并症的治疗
	- 呼吸系统感染
	- 气胸
	- 肺结核
	- COPD
	- 呼吸衰竭
	- 慢性肺心病
- 康复治疗
- 抗纤维化治疗
- 全肺灌洗
- 外科干预
