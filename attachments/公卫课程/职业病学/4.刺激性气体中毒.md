# 1.概述
==**刺激性气体（irritant gas）**== 主要是指那些由于本身的理化特性，而对呼吸道及肺泡上皮有**直接刺激作用**的**气态化合物**，是工业生产中最常用到的一类有害气体，不慎吸入后对人体，特别是对呼吸道有明显的损伤作用，轻者可引起上呼吸道刺激，重者则致喉头水肿、支气管炎及**急性肺损伤（acute lung injury, ALI）**、肺炎、肺水肿，**甚至导致==急性呼吸窘迫综合征（acute respiratory distress syndrome, ARDS）==**；以上这些因吸入刺激性气体而引起的以呼吸系统损伤为主的临床表现，被统称为“**刺激性气体中毒（irritant gas poisoning）**”

---
# 2.刺激性气体的种类
- 酸类：
    - 无机酸（硫酸、硝酸、盐酸、**氢氟酸**等） 
    - 有机酸（甲酸、乙酸、丙酸、丁酸、乙二酸、丙二酸、丙烯酸等）
- 成酸氧化物（酸酐）：如**二氧化硫**、**三氧化硫**、二氧化氮、五氧化二磷等  
- 成酸氢化物：如**氯化氢**、**氟化氢**、**溴化氢**
- 卤族元素：氟、**氯**、溴、碘
- 卤化物：如氟化物、**光气**、二氯亚砜、三氯化磷、三氯化硼、三氯化砷、四氯化硅等
- 氨和胺类：如**氨**、甲胺、乙胺、丙胺、乙烯胺、丙烯胺、环己胺、乙二胺等
- 酯类：如硫酸二甲酯、二异氰酸甲苯酯、氯甲酸甲酯、醋酸甲酯等
- 醛类：如**甲醛**、乙醛、丙烯醛、糠醛等
- 醚类：如氯甲基甲醚
- 强氧化剂：如氧化镉、羰基镍、硒化氢、五氧化二钒、三氧化二锰、四氧化锇等
- 金属化合物：氧化镉、羰基镍、硒化氢
- 失火烟雾：现代建筑失火烟雾至少含60种以上化合物，其中不少具有呼吸道毒性，如氮氧化物、光气、其他卤化物等

==**常见的刺激性气体**==：氯气、氨、光气、双光气、二氧化硫、三氧化硫、氮氧化物、甲醛、硫酸二甲酯、氯化氢、氟化氢、溴化氢、臭氧（$\text{O}_3$）等

# 3.毒性作用
**刺激性气体的共性**
- 对呼吸道、皮肤、粘膜都有程度不同的刺激作用
- 以局部损害为主
- 也可引起全身的反应
- 危害面广，常多人中毒

**局部症状：直接刺激腐蚀作用**
- 与毒物的水溶性和作用部位有关
	- 水溶性大的刺激性气体（如氨、氯气）接触潮湿的黏膜即起刺激作用，引起结膜炎、鼻炎、角膜炎、咽炎、喉炎、气管支气管炎等    
	- 水溶性低的刺激性气体（如二氧化氮、光气等）易深入呼吸道，引起支气管炎和细支气管炎，严重时合并肺炎或肺水肿
- 与毒物的浓度有关
	- 液态的刺激性毒物如氢氟酸等，直接接触皮肤、黏膜可以引起灼伤
- 与接触强度有关

**全身症状**
- 神经系统：头晕、头痛、乏力，严重者出现烦躁、昏迷、惊厥等
- 循环系统：常见心肌损害和心律不齐
- 血液系统：WBC（白细胞）升高，**氮氧化物中毒时可见高铁血红蛋白血症**
- 消化系统：恶心、呕吐、食欲减退及肝脏损害，如转氨酶升高
- 发热：**有机氟化物烟尘热及氧化镉中毒**早期出现体温升高，并发肺水肿时也常有发热症状
- 纵隔气肿、颈部皮下组织气肿：如氮氧化物、氨、有机氟裂解物等中毒均可发生
- 肺水肿：刺激性气体引起的严重症状就是肺水肿

# 4.中毒机理
- 直接刺激腐蚀作用
	- 主要见于酸、酸酐、氨、胺等具有强烈刺激腐蚀作用的物质，可直接引起组织细胞结构溶解坏死
	- 如酸可迅速由组织中吸收水分、凝固其蛋白质，使细胞坏死；碱则不仅吸收水分并可皂化脂肪，使细胞发生溶解性坏死
	- 此种机制引起的刺激作用与刺激性气体的水溶性有密切关系：水溶性大的刺激性气体，如氨、氯化氢、二氧化硫、三氧化硫等，会立即溶入覆盖于眼和上呼吸道粘膜表面的粘液中，产生刺激作用，引起流泪、流涕、咽痒、呛咳等症状，由于其刺激作用明显，反可能较少造成严重中毒
	- 氮氧化物、光气、氯气等水溶性较小的刺激性气体，因通过上呼吸道粘膜时溶解较少，刺激作用较轻，且可大量深入肺泡，故易引起化学性肺炎（chemical pneumonia）或肺水肿（pulmonary edema）
- 呼吸道炎症反应
	- 刺激性气体特有的气道刺激性会不同程度地造成呼吸道细胞损伤，使之释放出细胞因子（cytokines），后者具有明显的趋炎性，可吸引大量炎性细胞向损伤处集聚、激活，产生炎症介质及氧自由基，并通过“瀑布效应”不断扩大炎症损伤效应
- 自由基损伤作用
	- 不少刺激性气体本身即是自由基（free radical），或可迅速产生自由基，如氧气、臭氧、各种氧化物（氮氧化物、二氧化硫、三氧化硫、五氧化二磷、氧化镉、五氧化二钒、三氧化二锰、四氧化锇等）、具有氧化性的化合物（如卤素及其化合物）等
	- 即便不能直接产生自由基，也可通过损伤肺泡上皮和血管内皮，引起炎性细胞在肺内集聚、激活，并释放大量活性氧（reactive oxygen species, ROS）自由基，引起脂质过氧化反应（lipid peroxidation），造成细胞结构严重破坏，并进而导致低氧血症

（1）呼吸道上皮发生水肿、变性、坏死、凋亡，构成支气管/肺泡化学性炎症的病理学基础；大量粘液、炎性细胞、坏死组织堵塞气道则引起肺通气障碍
（2）血管内皮和肺泡上皮受损，引起细胞间隙增宽、通透性增加、血浆渗出增多，最终形成肺水肿，构成气体弥散屏障（diffusion barrier）；损伤细胞还会释放大量血管活性物质，如组织胺、5-羟色胺、缓激肽、前列腺素等，进一步增加血管的通透性
（3）肺泡Ⅱ型细胞受损，表面活性物质生成减少，间质水分大量透过肺泡上皮，并使肺泡表面活性物质（alveolar surfactant）更形减少，形成恶性循环

**肺水肿**
肺水肿的发生取决于化学物的理化性质、浓度、作用时间以及个体的应急能力，发病机理归纳为：
- 肺泡和血管通透性增强：毒物可损伤肺泡Ⅰ型、Ⅱ型肺泡上皮细胞和肺毛细血管内皮细胞，使其通透性增加，形成肺水肿
- 血管活性物质释放与肺毛细血管通透性增加：中毒使体内释放大量血管活性物质如组织胺、5-羟色胺、缓激肽、前列腺素等，这些物质可增加血管通透性

# 5.临床表现
**急性中毒**
- **眼和上呼吸道的刺激症状**：出现畏光、流泪、结膜充血、水肿、流涕、咽痛、咽部充血、发音嘶哑、呛咳、胸闷等
- **窒息状态**：吸入高浓度的氨气、氯气、硫酸二甲酯等可引起喉痉挛和声门水肿，导致呼吸道机械性阻塞，表现为极度呼吸困难和喉鸣
- **呼吸道炎症**：刺激性阵发性呛咳、胸闷、胸痛、气急等；体温、白细胞均可增高
	- 化学性气管炎、支气管炎肺部听诊为呼吸音粗糙、可有痰鸣音；化学性肺炎时可闻及肺内散在湿罗音
	- X线胸片检查：病变部位多以肺中、下野或右下明显，化学性气管、支气管炎仅见肺纹理增粗；化学性肺炎可见肺纹理增强，边缘不清，肺野内可见局灶性大片状密度增高阴影，沿支气管走向分布，其间夹杂散在点状阴影
- ==**化学性肺水肿**==
	- **刺激期stimulation period**
		- 肺水肿前期：出现呛咳、气短、胸闷、呼吸困难并伴有烦躁、头痛、乏力、恶心等全身症状。体检可见眼结膜、咽喉等部位充血，肺部呼吸音粗糙或降低，偶可见有哮鸣音。X线胸片显示肺纹理增强，边缘模糊；体温、血WBC升高    
	- **潜伏期latent period**
		- 病人自觉症状好转或消失，但病理改变仍在继续。潜伏期的长短取决于毒物的溶解度和浓度，一般在4-12小时，少数病人可以达到36或48小时
	- **肺水肿期pneumonedema period**
		- 症状突然加重，表现为剧烈咳嗽、呼吸困难，呼吸加快、吐大量粉红色泡沫痰。体征：皮肤苍白、皮肤黏膜紫绀；脉搏、呼吸次数增加；血压下降
		- X线：肺纹理增粗、边缘模糊；两肺可见1-10mm的斑片状阴影，严重时全肺透光度降低，出现蝴蝶状阴影
		- 血氧分压降低，WBC增高。如缺氧不能及时纠正，则出现呼吸衰竭，导致代谢性酸中毒和呼吸性酸中毒
	- **恢复期recovery period**
		- 如无并发症，肺水肿可在2-3天控制，病情进入恢复期，X线异常多在1-2周内消失
		- 大多数预后较好，有机氟聚合物、热解物可发生慢性肺间质纤维化
	- 肺水肿的严重程度及预后等往往随毒物种类、吸入剂量、个体差异、潜伏期处置等不同而异
	- 如吸入剂量较小，处理适时，可不发生或仅发生轻度肺水肿；反之，可引起严重肺水肿，甚至导致ARDS
	- 刺激性气体引起急性肺水肿常被诊断为ARDS，但其死亡率多在10%以下
	- 根据1992年的欧美联席会议的论果，ARDS实质上是全身性炎症反应的一部分，而从原发病因发展到ARDS，需要炎症积累过程，此一诱导期至少为24h-48h
	- 刺激性气体引起的肺水肿潜伏期仅数十分钟或数小时，如此短暂时间很难促成全身炎症反应并诱起ARDS，如此短暂的潜伏期实际上只是化学性肺水肿发生发展过程，故规范的抗肺水肿治疗常能生效，而不像ARDS那样难以纠正
- **==并发症==**
	- 气道粘膜坏死脱落：多见于重症氨气、氯气、硫酸二甲酯中毒，有时可因堵塞气道，导致突发呼吸困难、窒息
	- 肺不张：主因气道内水肿液、脓痰或脱落的坏死粘膜堵塞小气道而致。右肺下叶多见
	- **纵膈气肿、自发性气胸**：多见于重度氨、二氧化氮、氯、硫酸二甲酯、有机氟热裂解物等中毒后2-4天，因细支气管或肺泡腐蚀破裂引起
	- 肺部感染：组织坏死、局部引流不畅易引起细菌性支气管肺炎；应用糖皮质激素、气管切开也是继发感染的诱因
	- **肺纤维化**：多见于二氧化氮、有机氟聚合物或热解物、失火烟雾等中毒；患者有进行性呼吸困难，X线可见弥漫性网状、条索状、点状阴影，重者伴低氧血症
	- 阻塞性细支气管炎：常见于氮氧化物、氯气吸入后，容易进入呼吸道深部，引起细支气管黏膜损伤、脱落，造成局部堵塞
	- **中毒性心肌损害、休克**：有机氟热裂解物可直接损伤心肌；肺水肿可引起严重缺氧，间接引起心肌损伤；肺水肿导致低血容量、休克
	- ARDS：严重刺激性气体引起的ALI如未得到控制，可进展为ARDS
	- 如果肺水肿得不到有效治疗可发生酸中毒、脑水肿，或出现中毒性肾病、肝病

**慢性影响**
低浓度长期接触刺激性气体
- 可引起眼结膜、呼吸道等刺激症状，如慢性结膜炎、鼻炎、咽炎等；
- 慢性支气管炎发病率较高，个别有支气管哮喘发作，如接触TDI（二异氰酸甲苯酯）者；
- 少数可导致慢性阻塞性肺疾病（COPD）；
- 神经衰弱综合征；
- 消化不良；
- 有些刺激性气体对皮肤有刺激性或致敏性，如接触甲醛可导致过敏性鼻炎；长期接触无机氟及酸雾可发生牙齿酸蚀症

# 6.诊断及诊断标准
**急性刺激性气体中毒诊断原则**
- 明确的刺激性气体吸入史
- 典型的呼吸道刺激症状及肺部炎症体征
- X线胸部摄片显示肺纹理增粗、紊乱，肺门扩大，甚至出现散在点片状阴影
- 排除其他病因引起的支气管炎、肺炎、肺水肿
- 综合分析，作出诊断

**诊断分级**
- **接触反应：** 短期内接触较一定量化学物后，仅有轻微症状，但无相应靶器官（系统）损伤的阳性体征、实验室检查及其他相关辅助检查异常者
- **轻度中毒：** 除刺激与炎症表现外，两肺有干罗音或哮鸣音，少量湿性罗音，X线胸片符合支气管炎或支气管周围炎表现
- **中度中毒：** 有明显呼吸系统症状，两肺有干或湿罗音，X线胸片示支气管肺炎，间质肺水肿或局限性肺水肿表现
- **重度中毒：** 有典型肺水肿症状体征或出现严重并发症，如气胸、喉头水肿及窒息等，X线胸片示严重肺炎或肺泡性肺水肿改变

# 7.处理原则
刺激性气体的主要危害是肺水肿，**积极防治肺水肿**是中毒抢救治疗的关键
## 7.1处理灼伤，预防肺水肿
- 迅速把患者脱出现场，脱去污染衣服，用大量清水冲洗污染部位，并根据刺激性气体的特点，选用碱性5%碳酸氢钠或酸性2%-4%硼酸溶液中和剂冲洗或湿敷
- 及早应用地塞米松或氢化可的松以增加机体应激能力，减少肺泡壁及肺毛细血管的通透性
- 限制补液量，必须静脉补液时，每日一般不超过1000毫升
- 根据病情需要、使用镇静剂、镇咳去痰剂、支气管解痉剂等作对症治疗

## 7.2肺水肿的处理
原则上是采取**通气、利水、抗感染**等措施
**通气措施
- 保持呼吸道通畅、及早给氧。必要时气管插管或气管切开，加压给氧。但应注意压力不宜过大，以防发生纵隔气肿或气胸
	- 刺激性气体中毒时由于肺通气和换气功能均有损伤，一时很难将血氧恢复至正常水平，但血氧分压维持在80mmHg左右，即已可基本满足机体需求
- 采用二甲基硅油消泡剂雾化吸入，可不均匀地降低肺内泡沫的表面张力，使泡沫破灭而成液体，增加氧气与肺泡壁的接触面积，改善弥散通气功能，使症状迅速缓解。其疗效迅速、可靠，较吸入乙醇湿化气好，但作用时间短暂，可反复使用
- 使用氨茶碱，喷雾吸入糖皮质激素和异丙肾上腺素混合液可解痉，减少渗出

**利尿措施**
- 脱水疗法：可选用脱水剂或利尿剂，必要时，两者交替使用，以减少肺循环容量，改善肺水肿
- 地塞米松5-10毫克，1日3-4次静脉注射。症状改善后，剂量减少并改为口服
- 应用抗菌素，预防及控制感染。并注意纠正酸碱和水、电解质平衡
- 使用解毒剂：如**氮氧化合物中毒**时，高铁血红蛋白含量在20%以上时，可应用**美兰**（亚甲蓝）解毒治疗

# 8.预防措施
刺激性气体对人群的危害是突发性事故造成的群体性中毒和死亡，因此，预防控制的重点是消除事故隐患，早期发现和预防重度中毒，加强现场急救，预防控制并发症
- 病因控制
	- 消除事故隐患，控制接触水平
		- 加强对化学反应锅、输送管道、贮槽或钢瓶等的维修及灌注、储存和运送通道的安全防范，做好防爆防火防漏
		- 生产和使用刺激性气体的设备和过程实行密闭化、自动化及局部吸出式通风，做好废气的回收和利用
		- 定期进行环境检测，及时发现刺激性气体超过最高容许浓度的原因，提出改进措施
		- 提高作业人员素质与自我保健意识。加强职工上岗前安全培训，自觉执行安全操作规程，空戴防护衣帽和防毒口罩
	- 提高现场急救水平，控制毒物吸收
		- 有潜在事故隐患作业，配置急救设备，如防毒面具、冲洗设备等，开展急救训练，并定期对急救设备和防毒面具进行维修和有效性检验
		- 尽快使染毒者脱离接触，进入空气新鲜地带，迅速脱去污染衣服。皮肤、眼染毒即刻进行清洗或中和解毒，视吸入或接触气体为酸性或碱性，分别以3%～5%碳酸氢钠或3%～5%硼酸、柠檬酸冲洗，或湿敷，或呼吸道雾化吸入
- 预防和早期检出重度中毒
	- 依据接触者接触刺激性气体的性质、与事故现场的距离、风向及接触时间，估算吸入剂量，结合症状与体征，对接触人群中可能发生肺水肿者，进行密切观察至少48～72小时，绝对休息，保持安静，必要时给予镇静剂和地塞米松等处理；对可能发生喉头水肿或痉挛导致窒息者，进行及时处理
