# ==1.窒息性气体Asphyxiating gases==
**定义**：指那些主要以“取代氧气”或“干扰机体对氧的摄取、运输和利用”为主要毒理机制，导致机体组织缺氧，从而引起一系列病理生理变化甚至死亡的气体

**分类**
- 单纯性窒息性气体(simple asphyxiants)
	- 氮气、甲烷、水蒸气等
	- 空气：氧含量20.96%；氧含量小于16%为缺氧表现；氧含量小于10%为昏迷甚至死亡
- 化学性窒息性气体(chemical asphyxiants)
	- 血液窒息性气体：CO、苯胺、硝基苯等
	- 细胞窒息性气体：HCN、H<sub>2</sub>S

# 2.⼀氧化碳（Carbon Monoxide，CO）
**概述**
- CO是最常⻅的中毒之⼀
- 在北⽅、冬季多发。极端天⽓情况下发病率更⾼
- 死亡⼈中，⾃杀数量远⾼于意外事故（5-10倍）

**理化性质**
- ⽆⾊、⽆味、⽆刺激性⽓体，故称“silent killer”
- 密度是空⽓的0.97，故通常均匀地混合于空⽓中

**自然界中的CO**
- 含碳化合物部分氧化的产物
- 空气中有微量，小于0.001%（10ppm），城市大于农村
	- 来源于植物的代谢
	- 大气对流层及海洋的光化学反应
	- 甲烷及非甲烷碳氢化合物的氧化
	- 火山、森林大火、天然气等

**动物及人体内源性产生**
- 体内代谢也产生小量CO
	- 血红素氧化酶家族合成CO占绝大多数，约 50%
	- 脂肪酸代谢中的酶也可生成CO

**CO的生理功能**
- 血管调节，舒张血管平滑肌
- 抗炎作用
- 细胞分化、增殖及凋亡

**日常生活中的CO**
- 通常室内浓度0.5-5ppm
- 使用煤气炉后15ppm
- 室内木火5000ppm
- 未经催化转化的汽车尾气中高达 7000ppm。如果在密闭车库内着车，可在10分钟内达到致死量

**生活性接触**
- 汽车尾气、基于含碳燃料的加热系统及发电机
- ⼀些清洁燃料如甲烷丙烷虽然能更完全燃烧，也有CO产⽣

**职业性接触**
任何含碳物质不完全燃烧均可产生CO
- 煤气工业：任何生产环节漏洞
- 化学工业：氮肥生产、甲醇等化工产品制备、羰基金属制备等
- 采矿工业：爆破时会产生大量CO
- 冶金工业：用煤进行冶炼、炼焦等
- 交通运输：用煤、汽油、柴油为燃料的车辆开动时

**二氯甲烷(methylene chloride)**
- 用作脱漆剂、胶片生产等
- 无色液体，容易挥发
- 通过皮肤或蒸汽经肺吸收，在肝脏代谢后生成CO

**病理生理**
- CO中毒的核心是造成机体缺氧
    - 影响血液氧气的运输
    - 氧离曲线左移，影响外周组织对氧的摄取
- 1857年生理学家Claude Bernard发现CO与Hb结合，降低血液运输O₂能力，导致组织缺O₂
- **与红细胞中血红蛋白Hb中的铁结合→碳氧血红蛋白COHb**→失去携带氧气的能力（CO与Hb的亲和力约是O₂的**200多倍**）
- 1.34ml O₂/g Hb，约是血浆氧容量的70倍。在血液中的运输98.5%是与Hb结合，只有1.5%是物理溶解
- 吸收：容易经呼吸道通过肺迅速收入血
	- 吸收量：
	    - 每分钟通气量
	    - 暴露时间
	    - 暴露环境中CO浓度
- 排出
	- 主要经肺原型排出
	- 小于1%转变为CO<sub>2</sub>排出
- COHb与临床症状的关系
	- 10%COHb：开始出现症状
	- 50-70% COHb：抽搐、昏迷甚至死亡
- CO中毒的核心问题是机体缺氧，因此损伤的靶是对氧最敏感的器官——大脑。对心肌的损伤也不应忽视

**临床表现**
- 头痛、头晕、乏力、恶心、注意力降低、气短、视力改变、胸痛、意识丧失、腹痛、抽搐
- 环境中不同浓度时临床表现
	- 100ppm：头痛、头晕
	- 200ppm：判断力及定向力下降
	- 大于800ppm：恶心呕吐、昏迷
- 临床表现并无特异性
- 既往教科书中所描述的经典表现，如樱桃红嘴唇、视网膜出血罕见
- 心血管系统：心脏对缺氧的敏感性仅次于大脑，因此CO中毒有可能导致心肌损伤。对既往有心脏病史的病人尤其要注意
- 孕妇及胎儿：胎儿对缺氧敏感，而且血中的CO半衰期更长
- 迟发性脑病 (DNS) (Delayed neuropsychological sequelae)
	- 中毒后有发生DNS高风险 (10-30%)
	- 在数日或数周内发生，其间一切正常
	- 各种神经精神症状：运动异常、周围神经病、精神病、痴呆
	- 可能为永久性，目前没有症状或检查预测，机理不清。缺血后再灌注损伤可能起一定作用
	- 高龄是危险因素之一

**碳氧血红蛋白测定(Carboxyhemoglobin, COHb)**
- 测定方法：采用分光光度计，动静脉血均可，动脉血可了解酸中毒情况
- 血液中血红蛋白的种类：
	- 还原血红蛋白 (Hb)
	- 氧合血红蛋白 (HbO₂)
	- 碳氧血红蛋白 (HbCO)
	- 高铁血红蛋白 (HbMe)
- 正常情况下
	- 小于3% - non-smoker
	- 小于10% - smoker
- COHb与临床表现及预后相关性不好
	- 样本采取时间
	- 是否吸氧

**氧饱和度测定**
- 标准氧饱和度测定仪只测定氧合Hb及还原Hb两个吸收光谱的波长，然后根据两个光谱相对的吸收程度，计算出氧合Hb的百分比
- 新近的氧饱和度测定仪可测定四种Hb的光谱波长：氧合Hb、还原Hb、碳氧Hb及高铁Hb

**ECG检查**
- 所有中毒病人需做ECG及心电监测
- 可有心律失常、ST段改变

**心脏标记物检查**：如果ECG提示有缺血改变并有相应的心机缺血症状，则应进行心肌酶检查

**胸片**：利于了解其他疾病或合并症：肺水肿、肺炎、气胸、COPD

**CT检查**
- 了解伴随的创伤情况
- 鉴别诊断，除外中风等疾病
- CT检查发现CO中毒无特异性

**诊断**
- 有CO接触史，尤其是对轻、中度中毒的病例，因为在这些病人中，头痛、乏力、恶心及呕吐可能是唯一的症状，与胃肠炎、感冒等类似
- 有相应的症状和体征
- HbCO的浓度测定
- 排除其它原因，如安眠药中毒、中风、糖尿病性昏迷等
- 诊断中需要注意的几个问题
	- 无特异性临床表现，尤其是接触史不明确的病例，故应有高度警惕
	- 对疑难病例，最好做现场调查、CO浓度测定
	- 正确解释COHb测定结果：结果正常不能除外诊断；但结果升高多是有意义的
	- 应做好现场CO来源的调查，以防周围人群中毒及病人本人的再次接触
- 诊断主要依据：接触史、临床表现及COHb
- 对疑似病人：明确诊断、评估损伤的严重程度、了解对靶器官的损伤

**治疗**
- 在保障救援人员安全的前提下，将患者迅速移至安全处
- 根据呼吸道情况，迅速通过面罩或气管插管给予高流量纯氧
- 注意是否有其它的合并损伤，给予相应的治疗
- 尚无资料证明给纯氧优于空气，但因安全、便宜，故使用
- 治疗的核心：减少机体的缺氧时间，即迅速转移患者至安全处并给予吸氧治疗
- 目的：不仅要挽救生命，还要注意预防迟发的永久性损伤
- 目前没有证据表明高压氧治疗HBO能降低死亡率及改善预后

**预防**
- 加强公共教育，尤其是弱势群体，认识CO的危险性
- 设备的定期专业维护
- 密闭环境的通风
- 不允许汽车在封闭环境内处于发动状态
- 装配CO报警器

# 3.氰化氢(Hydrogen Cyanide, HCN)
**理化性质**：蒸气带杏仁样气味，易在空气中弥散，易溶于水。蒸气密度与空气接近为0.94

**职业接触**
- 化工工业
	- 直接制备HCN
	- 一些反应的副产物，如氰化钾与硫磺制备硫氰酸钾等
	- 用作化工合成的原料，如乙炔-氰氢酸合成法制备丙烯睛
- 冶金工业：贵重金属提取等
- 任何含氮有机物质干馏或不完全燃烧

**病理生理**
- 吸收：呼吸道、皮肤、消化道均容易吸收
- 排除：
	- **==主要在硫氰酸盐生成酶的作用下→硫氰酸盐（低毒，从尿排出）==**
	- 原形HCN（呼气、尿、粪）
	- 氰钴铵素转化VitB12
	- 氧化为CO2和NH3（呼气）
	- 与葡萄糖醛酸结合→低毒晴化物（尿）
	- 与半胱氨酸结合→亚氨基噻唑-4-羧酸（尿）
- 氰根与细胞中含三价铁的呼吸酶细胞色素氧化酶结合→使生物氧化过程不能进行

**临床表现**
- 剧毒，极低浓度（5～20 mg/m³）吸入数小时后可引起症状；较高浓度（大于200 mg/m³可迅速致死
- 可有黏膜轻微的刺激症状（眼、呼吸道）
- 余见CO中毒

**诊断**
- 有接触史
- 呼出气中有杏仁味
- 血、尿中硫氰酸盐明显增高（大于10mg/L）
- 注意与其他引起昏迷的疾病鉴别

**治疗**
- 脱离接触
- 给氧
- 解毒
	- 先亚硝酸钠（生成大量高铁血红蛋白methemoglobin与氰根结合成氰络高铁血红蛋白）
	- 然后硫代硫酸钠（与从氰络高铁血红蛋白解离出的氰根生成不易分解的低毒化合物硫氰酸盐从尿中排出）
- 余同CO中毒

# 4.硫化氢(Hydrogen Sulfide, H₂S)
**理化性质**：无色气体，有强烈的臭蛋样气味

**职业接触**
- 化学工业：含硫化合物的生产制造如对硫磷、乙硫磷等
- 石油工业：钻井开采石油中
- 制革工业：鞣制皮革
- 其它：含硫矿石冶炼
- 生活来源

**病理生理**
- 强烈刺激性：黏膜如眼、呼吸道
- 细胞窒息作用：与细胞色素氧化酶结合
- 中枢神经抑制作用：十分迅速

**临床表现**
- 黏膜刺激症状，如眼、呼吸道
- 高浓度吸入患者可立即昏迷，如电击一般
- 余见CO中毒

**诊断**
- 明确接触史，呼出气及衣物均带有臭蛋样气味
- 血中硫化Hb增高
- 临床表现

**治疗**
- 脱离接触
- 给氧
- 注意肺水肿
- 余同CO中毒
