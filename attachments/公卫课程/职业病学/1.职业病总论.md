# 1.职业病定义
- 职业病是指企业、事业单位和个体经济组织等用人单位的劳动者在职业活动中，因接触粉尘、放射性物质和其他有毒、有害因素而引起的疾病
- 职业病临床特点
	- 病因明确
	- 临床表现缺乏特异性
	- 具有特殊的靶部位和损伤机制
	- 有明显的剂量效应关系和时间效应关系
	- 疾病无传播性
	- 临床后果与作用强度密切相关

# 3.我国职业病发病情况
**职业病发病特点**：以2012年发病为例
1. 总职业病数27420，尘肺病报告病例数24206例，占2012年职业病报告总例数的88.28%
2. 急性中毒601例，引起急性职业中毒的化学物质主要是一氧化碳、二氯乙烷和氯气，此3种物质共发生中毒291例；病死率最高的是硫化氢中毒，中毒38例，死亡9例
3. 慢性职业中毒1040例，引起慢性职业中毒的化学物质主要是苯、铅及其化合物（不包含四乙基铅）、砷及其化合物，分别为329例、197例和164例
4. 职业性肿瘤95例，以轻工、化工行业为主。其中苯所致白血病53例，石棉所致肺癌、间皮瘤19例，焦炉工人肺癌17例，联苯胺所致膀胱癌3例，铬酸盐制造业肺癌3例
5. 职业性放射性疾病。共报告32例
6. 职业性耳鼻喉口腔等疾病1446例。职业性耳鼻喉口腔疾病639例，其中噪声聋597例；职业性眼病94例；物理因素所致职业病201例，其中手臂振动病130例，中暑54例；生物因素所致职业病293例，其中布氏杆菌病244例，森林脑炎49例；职业性皮肤病148例，其中皮炎64例；其他职业病71例，其中职业性哮喘为46例

**面临的严峻形势**
1. 我国职业病危害因素分布广泛。从传统工业，到新兴产业以及第三产业，都存在一定的职业病危害，接触职业病危害因素人群数以亿计，职业病防治工作涉及三十多个行业，法定职业病名单达132种。接触职业危害人数、职业病患者累计数量、死亡数量及新发病人数量，都居世界首位
2. 我国职业病发病形势严峻。近十年职业病发病情况呈现明显的凹形反弹倾向。发病人数从上世纪90年代初逐年下降，1997年降至最低后又呈反弹趋势。其中主要是尘肺病检出率显著回升
3. 我国的职业危害主要以粉尘为主，职业病人以尘肺病为主，占全部职业病的90％左右，尘肺病又以煤工尘肺、矽肺最为严重，尘肺病患者中有半数以上为煤工尘肺
4. 职业病所造成的经济损失严重。根据有关部门的粗略估算，每年我国因职业病、工伤事故产生的直接经济损失达1000亿元，间接经济损失2000 亿元
5. 职业性疾患是影响劳动者健康、造成劳动者过早失去劳动能力的主要因素，所波及的后果往往导致恶劣的社会影响
6. 对职业卫生机构和队伍现状调查表明，我国已经初步形成职业卫生监督与技术服务网络，但依然存在队伍数量少，质量不高；文化素质偏低，现场技术服务人员比例较低以及后备力量不足等问题
7. 对我国职业卫生投入调查表明，各级政府自1999年起职业卫生投入呈逐年增加的趋势。但由于基数低，人均职业卫生投入明显不足，与经济发展水平极不适应，造成职业卫生监督与技术服务得不到保证

**存在的主要问题**
1. 防治检测不到位。表现在检测企业数的减少和达标率的升高，这一相互背离的结果反映随着职业卫生技术服务市场化，检测的数据不能真实地反映用人单位工作场所职业危害的实际情况。疾病信息监测系统不健全
2. 企业不重视。一些乡镇企业、个体经济企业生产力低下，设备简陋，无任何防护设施；管理混乱，制度不全；人员整体素质低，法制观念淡漠和愚昧无知等；劳动者特别是农民工的健康权益得不到保护，其社会保障、职业防护等都难以得到保障健康影响难以估计和控制
3. 职业病危害因素转移非常严重。在我国对外开放、引进外资和先进技术的同时，一些具有风险性的产品由境外向境内转移，从城市和工业区向农村迅速转移，从经济发达地区向欠发达地区转移，从大中型企业向中小型企业转移
4. 职业病防治经费投入严重不足
5. 职业卫生资源整体效率低，配置不平衡，职业卫生技术服务水平不高
6. 职业卫生执法力度不够强
7. 职业卫生涉及多个部门，协同工作机制尚未充分建立。在一个部门内部，也往往有职能交叉，职业卫生决策、协调、指挥不够充分，部门之间缺乏协同机制，职业病危害前期预防措施得不到有效落实
8. 传统的职业危害尚未得到完全控制，新的职业危害不断产生，对劳动者的健康构成新的威胁
9. 职业卫生标准尚未与国际接轨

# 5.职业健康监护
以预防为目的，根据劳动者的职业接触史，通过定期或不定期的医学健康检查和健康相关资料的收集，连续性地监测劳动者的健康状况，分析劳动者健康变化与所接触的职业病危害因素的关系，并及时地将健康检查和资料分析结果报告给用人单位和劳动者本人，以便及时采取干预措施，保护劳动者健康

## 5.1职业健康监护的目的
1. 早期发现**职业病、职业健康损害和职业禁忌症**
2. 跟踪观察职业病及职业健康损害的发生、发展规律及分布情况
3. 评价职业健康危害与作业环境中职业病危害因素的关系及危害程度
4. 识别新的职业病危害因素和高危人群
5. 进行目标干预，包括改善作业环境、改革生产工艺、增添防护措施等
6. 评价预防和干预措施的效果
7. 为制定或修订卫生政策和职业病防治对策服务

## 5.2职业健康监护的内容
包括职业健康检查和职业健康监护档案管理

==**职业健康检查**包括==
- 上岗前职业健康检查
- 在岗期间职业健康检查
- 离岗时职业健康检查
- 离岗后健康检查
- 应急健康检查

**职业健康监护档案管理**
- 个人健康档案内容包括：个人基本信息、个人生活史、家族史、职业危害因素接触史、历次职业健康检查结果及职业病诊疗情况记录等
- 群体性健康监护内容包括：用人单位基本情况、工作场所职业危害因素种类、不同作业点的环境监测数据、健康体检总结和评价报告、落实健康监护评价报告的建议和干预措施的实施情况及效果评价

## 5.3职业健康监护的实施
常规医学检查和特殊医学检查
**常规医学检查内容**
1. 劳动者个人基本信息资料
2. 一般医学生理指标的监测
3. 症状询问
4. 内科常规检查
5. 神经系统常规检查
6. 其他专科的常规检查
7. 实验室常规检查

**特殊医学检查**：针对特定危害因素所做的相应的特殊医学检查，如铅作业者体检时需测定血铅和尿铅等

# 6.职业病的预防
**预防原则**：遵循三级预防的原则，特别是第一级预防，即病因预防

**一级预防措施**
1. 实施劳动卫生监督
2. 加强劳动过程中的预防措施：降低有害物质浓度、采用有效的防护措施、报警和急救设备、接触水平评价

**积极开展二级和三级预防**
- 职业健康监护工作是重要的二级预防措施，通过职业健康监护的群体评价，可了解某些职业病或工作有关疾病与环境中有害因素的具体关系，从而采取相应的预防措施
- 对于受到健康损害的接触者可尽早进行三级预防，如职业禁忌症者应及时脱离有害岗位，可疑职业病及时送往医疗机构进一步检查确诊

# 7.职业病的管理
**职业病报告**
用人单位和医疗卫生机构发现职业病病人或者疑似职业病病人时，应当及时向所在地卫生行政部门报告。确诊为职业病的，用人单位还应当向所在地劳动保障行政部门报告

**诊断争议**
- 当事人对职业病诊断有异议的，可以向作出诊断的医疗卫生机构所在地地方人民政府卫生行政部门申请鉴定
- 职业病诊断争议由设区的市级以上地方卫生行政部门根据当事人申请，组织职业病鉴定委员会进行鉴定
- 当事人对设区的市级职业病诊断委员会的鉴定结论不服的，可向省、自治区、直辖市人民政府卫生行政部门申请再鉴定

**疑似职业病**：采取干预措施

**职业病致残程度鉴定及待遇**
职业病病人的诊疗、康复费用，伤残以及丧失劳动能力，按照国家有关工伤社会保险规定执行。同时有获得赔偿的权利，可向用人单位提出赔偿要求

# 8.职业病诊断
**职业病诊断工作特点**
1. 须依照法定程序进行
2. 以国家《职业病诊断标准》为依据
3. 须在法定机构，由法定资格的医生诊断
4. 须做出病因诊断

**职业病诊断法定程序**
劳动者或用人单位向法定职业病诊断机构提出诊断申请，并提供职业史书面材料、历次健康检查结果和作业场所历年卫生监测资料

**职业病诊断法定依据**：《职业病目录》，十大类132种

**职业病诊断的法定机构**
1. 持有《医疗机构执业许可证》
2. 具有与开展职业病诊断相适应的医疗卫生技术人员
3. 具有与开展职业病诊断相适应的仪器、设备
4. 具有健全的职业病诊断质量管理制度

承担职业病诊断的医疗机构不得拒绝劳动者进行职业病诊断的要求
劳动者可以在用人单位所在地、本人户籍所在地或者经常居住地依法承担职业病诊断的医疗卫生机构进行职业病诊断

**职业病集体诊断的原则**：三人以上具有资格的医师作出诊断

**职业病诊断工作的具体内容**
1. 职业史
2. 职业危害因素接触情况
3. 临床情况
4. 实验室检查

**职业病的鉴别诊断**
没有证据否定职业病危害因素与病人临床表现之间的必然联系的，应当诊断为职业病

# 9.职业病治疗
## 9.1病因治疗
1. 脱离或减少病因接触：离开工作场所，清洗污染皮肤、眼睛，更换清洁衣物，经消化道中毒者尽快催吐、洗胃等
2. 加速毒物排出或代谢：饮水、利尿、血浆置换等
3. 特殊治疗措施
	1. 金属解毒药：依地酸钠钙、二乙酸三胺五醋酸钠钙、二巯基丙磺酸钠、二巯基丁二酸钠以及二巯基丁二酸
	2. 高铁血红蛋白还原剂：美篮
	3. 氰化物解毒剂：亚硝酸盐、硫代硫酸钠、4-二甲基氨基酚、氨基苯丙酮
	4. 农药中毒解毒药：阿托品、解磷定、氯磷定、盐酸戊乙奎醚、乙酰胺等

## 9.2对症支持治疗
中毒性脑病与脑水肿、中毒性肺水肿及ARDS、急性呼吸衰竭、中毒性肾病及急性肾衰竭

# 10.职业性器官系统疾病
## 10.1职业性神经系统疾病
**神经毒物**
1.  金属、类金属及其有机化合物：铅、汞、砷
2. 有机溶剂：汽油、苯、甲苯
3. 窒息性毒物：一氧化碳、硫化氢、氰化物
4. 农药：有机磷、氨基甲酸酯类、杀虫脒
5. 其他：氯丙烯、溴甲烷、碘甲烷

**临床类型**
1. 急性中毒性脑病
2. 慢性中毒性脑病
3. 中毒性周围神经病
4. 中毒性神经衰弱综合征

## 10.2职业性呼吸系统疾病
**有害物质**
1. 刺激性气体和烟雾：硫酸、硝酸、氯气
2. 无机粉尘：二氧化硅、硅酸盐尘、金属粉末
3. 有机粉尘：动植物及其蛋白成分、微生物粉尘
4. 呼吸道致癌物：砷、氯甲醚、焦油、石棉

**临床类型**
1. 急性化学性气管-支气管炎
2. 化学性肺炎
3. 化学性肺水肿
4. 急性呼吸窘迫综合征（ARDS）
5. 支气管哮喘
6. 慢性化学性支气管炎及慢性阻塞性肺病
7. 慢性肺纤维化性肺病
8. 肺部及胸膜肿瘤

## 10.3职业性心血管系统疾病
**有害物质**
1. 窒息性气体：一氧化碳、硫化氢、氰化物
2. 农药：有机氯、有机硫、有机磷
3. 金属、类金属：铅、汞、砷
4. 刺激性气体：氨、氯气、光气
5. 卤代烃类：四氯乙烯、氯乙烯、氯乙烷
6. 有机溶剂：笨、甲苯、汽油
7. 高铁血红蛋白形成剂：苯胺、硝基苯
8. 硝酸盐类：硝酸铵、硝酸钠、硝化甘醇
9. 其他：苯酚、氟乙酸、烯丙胺
10. 物理因素：噪声、振动、低气压、微波、电击

**临床类型**
1. 心脏损害：心肌损害、心源性休克、充血性心力衰竭
2. 心律失常：窦性、异位心律失常、传导阻滞
3. 猝死
4. 血管损伤：血压降低或增高、动脉粥样硬化、雷诺综合征、心绞痛

## 10.4职业性血液系统疾病
**再生障碍性贫血**
- 化合物有苯、三硝基甲苯、二硝基酚、砷化合物、四氯化碳、有机氯及有机磷杀虫脒等
- 放射性物质有X射线、伽马射线、中子流、放射性核素等
- 药物有氯霉素、保泰松、硫氧嘧啶、卡比马唑、甲硫咪唑、苯妥英钠、氯丙嗪、吲哚美辛、甲氨蝶呤等

**巨幼细胞性贫血**
砷化合物、慢性乙醇中毒

**铁粒幼细胞性贫血**
慢性铅中毒、酒精中毒

**溶血性贫血**
