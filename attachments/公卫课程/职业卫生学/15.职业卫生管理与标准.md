# 1.职业健康的基本概念
**劳动卫生与劳动保护**
- **劳动卫生labor hygiene**：与劳动和劳动条件有关的卫生学科，主要研究劳动条件对工人建康的影响，目的是识别、评价和控制不良劳动条件，创造安全、卫生、满意和高效、舒适的工作环境，保护和促进劳动者的身心建康
- **劳动保护labor protection**：为保护劳动者在劳动生产过程中的安全卫生所采取的立法、组织和技术措施。目的是消除和预防劳动生产过程中可能发生的伤亡和职业病，分为劳动安全保护和劳动卫生保护
	- **劳动卫生保护**：保护劳动者在劳动生产过程中的身体健康，避免有毒、有害物质的危害，防止、消除职业中毒和职业病

**工（产）业卫生与职业卫生**
- **工（产）业卫生industrial hygiene**
	- 致力于预期、识别、评估和控制工作场所或工作场所产生的可能导致工人或社区居民疾病、损害健康和福祉或严重不适的环境因素或压力的科学和艺术(OSHA)
	- 建筑、采矿、制造等传统产业，美国使用广泛，我国在60~70年代使用
- **职业卫生occupational hygiene**
	- 是对工作场所产生或存在的可能损害劳动者健康与福祉，或可能影响周围社区以及整体环境的危险进行预期、识别、评估和控制的科学(IOHA,2003)
	- 涵盖所有类型的产业

**职业医学occupational medicine**
- 研究职业性有害因素所致的人体健康损害，包括工作有关疾病、职业病和伤害等的诊断、治疗、康复和劳动能力鉴定的一门临床科学，也是研究预防控制职业性有害因素的预防医学
- 职业医学是评估工人健康，关联工人建康与工作条件和工艺，支持管理职业人群健康、技能和工作能力，以及管理个体工作能力及形成的医学学科，涉及对工人健康不良问题的一级、二级和三级预防工作，并对整个人口的建康产生潜在影响
- 职业医学是强调预防的医学分支，它从临床和行政角度处理个人和团体有关其工作环境以及包括职业相关疾病和伤害的识别、评价、控制、管理和康复，以及影响工作能力其他问题
- 职业医学旨在：
	- 促进和保特所有职业工人的最高程度的身体、心理建康和社会福祉
	- 预防因工作条件引起的建康问题
	- 使工人身体和心理能力与职业环境相适应

**职业健康occupational health**
- 旨在促进和维持所有职业工人最高程度的身心健康及社会福祉，预防工作条件对工人造成的健康损害；保护工作中的工人免受建康危害因素所致的风险，使工人处于并维持在与其生理和心理能力相适应的职业环境。即使工作适应人，使每个人适应其工作
- 公共卫生领域中的一个工作领域，涉及职业医学、护理学、工效学、心理学、卫生、安全及其他学科，旨在促进和维持所有职业工人的最高程度的身心健康和社会福祉
- 目标：
	- 维护和促进工人的健康和工作能力
	- 改善工作条件和工作环境，使其有利于安全建康
	- 建立旨在反映企业基本价值体系的工作组织和工作文化，包括以改善职业安全建康为目的的管理制度、人事政策、参与原则、培训政策以及自愿的质量管理实践

**工作与健康的相互作用**
- 当工作与人的目标、能力完全适应，且职业危害因素得到控制时，工作会促进人的身心健康
- 工作场所中存在的物理、化学和生物危害因素一旦超出容许限值，即可能成为职业病的致病因素
- 工作环境及其特征在复杂、多重病因的疾病发生发展方面，与其他危险因素共同发挥作用
- 职业危害因素可能加重影响职业人群的一般疾病

**职业病危害及危害因素**
- **职业病危害**：是指对从事职业活动的劳动者可能导致职业病的各种危害
- **职业病危害因素**包括职业活动中存在的各种有害的化学、物理、生物因素以及在作业过程中产生的其他职业有害因素

**职业性病损**
劳动者在职业活动中接触职业性有害因素以及影响劳动者健康、安全和作业能力的条件，一旦有害因素作用于人体的强度与时间超过一定限度，则有可能造成人体功能性或器质性病理改变，当人体失代偿并出现相应临床症状时，称为职业性病损

**职业病的定义及其范围**
- **职业病**：是指企业、事业单位和个体经济组织等用人单位的劳动者在职业活动中，因接触粉尘、放射性物质和其他有毒、有害因素而引起的疾病
- 定义包含四个方面的含义：
	- 患病主体为用人单位的劳动者（存在劳动关系）
	- 所患疾病在职业活动中产生（疾病与职业活动存在依从性）
	- 接触粉尘、放射性物质和其他有毒、有害因素（病因明确）
	- 为国家《职业病分类和目录》所列职业病（法律限定）
- **职业病目录**：一组因接触职业性有害因素引起的疾病，是国家通过法律或其他形式对职业病进行的定义、分类

==**工作相关疾病work related diseases**==
- 是由多因素引起的疾病，与工作有联系，但也见于非职业人群
- 与职业病比较，工作相关疾病与工作条件的关系可能微弱、不一致或者不明确，可能存在因果关系，但因果关系的强度和大小可能有所不同

| 职业病                                          | 工作相关疾病                          |
| -------------------------------------------- | ------------------------------- |
| 病因明确，接触职业危害因素是导致职业病发生的关键因素，疾病与职业接触建立了直接的因果关系 | 由多因素引起的疾病，工作条件和接触并不一定是疾病的危险因素   |
| 致病因素可以识别、测量并控制，采取有效干预措施可预防其发生或减轻其严重性         | 职业接触可能使原有疾病加重、加速或恶化；可能损害工作能力    |
| 一般只影响受具体危害因素影响的职业人群                          | 通常与工作相关，但也发生在一般人群，比职业病更常见       |
|                                              | 个人特征、其他环境和社会文化因素通常作为疾病的危险因素发挥作用 |

# 2.职业卫生法律体系及监管体制
**职业病防治法律法规**
依立法主体、法律效力、职业病防治法律法规等法律层次分为**法律、行政法规、地方性法规、部门规章**
- 职业病防治法律：核心法律是职业病防治法，其他包括劳动法、劳动合同法、安全生产法...
- 职业病防治行政法规
- 职业病防治法部门规章
- 地方职业卫生法规及规章
- 批准生效的职业卫生安全国际条约、公约

**新时期新发展格局**：开放国内国际双循环
建设更高水平开放型经济新体制，形成国际合作和竞争新优势
高举构建人类命运共同体旗帜，坚定不移维护和引导经济全球化

**职业病防治方针：预防为主，防治结合**
- 第一级预防：职业病危害源头控制、劳动过程中的防护、危害治理与管理
- 第二级预防：职业健康检查
- 第三级预防：职业病诊断与治疗

**职业病防治工作原则：分类管理、综合治理**
- 对从事放射性、高毒、高危粉尘等作业实行特殊管理
- 对职业病、职业病危害因素实行分类管理：职业病分类和目录、职业性有害因素分类目录、高毒物品目录
- 建设项目职业病危害分类管理

==**职业健康监护制度**==
- 上岗前职业健康检查
- 在岗期间职业健康检查
- 离岗时职业健康检查
- 离岗后健康检查
- 应急时的健康检查和医学观察

**职业禁忌症occupational contraindication**
劳动者个人的特殊生理或病理状态，在该状态下，从事特定职业或接触特定职业病危害因素时：
- 比一般职业人群更易于遭受职业病危害和罹患职业病
- 可能导致原有自身疾病病情加重
- 在作业过程中诱发可能导致对他人生命建康构成危险的疾病

**职业健康监护与职业健康检查的关系**：管理责任不同、实施主体不同、法律责任不同，但又有紧密联系

|      | 职业健康监护                            | 职业健康检查               |
| ---- | --------------------------------- | -------------------- |
| 责任主体 | 用人单位，法定责任                         | 医疗卫生机构，接受委托          |
| 范围   | 职业健康检查、应急健康检查、离岗后健康检查和职业健康监护档案管理等 | 职业健康监护的重要内容和主要资料来源   |
| 内容   | 连续监测劳动者健康状况                       | 对劳动者的健康损害进行医学检查（一次性） |
| 方法   | 定期或不定期的健康检查和健康相关资料收集              | 医学手段和方法              |
| 性质   | 职业健康管理行为                          | 医疗行为                 |
| 档案管理 | 职业健康监护档案                          | 职业健康检查档案             |

**职业病诊断与鉴定制度**：一级诊断、两级鉴定
- 分级管理制度
- 备案管理制度
- 资格管理制度
- 机构负责制度

**职业病的诊断**
- 劳动者及劳动者的用人单位均可提出职业病诊断
- 劳动者依法享有职业病诊断机构选择权，可以在用人单位所在地、本人户籍所在地或者经常居住地的职业病诊断机构进行职业病诊断
- 职业病诊断机构应当在备案的诊断项目范围内开展职业病诊断，在备案开展的诊断项目范围内不得拒绝劳动者进行职业病诊断的要求

**职业病诊断需要的资料**
- 劳动者的职业史和职业病危害接触史，包括在岗时间、工种、岗位、接触的职业病危害因素名称等
- 劳动者职业健康检查结果——职业健康监护资料
- 工作场所职业病危害因素检侧结果

**疾病认定原则**
- 疾病是指在病因作用下机体出现自稳调节紊乱，并引发一系列代谢、功能或结构变化的异常状态
- 临床表现和相应辅助检查是判定有无疾病及其严重程度的主要依据
- 应遵照循证医学要求做好鉴别诊断

**接触职业病危害因素的判定原则**
- 依据接触职业病危害因素的时间和方式、职业病危害因素的浓度（强度），参考工程防护和个人防护情况，估计劳动者的可能的累积接触水平
- 根据职业病危害因素检测结果及生物监测结果，结合OEL/BEL标准，判断机体有无过量接触（外接触或内接触剂量是否致病的水平）

**疾病与接触职业病危害因素之间因果关系判定原则**
- 时间或时序性原则
- 生物学合理性原则
	- 根据危害因素的理化特性、毒理学等属性，证实接触该因素可导致相应疾病且疾病的临床表现应与所接触的具体危害因素导致的建康效应相一致。即所接触的危害因素引起的疾病应符合该因素的毒理学、理化特性或其它特征
	- 符合致病因素所致疾病的生物学潜伏期和潜隐期的客观规律
		- 潜伏期incubation period：从开始接触职业性有害因素至出现相应疾病最早临床表现之间的间隔时间。超过最长潜伏期，任何疾病都不能归因于接触。如急性接触CO一年后发生的急性中枢神经症状不能归因于职业接触
		- 潜隐期latent period, latency：从开始接触危害因素到出现该因素所致健康损害时的最短时间，即发生疾病需要的最短接触时间，短于这个时间，因接触引起疾病的可能性不大。如首次接触石棉后一年内发生的职业性间皮瘤归因于接触的可能性不大
- 生物学特异性/一致性与连贯性原则
- 生物学梯度与关联程度
	- 职业病危害因素能否引起疾病存在最低累积接触量，只有接触的积业病危害因素达到致病所需的最低累积接触水平才有可能引起职业病的发生，尤其是化学毒物
	- 多数职业病与职业病危害因素接触之间存在剂量-效应（反应）关系。接触水平越高、接触时间越长，疾病的发病率越高或病情越严重；职业病危害因素对疾病的发生、发展影响越大，疾病与接触之间因果关系的可能性就越大
- 可干预性原则
	- 对接触的职业病危害因素采取干预措施，可有效的防止职业病的发生、延缓疾病的进展或使疾病向好的方向转归。消除或减少工作场所或职业活动中的职业危害因素，可减少相应疾病的发生或降低发病率
	- 多数职业病在脱离原工作场所后，经积极治疗，疾病可好转、减轻甚至消失

# 4.职业接触控制
**职业接触控制优先原则**
- 消除替代原则。优先采用有利于保护劳动者健康的新技术、新工艺、新材料、新设备，从源头控制劳动者接触职业性有害因素
- 工程控制原则。对生产工艺、技术和原材料达不到卫生学要求的，应根据生产工艺和化学有害因素的特性，采取相应的防尘、防毒、通风等一切合理可行的预防控制措施，使劳动者的接触或活动的工作场所化学有害因素的浓度符合卫生要求
- 管理控制原则。通过制定并实施管理性的控制措施，控制劳动者接触职业性有害因素的程度，降低危害的影响
- 个体防护原则。当所采取的控制措施仍不能实现对接触的适宜控制时，应联合使用其它控制措施和适当的个体防护装备；个体防护用品通常在其他控制措施不能理想实现控制目标时使用
- 成本效益原则。在评估预防控制措施的合理性、可行性时，还应综合考虑职业病危害的种类以及为减少风险而需要付出的成本

根据劳动者接触化学有害因素的浓度可将职业接触分为5级，应采取与其对应的控制措施

| 接触等级            | 推荐的控制措施                                         |
| --------------- | ----------------------------------------------- |
| 0（＜1% OEL）      | 不需采取行动                                          |
| Ⅰ（1%-10% OEL）   | 一般危害交流                                          |
| Ⅱ（10%-50% OEL）  | 一般危害交流、化学物质特殊危害交流                               |
| Ⅲ（50%-100% OEL） | 一般危害交流、化学物质特殊危害交流、接触监测、医学监护、作业管理                |
| Ⅳ（＞100% OEL）    | 一般危害交流、化学物质特殊危害交流、接触监测、医学监护、作业管理、呼吸防护用品和工程、工艺控制 |

**行动水平action level**：也称管理水平，是指劳动者职业接触水平已达到用人单位需要采取包括接触监测、职业健康监护、职业卫生培训、职业危害告知等控制措施的水平。行动水平根据工作场所环境、接触物质的不同而有所不同，一般为该化学有害因素容许浓度的一半
