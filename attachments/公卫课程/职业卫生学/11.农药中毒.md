# 1.农药概述
## 1.1农药的定义与分类
**定义**：**农药(pesticides)** 指用于防止、控制或消灭一切虫害的化学物质或化合物
农药也包括**植物生长调节剂**和**卫生杀虫剂**
《中华人民共和国农药管理条例》中<u></u>的定义：用于**预防、消灭或者控制**危害农业、林业的病、虫、草和其他有害生物以及有目的的调节植物、昆虫生长的**化学合成或来源于生物、其他天然物质**的一种或几种混合物及其制剂

农药包括：
- 预防、消灭或者控制危害农业、林业的病、虫（包括昆虫、蜱、螨）、草和鼠、软体动物等有害生物
- 预防、消灭或者控制仓储病、虫、鼠和其他有害生物
- 调节植物、昆虫生长
- 用于农业、林业产品防腐或者保鲜
- 预防、消灭或者控制蚊、蝇、鼠和其他有害生物
- 预防、消灭或者控制危害河流堤坝、铁路、机场、建筑物和其他场所的有害生物

**农药的分类**
- 按成分分类
	- 原药（产生生物活性的物质）
	- 制剂（含溶剂、助剂）
- 按单、混剂分类
	- 农药单剂
	- 农药混剂
- 按作用方式分类
	- 触杀剂(Contact Poison)
	- 胃毒剂(Stomach Poison)
	- 熏蒸剂毒剂(Fumigant Poison)
	- 内吸毒剂(Systematic Poison)
- 按靶生物分类（较普遍）
	- **杀虫剂**：有机磷杀虫剂，有机氯杀虫剂，拟除虫菊酯类杀虫剂，氨基甲酸酯类杀虫剂；**用量最大**
	- 杀菌剂：无机杀菌剂，有机磷杀菌剂，有机砷杀菌剂
	- 除草剂：燕麦敌；敌草隆；百草枯
	- 杀鼠剂
	- 杀螨剂
	- 杀螺剂：五氯酚钠
	- 杀卵剂
	- 杀线虫剂
	- 植物生长调节剂：矮状素，乙烯利
- 按来源分类
	- 矿物源农药(无机化合物)：硫磺
	- 生物源农药(天然有机物、抗生素、微生物)
	- 化学合成农药：大部分
- 按化学结构分类
	- 无机化学农药：极少
	- 有机化学农药：**有机磷、氨基甲酸酯、拟除虫菊酯**、有机氯化合物、有机硫化合物、酰胺类化合物、脲类化合物、醚类化合物、酚类化合物、苯氧羧酸类、三氮苯类、二氮苯类、苯甲酸类、脒类、三唑类、杂环类、香豆素类、有机金属化合物等

## 1.2农药的相关特性
- 种类繁多，使用范围广
	- 品种达上千种
	- 农、林、畜牧、卫生部门等
- 生产后，会有目的地投放到环境中
- 接触人群广泛，包括
	- 职业接触人群：从事生产、运输、保存、使用
	- 一般人群：通过污染的产品、水体、土壤等环境接触
- 农药的毒性相差悬殊，在我国根据农药的<u>大鼠急性毒性</u>的大小，进行毒性分级

## 1.3农药对人体健康的影响
**农药的危害**
- 职业危害：农药生产、运输、保管等过程；农业使用过程；经皮吸收和经呼吸道吸收
- 非职业危害：对蔬菜、水果的污染和残留
- 生态环境破坏：在某些动植物体内富集或蓄积
- 特殊危害：致突变（mutation）；致畸（teratogenesis）；致癌（carcinogenesis）；迟发性神经毒性（delayed neurotoxicity）
- 自杀：农村地区

**农药对人体健康的影响**
- 急性中毒：主要取决于
	- 农药的急性毒性大小：农药活性成分、溶剂或助剂
	- 人群短时间内可能的接触量
- 长期接触后的不良健康效应：
	- 致癌
	- 致突变
	- 致畸
	- 迟发性神经毒性
	- 生殖发育相关损伤
	- 免疫功能损伤

## 1.4我国农药生产使用现况
农药是重要的农业生产资料，主要用以提高单位面积粮食产量
- 世界范围内农药防治农业病、虫、草害挽回的损失占粮食产量的1/3
- 我国耕地少，人口多，需要使用农药来提高粮食产量

我国农药生产发展迅速，已成为世界第二大农药生产国，也是农药出口大国
- 产业规模不断扩大
	- 截至2010年，我国农药生产企业1800多家，其中原药生产企业500多家，全行业从业人员16万人
	- 2010年我国共生产农药234.2万吨，出口61.3万吨，销售产值1589.6亿元
- 产品结构不断改善
	- 削减高毒农药，优化生产高效、低毒、低残留产品
	- 杀虫剂比重不断下降，杀菌剂和除草剂比重逐渐提升

我国每年使用农药制剂100万吨左右
混配农药使用非常普遍，占使用品种60%以上
- 有机磷与拟除虫菊酯
- 有机磷与另一有机磷
- 两种不同有机磷与拟除虫菊酯
- 有机磷与氨基甲酸酯
- 有机磷、氨基甲酸酯、拟除虫菊酯的三元混配剂

毒性多呈相加作用，少数有协同作用
## 1.5农药的管理
《中华人民共和国农药管理条例》明确规定农药管理办法
- 国家实行农药登记制度
- 国家实行农药生产许可制度
- 国家实行农药经营管理制度
- 国家实行农药使用范围的限制

根据国家规定，未经批准登记的农药，不得在我国生产、销售和使用
禁止使用的农药：
- 没有生产厂家生产，从而没有申请登记，不一定是农药本身有什么问题
- 试验或使用中有安全方面的问题，而不被批准登记

限制使用：
- 逐步限制高毒类农药的登记，以低毒类农药逐步取代高毒类农药
- 每一种农药都有一定的限制使用条件，包括使用的作物、防治对象、施用量、方法、时期、土壤、气候、条件等（精准农业）
- 不同的毒性分级农药，在登记时其应用范围有严格的限制

其他法规：
- 《农药安全使用规定》
- 《农药合理使用准则》
- 农村农药中毒卫生管理办法（试行）

## 1.6农药中毒的预防
严格执行农药管理的有关规定：登记、生产许可、经营、使用范围、标签标识
有关人员安全管理和使用农药的宣传教育
改进农药生产工艺及施药器械，加强通风排毒措施
遵守安全操作规程
- 运输：专人、专车
- 营销：剧毒的农药要有专门的仓库或专柜
- 配药、拌种要有专门的容器。定点清洗容器，避免水源污染
- 喷药：不要逆风，避免正午、大风时作业
- 施药工具的保管、维修：防止泄漏，严禁用嘴吸喷头
- 个人防护：工作时长衣长裤，眼罩，严禁吸烟、吃东西。工作后洗澡，换洗衣服
- 显著标志：施药区域禁止人员和动物进入

医疗保健与预防措施
- 工人就业前和定期体检
- 给予健康指导
- 禁忌症：神经系统疾病、肝肾疾病等，妊娠期和哺乳期妇女

其他：研发低毒或无毒农药，在高毒类农药加入警告色

# 2.有机磷酸酯类农药(Organophosphate Pesticidcx, OPP)
- 应用最广泛，用量最大
- 多数品种为剧毒或高毒
- 可引起人畜中毒
- 农药中毒和死亡中，有机磷农药引起的占绝大多数
## 2.1理化性质
**结构：**
![[11.农药中毒-2.有机磷酸酯类农药(Organophosphate Pesticidcx, OPP)-20260409.png|161]]
R<sub>1</sub>或R<sub>2</sub>：烷氧基，甲氧基-OCH<sub>3</sub>，乙氧基-OC<sub>2</sub>H<sub>5</sub>，毒性：乙氧基>甲氧基
Z：可为氧（磷酸酯类），也可为硫（硫代磷酸酯类）
X：烷氧基、芳氧基或其他基团

纯品为白色结晶，有**大蒜或韭菜的特殊臭味**
比重>1，比水重
**常温常压下，有蒸汽溢出**
不耐热，易爆炸

## 2.2有机磷农药分类
按化学结构和取代基团种类
- 磷酸酯类：敌百虫
- 硫代磷酸酯类：对硫磷、马拉硫磷、乐果
- 磷酰胺、硫代磷酰胺类：甲磷胺、乙酰甲磷胺
- 焦磷酸酯、硫代焦磷酸酯和焦磷酰胺：双硫磷

## 2.3职业接触机会
工业生产：
- 跑、冒、滴、漏
- 检修设备或设备发生故障，农药外溢
- 分装：呼吸道吸入或经皮肤吸收

农业使用：
- 喷洒过程
- 粮库熏蒸灭虫、**拌种灭虫**（棉花、马铃薯、蚕豆等）
- 用具涂抹：运输、装卸、储存、供销与保管

日常生活接触：
- 如误食或误服、自杀、投毒等均可引起中毒
- 滥用有机磷农药防治卫生害虫，如灭蚊、蝇
- 治疗皮肤病如疥疮
- 浸泡衣服及床上用品
- 口服有机磷驱虫等
- 用污染较重的水源(水井、池塘)洗澡及洗衣服等

## 2.4毒理
- 吸收：消化道、呼吸道和完整皮肤及粘膜
	- **经皮吸收为职业中毒主要途径**
- 分布
	- 分布主要为肝脏，其次为肾、肺、脾
	- 可透过血脑屏障，胎盘屏障（脂溶性）
- 排泄
	- 随尿排出
- 代谢：代谢转化为**氧化和水解**两种方式
	- **氧化后毒性增高，水解后毒性下降（水解酶）**
	- 人和温血动物的**羧酸酯水解酶**含量丰富，水解>氧化，但昆虫体内该种酶水平很低——马拉硫磷是高效、人畜低毒的杀虫剂
	- 昆虫体内**酰胺酶**降解能力有限——乐果杀虫效果好
	- 乐果的生物标志物：尿中的二甲基磷酸酯

==**靶器官毒性表现及中毒机理**==
- 有机磷化合物是**神经毒物**，**乙酰胆碱酯酶**是它的靶点。进入体内的有机磷化合物与乙酰胆碱酯酶有高度的亲和性，结合成磷酰化胆碱酯酶，乙酰胆碱酯酶失活
- 毒性机制：抑制胆碱酯酶活性
	- 乙酰胆碱蓄积→毒蕈碱样症状、烟碱样症状和中枢神经系统症状表现
	- 酶的老化：磷酰化胆碱酯酶不能重活化，15-30天

## 2.5临床表现
- 急性中毒：**通常2~6小时**
	- **毒蕈碱样症状**：恶心，呕吐，多汗，流涎，瞳孔缩小，严重者呼吸困难，肺水肿
	- 烟碱样症状：肌肉震颤，痉挛（胸、面颊、上肢）；呼吸肌麻痹死亡
	- 中枢神经系统症状：头痛，头晕，多梦，失眠，昏迷；一般无后遗症，迟发性周围神经病；心肌毒性作用，发生“电击样”死亡；“癔病样”发作
	- 其他脏器系统损害
		- 上消化道损害
		- 中毒性肝病
		- 急性坏死性胰腺炎
		- 中毒性心脏损害
		- 肺部感染
		- 中毒性肾损害
	- 严重继发症
		- 脑水肿
		- 惊厥
		- 中间肌无力综合征(IMS)：近端肌、呼吸肌无力（神经-肌肉接头传导阻滞）
		- 迟发性神经病（OPIDN）：手脚无力，拖曳步态、继发肌强直；周围神经和脊髓长束脱髓鞘
- 慢性中毒
	- 胆碱酯酶活性下降（功能蓄积），类神经症，毒覃碱样症状
	- 视力变化：近视、眼睑充血、水肿、角膜上皮肿胀
- 致敏作用和皮肤损害
	- 支气管哮喘，接触性皮炎或过敏性皮炎
- 遗传毒性和生殖发育毒性

## 2.6治疗
- 脱离清洗
- 特效解毒药
	- **阿托品等抗胆碱药**
	- 胆碱酯酶复能剂
- 对症和支持治疗

# 3.氨基甲酸酯类农药
氨基甲酸酯类农药是在有机磷和有机氯农药后发展起来的一类**合成农药**。作为杀虫剂，具有速效、触杀、残留期短、对人畜毒性较低的特点
常用的有：呋喃丹、西维因、速灭威等
## 3.1理化性质
氨基甲酸酯类的共同基本结构
![[11.农药中毒-3.氨基甲酸酯类农药-20260409.png|303]]
R<sub>1</sub>、R<sub>2</sub>和X取代基不同，可形成多种化合物
R<sub>1</sub>、R<sub>2</sub>为芳香烃、脂肪族链或其他环烃
- R<sub>1</sub>如为甲基，杀虫剂
- R<sub>1</sub>如为芳香族集团，除草剂
- R<sub>1</sub>如为苯并咪唑，杀菌剂

白色结晶，无特殊气味
熔点50-150度
易溶于有机溶剂，难溶于水
酸性溶剂中稳定，遇碱易分解

## 3.2毒理
经口：中等毒性；经皮：低毒
- 呼吸道和胃肠道吸收，经皮吸收缓慢
- 进入机体后很快分布到全身组织和器官
- **代谢迅速，体内无蓄积，24小时70-90%排出**
- 主要从尿排出

氨基甲酸酯与胆碱酯酶分子结合（可逆）

## 3.3临床表现
类似有机磷农药中毒
潜伏期2～4小时
病情较轻，毒覃碱样症状为主
特殊毒性：动物致畸和致癌作用
- 人体：内分泌干扰（精子染色体畸形率增加）
- 雌性大鼠：动情周期延长

## 3.4诊断
职业性急性氨基甲酸酯杀虫剂中毒诊断标准（GBZ52-2002）
诊断原则：
- 职业接触史
- 临床表现
- 全血胆碱酯酶活性测定
- 现场调查资料
- 排除其他病因

分级标准：
- 轻度中毒：酶活性70%
- 重度中毒：酶活性30%

## 3.5处理原则
- 现场处理：脱离中毒现场，脱去受污衣服，清洗受污皮肤
- 阿托品+对症处理

## 3.6==有机磷农药与氨基甲酸酯类农药==

| 有机磷农药                     | 氨基甲酸酯农药                   |
| ------------------------- | ------------------------- |
| 磷酰基与胆碱酯酶CHE结合，氧化速度↑，毒性越大↑ | 以分子与胆碱酯酶CHE可逆结合           |
| 可用复能剂CHE复能                | 不用复能剂，CHE抑制               |
| CHE"酶老化"易引起严重中毒（磷酰基结合）    | CHE“易复能” ，不容易引起严重中毒（分子结合） |

**急性有机磷中毒与氨基甲酸酯类农药中毒的鉴别要点**

|    项目    | 有机磷中毒 | 氨基甲酸酯类农药中毒 |
| :------: | :---: | :--------: |
|   接触史    |  有机磷  |  氨基甲酸酯类农药  |
| 体表或呕吐物气味 |  蒜臭味  |    无蒜臭味    |
|    病程    | 相对较长  |    相对较短    |
|  阿托品用量   |   大   |     较小     |
| 复能剂治疗效果  |  显著   |    不明显     |
| 中毒胆碱酯酶活性 | 恢复缓慢  |    恢复较快    |

# 4.拟除虫菊酯类农药
- 人工合成
- 高效、广谱杀虫剂
- **环境残留低、人畜低毒**
- **用于家庭卫生杀虫剂**
- 混配制剂较多（与有机磷混配）
- 常用种类：溴氰菊酯（敌杀死）、氰戊菊酯（速灭杀丁）

## 4.1理化性质
- 物理性质：液态，粘稠状，黄色
- 化学性质：易溶于有机溶剂（甲苯）、难溶于水、遇碱分解
- 结构
	- Ⅰ型：不含氰基、低毒-卫生杀虫剂（电蚊香）
	- Ⅱ型：含氰基、拟除虫菊酯类农药、中等毒性-农药

## 4.2毒理
- 可经呼吸道、消化道、皮肤吸收。在田间施药过程，经皮肤吸收是最重要的吸收方式
- 肝脏代谢
	- 反式异构体——酶水解
	- 顺式异构体——酶氧化
	- 毒性：反式<顺式
- 尿排出：尿中拟除虫菊酯原型或代谢产物可作为接触指标
- 具有神经毒性，毒作用机制未完全阐明
- 拟除虫菊酯类农药影响哺乳动物等多种生物的钠离子通道活性，损伤神经细胞的DNA，诱发神经细胞的凋亡
- **尿Br<sub>2</sub>CA（顺式二溴乙烯二甲基环丙烷羧酸）** 可作为接触性生物标志物

## 4.3临床表现
- 急性中毒
	- 职业性接触，多为经皮、呼吸道吸收，症状一般较**轻**，主要表现为**皮肤粘膜刺激症状和全身症状**
	- 误服：大量口服，症状出现快，主要为上腹部灼痛、恶心、呕吐等**消化道症状**。另有：心慌、视物模糊、多汗等，四肢肌肉肌束震颤；昏迷和发作性抽搐；肺水肿
- 变态反应：诱发过敏性**哮喘**
- 生殖毒性：降低哺乳动物精子活动度，精母细胞畸形率增加，睾酮水平下降
- 心血管毒性：心肌酶谱活性增高，心悸

## 4.4职业性急性拟除虫菊酯中毒诊断标准及处理原则（GBZ43-2002）
**诊断原则**
- 接触史
- 神经系统症状
- 现场调查
- 排除其他类似疾病

**诊断分级**
- 观察对象
- 轻度中毒
- 重度中毒

**处理原则**
- 立即脱离现场，清洗受污皮肤
- **无特效解毒药**，对症治疗和支持治疗
- 如为拟除虫菊酯与有机磷混配农药中毒，**先根据急性有机磷中毒治疗**原则进行处理，后给予对症治疗

## 4.5有机磷与拟除虫菊酯类农药的区别
- 对害虫的毒力：有机磷<拟除虫菊酯
- 对人畜毒性：有机磷>拟除虫菊酯
- 抗药性、使用方式、应用范围：有机磷占优势
