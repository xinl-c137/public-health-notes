# 1.生命早期营养
**人的年龄段分期**

| 年龄段 | 生物学分期 |
| --- | --- |
| 从受精卵发育到胎儿娩出 | 胎儿期 |
| 出生～28天 | 新生儿期 |
| 28天～1周岁 | 婴儿期 |
| 1~3周岁 | 幼儿期 |
| 3〜6周岁 | 学龄前期 |
| 6〜12周岁 | 学龄期 |
| 12~18周岁 | 青春期 |
| 18~60周岁 | 成年期 |
| 60岁以上 | 老年期 |

**NCDs预防与控制的生命周期论（Life-course approach）**：NCDs预防与控制应贯穿于全生命周期各阶段，其中生命早期的干预是NCDs一级预防的重要内容

**胎儿起源假说(the fetal origins hypothesis)**：胎儿在怀孕中晚期营养不良，会引起其生长发育失调，从而导致晚年易患冠心病
出生后的生长方式也会对成人期疾病的发病存在重要影响

## DOHaD(Developmental Origins of Health and Disease)健康与疾病的发育起源
**理论发展历程**
- 成人疾病的传统认识
	- 基因+成人期环境=成人疾病
- 胎儿起源假说
	- 胎儿宫内不良反应使其自身代谢和器官的组织结构发生适应性调节，导致机体组织和器官在代谢结构上发生永久性改变，进而演变为成人期疾病。这一漫长的“程序化”变化可被许多后天的环境因素放大，从而增强和加速成人疾病的发展过程
- DOHaD理论
	- 除遗传和环境因素，在发育过程的早期（包括胎儿和婴幼儿时期） 经历不利因素，将会增加成年后罹患肥胖、糖尿病、心血管疾病等慢性代谢性疾病的几率

**理论学说——生命早期1000天**：《中国0～6岁儿童营养发展报告（2012）》梳理了中国近三十年来有关0～6岁儿童营养调查和监测数据，提出生命早期1000天，是决定孩子一生营养与健康状况的最关键时期

**理论基础**
- 发育可塑性假说(Developmental Plasticity)：在胎儿发育过程中，在不同环境的影响下，一种基因型能产生多种不同的生理和形态学状态的现象
- 节俭表型假说(Thrifty Phenotype Hypothesis)：胎儿在发育过程中，如果遇到不利的生长环境，将改变其发育轨道与之相适应，这种改变是永久性的
- 适应性反应(Adaptive Response, ARs)：机体面对不同的环境影响的干扰，为了抵消部分不良影响，做出的适应性反应，包括：即刻的适应性反应（Immediately Adaptive Response, IARs）和预测的适应性反应（Predictive Adaptive Response, PARs）

# 2.孕妇营养
**孕期生理特点**
- 内分泌改变：卵巢及胎盘激素分泌增加（HCG、HCS、雌激素、孕酮）
- 代谢改变：合成代谢↑、基础代谢率↑
- 消化系统功能改变：牙龈炎、牙齿松动、胃肠蠕动↓、消化液分泌↓，某些营养素的吸收↑
- 血液变化：生理性贫血，血甘油三酯和VE↑
- 肾功能改变：肾小球滤过率↑、肾血流量↑、肾小管回吸收～
- 体重增加：7水/3脂肪/1蛋白

**《妊娠期妇女体重增长推荐值标准》（WST801—2022）**

| 孕前体重 分类 | 孕前BMI (kg/m{2) | 孕期总增重范围 (kg) | 孕早期增重范围 (kg) | 孕中、晚期增重均值及范 围 (kg/周) |
| --- | --- | --- | --- | --- |
| 低体重 | <18.5 | 11.0-16.0 | 0-2.0 | 0.46 (0.37-0.56) |
| 正常体重 | 18.5-24.0 | 8.0-14.0 | 0-2.0 | 0.37 (0.26-0.48) |
| 超重 | 24.0-28.0 | 7.0-11.0 | 0-2.0 | 0.30 (0.22-0.37) |
| 肥胖 | ≥28.0 | 5.0-9.0 | 0-2.0 | 0.22 (0.15-0.30) |

**孕期营养不良对母亲的影响**
- 营养性贫血（20～50%，35%）
	- 缺铁性贫血
	- 巨幼红细胞性贫血
	- 原因：膳食铁、叶酸摄入量不足；非血红素铁吸收利用率低；对铁的需要量增加；失血
- 骨质软化症：VD、钙
- 营养不良性水肿：蛋白质、 VB1
- 维生素缺乏症

**孕期营养不良对胎儿的影响**：新生儿死亡、小于胎龄儿、早产儿、低体重儿、先天畸形、巨大儿、脑发育受损

**孕早期营养与饮食原则**：碳水化合物>130g/d；叶酸400μg/d；清淡、易消化、口感好、少食多餐

**孕中期营养与饮食原则**
- 孕中期
	- 生长发育增快，特别是脑的发育
	- 内脏系统开始分化，开始形成循环、肝、肾功能
- 充足的能量
- 注意铁的补充
- 保证充足的鱼、禽、蛋、瘦肉和奶的供给
- 孕中期容易缺乏的营养素：铁、钙、锌、碘、DHA

**孕晚期营养与饮食原则**
- 胎儿压迫，饭量减少，易“饱”
	- 宜选择体积小、营养价值高的食物
	- 少食多餐
- 胎儿生长最快，脑细胞、脂肪细胞增殖敏感期
	- 蛋白质、磷脂、维生素（+）
	- 碳水化合物、脂肪（能量–）

**孕期饮食需注意的问题**
- 咖啡因
	- 咖啡因可增加心率，兴奋剂和利尿剂（影响钙、铁吸收利用）
	- 进入胎儿血液循环，在胎儿血中存在时间比母体更长
	- 与流产、先天畸形有关
- 酒精
	- 致出生缺陷、智力障碍和发育障碍
	- 酒精进入胎儿体内，而胎儿体内缺乏酒精代谢酶
	- 酒精对器官形成、生长发育和智力发育有害，增加流产、死产、婴儿死亡以及胎儿酒精综合征的危险
	- 妊娠期应避免饮酒

==**备孕期和孕期妇女膳食原则==**
- 调整孕前体重至正常范围，保证孕期体重适宜增长
- 常吃含铁丰富的食物，选用碘盐，合理补充叶酸和维生素D
- 孕吐严重者，可少量多餐，保证摄入含必需量碳水化合物的食物
- 孕中晚期适量增加奶、鱼、禽、蛋、瘦肉的摄入
- 经常户外运动，禁烟酒，保持健康生活方式
- 愉快孕育新生命，积极准备母乳喂养

# 3.乳母营养
**哺乳期生理特点**
- 内分泌变化
	- 雌激素、孕激素、胎盘生乳素等水平⇊
	- 催乳素⇈
- 乳汁生成
	- 初乳(～1W)：Na、Cl、IgA、乳铁蛋白
	- 过渡乳(2W)：乳糖、脂肪⇈，蛋白↓
	- 成熟乳(>2W)：富含蛋白、乳糖、脂肪等
- 泌乳量
	- 第二天100ml
	- 第二周500ml
	- 1个月650ml
	- 3个月750~850ml

**哺乳对乳母健康的影响**
- 近期影响
	- 促进产后子宫恢复
	- 避免乳房肿胀和乳腺炎
	- 延长恢复排卵的时间间隔
- 远期影响
	- 促进体重恢复
	- 体内钙流失
	- 降低乳母乳腺癌和卵巢癌的危险性

**乳母合理膳食原则**
- 产褥期
	- 分娩后0.5～1周进食流质、半流质食品
	- 次日开始进食易消化食物，多用汤类
	- 会阴撕裂伤严重应给无渣膳食1周左右
	- 剖腹产术后24h给与流质食物1天，但忌牛奶、豆浆等胀气食品
	- 补充蛋白质和铁等造血原料，以弥补分娩失血
	- 适量摄入蔬菜、水果，补充维生素C和膳食纤维，防止便秘
- 哺乳期
	- 食物种类齐全多样化
		- 每日4～5餐
		- 主食粗细搭配
	- 供给充足的优质蛋白质
		- 每天200～250g
	- 多食含钙丰富的食品
	- 多食含铁丰富的食品
	- 摄入足够的新鲜蔬果和海产品
		- 蔬果每天500g以上
	- 少吃盐、腌制品和刺激性强的食物
	- 合理烹调
		- 肉类：煮、煨
		- 蔬果注意维生素的保留
- 肥胖孕妇分娩后不宜吃减肥膳食，母乳喂养稳定后，每月减重不多于2kg
- 慎服药物，避免接触有害物质

**乳母膳食原则**
- 产褥期食物多样不过量，坚持整个哺乳期营养均衡
- 适量增加富含优质蛋白质及维生素A的动物性食物和海产品，选用碘盐，合理补充维生素D
- 家庭支持、愉悦心情、充足睡眠，坚持母乳喂养
- 增加身体活动运动，促进产后恢复健康体重
- 多喝汤和水，限制浓茶和咖啡，忌烟酒

# 3.婴儿营养
**婴儿的生理特点**
- 生长发育
	- 0～1岁，是人体生长发育的第一个高峰期，尤其是6个月内
	- 体重：1岁时达出生时的3倍
	- 身长：1岁时可达出生时的1.5倍（75cm）
	- 脑重：6月龄达出生时的2倍（600～700g），1岁达900 ～ 1000g（成人脑重的2/3）
	- 头围：出生平均34cm，1岁时达46cm，脑细胞数目持续增加
	- 胸围：比头围小1～2cm，但增长速度快，到1岁时与头围基本相等，并开始超过头围
- 消化吸收
	- 消化系统尚处于发育阶段，功能不完善
	- 口腔
		- 粘膜柔嫩，易受损伤
		- 双颊脂肪垫发育良好，利于吮吸
		- 唾液腺发育不完善，唾液淀粉酶低
	- 牙齿：6～8个月开始萌出，咀嚼能力差
	- 食管和胃
		- 食管细短
		- 水平胃，容量小（新生儿25～50ml，6个月200ml，1岁300～500ml）
		- 幽门括约肌发育良好
		- 贲门括约肌发育不良
		- 胃酶活力低
	- 肠道
		- 固定性差，易肠套叠
		- 蠕动较成人差
		- 肠激酶和肽酶，有助于消化蛋白质
	- 胰腺：发育尚不成熟，淀粉和脂肪类消化酶活力低
	- 肝脏
		- 相对较大，但肝细胞分化不全
		- 肝功能较差，胆汁分泌少
- 神经系统
	- 大脑皮层细胞快速增殖、增大和分化
	- 是智力发育的关键时期
- 排泄系统：肾小球滤过率低、回吸收差

**婴儿喂养**
- 母乳喂养
- 人工喂养
- 混合喂养

==**母乳喂养的优点**==
- 母乳中营养素全面、质量好、比例合适，适应婴儿消化功能和发育需要
	- 优质蛋白---乳清蛋白
	- 脂肪含量高于牛奶且含脂酶，易消化，含丰富必需脂肪酸，避免婴儿湿疹；DHA
	- 乳糖---促进肠道钙吸收
	- 钙、磷比例合适
	- 除维生素D较少，其他可满足4～6个月
- 母乳富含免疫物质
	- 母乳中特异性免疫物质（初乳）：淋巴细胞、抗体、IgA
	- 母乳中非特异性免疫物质：吞噬细胞、乳铁蛋白酶、溶菌酶、补体C3因子、双歧杆菌因子
- 母乳不容易发生过敏
- 哺乳增进母婴交流，促进婴儿智力发育
- 哺乳促进乳母体形恢复
- 母乳卫生、经济、方便、温度适宜，新鲜不变质

**0~6月龄婴儿喂养指南**
- 母乳是婴儿最理想的食物，坚持6月龄内纯母乳喂养
- 生后1小时内开奶，重视尽早吮吸
- 回应式喂养，建立良好的生活规律
- 适当补充维生素D，母乳喂养无需补钙
- 任何动摇母乳喂养的想法和举动，都必须咨询医生或其他专业人员，并由他们帮助做出决定
- 定期监测婴儿体格指标，保持健康生长

**0~6月龄婴儿喂养注意事项**
- 初乳营养好，尽量让新生儿吃到
- 掌握正确的母乳喂养姿势和方法
- 纯母乳喂养的婴儿不需特别补钙
- 推荐新生儿出生后补充维生素K
- 婴儿配方奶不如母乳
- 体重增长并非越快越好

==**7~24月龄婴幼儿喂养指南==**
- 继续母乳喂养，**满6月龄起必须添加辅食**，从富含铁的泥糊状食物开始
- 及时引入多样化食物，重视动物性食物的添加
- 尽量少加糖盐，油脂适当，保持食物原味
- 提倡回应式喂养，鼓励但不强迫进食
- 注重饮食卫生和进食安全
- 定期监测体格指标，追求健康生长

**7~24月龄婴儿喂养注意事项**
- 添加辅食不等于断奶，单独加工制作辅食
- 应在婴儿健康、消化功能正常时添加辅食
- 一种辅食适应数日后再增加新品种
- 辅食中适量添加植物油(α-亚麻酸）
- 进餐规律、环境良好，不以食物作为奖励或惩罚
- 早期培养孩子自主进食
