# 1.概述
**定义**：维生素(vitamin)是一类维持机体正常生理功能及细胞内特异代谢反应所必需的低分子有机化合物

**特点**
- 体内不能合成或合成数量不能满足机体需要，必须由食物提供
- 需要量少：以微克或者毫克计
- 不构成机体原料、不提供能量，具有生理调节功能

**维生素的命名系统**
1. 以拉丁字母命名，按发现顺序，如维生素A、维生素B、维生素C、维生素D、维生素E...
2. 根据其化学结构特点命名，如核黄素、视黄醇等
3. 根据生理功能命名，如抗坏血酸、抗干眼病维生素

**维生素的分类**
- 脂溶性维生素(fat-soluble vitamins)
	- 维生素A
	- 维生素D
	- 维生素E
	- 维生素K
- 水溶性维生素(water-soluble vitamins)
	- B族维生素
		- 硫胺素(B1)
		- 核黄素(B2)
		- 尼克酸(B3)
		- 叶酸(B9)
		- 维生素B12
		- 维生素B6
		- 生物素(B7)
		- 泛酸(B5)
	- 维生素C

|         | 脂溶性维生素 (lipid-soluble) | 水溶性维生素 (water-soluble)          |
| ------- | ---------------------- | ------------------------------- |
| 化学组成    | 仅含C、H、O                | 除C、H、O外，常含N、S等元素                |
| 容解性     | 溶于脂肪和有机溶剂，不溶于水         | 可溶于水，不溶于脂肪和有机溶剂                 |
| 体内贮存    | 在肝脏和脂肪组织中大量贮存          | 体内贮存量极少 $(\mathrm{B}_{12} 例外 )$ |
| 缺乏症出现速度 | 摄入不足时，缺乏症状出现缓慢         | 摄入不足时，缺乏症状出现较快                  |
| 过量中毒风险  | 易在体内蓄积，过量易中毒           | 多余部分随尿液排出，不易中毒                  |

**维生素缺乏的原因**
- 维生素的摄入量不足
	- 食物短缺、或选择食物不当，运输、加工、烹调、储藏过程中维生素丢失或被破坏
- 机体的吸收利用率降低
	- 老人胃肠道功能降低，对维生素的吸收利用降低；肝、胆疾病病人由于胆汁分泌减少会影响脂溶性维生素的吸收；膳食拮抗因素
	- 共同作用：维生素B1、维生素B2、烟酸共同参与能量代谢（糖、脂肪、蛋白质分解）
	- 协同作用：维生素E（配合维生素C、硒）保护维生素A不被氧化，促进其在肝脏储存
	- 拮抗作用：高剂量维生素E会抑制维生素K吸收，影响凝血功能
- 维生素的需要量相对增高
	- 由于维生素的需要量增多，或丢失增加，使体内维生素需要量相对增高。如妊娠和哺乳期妇女、生长发育期儿童、特殊生活及工作环境的人群、疾病恢复期病人

**维生素缺乏的分类**
- 按缺乏原因：原发性和继发性维生素缺乏
	- 原发性维生素缺乏：是由于膳食中维生素供给不足或其生物利用率过低引起
	- 继发性维生素缺乏：是由于生理或病理原因妨碍了维生素的消化、吸收、利用或因需要量增加、排泄或破坏增多而引起的条件性维生素缺乏
- 按缺乏程度：临床和亚临床维生素缺乏
	- 维生素缺乏在体内有一个渐进过程。初始时表现为贮备量降低，继而出现有关生化代谢异常、生理功能改变，并出现临床症状和体征
	- 症状
		- 维生素亚临床缺乏或不足症。轻度缺乏时，出现体力和脑力劳动效率下降，对疾病的抗力下降（较为常见）
		- 维生素临床症状或缺乏症。严重缺乏时，临床上可见相应的维生素缺乏的独特症状和体征

**维生素摄入过量**
- 过量的危害
	- 当水溶性维生素摄入过多时，常以原形从尿中排出，但超过生理需要量时有不良影响，如维生素代谢异常，干扰其它营养代谢
	- 当脂溶性维生素摄入过多时，可导致体内贮存过多引起中毒
- 摄入维生素应遵循的原则：必须注意某些含维生素丰富食物的过量摄入，也要注意维生素制剂的大量服用，要遵循合理原则

# 2.脂溶性维生素
## 2.1维生素A
**概述**
- 维生素A，是指含有**视黄醇**（Retinol）结构，并具有其生物活性的一大类物质，又称抗干眼病维生素
- 视黄醇：贮存在肝脏中，由肝脏进入血液
- 视黄醛和视黄酸：在细胞需要的时候，由视黄醇转化而来
- 维生素A对异构、氧化和聚合作用敏感，应避免与氧、高温和光接触
- 对酸和碱比较稳定，食物中含有抗氧化剂时较为稳定，脂肪酸败时破坏严重

**维生素A原**
- 已形成的维生素A：动物性食物中含有的维生素A
- 维生素A原：植物中存在的能在人体内转化为维生素A的**类胡萝卜素**
- 目前已发现的具有维生素A活性的主要有玉米黄素、α-胡萝卜素、β-胡萝卜素、γ-胡萝卜素、β-隐黄素，其中以**β-胡萝卜素**的生物活性最高，是维生素A的重要来源
	- 1分子β-胡萝卜素可以生成2分子视黄醛
	- 在动物实验中，相同质量的β-胡萝卜素与视黄醇相比：β-胡萝卜素的生理活性只有视黄醇的一半；β-胡萝卜素的吸收率只有视黄醇的三分之一

**维生素A的吸收与代谢**
![[4.维生素与健康-20260515.excalidraw]]

**维生素A的生理功能**
- 视觉
	- 构成视觉细胞内感光物质-**视紫红质**的成分
		![[4.维生素与健康-2.脂溶性维生素-20260515.png]]
	- 保持角膜的健康
		- 如果摄入的维生素A极为不足，则角蛋白会聚集于角膜中，并引起眼球干燥、干眼病、失明
- 维持细胞的正常生长和分化：视黄酸及其代谢物可与核受体超家族视黄酸受体/类视黄醇X受体（RAR/RXR）结合，影响基因的表达，调节组织细胞的生长和分化
- 维护上皮组织细胞功能：皮肤和上皮组织细胞（呼吸道、消化道、泌尿生殖道等）的糖蛋白合成
- 调节机体免疫功能：调节细胞和体液免疫
- 抗氧化：类胡萝卜素捕捉自由基，淬灭单线态氧
- 抑制肿瘤生长：与调节细胞的分化、增殖、凋亡、抗氧化有关

==**维生素A的缺乏**==
- 使眼暗适应能力下降，患夜盲症和干眼病
- 使粘膜、上皮组织结构破坏
	- 皮肤粗糙、干燥、鳞状等角质化变化
	- 口腔及消化道、呼吸道和泌尿生殖道的粘膜失去滋润和柔软性，使细菌易于侵入，儿童易患支气管肺炎
	- 泌尿道上皮损伤脱落，角化过度，患泌尿系统结石
- 生长发育受阻，特别是儿童的骨骼、牙齿发育受影响
- 味觉、嗅觉减弱，食欲下降
- 比托斑：儿童维生素A缺乏最重要的临床体征，角膜两侧和结膜外侧因干燥而出现褶皱，角膜上皮堆积，形成大小不等的形状似泡沫的白斑

**维生素A的过量**
- 急性中毒：恶心、呕吐
- 慢性中毒：头痛、脱发、肝大、肌肉僵硬、皮肤瘙痒、复视
- 致畸毒性：胚胎吸收、流产、畸胎、出生缺陷
- 高胡萝卜素血症：皮肤呈黄色或橙黄色

**维生素A的营养状况评估**
- 血清维生素A水平：可用来判断缺乏或边缘性缺乏，不能判断营养充足。成人<0.35μmol/L缺乏；0.35~0.70μmol/L边缘性缺乏
- 相对剂量反应试验（relative dose response test, RDR）：间接估计肝脏维生素A水平
- 稳定同位素测定：金标准，反映肝脏维生素A储存
- 视觉暗适应功能测定：人群现场调查
- 血浆视黄醇结合蛋白：与血浆视黄醇水平呈正相关
- 眼结膜印迹细胞学法：临床检查
- 眼部症状检查：临床诊断体征

## 2.2维生素D
**概述**
- 又称抗佝偻病维生素，与体内其它的营养物质不同，**身体借助于阳光就可合成维生素D**
- 最常见的是维生素D2（麦角钙化醇）和维生素D3（胆钙化醇）
- 维生素D的活性形式为1，25-二羟基胆钙化醇
- 维生素D2和D3化学性质稳定，在中性和碱性溶液中耐热，不易被氧化；在酸性溶液中逐渐分解，烹调加工不会引起损失；脂肪酸败可导致其破坏

**来源、吸收与生理功能**
![[4.维生素与健康-2.脂溶性维生素-20260515-1.png]]

**维生素D的生理功能**
- 促进钙、磷在小肠内的吸收
- 促进肾对钙、磷的重吸收
- 维持血钙水平
- 促进骨头、软骨和牙齿的钙化
- 免疫调节、炎症反应
- 高血压、糖尿病、心血管疾病

==**维生素D缺乏**==
- 危害骨骼——佝偻病(rickets)
- 成年人：骨软化症、骨质疏松

**维生素D过量的危害**
- 维生素 D 在所有维生素中潜在的毒性最大，婴儿最有可能发生维生素 D 中毒
- 维生素D长期过量摄入会引起食欲减退、恶心和呕吐等中毒症状；并可导致心脏和肾脏钙沉积
- 由于维生素 D 对中枢神经系统的作用还可能导致严重的心理压抑

**维生素D的机体营养状况评估**
血25-(OH) D₃作为首选评价机体维生素D状况的指标
- ＜10 ng/ml（25 nmol/L）：严重缺乏
- ＜20 ng/ml（50 nmol/L）：缺乏
- 21~29 ng/ml（52~72 nmol/L）：不足
- ≥30 ng/ml（75 nmol/L）：充足
- 100 ng/ml（259 nmol/L）：正常值上限
- ＞150 ng/ml（375 nmol/L）：可发生中毒

**维生素D的食物来源**
- 经常晒太阳是人体获得充足有效的维生素D的最好来源
- 主要存在于海水鱼（如沙丁鱼）、肝脏、蛋黄等动物性食品及鱼肝油制剂中
- 蔬菜、谷类及其制品和水果只含少量或几乎不含维生素D
- 婴儿出生后两周左右，补充维生素D

# 3.水溶性维生素
## 3.1B族维生素
B族维生素是**辅酶（coenzyme）**的组成成分

**B族维生素与代谢**
- 硫胺素、核黄素、尼克酸、泛酸和生物素参与了糖、脂肪和蛋白质的代谢产能过程
- 维生素B6帮助身体利用氨基酸制造蛋白质
- 叶酸和维生素B12帮助细胞增殖

**B族维生素缺乏**
B族维生素广泛参与了人体的能量和物质代谢，缺乏时会产生系列不良影响
- 恶心
- 严重虚脱
- 易怒
- 抑郁
- 健忘
- 食欲不振
- 体重下降
- 肌肉疼痛
- 免疫系统受损
- 无法控制四肢
- 心脏活动异常
- 严重的皮肤病
- 流泪或眼球充血
- 消化道损伤
- 脑部永久性损伤

人类通常都是同时缺乏多种B族维生素，而最明显的症状掩盖了不明显的症状

### 维生素B1（抗脚气病维生素）
**概述**
- 也称抗脚气病因子和抗神经炎因子，由含氨基的嘧啶环和含硫的噻唑环通过亚甲基桥相连而成，因分子中含有“硫”和“氨”，故又称**硫胺素**（thiamin）
- 纯品为白色粉末状结晶，易溶于水，微溶于乙醇
- 酸性环境下较稳定，加热120℃仍不分解
- 中性和碱性环境中不稳定，易被氧化和受热破坏
- 粗粮、全谷物的加工精度越高，维生素B₁损失越严重；同时，蒸煮、炖等低温、偏酸性的烹饪方式更有利于保留食物中的维生素B₁

==**维生素B1的生理功能**==
- 辅酶功能：参与体内物质和能量代谢
- 非辅酶功能：调节神经生理活动，与心脏活动、食欲维持、胃肠道蠕动有关

==**维生素B1的缺乏**==
- 硫胺素缺乏症：==脚气病（beriberi）==（以抛光的大米为主食）
	- 损害神经系统、心血管系统和消化系统。出现神经炎、水肿、食欲不振、大便秘结，严重时可出现心力衰竭
- Wernicke-Korsakoff（韦尼克- 科尔萨科夫）综合征
	- 嗜酒，妨碍对硫胺素的吸收并促进硫胺素随尿液排出，导致Wernicke-Korsakoff综合症。患者会陷入精神紊乱、协调性不足，还可能出现记忆和学习障碍

**维生素B1的机体营养状况评估**
**尿负荷试验**：通过口服负荷剂量后测定尿中排出量，反映体内维生素的储备水平
清晨口服5mg维生素B₁，测定4h内尿中维生素B₁含量（μg）
- ＜100 μg：缺乏
- 100~199 μg：不足
- ≥200 μg：正常
- ≥400 μg：充裕

**维生素B1的食物来源**
维生素B1含量丰富的食物有谷类、豆类及干果类。动物内脏、瘦肉、禽蛋 中含量较高。加工和烹调可造成维生素B1的损失，其损失率30～40%

### 维生素B2
**概述**
- 又称**核黄素**（riboflavin），是具有一个核糖醇侧链的异咯嗪类衍生物
- 黄色粉末状结晶，味苦，熔点高，水溶性相对较低
- 酸性及中性环境中对热稳定，碱性环境中易被热和紫外线破坏
- 有游离及结合两种形式，游离状态易发生光裂解，结合状态较稳定
- 主要在胃肠道上部吸收
- 体内多余的维生素B2主要随尿液排出，未被吸收的维生素B2随粪便排出，汗液亦可排出少量维生素B2

**维生素B2的生理功能**
- 辅酶的功能
	- 维生素B2在体内转化为活性形式的黄素单核苷酸（FMN）和黄素腺嘌吟二核苷酸（FAD），它们是体内多种氧化酶系统不可缺少的辅基
	- 参与体内生物氧化与能量代谢酶功能：以黄素蛋白参与许多酶系，如氨基酸氧化酶、细胞色素C还原酶、丙酮酸脱氢酶等，维持蛋白质、脂肪和碳水化合物的正常代谢，促进生长发育，维持皮肤和粘膜的完整性
- 激活VB6，参与色氨酸转化为尼克酸
	- 作为辅酶参与维生素B6转变为磷酸吡哆醛，色氨酸转变为烟酸
- 其他的功能
	- FAD作为谷胱甘肽还原酶的辅酶，参与氧化防御；FAD与细胞色素P450结合，参与药物代谢等

**维生素B2的缺乏**
- 维生素B2缺乏主要的临床表现为**眼、口腔和皮肤的炎症反应**，称为“口腔生殖系统综合征”
	- 眼：表现为睑缘炎、畏光、视物模糊和流泪等
	- 口腔：口角炎、唇炎、舌炎、地图舌
	- 皮肤：脂溢性皮炎

**维生素B2的食物来源**：牛奶、肝脏、强化谷物、蘑菇、菠菜

### 维生素B3（尼克酸）
**概述**
- 又称烟酸、维生素PP、抗癞皮病因子
- 尼克酸也参与人体的能量代谢。在体内主要以尼克酰胺的形式存在
- 烟酸是所有维生素中最稳定的一种，不易被空气中的氧、热、光、高压所破坏，酸、碱也很稳定
- 成年人体内的烟酸可由色氨酸转化而来，需要维生素B₁、维生素B₂和维生素B₆参与这一过程

**尼克酸的生理功能**
- 为辅酶I（NAD+）与辅酶II（NADP+）的组成成分
	- 辅酶I、辅酶II为体内一系列脱氢酶的辅酶，是生物氧化过程中氢和电子的传递体，参与碳水化合物、脂肪和蛋白质在体内代谢过程中的脱氢作用
- 在维生素B6、泛酸和生物素存在下，参与脂肪、蛋白质和DNA的合成
- 降低血清胆固醇和扩张末梢血管水平
- 维护皮肤、消化系统及神经系统的正常功能

==**尼克酸的缺乏**==
- **尼克酸缺乏症-癞皮病/糙皮病**（pellargra），主要损害皮肤、口、舌、胃肠道粘膜以及神经系统
- 3D综合征：Diarrhea（腹泻）、Dermatitis（皮炎）、Dementia（痴呆）
- 尼克酸缺乏常与维生素B1、B2及其它营养素缺乏同时存在

**尼克酸的食物来源**
- 膳食中尼克酸的参考摄入量以烟酸当量（niacin equivalence, NE）表示
- 烟酸当量（mg NE）= 烟酸（mg）+ 1/60 色氨酸（mg）
- 色氨酸在几乎所有的蛋白质中含量都很丰富，但玉米蛋白质中含量极少

### 叶酸（维生素B9）
**概述**
- 叶酸（folic acid）也称为维生素 B₉、其化学名称是蝶酰谷氨酸
- **辅酶形式是四氢叶酸**
- 以单谷氨酸盐的形式被吸收，到达肝脏后重新转变成多聚谷氨酸盐的形式，大部分储存于肝脏，通过尿和胆汁排泄

==**叶酸的生理功能**==
- 作为一碳单位的载体参与代谢
- 通过调控核苷酸代谢影响DNA和RNA的合成
- 通过蛋氨酸的代谢影响磷脂、肌酸、神经介质以及血红蛋白的合成

**叶酸的缺乏**
- 可导致**贫血和消化机能紊乱**
- 高同型半胱氨酸血症
- 巨幼红细胞性贫血；神经系统紊乱
- 孕妇：先兆子痫、胎盘早剥、流产
- 胎儿：宫内发育迟缓、新生儿低出生体重、神经管畸形（孕早期）

**叶酸的来源**：肝脏、斑豆、小扁豆、芦笋、菠菜

## 3.2维生素C
**维生素C的吸收与代谢**
- 主要通过主动转运形式由小肠上段吸收进入血液循环。胃酸缺乏时，维生素C的吸收减少
- 与大多数水溶性维生素不同，维生素C在体内有一定量的储存
- 主要随尿排出，其次为汗和粪便
- 尿中排出量与体内贮存量、摄入量和肾功能有关。一般情况下，血浆维生素C含量与尿排出量有密切联系

==**维生素C的生理功能**==
- 作为抗氧化剂
	- 保护铁不被氧化，促进铁的还原和吸收
	- 促进叶酸还原和利用
	- 清除自由基、抗衰老
- 羟化过程底物和酶的辅助因子
	- 促进胶原蛋白合成
	- 促进去甲肾上腺素和5-羟色胺合成
	- 胆固醇转变为胆汁酸
- 促进抗体生成，增强免疫功能
- 解毒作用

==**维生素C的缺乏与过量**==
- 若体内维生素C储存量低于300mg可引起坏血病，临床表现如下
	- 前驱症状：起病缓慢，多有全身乏力、食欲减退
	- 出血
	- 牙龈炎
	- 骨质疏松
- 维生素C毒性很低。但是一次口服2~3g时可能会出现腹泻、腹胀；患有结石的病人，长期过量摄入可能增加尿中草酸盐的排泄，增加尿路结石的危险
