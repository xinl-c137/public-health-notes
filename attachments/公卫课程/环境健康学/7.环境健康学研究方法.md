**环境流行病学研究的方法**
- 描述性研究（如横断面研究）
- 分析性研究（如队列研究、病例对照研究、定组研究、时间序列研究)
- 实验性研究（如干预试验）
# 1.横断面研究
伦敦宽街霍乱
**介水传染病(water-borne communicable diseases)**
- 定义：指通过饮用或接触病原体污染的水而传播的疾病。肠道传染病多见，如霍乱、伤寒、痢疾、甲肝等
- 特点
	- 水源污染后，可呈暴发流行，短期内出现大量病人
	- 病例分布范围与供水范围一致
	- 对污染源采取措施，疾病流行能迅速控制

**横断面研究**又称<u>横断面调查</u>，因为所获得的描述性资料是在某一时点或在一个较短时间区间内收集的，所以它客观地反映了这一时点的疾病分布以及人们的某些特征与疾病之间的关联。由于所收集的资料是调查当时所得到的现况资料，故又称<u>现况研究或现况调查(prevalence survey)</u>。分为普查和抽样调查。

# 2.病例对照研究
日本痛痛病
**病例对照研究**也称为回顾性研究，是比较患某病者与未患某病的对照者暴露于某可能危险因素的百分比差异，分析这些因素是否与该病存在联系。是环境流行病学分析性研究中最基本、最重要的研究类型之一

**病例对照研究的特点**
- 设立对照组
- 从果到因
- 可用于探索疾病的可疑危险因素

**病例对照研究的内容与方法**
OR(比值比)：病例对照研究中，病例组和对照组中某事件发生的比值之比
比值：某事件发生的概率和未发生概率之比
病例组的暴露比值=a/b
对照组的暴露比值=c/d
OR(比值比)=病例组的暴露比值/对照组的暴露比值=ad/bc

**病例对照研究的优缺点**
- 优点
	- 特别适用于罕见疾病的研究
	- 省力、省时、省钱，相对容易组织
	- 可以同时研究多个因素与某种疾病的联系，特别适合于探索病因
	- 对研究对象多无损害
- 不足
	- 不适用于研究人群中暴露比例很低的因素
	- 信息的真实性难以保证，暴露与疾病发生的先后难以判断
	- 回忆偏倚
	- 不能测定暴露组与非暴露组疾病的率

# 3.队列研究
哈佛六城市研究
**定义**
队列研究是将某一特定人群按是否暴露于某可疑因素或暴露程度分为不同的亚组，追踪观察两组或多组成员结局(如疾病)发生的情况，比较各组之间结局发生率的差异，从而判定这些因素与结局之间有无因果关联及关联程度的研究方法

**特点**
- 设立对照组
- 从因到果
- 能确证暴露与结局的关系

**内容与方法**
RR(相对危险度)：队列研究中，试验组和对照组某事件发生的风险之比
暴露组（高暴露组）的发病率或死亡率=a/(a+b)
对照组（低暴露组）的发病率或死亡率=c/(c+d )
RR(相对危险度)=暴露组的发病率或死亡率/对照组的发病率或死亡率=a(c+d)/c(a+b)

**优缺点**
- 优点
	- 资料可靠，一般不存在回忆偏倚
	- 获得暴露组和对照组人群的发病率或死亡率，计算反应疾病危险强度的指标
	- 检验假设的能力较强，一般可证实病因关联
	- 了解疾病的自然史，可获得多种预期之外的疾病结局，一因多果
- 缺点
	- 不适用于发病率低疾病的病因研究
	- 容易产生失访偏倚
	- 研究耗费人力、物力和财力较多
	- 随访过程中，未知变量引入人群，或人群中已知变量发生变化，可能使结局受到影响，分析复杂化

# 4.时间序列研究
**定义**
用于研究某一时间段暴露与结局变量之间的关系，暴露与结局变量之间的关系，暴露和结局变量均为同一时间单位（如几天或几周）的累积测量数据。每一个变量的测量构成一个时间序列。
- 目标是评估随暴露而改变的健康结局序列的短期变化
- 健康结局变量可能在暴露改变后短期内出现反应
- 暴露变化与其导致的结局变化之间的时间段称为“滞后”

**特点**
- 同时收集某一时间段暴露与结局变量
- 通常收集常规监测资料
- 样本量大
- 用于分析随暴露而改变的健康结局序列的短期变化

**内容与方法**
- 同时收集某一时间段暴露与结局变量
- 通常收集常规监测资料
- 样本量大
- 用于分析随暴露而改变的健康结局序列的短期变化

**暴露评价**
- 主要利用常规监测数据，例如常规空气污染监测数据、气象监测数据
- 覆盖较大的区域

**健康效应**
- 基于常规收集的资料
- 覆盖的人群较大
- 关注的结局包括死亡、住院或门诊的资料

**优缺点**
- 优点
	- 费用低
	- 容易实施
	- 可获得较大的数据量，增加统计功效
- 不足
	- 监测污染物按标准选择，通常未考虑健康因素
	- 数据无法评估个体间暴露的变异
	- 健康结局数据受死亡或住院诊断错误等因素影响

# 5.定组研究
**定义**：收集研究人群中每个个体的暴露和健康变量的变化水平，目的是研究随时间变化的环境暴露对健康结局的短期效应

**内容与方法**
- 暴露评价
	- 短期评价
	- 可在个体水平上进行评估
	- 暴露评价频率密集
	- 可对多种环境因子实施监测
- 健康效应
	- 亚临床健康结局
	- 生物标志物（呼吸、心血管、组学指标等）
	- 用于探讨机制

**优缺点**
- 优点
	- 暴露评价和健康效应测量的精确度提高
	- 研究对象自身为对照，降低混杂因素影响
	- 纵向多个时间点测量，增大样本量
	- 可用于探索机制
- 缺点
	- 随访次数多，增加研究对象负担，降低依从性
	- 密集随访增加成本，限制研究对象数量
	- 研究对象退出导致数据缺失引起较大偏倚

# 6.干预研究
**概念**
- 研究者根据研究目的人为地对受试对象设置干预措施，通过随机化、对照等控制非干预措施的影响，从而分析干预措施的效果研究
- 除了考虑随机化分组、设置对照和盲法观察外，还需要考虑伦理、失访、依从性和主观感觉对研究结果的影响

**特点**
- 研究者可以通过随机抽样和随机分配的方法主动地控制与消除某些因素的干扰，使结果具有更好的真实性
- 研究的措施是人为施加的，而不是自然发生的
- 设立的对照组有利于结果的比较
- 研究组与对照组研究条件相同，研究时间同步
- 在研究结束时，即可获得研究结果

**优缺点**
- 优点
	- 随机分组，可比性好
	- 防止选择性偏倚
	- 研究对象明确
	- 盲法观察，结果可靠
- 不足
	- 较费人力、物力和时间
	- 研究对象均来源于合格的研究对象，研究结果外推到一般人群时受到一定程度的限制
	- 对照组不使用药物或疗法，只能使用安慰剂，因此可能存在不符合伦理的情况

# 7.环境毒理学研究
**环境毒理学研究内容**
- 环境毒物在机体的吸收、分布、代谢和排泄
- 环境毒物对机体的影响及毒作用机理
- 环境毒物对机体影响的作用条件和影响因素
- 环境毒物及其转化产物的毒性及其评定方法
- 制订环境卫生标准

**环境化学毒物的毒性评定方法**
- 急性毒性试验：目的是在短期内了解试验物质毒性大小，为其他毒性试验提供设计依据
	- 急性致死毒性试验：LD100与LD50
	- 急性非致死毒性试验
		- 急性阈剂量（浓度）的测定
		- 局部刺激试验
			- 急性皮肤刺激试验
			- 急性眼刺激试验
- 蓄积性和耐受性试验
	- 蓄积性试验：蓄积系数
		- 固定剂量法
		- 递增剂量法
		- 二十天试验法
	- 耐受性试验
- 亚急性、亚慢性毒性试验
	- 目的是在急性试验的基础上，在较短时间内了解受试物对机体的主要毒性作用及其靶器官，探讨敏感的观测指标和剂量一反应关系，为慢性毒性试验提供设计依据
- 慢性毒性试验
	- 目的是评价受试物在长期小剂量作用的条件下对机体产生的损害及其特点，确定其慢性毒作用阈剂量和最大无作用剂量，为制订最高容许浓度(MAC)提供依据
- 特殊毒性试验
	- 致突变试验
	- 致畸试验
	- 致癌试验
