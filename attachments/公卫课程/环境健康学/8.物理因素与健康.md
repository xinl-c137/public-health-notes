# 1.概述
**物理因素的种类**
- 气象因素：气温air temperature、湿度humidity、气流air pressure、气压等
- 电磁辐射：可见光、紫外线、红外线、X射线、γ射线
	- 电离辐射ionizing radiation：波长短，频率高，能量高，能够引起物质电离（产生电子）的射线，对人体危害大
	- 非电离辐射non-ionizing radiation：波长较长，频率较低，能量较低的射线或电磁波
	- 非电离辐射与电离辐射的区别在于其无法从（绝大多数）原子或分子里面电离出电子
- 噪声noise：在生活中，凡是干扰人们正常学习、工作和休息的声音；物理学上，振幅和频率杂乱无章
- 地质灾害geological disasters：崩塌、滑坡、泥石流、地面塌陷和地裂缝

**物理因素的特征**
- 常见的物理因素，除了激光之外，在自然界中均有存在
- 大部分物理因素在正常范围内不但对人体无害，反而是人体生理活动或从事生产活动所必需的因素

**物理因素的作用及危害**
- 物理参数
	- 每一种物理因素都有其特定的物理参数，如温度、振动的频率和速度、电磁辐射的能量或强度
	- 这些因素的物理参数与其所产生的效应程度之间存在密切的相关性
- 产生来源
	- 工作和生活中的各种物理因素一般都有其明确的来源
	- 如果产生某种有害物理因素的来源处于工作状态，则其产生的因素可能对人体造成健康危害；如果来源停止工作，则相应的有害物理因素便消失，不会造成健康损害
- 距离
	- 多数物理因素的强度一般是不均匀的，表现为以源为中心，向四周传播。如果没有阻挡，物理因素的强度常常随距离的增加呈指数关系衰减
- 传播形式
	- 某些物理因素，如噪声、微波等，可以有**连续波和脉冲波**两种传播形式。而不同的传播形式使得这些因素对人体的危害程度会有较大的差异
- 接触强度
	- 许多情况下，物理因素对人体的危害程度与物理参数之间不是呈直线的相关关系，而是表现为在某一强度范围内对人体无害，高于或低于这一范围才对人体产生不良影响
- 机体影响广泛
	- 噪声主要影响听觉系统，引起听力损伤和耳聋。还影响神经系统、心血管系统和内分泌系统
	- 高温则会破坏体温调节、水盐代谢和心血管功能等，严重者则会出现中暑
- 在体内残留少
	- 除了某些放射性物质，其他绝大多数物理因素在脱离接触后，体内便不再残留
	- 对物理因素所致损伤或疾病的治疗，无需采用“排出”的方法，主要是针对性处理受损组织器官
	- 机体在面对物理因素影响时，会产生一定自我保护的适应现象

**物理因素的危害防护**
- 根据物理因素的特点，在对环境进行检测时应对有关参数进行全面测量
- 同时，针对物理因素的健康危害所采取的相关防护措施，并不是设法消除这些因素，而是设法将这些因素控制在正常范围内，避免对人体健康构成损害

# 2.气象、气候与健康
## 2.1概念
**气候**：一个地区大气的多年平均状况，主要的气候要素包括气温、降水和光照等
**气候变化**：经过相当一段时间的观察，在自然气候变化之外由人类活动直接或间接地改变全球大气组成所导致的气候改变
**温室效应**
- 太阳光线透过温室的玻璃加热室内温度，但玻璃却阻碍温室内热量流失
- 云、水蒸气和自然温室气体二氧化碳、甲烷、一氧化二氮、臭氧让太阳辐射进入地球，但阻碍红外线热量外流，于是自然的温室效应(greenhouse effect)产生

**气象灾害和极端天气**
- 常见气象灾害：暴雨洪涝、台风、龙卷风、干旱、霜冻、暴风雪、冰雹、沙尘暴等
- 极端天气：主要指超常的高温、寒冷等天气
	- 高温天气：日最高气温≥35℃为高温天气，持续多日35℃以上的高温天气叫**热浪(heat wave)**
	- 寒冷天气：凡使当地24小时降温10℃以上或48小时降温12℃以上，且最低气温降至5℃以下的强冷空气称为**寒潮(cold wave)**

## 2.1健康影响
**气候变化的健康影响**
![[8.物理因素与健康-2.气象、气候与健康-20260501.png]]

**气象灾害和极端天气对健康的影响**
- 高温天气
	- 导致人体感到不适，工作效率降低
	- 导致中暑、胃肠道疾病、”空调病“和心血管病等增加
	- 机体抵抗力较差的老人、病人等因暑热死亡的人数将增加
- 寒潮
	- 主要由聚集在高纬度地区的强冷空气南下造成
	- 大气压和氧分压受温湿度、风速和海拔高度等影响，尤其是海拔高度影响较大
	- 高原移居者可因高原低氧引起高原反应、高原肺水肿等不良健康影响

**气候变化的应对策略**
- 加强气候监测系统建设
	- 对气候变化健康影响研究需要长期的、连续的、精确的一系列气象监测数据
- 建立和加强公共健康系统
	- 公共健康系统包括早期预警系统（主要监测极端气候事件的发生、感染性疾病的爆发、虫媒的数量等）和卫生保健系统
- 减缓和适应气候变化的复合效益分析
	- 未来需要将温室气体减排的空气质量改善和健康协同效应纳入全球气候变化政策制定的成本效益分析中，以进一步明确气候变化政策的健康和经济效应
- 气候变化相关的健康风险评估
	- 建立可以综合考虑未来社会、人口变化、技术进步和人类适应性改变等因素的模型

# 3.噪声与健康
**噪声的来源**：生产性噪声、交通噪声、社会生活噪声、建筑施工噪声
**噪声对健康的影响**
- 听觉系统：噪声可以引起听觉器官的损伤，这种损伤一般经历由生理变化到病理变化的过程，即先出现暂时性听阈位移，逐渐发展为永久性听阈位移
- 非听觉系统：可影响人体神经系统、心血管系统、内分泌及免疫系统、消化系统、生殖系统
- 日常工作生活：噪声会影响人们日常工作和生活

**噪声对听觉系统的影响**
- 暂时性听阈位移temporary threshold shift, TTS
	- 听觉适应auditory adaptation：指短时间暴露在强烈噪声环境中，机体听觉器官敏感性下降，听力检查听阈可提高10~15dB(A)
	- 听觉疲劳auditory fatigue：指较长时间停留在强烈噪声环境中，引起听力明显下降，听力检查听阈提高15~30dB(A)
- 永久性听阈位移permanent threshold shift, PTS
	- 听力损伤hearing loss：指听力曲线在3000~6000Hz（多在4000Hz）出现V字型下陷
	- 噪声性耳聋noise-induced deafness：指人们在工业生产和生活中，由于长期接触噪声而发生的一种进行性的感音性听觉损伤
	- 爆震性耳聋：爆震性耳聋是指在爆破等特殊条件下，由于防护措施不当，强烈爆炸所产生的冲击波造成听觉系统的急性外伤而引起的听力损失

**噪声对非听觉系统的影响**
- 神经系统：头痛、头晕、睡眠障碍、全身乏力、视觉运动迟缓
- 心血管系统：心率改变、心电图异常、血压升高
- 内分泌及免疫系统：肾上腺皮质功能改变、免疫功能降低等
- 消化系统：胃肠功能紊乱、食欲降低、脂代谢障碍
- 生殖功能及胚胎发育：女性月经不调、妊娠期高血压、男性不育、流产、畸形儿

**噪声对日常工作和生活的影响**
- 干预日常生活：超过50dB(A)的噪声就会影响正常休息和睡眠。65dB(A)以上噪声可干扰普通谈话
- 影响睡眠：噪声可以导致入睡时间延长、睡眠深度变浅、缩短醒觉时间、多梦。长期会引起神经衰弱
- 影响工作：由于噪声的干扰，人会感到烦躁，注意力无法集中，反应迟钝，不仅影响工作效率，而且降低工作质量

**噪声的卫生防护**
- 立法保障：《声环境质量标准》、《环境噪声检测技术规范 城市声环境常规检测》
- 控制噪声源：控制工业、道路、建筑、社会生活噪声源
- 控制噪声传播：隔声、消声、吸声、隔振和减振
- 个体防护：耳塞、耳罩、隔音头盔

# 4.电离辐射与健康
**电离辐射的分类**
- 根据来源
	- 天然放射性辐射
		- 天然放射性辐射主要来自于氡及其衰减产物，其次是宇宙射线
		- 天然放射性辐射还包括陆地源性辐射和内源性辐射。陆地源性辐射主要来自岩石、土壤等各种地理因素。而内源性辐射主要来自食物、饮水以及人体自身的放射性同位素
	- 人为放射性辐射
		- 医疗领域使用的X射线，生活中接触的各种建筑材料、家用电器，工业作业中对放射性微尘和核物质的暴露等
- 根据电离作用方式
	- 直接电离辐射
		- 带电的粒子（如电子、质子、α粒子和β粒子等）与物质作用时直接引起物质的原子或电子发生电离
	- 间接电离辐射
		- 不带电的粒子（如光子和中子等）与物质作用时无法直接引起物质电离，而使受暴露物质自身释放电离粒子或引起核反应
- 根据对人体作用方式
	- 外照射：电离辐射在人体外的辐射源对人体产生作用，主要为体外的γ射线、中子等。作用强度主要取决于机体吸收剂量的大小
	- 内照射
		- 进入人体的放射性物质直接在人体内部产生作用，一般主要为释放α和β射线的核素
		- 这些放射性核素进入人体内部，将会引起生物体内高能量的局部吸收，导致内照射，产生特异的生物学效应

**电离辐射对健康的影响**
- 电离辐射以外照射和内照射两种方式作用于人体
- 电离辐射对机体的损伤主要受辐射因子和机体两方面的因素影响
- 人们时刻处于低剂量的电离辐射中：天然本底辐射
- 日常生活常接触，但是剂量小
- 目前最关注：CT检查的电离辐射危害

**放射病(radiation sickness)**
- 一定剂量的电离辐射作用于人体所引起的全身性或局部性放射损伤被统称为放射病
- 临床上分为急性、亚急性和慢性放射病
- 我国的职业病目录将放射病分为以下11种
	- 外照射急性放射病
	- 外照射亚急性放射病
	- 外照射慢性放射病
	- 内照射放射病
	- 放射性皮肤疾病
	- 放射性肿瘤
	- 放射性骨损伤
	- 放射性甲状腺疾病
	- 放射性性腺疾病
	- 放射性复合伤
	- 其他放射性损伤

**电离辐射的卫生防护措施**
- 辐射防护的三原则
	- 放射实践的正当性：如果对受照射个体或社会产生的利益不足以抵消它带来的危害，则不予采用
	- 放射防护的最优化：任何电离辐射的实践应当避免不必要的照射。而任何必要的照射，应当在基于经济和社会因素后控制在能够降到的最低水平
	- 个人剂量和限值：人体所受照射剂量必须低于剂量限值。在潜在照射情况下，应低于危险度控制值
- 外照射防护：时间防护、距离防护、屏蔽防护、控制辐射源强度
- 内照射防护：设施防护为主，个人防护为辅；选择适当的放射性核素；兼顾外照射防护
- 其他
	- 个人防护措施：使用个人防护用品
	- 辐射监测：个人剂量监测
	- 辐射监测：放射性工作场所监测

# 5.非电离辐射与健康
**非电离辐射的分类及其健康危害**
- 静磁场
	- 定义：地球磁场、磁铁和稳恒电流等产生的频率为0Hz的磁场
	- 健康危害
		- 生物体在静磁场中运动可能会产生急性生物学效应，例如个体的运动或身体内部的运动（血液流动和心脏跳动等)
		- 人在强度超过2T的静磁场中运动会产生眩晕和恶心，口腔偶尔有金属异味感等
- 极低频电磁场
	- 定义：来源于输电线路、变电站、电气设备等产生的0~300Hz的电磁场
	- 健康危害
		- 世卫组织评估认为目前缺乏极低频电磁场与健康危害的充分证据
		- 不过有研究显示，极低频电磁场暴露可能会增加阿尔茨海默病等神经退行性疾病的发病风险，而短时间高强度（>100μT）的暴露会刺激神经等组织产生急性生物学效应
- 射频电磁场
	- 定义：频率在100kHz~300GHz的电磁辐射，也称为无线电波
	- 健康危害
		- 主要表现为非特异性类神经征，如全身无力、易疲劳、头晕、头痛、胸闷、心悸、睡眠不佳、多梦、记忆力减退、多汗、脱发和肢体酸痛等
		- 严重者可能出现局部器官的不可逆性损伤，如微波辐射会引起眼晶状体浑浊，发展为白内障
- 红外辐射
	- 定义：又称红外线，波长范围在760nm-1mm
	- 健康危害
		- 主要影响皮肤和眼。短时间可以导致皮肤局部红斑反应。反复照射可导致色素沉着。过量照射可能会发生皮肤急性灼伤，甚至透入皮下组织
		- 导致眼部的慢性损伤，常见表现为慢性充血性睑缘炎、角膜虹膜损伤、白内障和视网膜损伤
- 紫外辐射
	- 定义：又称紫外线，波长范围在100~400nm
	- 健康危害
		- 主要也是影响皮肤和眼。太阳光辐射中适量的紫外线可以使人体产生必需的维生素D3，对健康有积极的作用。但过强的紫外线辐射则对机体有害
		- 可诱发皮肤老化、皮肤癌、电光性眼炎、雪盲症等
- 激光
	- 定义：一种人造的、特殊类型的非电离辐射
	- 健康危害
		- 主要伤害的人体靶器官也是皮肤和眼。激光对皮肤的损伤轻度表现为红斑和色素沉着，而随着照射量的增加，可出现水泡、皮肤褪色、焦化乃至深部器官受损
		- 激光对眼部损伤的典型表现为视网膜烧伤，黄斑部损伤，最后导致视力急剧下降

**非电离辐射的防护措施**
- 电磁屏蔽
- 距离防护
- 个体防护
