# 1.概述
## 1.1毒性与毒效应
**毒性(toxicity)**：化学物引起有害作用的固有的能力。毒性是物质一种内在的，不变的分子性质，取决于其化学结构
**毒效应(toxic effect)**：化学物对机体健康引起的有害作用
**毒性和毒效应的区别**：毒性是化学物固有的生物学性质，而毒效应是化学物毒性在某些条件下引起机体健康有害作用的表现，改变条件可能影响毒效应

## 1.2癌
毒理学中“癌”的概念是广义的，包括
- 上皮组织的恶性变（癌）
- 间叶组织的恶行变（肉瘤）
- 良性肿瘤

**癌症的病因(causes of cancer)**
- 环境因素
	- 物理因素
	- **化学因素，90%以上**
	- 生物因素
- 遗传因素

**概念**
- **化学致癌作用(chemical carcinogenesis)**：化学物质引起正常细胞恶性转化并发展成肿瘤的过程
- **化学致癌物(chemical carcinogen)**：具有化学致癌作用的化学物质称为化学致癌物

**IARC基于证据权重的化学致癌物分类**
- 组1：对人类是致癌物

# 3.化学物致癌性的鉴定
## 3.1哺乳动物致癌试验
**哺乳动物致癌试验(long-term carcinogenicity study)**：鉴定哺乳动物化学致癌物的标准体内试验
- 目的
	- 确定受试化学物对实验动物的致癌性
	- 剂量-反应关系
	- 诱发肿瘤的靶器官
- 实验要点
	- 实验动物的选择
		- 常规选用大鼠和小鼠，也可用仓鼠
		- 品系：应选择较敏感、自发肿瘤率低、生活力强及寿命较长的品系。NCI推荐Fischer344大鼠和B6C3F1小鼠
		- 性别：应使用同等数量的雌雄两种性别的动物
		- 年龄：使用刚离乳的动物
	- 剂量设定与实验分组
		- 3个剂量组和1个对照组，每组至少雌雄各50只动物，希望在出现第1例肿瘤时，每组还有不少于25只动物
		- 美国NCI推荐以最大耐受剂量（MTD）为高剂量
		- ICH（1995）提出，高剂量选择可根据：①毒性终点，即最大耐受剂量（MTD）；②药代动力学终点，啮齿动物血浆AUC（时量曲线下面积）为人的25倍；③选择吸收饱和剂量；④药效学终点，不应产生生理学和内稳态紊乱；⑤最大可行剂量，受试物在饲料中最高含量为5%。限制剂量为1500 mg/kg/天
		- 中、低剂量组则按等比级数下推，如分别为上一个剂量水平的1/2或1/3。低剂量组应不影响动物的正常生长、发育和寿命，即不产生任何毒效应。但低剂量组应高于人的接触剂量，一般不低于高剂量的10%。对照组除不给受试物外，其它条件均与试验组相同
	- 染毒途径
		- 应尽可能模拟人体可能的接触途径：经口、经皮肤和吸入
	- 实验期限
		- ICH（1997）建议实验期限参考下面几条准则：
			- 一般情况下，试验期限小鼠和仓鼠应为18个月，大鼠为24个月；然而对于某些生命期较长或自发肿瘤率低的动物品系，小鼠和仓鼠可持续24个月，大鼠可持续30个月
			- **当最低剂量组或对照组存活动物只有25%时，可结束试验**；对于有明显性别差异的试验，则试验结束的时间对不同的性别应有所不同；在某种情况下因明显的毒性作用，只造成高剂量组动物过早死亡，此时不应结束试验
	- 结果观察与分析
		- 一般观察和病理学检查
		- 统计肿瘤发生情况，计算肿瘤发生率
		- 肿瘤发生率（%）=（实验结束时患肿瘤动物数/有效动物总数）×100%
		- **有效动物总数：检测到第一例肿瘤时存活的动物数**
		- 致癌试验阳性的判断标准（WHO）：符合以下任1条即认为有哺乳动物致癌性
			- 对于对照组也出现的一种或数种肿瘤，试验组肿瘤发生率增加
			- 试验组发生对照组没有的肿瘤类型
			- 试验组肿瘤发生早于对照组
			- 与对照组比较，试验组每只动物的平均肿瘤数增加

**哺乳动物致癌试验的实验条件要求较高，人力、经费及时间耗费巨大**
以下情况考虑做哺乳动物致癌试验：
- 人体可能长期接触该化学物
- 该化学物或其代谢物与已知的致癌物相似
- 反复染毒毒性试验提示该化学物可能引起癌前病变

	- 小鼠肺瘤诱发试验

	- 雌性大鼠乳腺癌诱发试验

	- 转穿梭质粒的转基因小鼠

>- 体外遗传毒理学试验，体外细胞转化是一重要的遗传学终点

# 4.化学致癌作用的机制
**遗传机制学派(genetic theory)**：亲电子剂学说，体细胞突变学说，**癌基因学说，多阶段学说**
**非遗传机制学派(non-genetic theory)**：细胞异常增生，免疫抑制，内分泌激素失衡，表观遗传改变（Epigenetic Change）等
## 4.1癌基因学说
**癌基因(oncogene)**
- 一类能引起细胞恶性转化及癌变的基因，是化学致癌物作用的主要靶分子，在细胞癌变过程中起关键作用

**原癌基因(proto-oncogene)**
- 本身是正常的细胞基因，是一类调控细胞生长分化的基因，在进化过程中高度保守
- **在胚胎期或组织的再生和修复过程中表达**，通常处于静止状态
- 在化学致癌物、某些物理因素和生物因素的作用下被激活而进行表达，促进细胞分裂增殖，引起细胞转化，最终形成肿瘤
- 活化的原癌基因才被称为癌基因

**抑癌基因(tumor suppressor gene)**
- 作用方式与癌基因相反
- 它们**在正常细胞中起抑制细胞增殖和促进分化**的作用，在环境致癌因素的作用下，抑癌基因发生纯合缺失或失活亦会引起细胞的恶性转化而致癌
- 抑癌基因必须一对等位基因丢失或突变后失活，才能对细胞的恶性转化起作用，故又称为隐性癌基因（recessive oncogene）
- TP53基因
	- 野生型TP53
		- 肿瘤抑制基因作用。其表达产物p53蛋白参与细胞周期G1/S关卡（checkpoint）的检查机制，负责检查细胞基因组的完整性
		- 如果DNA有损伤，则p53使细胞阻滞于G1期，以使其修复，如修复失败，则p53启动细胞发生凋亡
	- 突变性TP53
		- 癌基因的作用。促进细胞恶性转化，抑制细胞凋亡。在多种肿瘤中高表达

**致癌过程中涉及的两类基因比较**

| 原癌基因               | 肿瘤抑制基因                       |
| ------------------ | ---------------------------- |
| 涉及细胞生长和分化          | 功能不清楚，但可能涉及生长和分化             |
| 存在基因家族             | 存在基因家族                       |
| 在癌中被活化或扩增          | 在癌中被灭活或丢失                    |
| 因点突变、染色体易位或基因扩增而活化 | 因染色体丢失、染色体缺失、点突变、转换、体细胞重组而灭活 |
| 几乎没有证据表明与遗传性癌有关    | 有证据涉及遗传性和表观遗传性癌              |

**DNA保真性相关基因**
- 含有30亿个碱基对，20,000~25,000个蛋白质编码基因的人类基因组经常遭受内源性活性代谢产物、治疗药物和大量环境致突变物的攻击而使完整性受到影响
- 在维持细胞基因组完整性中，DNA损伤修复机制扮演着重要的角色
- DNA损伤修复机制包括直接修复、碱基切除修复、核苷酸切除修复、双链断裂修复和交联修复
- 此类基因功能异常，将可能导致损伤的DNA不能被正确修复，突变积累

**Hanahan和Weinsberg提出正常细胞转变为癌细胞的6种能力，还将遗传不稳定性列为另一获得性能力**
- 逃避细胞凋亡的能力
- 自足的生长信号
- 对负性生长信号的不敏感性
- 持续的血管生成
- 无限的增值潜能
- 对组织的侵袭和转移潜能

## 4.2化学致癌的多阶段学说（Multi-Stage theory of Carcinogenesis）
正常细胞经过遗传学改变的积累，才能转变为癌细胞，癌症的发生是多阶段过程，至少包括**引发、促长和进展**阶段

**引发剂、促长剂、进展剂**
- 引发剂（initiator）：本身或其代谢产物能与DNA作用，导致体细胞突变为引发细胞的化学物
- 促长剂（promoter）：作用于引发细胞增殖成为癌前病变或良性肿瘤的化学物
- 进展剂（progressor）：作用于促长阶段的细胞转变成进展期的化学物

**引发阶段(initiation)**
- 是指化学物或其活性代谢产物与DNA作用，导致体细胞突变成引发细胞（initiated cell）的阶段
- 引发是一个遗传毒性事件
	- 引发剂本身有致癌性，大多数是致突变物，没有可检测的阈剂量，其作用的靶主要是原癌基因和肿瘤抑制基因
- 引发阶段涉及正常细胞发生不可逆的变化（通常是基因组结构的改变），从而使其能够不受限制地生长
- 引发阶段特点
	- 引发阶段很短，引发细胞不是肿瘤细胞，引发作用**不可逆**，并且是累积性的
	- 引发剂本身有致癌性，大多数是致突变物，**没有可检测的阈剂量**
	- 细胞对致癌物的代谢以及DNA修复等决定了引发作用的个体变异、物种差异及器官特异性

**促长阶段(promotion)**
- 引发细胞增殖成为癌前病变或良性肿瘤的过程
- 促长阶段是指已启动的细胞进一步扩展，从而形成一种活跃增殖的多细胞前癌性肿瘤细胞群体
- 促长阶段主要是非遗传机制
	- 细胞异常增殖：细胞周期调控紊乱
	- 表观遗传修饰：
		- DNA甲基化
		- 组蛋白修饰
		- 染色质重塑
		- 非编码RNA对转录的调控等
- 促长阶段特点
	- 历时较长，持续给以促长剂是必需的，**早期有可逆性，晚期不可逆**
	- 对生理因素调节敏感，衰老、饮食和激素可影响促长作用
	- 促长剂本身不能诱发肿瘤，常是**非致突变物**，通常**具有阈剂量**，其剂量反应关系呈S形曲线
	- 促长剂通过选择性促进引发细胞增殖和/或抑制正常细胞增殖使引发细胞克隆扩增

**进展阶段(progression)**
- 从引发细胞群（癌前病变、良性肿瘤）转变成恶性肿瘤的过程。在进展期肿瘤获得生长，侵袭和转移
- 进展阶段是不可逆的过程，它会产生一种具有增强增殖能力的肿瘤细胞克隆
- 进展阶段特点
	- 进展期主要表现为核型不稳定及表型的异质性
	- 在进展期癌前病变或良性肿瘤**获得生长，侵袭和转移能力**
	- 兼有引发剂、促长剂和进展剂作用的化学致癌物称为**完全致癌物**（complete carcinogen）

**多阶段致癌的形态学和生物学特征**

| 引发                             | 促长                          | 进展                       |
| ------------------------------ | --------------------------- | ------------------------ |
| 1.不可逆性                         | 1.在基因表达和细胞水平上有可逆性；          | 1.不可逆性；                  |
| 2.经引发的“干细胞”在形态学上不能鉴定；          | 2.持续给以促长剂才可维持促长细胞群；         | 2.核型不稳定性导致细胞基因组结构的形态学改变； |
| 3.对外源化学物及其他化学因子敏感；             | 3.对衰老、饮食和激素因子敏感             | 3.在进展阶段早期，已改变的细胞对环境因子敏感； |
| 4.引发细胞可能自发(内源性) 发生；            | 4.内源性促长剂可起“自发性”促长作用；        | 4.在进展阶段观察到良性或恶性肿瘤；       |
| 5.需经细胞分裂“固定”突变；                |                             | 5.进展剂使已促长的细胞进入此期。        |
| 6.剂量-反应关系没有易于确定的阈值；            | 5.剂量-反应关系显示有阈值和最大作用；        |                          |
