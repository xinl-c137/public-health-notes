# 1.概念
**毒作用模式(Mode of Action, MOA)**：指从化学物与生物分子相互作用开始的、证据权重支持的、可能导致毒性有关终点的一组事件或过程
**分子起始事件(Molecular Initiating Event, MIE)**：在机体内化学物-生物分子相互作用，启动通路扰动的初始点，沿被扰动通路引起进一步的关键事件，并与有害结局相关联
**有害结局通路(Adverse Outcome Pathway, AOP)**：是在个人或群体的水平，关于分子起始事件和有害结局之间相互联系的已有的知识汇总
**毒性通路(Toxicity Pathway)**：在受到一定程度的干扰时会导致有害效应的细胞应答通路
![[4.毒作用机制、急性毒性和局部毒性试验-1.概念-20260401.png|534]]
**化学物引起毒作用的4个阶段、3种途径**
- **4个阶段**：处置→与靶分子相互作用/生物环境的改变→细胞功能障碍→修复失调
- 3种途径
	- 化学物不与靶分子发生作用而直接作用于接触的部位
	- 毒物与靶分子发生作用造成细胞功能障碍或损伤
	- 毒物造成细胞损伤后，机体出现异常修复（包括分子水平、细胞水平和组织水平的修复)

毒物对靶分子的影响
- 毒理学上的常见的靶分子：膜脂质、核酸（特别是DNA)、蛋白质
- 靶分子功能失调、靶分子结构破坏、新抗原形成

**细胞功能障碍与毒性**
![[细胞功能障碍与毒性|700]]
**细胞应激与细胞损伤**
外源化学物作用于细胞后，细胞具有一定的抗损害能力，即首先发生适应性应激反应，当外源化学物的作用强度超过细胞的抗损害能力后，就会引起细胞动能障碍（细胞损伤），最后还可导致死亡
**细胞应激cell stress**：指细胞处于不利环境和遇到有害刺激时所产生的防御和适应性反应
- 热应激
- 氧化应激
- 缺氧应激
- 遗传损伤应激
- 内质网应激等
- 线粒体应激

**毒作用机制的研究**
- 从外源化学物的化学结构和理化特性，毒动学/生物转化和毒效应等多个方面；从分子一细胞一组织一器官一整体多个层次的研究
- 是点一线一面，也就是从生物靶分子一途径一网络，即说明毒作用的靶分子，其上下游关系即毒作用的途径，不同途径的联系和交互作用即为毒作用的网络

# 2.急性毒性作用acute toxicity
**急性毒性作用**通常是描述毒理学研究的最初步的资料，包括经口、吸入、经皮和其他途径的急性毒性
![[4.毒作用机制、急性毒性和局部毒性试验-20260715.excalidraw|900]]
## 2.1概念
**急性毒性（acute toxicity）**：实验动物一次接触或24小时内多次接触某一化学物所引起的毒性效应，甚至死亡
实验动物接触化学物的方式或途径不同，“一次“的含义也有所不同
- 经口和注射接触：“一次“是指在瞬间给予试验动物染毒
- 经呼吸道吸入与经皮肤染毒：“一次”是指在一个规定的期间内实验动物持续地接触受试化学物

“多次“的含义是指当外源化学物毒性很低时，即使一次给予实验动物最大染毒容量还观察不到毒性作用便需要在24小时内多次染毒，以达到规定的限制剂量

## 2.1研究目的
- 为亚慢性、慢性毒性研究及其它毒理试验的剂量设计和观察指标的选择提供依据
- 评价化学物对机体的急性毒性的大小，并根据LD50值进行急性毒性分级
- 确定急性毒效应特征、毒作用靶器官及损害的可逆性
- 为机制研究提供线索

## 2.3急性毒性试验及其设计要点
以死亡为终点急性毒性实验：检测急性毒性上限指标，主要是LD50/LC50或近似LD50
非致死性急性毒性实验：可检测非致死性指标LOAEL和NOAEL
典型急性毒性试验设计要点：
- 实验动物的选择和处置
- 试验设计
- 毒作用观察
- LD50的计算

### 2.3.1实验动物的选择和处置
**物种和品系**
一般常用大、小鼠和狗进行急性毒性实验，首选大鼠
用大、小鼠进行致死性毒性测LD50
利用狗观察毒性反应

**年龄、体重**
体重范围是：大鼠180~240g，小鼠18~25g，狗10~15kg
试验动物体重变异不应超过平均体重的20%

**性别**
通常雌、雄各半

**动物检疫**
通常情况下观察5~7天

**动物分组**
根据实验设计随机分组，至少设3个剂量组
Horn's法：设4个剂量组，4~5只/组
简化寇氏法：设5~7个剂量组，8~10只/组
概率单位法、Bliss法：5~7个剂量组，10只/组

**染毒途径**
总的原则是应尽可能模拟人可能的接触途径，灌胃染毒应注意禁食，染毒后继续禁食2~4小时

### 2.3.2实验设计
了解受试物用途、理化特性
确定LD50计算方法
**剂量设计**：
- 原则上至少3个剂量
	- 高剂量：全部/大部分动物死亡
	- 中剂量：50%动物死亡
	- 低剂量：不引起/很少动物死亡

通常，先进行预实验：以化学结构和性质类似的化学物的毒性资料作参考，用少量动物进行，获得10-90%的致死剂量范围。在此剂量范围内，以合适的组距设置剂量组
- Horn's法：设4个剂量组，组距2.15/3.16
- 简化寇氏法：设5~7个剂量组，组距
- 概率单位法、Bliss法：5~7个剂量组

### 2.3.3观察
染毒后一般要求观察14天
详细观察和记录实验动物的中毒体征、发生时间、体征发展的过程、死亡前特征等，有助于了解该化合物的靶器官并可能为中毒机制提供线索
实验动物的体量变化：染毒前、后和死亡时测体重
非致死性指标的可逆性
**啮齿类急性毒性试验中常见的体征和观察**

| 系统和器官         | 中毒后常见的表现                                                   |
| ------------- | ---------------------------------------------------------- |
| 中枢神经系统与神经肌肉系统 | 体位异常，叫声异常，活动增多和呆卧少动，肌颤，痉挛，抽搐，麻痹，后肢无力，运动失调，管状尾，对外界刺激反应过敏或迟钝 |
| 植物神经系统        | 瞳孔扩大，流涎，流泪，出汗                                              |
| 呼吸系统          | 鼻孔溢液，鼻翼扇动，呼吸深缓，呼吸过速，张口或腹式呼吸                                |
| 泌尿生殖系统        | 会阴部污秽，有分泌物，阴户或乳腺肿胀。阴茎突出，遗精                                 |
| 皮肤和背毛         | 皮肤充血，紫绀，背毛蓬松，污秽                                            |
| 眼睛            | 眼球突出，充血，角膜混浊，血性分泌物                                         |
### 2.3.4病理学检查
死亡动物应及时进行大体解剖及病理检查。试验结束时，将存活动物处死，亦应进行大体解剖，对发现病变的组织器官应作病理检查
### 2.3.5LD50（LC50）的计算
求出回归方程：Y=a+bX
X为剂量对数值，Y为死亡率的概率单位
利用此式即可求出受试化学物的LD50及其95%可信区间

**利用LD50比较不同外源性化学物急性毒性**
![[4.毒作用机制、急性毒性和局部毒性试验-2.急性毒性作用acute toxicity-20260401.png|304]]
- 化学物A、B、C剂量反应关系曲线的斜率相同，而LD50：A>B>C,因此，急性毒性大小的次序为：C>B>A
- 化学物D的LD50与C相同。但斜率比C大，可见在低于LD50的剂量时急性毒性C>D

## 2.4==LD50的毒理学意义和局限==
### 2.4.1意义
- 比较毒物急性毒性的大小
- 计算药物的治疗指数(LD50/ED50)，了解药效学剂量和毒性剂量的距离
- 为后续的重复给药毒理试验剂量选择提供参考
- 比较不同途径的LD50值，获得生物利用度的信息
- 试验结果可用来推测人类的致死剂量以及中毒后的体征

### 2.4.2局限性
- LD50值给予的有效信息少，作用有限：LD50值不能等同于急性毒性，死亡仅仅是评价急性毒性的许多观察终点之一
- LD50的波动性很大，无稳定的数值
- 物种差异对LD50的影响大，LD50值对于人急性中毒的诊疗意义很小
- 对于开发新药，无需得到精确的LD50值
- 消耗的动物数量大：经典法一次需30-70只动物

# 3.急性毒性替代试验Alternative Methods for Acute Toxicity
- 固定剂量法(fixed dose procedure)：不以动物死亡作为观察终点
- 急性毒性分级法(acute toxic class method)：以死亡为终点，分阶段试验
- 上-下移动法(up/down method)
- 限量试验(limit test)

# 4.局部毒性作用Local Toxicity
 - 局部毒性为外源化学物**对机体接触部位的毒效应**，不包括经接触部位的吸收而引起的系统毒性
 - 局部毒性又称耐受性试验

## 4.1皮肤原发性刺激试验
皮肤原发性刺激试验(skin primary irritation test)包括单次和多次皮肤刺激试验，完整皮肤和破损皮肤刺激试验等。观察终点为皮肤刺激和皮肤腐蚀
- 皮肤刺激(dermal irritation)是指其皮肤接触化学物后产生的局部可逆性的炎症变化
- 皮肤腐蚀(dermal corrosion)是指其皮肤接触化学物后产生的局部不可逆性组织损伤

### Draize试验
动物：健康、皮肤无损伤的白色家兔或豚鼠，至少4只
方法：
- 脱毛：试验前24小时，机械或化学法脱毛，范围各约：3cm×3cm
- 染毒
	- 取受试物0.5ml（或0.5g）直接涂在皮肤上，然后用纱布及玻璃纸覆盖，再用无刺激性胶布和绷带加以固定，敷用时间为4小时
	- 试验结束后，用温水或无刺激性溶剂除去残留受试物，在一只家兔或豚鼠背部脊柱两侧可各准备3个去毛区，可进行剂量-反应研究，或测试多种受试物
- 观察：涂抹部位于染毒后1、24、48小时观察皮肤反应。皮肤反应：红斑，水肿。按其严重程度评分

### 皮肤刺激/腐蚀试验替代方法
皮肤组织培养：人皮肤组织、小鼠皮肤组织体外培养
重组人体皮肤模型：重组人皮肤模型是用正常人皮肤角质细胞通过培养形成多层的高度分化的表皮模型如EpiSkin<sup>TM</sup>、EpiDerm<sup>TM</sup>
## 4.2眼原发性刺激试验
眼原发性刺激试验(eye primary irritation test)包括单次和多次眼刺激试验等。观察终点：眼刺激和眼腐蚀
- 眼刺激性(eye irritation)：眼睛前表面接触受试物后产生的眼睛可逆性炎性变化
- 眼腐蚀性(eye corrosion)：眼睛前表面接触受试物后产生的眼睛不可逆性组织损伤

### Draize试验
动物：健康白色家兔，至少4只
方法：
- 染毒：0.1ml受试物滴入每只兔的一只眼晴，另一只做为对照，滴入后不冲洗/滴入后不同时间（如4秒及30秒）冲洗。、
- 观察：染毒后24、48、72小时、4天和7天时以肉眼或手提式裂隙灯检查眼的反应，包括结膜（发红、球结膜水肿和分泌物)、角膜（混浊程度和范围）和虹膜（充血、肿胀和角膜周围充血）的反应，按规定分级标准进行评分

### 眼刺激/腐蚀试验的替代方法
- 离体牛眼角膜浑浊度和通透性试验（BCOP）
- 离体鸡眼试验（ICE）
- 重组人角膜上皮细胞模型试验（Human corneal epithelium models, RhCE）

## 4.3其他局部刺激试验
根据实际人可能接触途径，如眼、耳、鼻、口腔、呼吸道、关节腔、皮肤、直肠、阴道、静脉、动脉、肌肉、皮下、静脉旁和鞘内等非口服途经给药，观察用药局部产生的毒性（如刺激性和过敏性）都应进行局部刺激性试验
