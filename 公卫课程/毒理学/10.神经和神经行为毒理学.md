# 1.对神经系统的简单回顾
**神经系统的细胞组成及功能**
神经细胞（神经元）：具有接受刺激和迅速传导神经冲动的能力
神经胶质细胞：对神经元起支持、保护、分隔、营养等作用

**神经系统的解剖、生理学特点**
- 中枢神经系统的新陈代谢非常活跃
- 神经系统的功能最复杂、反应最迅速、和其它器官联系最广泛
	- 尤其是中枢神经系统具有整合、协调、学习、记忆以及思维等功能
- 神经系统的递质系统可以成为神经性毒物的靶点
	- 递质前体、合成酶、储存囊泡、摄取及释放、受体、灭活以及降解等
- 神经细胞再生能力差
	- 死亡：不能再生，神经胶质细胞形成瘢痕
	- 存活：轴突损伤（可再生，但是缓慢）
- 神经系统的保护机制
	- 有坚固的颅骨以及坚韧的脑脊膜
		- 脑脊膜：硬膜、蛛网膜和软膜组成
	- 血脑屏障（blood-brain barrier, BBB）
		- 血脑屏障由脑毛细血管内皮细胞通过紧密连接封闭形成的，内皮外有基膜、周细胞及星形胶质细胞突起的脚板围绕而形成。血脑屏障有许多不同类型的转运器，可以识别、转运特定的分子

# 2.神经毒性的损伤类型及机理
**神经毒理学neurotoxicology**：研究外源化学物引起神经系统结构和功能损害作用及其损伤机制的一门学科

==**神经毒性类型及机制**==
- 血-脑及血-神经屏障与神经毒性损伤
- 神经元损伤neuronopathy
	- 神经元损伤有关的化学物：氨基糖甙类抗生素、铅、汞、锰、铊等等
- 轴病axonopathy
	- 轴索损害有关的化学物：秋水仙素、紫杉醇、长春新碱、**丙稀酰胺**、二硫化碳、金、有机磷、己烷等等
- 微管相关的神经毒作用—周围神经病peripheral neuropathy
- 髓鞘病myelinopathies
	- 髓鞘损害有关的化学物：碘胺酮、哌克昔林、溴化乙啶，EB
- 神经传递与神经毒性transmission toxicity
	- 常见神经毒性相关化学物：苯丙胺、阿托品、氯丙嗪、可卡因、烟碱等等

**外源化学物对神经系统毒作用的表现**
- 中毒性神经官能症
	- 以脑功能失调和精神障碍为主的疾病。表现为兴奋和抑制症状
	- 头痛、头晕、无力。肌肉关节酸痛，失眠、记忆力减退等
	- 可逆（早期症状及轻度中毒）
- 中毒性脑病
	- 毒物引起的中枢神经系统器质性病变
	- 急性中毒性脑病：恢复正常或恢复不全（如一些药物引起的急性脑水肿等）
	- 慢性中毒性脑病：神经毒物慢性重度暴露
		- 震颤麻痹综合症（吩噻嗪类、丁酰苯类以及三环类抗精神失常药物；锰、二硫化碳等）
		- 中毒性精神分裂症（异烟肼、四乙基铅、二硫化碳等）
		- 中毒性痴呆（铅、汞、铊、锰等）
- 中毒性神经炎
	- 有毒化学物损害周围神经导致的病变，表现为单神经炎或多神经炎
	- 毒物损伤单一的周围神经而导致单神经炎，如铅→桡神经麻痹；三氯乙烯→三叉神经麻痹
	- 多神经炎可在急性中毒的早期出现，如铊、CO中毒等。这类药物如抗肿瘤药、抗结核药等等

# 3.迟发性神经毒性delayed toxicity
**定义**：在中毒反应发生后8～14天，再次出现的较持久的神经中毒反应，主要表现为迟缓性麻痹或轻瘫，而后出现脊髓损伤体征，如共济失调或强直。多见于有机磷农药、CO中毒
**中毒作用的机理**
![[10.神经和神经行为毒理学-20260419.excalidraw|500]]
- 第一步：神经毒靶酯老化
- 第二步：钙稳态的失衡，引起依赖钙激活蛋白酶（calcium-activated neutral protease（CANP））的活化
- 第三步：CANP引起远端轴突的降解

**常见的引起迟发性神经毒性作用的有机磷农药**
TOCP（三磷甲苯磷酸酯）、丙胺氟磷、丙氯磷、对溴磷、三硫磷、苯硫磷、脱叶磷、皮蝇磷、壤虫磷、草特磷、敌敌畏和敌百虫等

# 4.神经毒性的研究方法
- 人体的研究：临床观察或者检查（神经学检查、电生理检查、生化检查）
- 动物研究
	- 功能试验组合（functional observational FOB，US EPA 1994）：自主体征、感觉、运动、生命体征
	- 神经学检查
	- 形态学检查：病理
	- 电生理
	- 生化检查：相应的酶、靶蛋白
	- 分子生物学检测：信号通路，表观遗传改变等
- 细胞组织研究
	- 神经元细胞的体外培养
	- 干细胞的定向诱导和神经元干细胞的应用
	- 中脑微团的培养
	- 脑片的体外培养

**神经干细胞在毒理学中的应用**
- 神经干细胞主要分布在大脑皮层、纹状体、海马、室管膜下层和中脑等区域
- 正常的哺乳动物成年后，中枢神经系统任然有神经干细胞存在，但是处于静止状态。成年脑区神经干细胞主要存在于海马齿状回、纹状体和环绕侧脑室的室脑膜下层
- 毒理学中，可以应用神经干细胞评价外源化学物的神经毒性和神经发育毒性

# 5.神经行为毒理学及研究方法
**神经行为毒理学(neurobehavioral toxicology)**：主要是研究外源性化学物特别是低剂量长期接触对人神经行为的毒性效应，亦及人的心理功能的毒性效应
**特点**
- 研究低剂量慢性神经毒性的方法
- 敏感
- 能定量
- 可早期检出亚临床表现
- 是整体试验

**研究方法**
- 动物实验
	- 主要监测学习记忆
		- 操作条件反射
		- 空间辨别试验：迷宫（Y迷宫、Morris水迷宫等）
		- 厌恶学习：穿梭箱、避暗、跳台试验等
	- 自主活动度测定：开阔场地等
	- 动物的焦虑测定：高架十字迷宫
	- 药物成瘾性：条件位置偏好
- 人的观察
	- 心理功能测试量表，包括学习、记忆、思维、智力、注意力等（韦氏智力量表、韦氏记忆量表等）
	- 常用的是NCTB（Neurobehavioral core test battery），在1983年由WHO提出。包括：情绪状态试验、手提转捷度试验、目标追踪试验、简单反应时、数字译码试验、视觉保留试验和数字广度试验
	- 计算机化神经行为评价系统（Computer-administered Neurobehavioral Evaluation System, NES）
	- 结果评价：指标多、变量多、混杂因素多、应选用适当的数学模型
