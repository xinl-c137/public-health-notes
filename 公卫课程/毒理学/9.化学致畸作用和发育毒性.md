# 1.发育毒理学
**发育毒性(developmental toxicity)**：外源化学物对发育过程的不良影响，包括对胚胎发育、胎仔发育及出生幼仔发育的损害作用，指出生前后接触有害因素，子代个体发育为成体之前诱发的任何有害影响。主要表现有：发育生物体死亡、生长改变、结构异常、功能缺陷

**影响发育毒性的因素**
- 社会因素：行为、社区、医疗保健
- 生物学因素：遗传、性别、年龄
- 环境因素：饮食、烟草、化学物、射线

**发育毒性的主要表现**
- 发育生物体死亡death of developing organism
	- 受精卵未发育即死亡
	- 胚泡未着床即死亡
	- 着床后出生前死亡
	- 早期-自然流产；晚期-死胎
- 生长改变altered growth
- 结构异常structural abnormality
- 功能缺陷functional deficiency

**发育毒性观察终点**
- Ⅰ型改变：永久性的、危及生命的、常伴有大畸形的结局
	- 减少活产数
	- 增加死产数
	- 增加吸收胎的数量
	- 增加畸形胎儿的数量
- Ⅱ型改变：非持久性、不会危及生命、往往与畸形无关的结局
	- 出生体重降低
	- 出生后存活率降低
	- 出生后的生长发育减缓、生殖能力下降
	- 增加发育迟缓胎儿的数量

# 2.致畸作用
**概念**
**致畸物(teratogen)**：可能导致人类或动物出现畸形的环境因素
**致畸性(teratogenicity)**：致畸物引起动物结构畸形的特性

**致畸作用的影响因素**
- 关键期critical period（暴露期exposure period）
	- 致畸试验的染毒时间必须安排在器官形成期(organogenesis)，即致畸作用关键期(critical period)
	- 不在器官形成期暴露也可能会形成其他发育毒性
	- **概念**：胚胎发育过程中，易于发生形态结构异常的整个时期。相当于或略长于胚胎的器官形成期。器官形成期为发生形态结构畸形的关键期，迅速改变细胞分裂速度对畸形发生极为重要。大多数器官又都有对其致畸作用的特殊敏感期，即“靶窗”。
	- **意义**：决定致畸试验染毒时间，只有安排在器官形成期，才有可能观察到形态畸形的致畸效应。
- 遗传类型heredity type
	- 主要表现为物种差异(species differences)
	- 如反应停引起灵长类动物畸形，而不引起啮齿类动物畸形
- 接触致畸物的剂量
	- 致畸作用具有阈剂量
	- 不同致畸物剂量反应关系不同（发育迟缓、结构畸形、胚胎死亡等效应出现先后不同）
	- 致畸带越宽，就有更广范围的剂量可以在不引起胚胎死亡的情况下导致畸形
	- 剂量设计要适中，过低不足以显示致畸性；过高引起大量胚胎死亡，畸胎率反而降低，或母体毒性过强，不能判断是致畸物的作用还是母体毒性的继发作用
- 双亲因素
	- 母体因素
		- 营养缺乏：碘、叶酸
		- 遗传背景：唇腭裂（白人多于黑人）
		- 疾病：糖尿病、病毒感染
		- 压力
		- 胎盘：羟基脲、苯妥英
	- 父亲因素
		- 遗传背景
		- 年龄
		- 环境因素：吸烟、饮酒、辐射
- 其他
	- 暴露途径
	- 化学物的理化性质

**致畸物质和发育毒性物质的风险评估**
- 致畸指数(teratogenic index)
	- 致畸指数=母亲的LD50/胎体的最小致畸量
	- 致畸指数<10，认为无致畸性；10~100有致畸性；>100致畸性强
- GHS对生殖毒性的分类
	- 1类（1A、1B）、2类、附加类型
- ICH分类
	- 综合人群流行病学数据和动物实验结果进行分类

# 3.生殖毒性
**生殖毒性(reproductive toxicity)**：外源性物质对生殖过程造成的不良影响。包括对生殖细胞形成、受精、胚胎发育、妊娠、分娩、哺乳等环节的损害。生殖毒性着重研究外源化学物对亲代的生殖功能的有害作用

**生殖毒性**
- 雄性生殖组织
	- 支持细胞(sertoli cells)：血睾屏障
	- 间质细胞(leydig cells)：产生睾酮
- 雄性生殖毒性的作用靶点
	- 间质细胞(leydig cells)：雌二醇、钙离子通道阻断剂CCB、降脂药、组织胺阻断剂、抗痉挛、抗病毒药
	- 支持细胞(sertoli cells)：邻苯二甲酸酯
	- 生精上皮：棉酚
	- 附睾上皮：LTQ
- 雌性生殖毒性的作用靶点
	- 卵巢、输卵管、子宫、激素分泌

**发育毒性和生殖毒性机制**
- 基因突变和染色体畸变
- 细胞凋亡和死亡
- 细胞间交互作用
- 胎盘损伤
- 间接效应
- 宫内重编程

# 4.生殖和发育毒性的实验
**致畸实验设计要点**
- 动物选择：大鼠、小鼠、家兔（啮齿类、非啮齿类哺乳动物两个物种）
- 接触选择
	- 剂量：高剂量应在母体中产生轻度的毒性（LD50的1/5~1/3）；中剂量应引起最小的毒作用（LOAEL）；低剂量为NOAEL或LD50的1/100~1/30
	- 接触途径与频率：与人的暴露途径相同
- 对照组：阳性对照、阴性对照
- 结果观察
	- 母体观察
	- 胎仔检查：一般检查、外观检查、内脏检查、骨骼检查
- 结果评定
	- 致畸指数=LD50/最小致畸剂量
	- <10不具有致畸作用；10~100有致畸作用；>100为强致畸作用

==**三段生殖毒性试验**==
![[9.化学致畸作用和发育毒性-4.生殖和发育毒性的实验-20260413.png]]
三段生殖毒性试验主要为以下三个阶段：各生殖阶段之间无空隙，3个相关阶段接触至少1天重叠，并能直接或间接的评价生殖过程的所有阶段
- Ⅰ：生育力和早期胚胎发育毒性试验——观察雌雄性生殖能力
- Ⅱ：胚体胎体毒性实验：致畸试验（器官形成期）——观察结构畸形
- Ⅲ：出生前和出生后发育毒性实验（着床后直至断乳）——观察功能异常

| 负向地性（出生后4和7天）  | 将子鼠头朝下放在一个30度倾斜的面板上，测它们调转自身，成为头朝上位置所需的时间                                             |

**一代生殖毒性试验**
亲代(F<sub>0</sub>)动物直接接触受试物，子一代(F<sub>1</sub>)将在母体子宫内及经哺乳接触受试物

**两代（多代）生殖毒性试验**
为什么要做两代或者多代？
一些外源化学物，如转基因食物、食品添加剂、农药、环境污染物等，人类反复长期接触，欲查明其与生殖有关的影响，仅做三段生殖毒性试验是不够的，应进行多代生殖试验

**发育毒物体外预筛试验**
- 小鼠胚体干细胞培养(embryonic stem cell test, EST)
	- 胚胎干细胞与受试物接触后诱导分化为心肌细胞，形成可以搏动的心肌细胞
- 胚芽培养
- 中脑微团培养(micromass, MM)
- 全胚胎培养
