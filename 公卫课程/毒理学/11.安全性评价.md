# 1.概念
**化学品安全性评价**
- 对生物体健康风险的安全性评价
- 对环境风险的安全性评价

**化学品管理的组织机构**
- 美国
	- FDA—食品、药品和化妆品
	- EPA—农药和环境化学物
	- OSHA—工业化学物
- 中国
	- 卫健委—涉及饮用水卫生安全产品和消毒药剂
	- 食品药品监督管理局—化妆品、药品、食品
	- 农业部—农药、兽药
	- 国家环保部—环境化学品
	- 国家安监总局—爆炸品、作业环境化学物

**安全性评价(safety evaluation)**：按照一定的程序要求对外源化学物的毒作用进行检测，以评价化学物的毒作用特点、毒作用的剂量-反应关系，经足够的试验之后，提出NOAEL和LOAEL，再除以一定的安全系数，制订出安全限值

**开展安全性评价必备的几个基本要素**
- GLP(good laboratory practice)：实验室认证认可
- SOP(standard operation procedures)：实验开展过程中依据的程序
- QAU(quality assurance unit)：质量保证部门

# 2.化学品安全性评价
**四阶段毒理学安全性评价**
- 第一阶段：急性毒性试验，局部毒性试验
	- LD50，毒作用性质，特点和靶器官
- 第二阶段：蓄积试验、遗传毒理学试验和致畸试验
	- 了解受试物在体内的蓄积毒性；是否具有遗传毒性；判断受试物的胚胎毒作用以及对胎仔是否有致畸作用
- 第三阶段：亚慢性毒性试验、生殖试验和代谢试验
	- 进一步确定毒效应强度，靶器官及可逆性，得到亚慢性毒性试验的NOAEL和LOAEL，预测人体的危害；检测生殖发育毒性；检测代谢产物的毒性及代谢酶
- 第四阶段：慢性毒性试验和致癌试验
	- 靶器官，NOAEL和LOAEL进行安全性评价；致癌作用

**毒理学安全性评价为什么要分阶段进行**
- 不同的评价程序对毒性试验划分的阶段性有不同的要求
- 原则：各种毒性试验按一定顺序进行，明确先进行哪项试验，再进行哪项试验
- 分阶段原因和目的
	- 毒理学的试验设计本身决定的，各种毒理学试验之间有联系的。四个阶段间是有联系的，某些实验是其他实验的基础，如毒性试验是绝大多数毒理学实验的基础；为了尽量减少资源的消耗，可根据前一阶段试验的结果判断是否进行下一阶段的实验
	- 从经济的角度考虑：以最短的时间，用最经济的办法，取得最可靠的结果。实际工作中常常是先安排试验周期短、费用低、预测价值高的试验
	- 如果是非食入的化学物只要进行前三阶段的试验即可，但对于食品和药品必须进行完整的四阶段试验

**食品安全性毒理学评价程序（GB151931-2014）**
- 急性经口毒性试验
- 遗传毒性试验
- 28天经口毒性试验
- 90天经口急性毒性试验
- 致畸试验
- 生殖毒性试验和生殖发育毒性试验
- 毒物动力学试验
- 慢性毒性试验
- 致癌试验
- 慢性毒性和致癌试验合并试验

**不同受试物选择开展试验的原则**
- 首创物质，构效分析（潜在毒性），使用范围广，摄入量大
	- 急性到致癌试验一整套
- 已知食品中相关物质的类似物，或者部分地区有食用历史
	- 急性毒性、遗传毒性、28天经口毒性试验
	- 根据具体情况确定是否继续
- 在多个国家有食用历史的物质，质量规格与国外产品一致
	- 急性毒性、】遗传毒性、28天经口毒性
	- 根据具体情况确定是否继续

**实验结果判定原则**
- 急性毒性：小于人摄入量100倍，放弃
- 遗传毒性
	- 若两项研究为阳性，弃用
	- 若一项为阳性，再选择两个备选试验（其中一个为体内试验），备选试验一阳，弃用
- 28天经口毒性试验
- 90天经口毒性试验：100，300倍
- 生殖毒性和生殖发育毒性：100，300倍
- 慢性毒性和致癌试验：50，100倍

**生物药物的特点（区别于化药）**
- 空间结构很复杂致使结构确定存在不完整性
- 生理结构差异导致生物活性差异——物种差异
- 生物药物受体广泛决定其具有多功能性
- 生物药物是异源性大分子，具有免疫原性

**生物药物与小分子化药的安全性评价策略有何不同**
- 总的基本框架是一致的
- 生物药物特别需要考虑的问题
	- 选择最相近的动物物种进行研究
	- 生殖毒性
		- 如果药理活性只有在非人的灵长类动物出现
			- 可以在一般毒性研究中观察生育力终点和生殖器官毒性，如生殖周期和病理组织学检查
			- 如果需要进行围生期和出生后毒性研究，则可以在猴子观察胚体-胎体发育毒性，一般考虑将这两个阶段合并起来进行观察
	- 致癌性：表遗传（非遗传毒性）致癌
		- ICH S6对于生物药物，标准的致癌试验不适用于其评价，但是有关致癌可能性的产品特异性的评估还是需要根据以下方面来考虑进行
			- 临床用药时间
			- 病人的特征
			- 产品的生物学活性
		- 潜在的致癌性表遗传机制：增强细胞增殖、器官增生、干扰激素分泌、减少细胞凋亡和免疫抑制
		
致癌试验模式

| 特征       | 小分子化药              | 生物药物                           |
| -------- | ------------------ | ------------------------------ |
| 模式       | 标准试验<br>2-年大鼠&小鼠实验 | 个案原则<br>啮齿类动物试验<br>体外/体内细胞增殖试验 |
| 策略       | 法规需求，ICH S1        | 科学需求，ICH S6                    |
| 观察效应     | 遗传毒性和表遗传毒性作用       | 非遗传毒性（表遗传）作用                   |
| 与人类风险的关联 | 有争议                | 需要更多的数据/经验                     |

- 安全药理学
	- ICH S6：在合适的动物模型观察可能的非预期药理活性是非常重要的。
	- 一般用猴子作为动物模型进行研究
	- ICH S7A ：对于到达高度特异性受体靶点的生物药物，在毒理学研究中评价安全药理学终点是很重要的

**安全评估试验：生物药物与化学药物**

|        | 生物药物(ICH S6)                             | 化学药物(ICH M3)                           |
| ------ | ---------------------------------------- | -------------------------------------- |
| 普通毒理学  | 啮齿动物和非啮齿动物-通常6个月<br>可以只用一种相关的动物          | 啮齿动物-长达6个月<br>非啮齿动物-长达9至12个月<br>需要2种动物 |
| 组织交叉反应 | 单克隆抗体及其衍生物-需要                            | 不需要                                    |
| 遗传毒理学  | 通常不需要                                    | 需要                                     |
| 安全药理学  | 通常不需要<br>可以与普通毒理学试验合并进行                  | 单独的动物试验（啮齿动物和非啮齿动物）                    |
| 生殖毒理学  | 可以只用一种有药物活性的非啮齿动物<br>啮齿动物-可考虑使用蛋白质的相应代理品 | 需要<br>需要2种动物试验（大鼠和兔）                   |
| 致癌性试验  | 个案考虑<br>如果具有药理活性和无免疫性，可只用一种啮齿动物          | 需要<br>大鼠和小鼠致癌试验                        |
| 光毒性试验  | 通常不要求                                    | 如果药物吸收290-700 nM范围内的光谱，则需要做光毒性试验       |
| 代谢特征鉴定 | 不做                                       | 通常需要                                   |

**生物药物的特别关注点**
- 组织交叉反应性
	- 抗体及相关产品对于人和动物组织的免疫组化标记模式
		- 主要是针对单克隆抗体和一些相产品进行检测，目的是为了了解靶分布或者可能的靶器官
		- 目前针对这些研究的价值性还存在一定的争议
- 细胞因子释放
	- 对于某些产品推荐使用体外实验，确定引起细胞因子释放或者细胞因子风暴的可能性
	- 如果产品与细胞表面的受体结合并且具有激动剂活性时，该实验很重要
	- 挑战：实验条件的一致性或者稳定性；结果的解释以及与人的相关性
- 免疫原性
	- 许多生物药物在动物和人类具有免疫原性的风险
		- 刺激产生抗药抗体（ADA）和抗人抗体
		- 刺激其他效应功能——如细胞因子，补体，溶血
	- 免疫系统识别药物为异物，从而产生针对药物的抗体
		- 结合性ADA——几乎没有影响
		- 清除抗体——增加药物清除，减少组织的分布
		- 中和ADA——降低药物的药理活性
		- 维持性ADA——降低药物清除，延长半减期
		- 与内源性蛋白的交叉反应性ADA——抑制或者改变药物的功能
	- 动物的免疫原性不能很恰当的预测人的免疫原性
		- 动物产生的ADA有助于解释毒理学研究的结果，但是不能预测在人体的免疫原性
		- 在不同器官免疫复合物沉积及相关的病理学变化

**蛋白类药物的关注问题**
- 免疫原性
	- 随着时间延长ADA可能减少药物的暴露和药理活性-可能导致长期毒性和药效研究的结果在动物试验中难以解释
- 急性血清病样反应
	- 重复暴露毒性研究中系统性的Ⅲ型超敏反应
	- 最初是在病人接受大剂量的外源性动物/人血清（治疗感染性疾病）出现的反应
	- 一般在暴露于外源血清/抗原10天后开始出现（常常是一次给药）：持续的产生抗体、抗原抗体复合物及其沉积
		- 发热、皮疹、出血
		- 急性系统性炎症反应（包括较低程度的凝集性病变）
		- 药物和ADA形成同时迅速清除
		- 尸检缺乏内生性的疾病
	- 免疫原性/过敏性往往在非自身蛋白的动物研究中会出现，但是不能代表在人类就会出现
- 免疫复合物疾病
	- 膜性肾小球肾病
- 免疫激活
- 靶器官和物种特异性的安全性问题

**核酸类药物的特殊关注点**
- 定义：寡核苷酸药物是较短的单链或者双链DNA或者RNA分子
	- 作用机制不同
	- 10-50核苷酸，有时候有结合物支持（e.g. PEG）
	- 调节基因的表达/蛋白质翻译/功能
- 安全性的问题
	- 药理效应过大
	- 非-药理学（脱靶）毒性
		- 聚阴离子效应——在高剂量的时候通过替代途径活化补体；抑制凝集链反应（寡核苷酸适配子）
		- 免疫激活：依赖于序列信息，基元/TLR介导的免疫活化；刺激细胞因子和趋化因子产生
		- 靶器官和组织蓄积
			- 靶器官
				- 主要是肝脏毒性、肾脏毒性和骨髓抑制
				- 肝毒性小鼠更为敏感
				- 大鼠对肾脏毒性更为敏感
			- 组织蓄积
				- 在不同组织广泛分布——常见于肝脏、肾脏和骨髓
				- 嗜碱性粒细胞和/或空泡化
		- 血液稀释——聚乙二醇连接的寡核苷酸

**干细胞治疗药物的安全性问题**
- 多能干细胞能够无目的性地自我更新和分裂
	- 胚胎干细胞-胚胎源性内细胞团
	- 成年干细胞-血液、骨髓、脂肪、脐带等
	- 可诱导干细胞-在体外可以诱导分化成成年细胞
- 干细胞可以分化成许多不同的细胞类型：内胚层、中胚层、外胚层，是生物药物研究的重要工具
- 共同的安全性问题
	- 免疫原性
	- 细胞迁移和移植
	- 畸胎瘤和异位性组织形成

**疫苗的安全性关注问题**
- 疫苗本身是免疫调节物，设计用于诱导人或者动物的特定免疫应答
- 初期的预防性疫苗用于阻断感染，现在增加的治疗性疫苗用于特定疾病的治疗
- 疫苗使用后的潜在问题就是产生免疫毒性

**生物药物的安全性评价**
- 每一个生物药物剂型都是独特的，需要设计和执行特定的安全性评价内容
- 良好设计和进行的药理学研究、毒理学研究和药代研究对于支持该药物安全应用于人体临床试验是很重要的
- 对每一个生物药物的非临床安全性评价个案分析决策是基于科学的推理和法规要求来执行的
- 新的技术保证不同生物药物剂型更有效的进行非临床安全性评价

# 4.转基因植物的安全性评价
**食用安全性评价**
- 新表达蛋白质与已知毒蛋白质和抗营养因子（蛋白酶抑制、植物凝集素等）的氨基酸比对研究
- 营养学评价（主要营养素的吸收、利用）
- 新表达物质及可食部位的毒理学评价（急性毒性、28天喂养等）
- 致敏性评价
- 蛋白热稳定性评价、体外模拟胃液蛋白消化稳定性试验

**中间试验**
- 提供新表达蛋白质的分子和生化特征信息
- 提供新表达蛋白质与已知毒蛋白质氨基酸序列相似性比较资料
- 提供新表达蛋白质与抗营养因子氨基酸序列相似性比较资料

**环境释放**：新蛋白质在植物食用和饲用部位的表达含量

**生产性试验**
- 新蛋白质体外模拟胃液蛋白消化稳定性试验
- 必要时提供全食品毒理学评价资料

**安全证书**
- 提供完整的毒性、致敏性、营养成分、抗营养因子等食用完全性资料

**环境安全性评价**
