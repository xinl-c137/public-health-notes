# 1.概念
**遗传毒理学(Genetic Toxicology)**
是研究化学、物理因素对遗传物质(DNA)及活细胞遗传过程的作用，以及人类接触致突变物可能引起健康效应的学科。主要研究致突变的作用机理，应用检测系统发现和探究致突变物，提出评价致突变物健康危害方法
- 遗传毒性(genetic toxicity)：外源因素引起生物体遗传物质表达异常，结构和功能损害的能力。可引起遗传毒性发生的物质称为遗传毒物
- 致突变性(mutagenicity)：指引起生物体细胞遗传物质发生突变的能力，在一个实验群体中突变率可以定量检测。
- 遗传毒性的概念更为广泛，它包括对基因组毒作用引起的致突变性及其他不良效应

**突变(mutation)**
遗传结构本身的变化及引起的变异，是遗传物质的一种可遗传的变异
- 自发突变(spontaneous mutation)：发生频率低、渐进、有益
- 诱发突变(induced mutation)：发生频率高、突然、可能有害
- 化学致突变作用(chemical mutagenesis)
- 诱变剂(mutagen)
	- 直接诱变剂(direct-acting mutagen)：原型
	- 间接诱变剂(indirect-acting mutagen)：本身无生物毒性，经代谢后活化成为诱变剂

**毒作用效应谱**
- 死亡
- 中毒、患病
- 亚临床变化
- 体内过量负荷
- 致突变作用
- 致癌作用

**有阈和无阈毒性作用**
一般认为，外源化学物的一般毒性（器官毒性)和致畸作用是有阈值的，即达到一定的剂量水平才对机体产生毒性作用；而遗传毒性致癌作用和致突变作用则无阈值，即只要接触就可能产生有害作用

### 基因突变(gene mutation)
- 碱基置换突变base-pair substitution mutation
- 移码突变frame shift mutation
- 整码突变codon mutation
	- 又称为密码子的插入或缺失，基因产物多肽链中会增加或减少一个或几个氨基酸
- 片段突变fragment mutation
	- 基因中小片段核苷酸序列发生改变，可跨越两个或数个基因，涉及数以千计的核苷酸
	- 主要包括核苷酸片段缺失、重复、重组及重排等。
### 染色体突变(chromosome mutation)
染色体畸变chromosome aberration
染色单体畸变chromatrid aberration

| 损伤类型              | 特点                                                     |
| ----------------- | ------------------------------------------------------ |
| 裂隙(gap)           | 在一条或两条染色单体上出现无染色质的区域，但该区域所分割的两段染色体仍保持线性。               |
| 断裂(break)         | 同裂隙，但无染色质区域所分割的两段染色体不再保持线性。                            |

| 倒位(inversion)     | 染色体或染色单体发生两处断裂，其中间节段旋转180°后再重接。可有臂间倒位和臂内倒位。            |

# 3.遗传毒性作用机制
## 3.1以DNA为靶的直接诱变作用
**基因突变、染色体畸变**
- 碱基类似物的取代
	- 有些化学物的结构与碱基非常相似，称碱基类似物，如5-溴脱氧尿嘧啶取代胸腺嘧啶，2-氨基嘌呤取代鸟嘌呤
	- 在DNA复制时与正常的碱基竞争，取代其位置，并与互补链上的碱基配对，常常引起碱基置换突变
- 碱基烷化
	- 烷化剂可使DNA碱基发生烷化，引起配对特性的改变，导致碱基置换型突变；也可能导致碱基与脱氧核糖结合力下降，引起脱嘌呤、脱嘧啶作用，最终导致移码突变、DNA链断裂等
	- 硫酸二甲酯、甲基磺酸乙酯、乙基黄酸乙酯、氮芥和硫芥等是常见的烷化剂
	- 各类烷化剂其分子上的烷基不同，烷化活性也不同，一般情况下，甲基化＞乙基化＞高碳烷基化
	- 最常发生烷化作用的是鸟嘌呤的N-7位，其次是O-6位，腺嘌呤的N-1、N-3和N-7也易烷化
	- 多功能烷化剂除了可使碱基发生烷化外，还常引起DNA发生链内或链间的交联，或与蛋白质的交联，交联常可导致染色体或染色单体的断裂
- 与DNA分子共价结合形成加合物
	- 亲电子性化学物可与DNA作用形成共价结合物一加合物(adduct)
	- 芳香族化学物：经代谢活化后形成亲电子基闭，可与DNA碱基上的亲核中心形成加合物
- 改变碱基的结构
	- 某些诱变剂可与碱基发生相互作用，使碱基发生除形成加合物以外的化学结构改变，引起错误配对或DNA链断裂
	- 亚硝酸可使胞嘧啶、腺嘌呤氧化脱氨基，分别形成尿嘧啶和次黄嘌呤，新的碱基形成后，配对关系发生变化，尿嘧啶、次黄嘌呤分别可与腺嘌呤和胞嘧啶配对，导致C：G→T：A和A：T→G：C的转换
- 平面大分子嵌入DNA链
	- 一些具有平面环状结构的化学物可以非共价结合的方式嵌入核苷酸链之间或碱基之间，干扰DNA复制酶或修复酶，引起碱基对的增加或缺失，导致移码突变
	- 如果嵌入到新合成的互补链上，就会缺失一个碱基，如果嵌入到两模板链的碱基之间，就会使互补链插入一个碱基。会造成移码突变
- 形成二聚体
	- DNA同一条链上的两个相邻的嘧啶核苷酸会发生共价连接，形成嘧啶二聚体
	- 主要的二聚体类型是胸腺嘧啶二聚体（TT），此外还有胞嘧啶二聚体（CC）以及胞嘧啶-胸腺嘧啶二聚体（CT）
	- 二聚体的形成会严重影响DNA的复制和转录，诱导突变的发生
- DNA-蛋白交联
## 3.2不以DNA为靶的间接诱变作用
干扰细胞分裂过程——染色体数目改变
- 对纺锤体的毒作用
	- 与微管蛋白二聚体结合，妨碍微管的正确组装，发生细胞分裂完全抑制。如秋水仙碱、长春花碱、长春新碱等
	- 与微管上的巯基结合，细胞分裂不完全抑制。如铅、锌、汞、砷等
	- 破坏已组装好的微管。如灰黄霉素、秋水仙碱、乙酰甲基秋水仙碱、长春花碱可导致组装好的微管解聚；毛地黄皂苷能通过非特异的蛋白质变性作用而破坏微管
	- 妨碍中心粒移动。秋水仙碱能妨碍有丝分裂早期两对中心粒的分离和移向两极

对DNA合成及修复酶的影响
## 3.3DNA损伤修复机制
![[6.遗传毒理学-3.遗传毒性作用机制-20260401.png]]
其中DNA修复是最主要途径

- 直接修复（特异性太强）
	- 光复活修复
		- 一种依赖光的过程，在光裂合酶作用下可修复紫外线诱发的嘧啶二聚体，使其在原位上恢复为单体
	- 烷基转移酶修复
		- O<sup>6</sup>烷基鸟嘌呤DNA烷基转移酶是广泛存在的一种烷基转移酶，主要修复DNA烷化损伤
- 切除修复（最主要）
	- 碱基切除修复
		- 由DNA糖基酶作用于受损的DNA，识别异常的碱基，切断碱基与脱氧核糖的连接，使受损的碱基脱落
		- 碱基切除修复的DNA糖基酶特异性很强，是细胞对碱基氧化损伤修复的最主要防御系统
	- 核苷酸切除修复
		- 核苷酸切除修复的基本步骤：
			- 损伤识别
			- 损伤两侧切开损伤链，释放出一个24-32核苷酸长度的低聚体
			- 切除寡聚核苷酸
			- 修复合成填补产生的缺口
			- DNA连接酶封闭，恢复原有DNA序列
		- 核苷酸切除修复是生物体内最常见的修复机制，基本上能修复所有种类的DNA损伤
	- 错配碱基修复
- 双链断裂修复
	- 同源重组：从同源双螺旋中重新获得因双螺旋断裂而丢失的信息
	- 非同源末端连接：是高等真核细胞中双链断裂修复的主导机制，与同源重组修复最大的差异在于完全不需要任何模板的帮助，修复蛋白直接将双链断裂的末端彼此拉近，再由DNA连接酶将断裂的双链重新结合
- 交联修复
	- 细胞内发生DNA链内交联，会造成双链的断裂，机体主要启动无误交联修复机制进行修复，还存在另一种次要的易误交联修复机制

## 3.4DNA损伤耐受机制
易误修复(error-prone repair, SOS repair)
- 切除修复、复制后修复等不能进行时，在损伤因素的诱导下才发生
- 诱导产生特殊的DNA聚合酶，以不严格碱基配通过损伤部位
- 常为易误修复

复制后修复(post-replication repair)
- 多见于复制时产生裂隙的DNA损伤，仅在DNA复制时发生
- 受损碱基阻滞DNA正常复制，新合成的互补链相应部位出现空隙
- 以重组作用及链延长作用填补空隙
- 避免了致死性的后果，但DNA损伤并未移除

## 3.5表观遗传调控机制
- **DNA甲基化**
	- 化学物的遗传毒性会影响DNA甲基化
	- DNA甲基化可调控遗传损伤的进展
- **非编码RNA**
	- 遗传损伤可影响微小非编码RNA的表达，微小非编码RNA的表达水平也可以影响化学物的遗传毒性作用及DNA的损伤修复
	- 长链非编码RNA可以富集到DNA损伤部位，参与遗传损伤或损伤后修复
	- 环状非编码RNA可通过与微小RNA结合，影响多种DNA损伤应答基因的表达
- **组蛋白修饰**
	- 组蛋白修饰对于遗传毒性作用的识别和信号传递，以及染色质的损伤和修复是必要的
	- 组蛋白修饰可调控DNA修复基因的表达
- **染色质重塑**
	- 在基因活化和转录时，染色质发生系列变化，如核小体结构变疏松，使核小体DNA易于暴露而被其它分子结合
	- 调节外来化学物对DNA的损伤作用，控制细胞对遗传损伤剂的敏感性
	- 染色质重塑对DNA损伤修复也具有关键的作用
- **RNA修饰**
	- RNA上的修饰包含了m6A、mlA、m5C和假尿(嘧啶核)苷，对调控基因表达和细胞功能至关重要
	- RNA修饰则是通过影响RNA的稳定性、翻译效率及调控基因表达来参与调节这些过程中遗传损伤的反应

# 4.遗传毒性的健康效应
![[6.遗传毒理学-4.遗传毒性的健康效应-20260401.png|453]]
## 4.1体细胞遗传毒性效应
- 突变致癌
	- 原癌基因经点突变或染色体畸变可转变成活化的癌基因
	- 抑癌基因的突变、失活或缺失在许多肿瘤的发生过程中起重要作用
- 突变致畸
	- 人类妊娠最初3个月流产中，有60%存在染色体畸变，在一定程度上这是致突变物透过胎盘作用于胚胎体细胞所致
- 其他
	- 突变与动脉粥样硬化症有关
	- 突变可能是衰老的原因之一

## 4.2生殖细胞遗传毒性效应
- 致死性
	- 显性致死：突变的精子不能受精，或合子在着床前死亡或着床后的早期胚胎死亡，对基因库不会产生影响
	- 隐性致死：纯合子或半合子才能出现死亡效应，可能出现显性或隐性遗传病，包括先天性畸形
- 非致死性
	- 显性存活突变：不会引起胚胎死亡，遗传缺陷可在子一代表现出来，即为显性遗传病
	- 隐形存活突变：在杂合子不能表现出来，必须在纯合子才能出现疾病

## 4.3遗传负荷和疾病风险
- 人类基因库(gene pool)
	- 指人群生殖细胞内所具有的能传给下一代的所有基因总和
	- 人类基因库的组成决定了人类的遗传整体性
	- 人类基因库的组成并非完美无缺，存在有部分致病基因
- 遗传负荷 (genetic load)/突变负荷 (mutation load)
	- 人类每一个体携带的可遗传给后代的有害基因的平均频率或水平
	- 当代人遗传负荷的大小直接影响到下一代或几代人的健康，遗传负荷的增加将引起人群发病率和死亡率上升，出生缺陷增多，群体健康水平下降
	- 环境有害因素的大量和长期暴露无疑会破坏正常的人类基因库，会增加遗传负荷

# 6.检测基因突变的试验
## 6.1细菌回复突变试验——Ames试验
**实验原理**
- 鼠伤寒沙门氏菌的组氨酸营养缺陷型菌株（His-）在某个调控组氨酸合成的基因发生了点突变，丧失了合成组氨酸的能力。这些菌株在无组氨酸的培养基上不能生长，在有组氨酸的培养基上可以正常生长。致突变物可以使鼠伤寒沙门氏菌由突变型（His-）回复突变为野生型（His+），即恢复合成组氨酸的能力，在不含组氨酸的选择培养基上生长成可见的菌落。如果选择培养基上的菌落显著超过了自发回变数，即可判定受试物为鼠伤寒沙门氏菌的致突变物
- 为了监测直接或间接致突变物，可加或不加代谢活化系统，通常为**大鼠肝微粒体酶（S9）**
- 为了增加对诱变物的敏感性——附加突变，多种试验株

| 特性     | 意义                        |
| ------ | ------------------------- |
| raf改变  | 细菌荚膜脂多糖屏障，使致突变物有更大的通透性    |
| ΔuvrB  | 切除修复系统缺失，增加对很多致突变物的敏感性    |
| pKM101 | 增强对某些其活性依赖于SOS系统的致突变物的敏感性 |
| pAQ1   | pAQ1上hisG428的回复性可被测试      |
- 为检测各种不同的突变——用一组菌株进行试验（TA97、98、100、102）

**优缺点**
- 优点
	- 致突变检测首选方法
	- 积累资料多
	- 简单、快速
- 缺点
	- 指示生物为原核生物
	- 假阳性和假阴性
	- 不适用某些化学物

**S9混合液**
- 组成：S9、MgCl<sub>2</sub>、KCl、磷酸盐缓冲液、葡萄糖-6-磷酸溶液、辅酶Ⅱ
- S9混合物是大鼠肝脏细胞匀浆经过9000转离心的微粒体上清液，加上相应的缓冲液和辅助因子
- 意义：由于微生物和培养的哺乳动物细胞缺乏整体哺乳动物的代谢能力，作用在许多遗传试验中都要加入活化体系以检测出间接致突变物，应用最广泛的就是S9混合液

**实验结果鉴定**
- 组氨酸营养缺陷鉴定：培养基无组氨酸不能生长；四种都不能生长
- rfa突变鉴定（结晶紫抑菌实验）：深粗糙型突变的鉴定，由于缺少脂多糖屏蔽，大分子物质能进入菌体，此四种菌在结晶紫溶液渗透区都出现抑菌，证明实验菌株有rfa突变
- ΔurvB缺失鉴定（紫外线敏感实验）：ΔurvB缺失即是切除修复系统缺失。TA97，TA98，TA100仅在没有照射过紫外线的一半生长，在照射过紫外线的一半不能生长，即这三种菌株有ΔurvB缺失，而TA102在没有照射和照射过的均能生长，证明TA102不具有这种缺失
- R因子和pAQ1：R因子具有抗氨苄青霉素的特性，pAQ1质粒具有抗四环素的特性。四个菌都不受氨苄青霉素的抑制，即都带有 R 因子。TA102不受四环素的抑制，证明带有pAQ1质粒
- 自发回变实验
	- 回变数：TA97：90-180；TA 98：30-50；TA 100：120-200；TA102：240-360
- 点实验：叠氮钠
	- 随着叠氮钠量的增加，菌落的数量增加，但当叠氮钠的量过高时，由于浓度过高对细菌产生细胞毒性，从而抑制了细菌的生长，出现抑菌环。抑菌环相邻的外圈有很密集的菌落生长，随着叠氮钠向周围扩散，浓度逐渐降低，菌落的数量也由抑菌环周围向外逐渐减少。由此可以得出，在一定浓度的范围内，随着叠氮钠量的增加，细菌的回复突变增加，叠氮钠起到促进回复突变的作用，但浓度过高反而出现细胞毒性
	- 是不是只有呈剂量反应关系才能推断叠氮钠是致突变物？
		- 答案：否，只要满足任意剂量超过自发回变两倍以上、呈剂量反应关系两个条件之一即可

## 6.2哺乳动物细胞基因突变试验Mammalian cell gene mutation assay
### 小鼠淋巴瘤（L5178Y）细胞tk位点突变
位于常染色体上，检测遗传损伤范围广
**TK试验基本原理**
- TFT对TK细胞具有毒性，TFT引起细胞分裂停止
- tk基因突变，引起细胞缺乏胸苷激酶TK
- 缺乏TK的细胞对嘧啶类似物三氟胸苷（TFT）的细胞毒作用具耐受性
- 突变细胞在TFT存在时能增殖，而普通细胞不能
### 中国仓鼠卵巢（CHO）细胞hgprt位点突变
位于X染色体上，可能与必须的基因相接
- HGPRT 突变的细胞可以抗6-硫代鸟嘌呤（TG）的毒性作用
- 在含有**6-TG的选择性培养基**中，只有发生了HGPRT基因突变、丧失酶活性的细胞才能存活并增殖形成克隆

## 6.3染色体畸变试验Chromosome aberration test
- 体外in vitro：CHO、CHL、V79、淋巴细胞
- 体内in vivo：骨髓细胞、睾丸细胞

## 6.4微核试验Micronucleus assay
**微核**：在有丝分裂（减数分裂）末期迟滞的染色体断片或整条染色体生成的，分离于细胞主核之外的小核
胞浆里面看微核
应用最多：小鼠骨髓多染红细胞微核试验Micronucleus assay in bone marrow polychronic erythrocytes
**小鼠骨髓多染红细胞微核实验**：当骨髓成红细胞发展为红细胞时，主核排出，成为多染红细胞，这些细胞保持其嗜碱性约24小时，成为正染红细胞，并进入外周血中。在主核排出时，已形成的微核可留在胞浆中，可通过计数有微核的多染红细胞率反映染色体断裂和纺锤体损伤的情况
细胞增殖速率影响微核——双核细胞微核试验-将经历过一次复制的细胞挑出来看微核发生情况（细胞增殖素B）

## 6.5显性致死试验Dominant lethal test
**显性致死**：引起胚胎或胎儿死亡
**显性致死突变**
- 发生于生殖细胞的突变，它不引起受精障碍，但能引起受精卵或发育中的胚胎死亡
- 引起显性致死的突变主要是染色体断裂

## 6.6姐妹染色单体交换试验Sister chromatid exchange assay
姐妹染色单体交换SCE：染色体同源位点上DNA复制产物的相互交换，其频率与DNA的断裂和修复有关

## 6.7单细胞凝胶电泳试验Single cell gel electrophoresis, SCGE（彗星试验Comet Assay）
- 在单细胞水平上定量检测DNA损伤
- 在电泳槽中，DNA断片在电场的作用下由细胞核中移出，并向阳极泳动，经荧光染色后可见到细胞核和移出的DNA断片，形成如同彗星一样的彗头和彗尾，DNA迁移的程度与单链DNA断裂的水平有关
- 技术简单、快速、价廉、检测DNA损伤的敏感性高
- 可进行体外试验和体内试验，也能用于人体监测

## 6.8程序外DNA合成实验（unscheduled DNA synthesis test, UDS）
- 正常细胞仅在S期进行DNA复制合成
- 当DNA受损后，DNA修复合成可发生在S期以外
- 用同步培养将细胞阻断于G1期，并将正常的DNA半保留复制阻断
- 培养液中加入3H-胸腺嘧啶核苷
- 测定掺入DNA的放射活性，评价DNA修复合成的程度
- 间接反映DNA的损伤程度
- 常用的细胞：大鼠原代培养肝细胞，人成纤维细胞，外周血淋巴细胞单核细胞，Hela细胞及人羊膜细胞FL株等

## 6.9转基因动物致突变试验Transgenic animal mutagenicity assay
**转基因动物（transgenic animal）**：基因组中整合有用实验方法导入的DNA（可以是完整的基因，也可是不完整的 DNA片段），并能遗传给后代的一类动物
Big blue mouse：以大肠杆菌 LacI为靶基因
Muta mouse：以大肠杆菌LacZ为靶基因

# 7.遗传毒理学试验在预测致癌性、评价遗传危害性中的应用
**灵敏性(sensitivity)**：受试致癌物中，在遗传毒理学试验中呈致突变性阳性结果的比例
**特异性(specificity)**：受试非致癌物中，在遗传毒理学试验中呈阴性结果的比例

**常用的哺乳动物生殖细胞突变检测方法**

| 生物分析       | 技术                                                                 |
| ---------- | ------------------------------------------------------------------ |
| 小鼠特异位点试验   | 分析受处理小鼠F1代的可见隐性突变。利用包括7个以上隐性位点的多标记小鼠测试品系检测可见的常染色体隐性突变              |
| 小鼠白内障突变分析  | 用裂隙灯对受处理小鼠F1代3周龄小鼠眼睛进行生物显微测试观察可见的显性突变。对小鼠无特殊品系要求。这一手段在大鼠可以直接计算诱发突变 |
| 小鼠骨骼突变分析   | 检测影响小鼠骨骼系统的可见显性突变。需要对骨发育缺陷进行认真的分析                                  |
| 小鼠可遗传易位试验  | 对受处理雄性F1代进行生育能力分析并以细胞遗传学分析证实                                       |
| 小鼠隐性致死试验   | 受处理小鼠F2代回交，分析胚胎死亡率                                                 |
| 小鼠生化特异位点试验 | 分析受处理小鼠F1代的蛋白电泳图谱以了解生殖细胞突变                                         |
| 小鼠酶活性分析    | 分析受处理小鼠F1代特异性酶活改变。                                                 |

体细胞和生殖细胞试验诱发突变的敏感性比较：体细胞敏感性>生殖细胞，先做体细胞试验

**遗传毒性试验成组应用组合原则**
- 应包括多个遗传学终点
- 试验指示生物包括若干进化阶段的物种
- 应包括体外试验和体内试验
- 在预测可遗传的危害时，应包括体细胞及性细胞的试验

**《食品安全性毒理学评价程序》(2014)**
- 组合1：细菌回复突变试验，哺乳动物红细胞微核试验或哺乳动物骨髓细胞染色体畸变试验；小鼠精原细胞或精母细胞染色体畸变试验或啮齿类动物显性致死试验
- 组合2：细菌回复突变试验，哺乳动物红细胞微核试验或哺乳动物骨髓细胞染色体畸变试验；体外哺乳动物细胞染色体畸变试验或体外哺乳动物细胞TK基因突变试验
- 其他备选试验：显性致死试验，UDS试验，体外哺乳动物细胞HGPRT基因突变试验

**《农药安全性毒理学评价程序》(2017)**
1. 细菌回复突变试验
2. 体外哺乳动物细胞基因突变试验
3. 体外哺乳动物细胞染色体畸变试验
4. 体内哺乳动物骨髓细胞微核试验
- 1-3任何一项阳性，4阴性：增加另一项体内试验
- 1-3阴性，4阳：增加体内哺乳动物生殖细胞染色体畸变试验或显性致死试验

**ICH推荐的试验组合方案（一)**
 - 细菌基因突变试验
 - 染色体损伤的细胞遗传毒性试验（体外的染色体畸变或体外的微核试验)，或者体外的小鼠淋巴瘤tk位点基因突变试验
 - 体内的遗传毒性试验，通常是啮齿类造血细胞染色体损伤试验，可选择微核试验也可选择细胞染色体畸变试验

**ICH推荐的试验组合方案（二）**
- 细菌基因突变试验
- 两个体内遗传毒性试验（通常一种是啮齿类的微核试验）
