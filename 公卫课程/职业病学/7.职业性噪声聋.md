# 1.噪声与听力介绍
**噪声的概念**
- 噪声(noise)是指环境中不需要、使人厌烦的、不同频率和强度杂乱无章的声音
- 从物理学角度讲，噪声是一种在频率和强度上毫无规律的随机组合的声音
- 从生理和心理学角度讲，噪声是人们不喜欢的、使人烦恼的、有害身心健康的声音

**噪声的来源**
- **交通运输噪声**：各种交通工具的喇叭声、汽笛声、刹车声、排气声、机械运转声等
- **工业噪声**：纺织厂、印刷厂、机械车间的噪声
- **施工噪声**：筑路、盖楼、打桩等
- **社会生活噪声**：家庭噪声、娱乐场所、商店、集贸市场里的喧哗声

**噪声的分类**
- 按噪声源的物理特性分类
	- 气体动力噪声：叶片高速旋转或高速气流通过叶片，会使叶片两侧的空气发生压力突变，激发声波
	- 机械噪声：物体间的撞击、摩擦、交变的机械力作用下的金属板，旋转的动力不平衡，以及运转的机械零件轴承、齿轮等都会产生机械性噪声
	- 电磁性噪声：由于电机等的交变力相互作用产生的声音。如电流和磁场的相互作用产生的噪声，发动机、变压器的噪声均属此类
- 按噪声源的时间特性分类
	- 稳态噪声：指噪声声压级的变化较小（一般不大于3dB），且不随时间有大幅度的变化，如电机、风机及其他电磁噪声，固定转速的摩擦、转动等噪声
	- 非稳态噪声：指噪声强度随时间而有起伏波动（声压变化大于3dB）。有的呈周期性噪声，如锤击；有的呈无规律的起伏噪声，如交通噪声
	- 脉冲噪声：指持续时间小于1s的单个或多个突发声组成的噪声，声压级原始水平升至峰值又回至原始水平所需的特续时间短于500ms，其峰值声压级大于40dB。脉冲噪声往往是突发的高强噪声，如爆破、火炮发射等所产生的噪声
- 按噪声的频率成分分布分类
	- 低频噪声：主频率＜400Hz
	- 中频噪声：主频率在400~1000Hz
	- 高频噪声：主频率高于1000Hz
	- 宽频带噪声：从低频到高频较为均匀的噪声
	- 窄频带噪声：主要成分集中分布在狭窄的频率范围内的噪声
	- 有调噪声：既有连续噪声，又有离散频率成分存在噪声

**声音强弱等级**
- 声级：与人们对声音强弱的主观感觉相一致的物理量
- 为保护听力，声音不能超过85dB
- 为保证学习和工作，声音不能超过70dB
- 为保证休息和睡眠，声音不能超过50dB

**职业性噪声(occupational noise)**：生产和工作过程所产生的噪声
==**职业性噪声聋(occupational noise-induced deafness)**==
- 人们长期接触“职业性噪声”
- 感音神经性听力损失(sensorineurol hearing loss)

**如何听见声音？**
- 声音是空气振动产生的疏密波，被人感受后就产生了听觉
- 声音的传导途径：声波振动→耳廓收集→耳道传入→振动鼓膜+听骨链→传入内耳听觉感受器→毛细胞兴奋产生神经冲动→听神经传入→皮层听中枢→听觉

# 2.噪声的危害
**噪声对人体的危害与以下因素有关**
- 性质（强度、频率）
- 接触时间与防护
- 人的身体易感性

**噪声危害的三要素**：噪声源、传播途径、接受者

## 2.1特异性危害（听觉系统）
**暂时性听阈位移(temporary threshold shift, TTS)**
- 接触噪声后在听觉疲劳的基础上产生，当与噪声源脱离接触后，可在不同时间内完全恢复
- 听觉适应(auditory adaptation)：短时间暴露于强烈噪声，听觉器官的敏感性下降，听阈可上升10dB~15dB，脱离噪声环境后数分钟内即可恢复正常，这种现象称为听觉适应，听觉适应是一种生理保护现象
- 听觉疲劳(auditory fatigue)：较长时间暴露于强噪声，听力可出现明显下降，听阈上升超过15dB~30dB，脱离噪声环境后，需数小时甚至数十小时听力才能恢复，此现象称为听觉疲劳

**永久性听阈位移(permanent threshold shift, PTS)**
- 随着接触噪声时间的延长，前一次接触噪声引起的听力改变尚未完全恢复前再次接触噪声，听觉疲劳则加重，听力改变不能恢复而成为永久性听阈位移
- 永久性听阈位移属不可逆的病理改变
- 根据听力受损程度，永久性听阈位移可分为听力损伤和**噪声聋(noise-induced deafness)**
- 噪声聋属于我国法定职业病

## 2.2非特异性的危害
- 产生疲倦生理效应
- 心跳加速、血压升高
- 头痛头晕、精神紧张
- 无法集中精神导致安全工伤事故

**生产活动影响**
- 噪声可诱发事故，在高噪声环境中工作，人的心情烦躁，易疲劳，反应迟钝，工作效率下降，工伤事故增多
- 噪声还对物体产生破坏作用，如抹灰开裂、墙裂缝
- 强噪声能使设备和仪器受干扰至破坏

**心理影响**：干扰人们的交谈、休息和睡眠，从而使人产生烦躁、焦急，妨碍注意力集中，降低工作效率

# 3.噪声聋的机制
**耳聋的定义**：听觉系统中，传音、感音、听神经及各级中枢发生病变，引起听功能障碍，导致不同程度听力下降，称为耳聋

**耳聋的分类**
- 按病变性质
	- 器质性耳聋：听觉系统有器质性变化
	- 功能性耳聋：听觉系统无器质性变化，但听觉功能异常
- 按发病时间
	- 先天性耳聋：妊娠期和产程中由各种因素导致胎儿听觉系统病变或损伤
	- 后天性耳聋：胎儿出生后听觉系统发生病变和损伤
- 按语言形成和发展的关键期
	- 学语前耳聋（4岁前）
	- 学语后耳聋（4岁后）
- ==**根据病变的发生部位**==
	- 传导性聋(CHL)：外耳、中耳
	- 感音神经性聋(SNHL)：内耳、听神经、听中枢
		- 感音性聋（耳蜗性）；蜗后聋；中枢性聋
	- 混合型聋(MHL)：中耳内耳均有病变

**引起听力减退的外耳疾病**
- 先天畸形
- 炎性肿胀
- 异物或肿瘤阻塞

**引起听力减退的中耳疾病**
- 先天畸形
- 炎症（中耳炎）
- 外伤（鼓膜穿孔、听骨链中断）
- 肿瘤

**引起听力减退的内耳疾病**
- 先天遗传
- 急性传染病
- 药物中毒
- 强声损伤
- 免疫缺陷

**噪声性聋的影响因素**
- **噪声强度**
	- 环境噪声≤80dB(A)时，即使终身暴露亦不会引起噪声性聋
	- 当坏境噪声>85dB(A)时，接触噪声的强度才与听力损失发病率呈正相关
		- 85dB(A)下发病年限为20年
		- 90dB(A)则为10年
		- 95dB(A)为5年
		- 100dB(A)以上均在5年之内
		- 随暴露年限增加听力损失发生率也逐年增高
- **噪声频率**
	- 人耳对低频噪声的耐受性比中频和高频噪声高
	- 2000Hz-4000Hz的声音最易导致耳蜗损害
	- 窄带音或纯音较宽带声影响大
	- **特续性噪声较断续的噪声损伤性大**
	- 突然出现的噪声危害性更大
	- 噪声伴振动对耳蜗的损害也较单纯噪声显著
- **接触噪声时间**
- **个体差异**
	- 噪声易感者约占人群5%，易感基因
	- 年龄越大，噪声导致的内耳损伤越重，并且受损耳蜗恢复能力越差
	- 原患有感音神经性听力损失的工人，暴露于噪声后，更易发生噪声性聋
	- 噪声性聋发病快慢以及听力损失程度更与个人防护密切相关

**存在职业性噪声的工种**：铆工、锅炉工、蒸汽锤工、铲工、锻工、锤工、锻冶工、织布工、纺纱工、飞机驾驶员等

**噪声导致内耳损伤的机制**
- 机械破坏：高强度短峰间期噪声
	- 细胞发生溶解、内外淋巴液混合、螺旋器自基底膜上分离脱落、毛细胞与神经纤维之间的突触连接撕脱
- 代谢异常：较低强度长峰间期噪声
	- 引起酶系统代谢障碍、内耳能量储备和供应耗竭、内耳缺氧、氧能量代谢明显抑制、毛细胞破坏
- 内耳微循环障碍：强噪声
	- 内淋巴液氧张力降低、耳蜗血流灌注减少

# 4.噪声聋的临床表现
**症状体征**
- 耳鸣
	- 耳鸣常常是早期症状，多为双侧高调性耳鸣
- 听力下降
	- 噪声引起的听力损失多为**双侧、对称、渐进性**感音神经性聋
	- 噪声损害首先累及高频区域，所以发病早期病人并未感到有听力下降，仅在纯音听阈测试时表现为3000Hz、4000Hz或6000Hz听阈的提高，这三个频率也是噪声性聋最早受影响的频率
	- 随着接触噪声时间延长，耳蜗受损频率可逐渐向中、低频率区发展，此时病人开始主诉听力下降；纯音听阈测试表现为音频听阈（500Hz、1000Hz、和2000Hz）提高，纯音听力图示为感音神经性聋
- 全身影响
	- 头痛、头晕、失眠、多梦、注意力减退、抑郁等症状
	- 长期暴露于噪声环境中出现前庭功能减退、平衡功能障碍

==**听力测试方法**==
- 主观测听法：定性、定量
	- 纯音测听
	- 言语测听
- 客观测听法：定位
	- **声导抗测听（中耳）**
	- 耳声发射测听（内耳）
	- 听性脑干反应（听神经及低级听中枢）
	- 40Hz听觉相关电位

**言语测听**
- 根据特制的词表发出口声或用录声磁带放声与听力计相结合的测试法
- 用言语信号检查听力，以能否听清和听懂言语来判断听功能，比较符合实际。可了解病变部位、评价社交能力、帮助选配助听器
- 言语识别阈和言语识别率

**纯音听阈测试**
- 听阈：声信号引起听觉的最小声压级
- 方法：用纯音可测气导和骨导听阈
- 意义：是听力测试中最基本的检查，耳聋定性、定量
- 纯音听阈测试是用以了解听敏度，并对听力损失进行定性和定量的**主观行为测试**
- 优点：可靠性、重复性好，具有良好的频率特异性
- 缺点：检查会受到受试者反应动机和反应能力等非听觉性因素的影响
- 传导性聋
	- 骨导曲线在≤25dB的范围内，气导曲线分布大于25dB，不在正常范围内。两条曲线距离大于10dB
- 感音神经性聋
	- 骨导曲线和气导曲线分布都大于25dB，不在正常范围内，而且两条曲线距离小于10dB
- 混合性聋
	- 骨导曲线和气导曲线分布都大于25dB不在正常范围内，而且两条曲线距离大于10dB

**声导抗**
- 测试中耳功能，了解人耳对声能传递状态的客观测听法
- 意义：对耳聋的定性、定位诊断，面瘫定位和预后估计，非器质性聋的提示，婴幼儿听力筛选有价值
- 镫骨肌声反射阈
	- 声反射阈是指能重复引出声反射的最小声音强度，以dBHL表示
	- 正常耳的声反射阈为70~95dBHL。一般认为同侧声反射阈比对侧低2~16dB，也就是对侧声反射阈的正常阈值约为72~111dBHL
	- 测试方法：在相对安静的环境内，选择大小合适的耳塞，使探头密封于外耳道内，做鼓室导抗图，于峰压点处给予不同频率的刺激声，应用上下升降5dB的等级法确定其阈值
	- 根据声反射的传导路径，以下几种情况会导致声反射的消失
		- 1、刺激耳重度听力损失
		- 2、刺激耳听神经病变
		- 3、记录耳传导性听力损失
		- 4、记录耳面神经病变
		- 5、记录耳镫骨肌健缺失
	- 刺激耳为左耳，记录耳亦为左耳，则为同侧声反射测试，在各个频率点上其则试结果在70~95dBHL范围内，为正常值
	- 刺激耳为左耳，记录耳为右耳，则为对侧声反射测试

**耳声发射（OAE）**
- 是产生于耳蜗，经听骨链及鼓膜传导，逆向释放于外耳道的音频能量。其来源于耳蜗外毛细胞
- 意义：代表耳蜗的主动性机械活动及正常的耳蜗功能。对新生儿及婴幼儿听力筛查、感音神经性聋定位诊断有价值

**电反应测听**
- 听觉系统在接受声刺激后，从耳蜗毛细胞到各级中枢均有相应的生物电活动，统称听觉诱发电位，将其作为指标来客观评价听力的方法，即电反应测听
- 意义：耳聋病变定位、鉴别器质性与功能性聋、客观估计听阈、婴幼儿听力筛查等，是现代测听方法中的重要技术之一
- 听性脑干反应（ABR）
	- 是声刺激后10ms内的诱发电位，可能来源于脑干的听通路，通称听性脑干反应，临床应用最多
	- 意义：对婴动儿听力筛选、器质性和功能性聋的鉴别具有重要价值。由于用含高频为主的短声刺激，其主要反映高频听力
- 40Hz听觉相关电位（40HzAERP）
	- 用重复率为40次/秒的声刺激所诱发的一组电位，称为40HzAERP
	- 意义：具有频率特性，能判断阈值，可客观评估听力。对ABR无反应的婴幼儿，可以此了解其有无中低频残余听力

# 5.诊断原则
**职业性噪声聋诊断原则**
- 明确的噪声作业史：8h等效声级≥85dB(A)
- 噪声作业工龄连续3年以上
- 纯音测听为SNHL，双耳高频平均听阈≥40dB，根据较好耳语频和高频4000Hz听阈加权进行诊断和分级
- 多次测试结果听阈变化≤10dB(HL)
- 应脱离噪声作业环境48测定听力，若达到噪声聋水平，复查应为脱离噪声环境后1周
- 排除中、内耳其他疾病所致的听力下降

**职业性噪声聋诊断步骤**
- 耳鼻喉科常规检查
- 至少进行3次纯音测听检查，2次检查时间间隔至少3天，且各频率听阈偏差应≤10dB
- 听力检查结果需进行**年龄、性别**修正
- 声导抗除外中耳疾病
- 对纯音测听不配合或结果不可靠者，应进行客观检查

**如何做正确的噪声测试分析**
- 测听方法的选择
	- 根据不同病情
	- 推荐听力测试组合
	- 主客观结合
- 测听结果分析
	- 综合评价
	- 动态观察
	- 其他考虑因素

**职业性噪声聋诊断分级**
- 年龄修正并计算双耳高频（<u>3000Hz、4000Hz、6000Hz</u>）平均听阈≥40dB，根据较好耳语频（500、1000、2000）和高频4000Hz听阈加权值进行诊断分级
- MTMV=(HL500Hz+ HL1000Hz + HL2000Hz)/3×0.9+ HL4000Hz×0.1
- 轻度：26~40dB；中度：41~55dB；重度：≥56dB

**职业性噪声聋鉴别诊断**
-  伪聋：客观测听
- 药物中毒性聋：用药史
- 老年性聋：无噪声史
- 突发性聋：突然发病
- 中耳疾病：鼓膜穿孔、积液

# 6.治疗与预防
- 目前无有效的药物治疗方法
- 加强个人防护，定期复查听力
- 轻、中、重度聋者应调离噪声作业环境
- 对于噪声敏感者（上岗前听力正常，接触噪声1年，任一耳的高频3、4、6kHz任一频率>65dB），应调离噪声环境
- 重度噪声聋者应佩戴助听器

**噪声作业职业健康检查**
- 根据职业健康监护技术规范(GBZ188-2014)的要求，确定上岗前、在岗期间、离岗时职业性噪声检查项目
- 使每位受试者脱离噪声环境12一48h后再行听力筛选测定
- 纯音测听明显异常者应进行复查，复查时间为脱离噪声环境后一周

**预防措施**
- 控制噪声来源，尽量降低噪声强度，用人单位在强噪声作业场所要进行噪声职业危害警示标示和告知
- 减少噪音接触时间。减少连续暴露在噪声环境中的时间，在强噪声下工作时，可每隔一小时，休息数分钟
- 采取个人防护措施，如佩带耳塞、耳罩、隔音帽等防声器材。一般在80dB噪声环境长期工作即应配用简便耳塞，90dB以上时必须使用防护工具
- 争取早期治疗。定期参加职业健康体检，发现问题及时治疗并调离强噪声岗位
