# 1.理化性质
- 原子量200.59，密度13.6
- 熔点/凝固点：-38.87℃，常温呈液态，银白色，又称水银
- 汞导热性能差，而导电性能良好
- 常温可蒸发，汞蒸汽比空气重7倍
- 汞的黏度小，易流动，无孔不入，不易清除
- 蒸气易于吸附衣物、墙壁等
- 不溶于水、有机溶剂
- 汞可溶解多种金属生成合金（汞齐，amalgam)，如金、银、锡、铅等，不包括铁
- 加热汞齐使汞蒸发，即可得到另一纯净金属。用于金属治炼、镀金、镶牙

**汞的无机化合物**
- 亚汞（Hg<sup>+</sup>）
	- 氯化亚汞(Hg<sub>2</sub>Cl<sub>2</sub>，也称甘汞，calomel)，以前曾用过缓泻剂
	- 氧化亚汞(Hg<sub>2</sub>O，merurous oxide)，汞在空气中的氧化产物
- 二价汞(Hg<sup>2+</sup>)
	- 氯化汞(HgCl<sub>2</sub>，也称升汞，sublimate)，消毒剂和化工原料
	- 氧化汞(HgO，也称三仙丹)
	- 硫化汞(HgS，也称朱砂)

**汞的有机化合物**
汞的有机化合物分两类，主要作为农药
- 在体内分解出汞离子的有机汞，毒性与无机汞相似，如烷氧基汞、苯基汞等
- 在体内不易分解出汞例子的有机汞，易透过血脑屏障以原形在脑内长期蓄积不易排出，可引起中枢神经系统损伤，如氯化甲基汞、氯化乙基汞等

**汞的来源——采矿**
- 朱砂(HgS)、氯硫汞矿、硫锑汞矿和其它一些与朱砂相关的矿物是汞最常见的矿藏
- 大约世界上50%的汞来自西班牙和意大利，其他主要产地是斯洛文尼亚、俄罗斯和北美
- 中国汞矿主要分布于贵州东部、湖南西部

**人类生产活动是环境中汞的重要来源**
- 全世界燃煤排入大气的汞为3000吨/年
- 炼制石油、石油燃烧、金属冶炼等均可向自然界排放汞
- WHO估计，每人每天从空气摄入0.06μg，从饮食摄入3μg（其中80%为有机汞化合物，20%为无机汞）

# 2.接触机会
**生产**
- 汞矿开采及冶炼：硫化汞，“火法冶炼”汞产生汞蒸汽
- 化学工业：氯碱行业，以水银电解法制碱
- 有机合成行业：乙炔法生产氯乙烯；以汞为原料生产医药、农药、试剂等含汞的化合物，如升汞、甘汞等
- 仪表行业：温度计、血压计、气压计、流量计等的制造、校验、维修等
- 电气行业：荧光灯、石英灯、电子管、汞电池等生产或维修
- 冶金工业：汞齐法提取金、银等，或汞齐镀金、镀银等
- 国防工业：制作炸药（雷汞）

**生活**
- 含汞偏方（熏蒸、吸入、皮肤涂抹等）
- 汞化合物制成的化妆品
- 银汞齐补牙等
- 环境污染：工业用汞随污水排放进入环境中，经甲基化，引起动物和人中毒

**汞接触的影响因素**
- 汞可以在室温下蒸发，蒸发速率取决于表面积、温度、通风情况
- 如果大量的小的汞滴散落于地面上，蒸发速度加快
- 40℃时，汞的蒸发量是20℃的4倍
- 20℃时，空气中汞浓度达到15mg/m<sup>3</sup>，即达到饱和状态（是职业接触限值0.1mg/m<sup>3</sup>的100多倍）

# 3.吸收、分布和排泄
**吸收**
- 金属汞**主要以蒸气形式经呼吸道**吸收入体内，可很快进入血液，吸收量达75%-100%
- 消化道对金属汞的吸收量很少，约万分之一，除引起腹泻外，其他影响很小，但是对溶解度高的化合物如氯化汞、硝酸汞和有机汞等吸收量大
- 皮肤不易吸收汞（汞化合物穿透皮肤的时间约为8h）
- 皮肤吸收途径
	- 不合格美白化妆品中主要含氯化亚汞（甘汞）、氯化氨汞（白降汞）和氯化汞（升汞），它们均可通过皮肤吸收（大于8h）
	- 动物实验发现含汞油膏中以黄色氧化汞经皮肤吸收率最高，其次为氨基汞、金属汞和甘汞

**分布和排泄**
- 汞进入体内后主要分布在红细胞，大部分被氧化成二价汞，进入血浆
- 二价汞可分布至各组织器官蓄积，在体内含量以肾脏最高，其次为肝、脾、睾丸、甲状腺、脑等
- 甲基汞是脂溶性的，可通过**生物膜**（血脑屏障和胎盘）
- 血浆汞的半衰期2-4天；无机汞在全身的半衰期约为30天，甲基汞半衰期为60天，但脑内汞的半衰期可达数年
- 体内的汞大部分由尿排出，小部分经肺、皮肤、粪便、唾液、胆汁、乳汁排出

# 4.毒性机制
汞主要以Hg<sup>2+</sup>形式发挥毒性
- 抑制许多重要酶的活性，与体内含硫、氧、氮等的基团发生作用，干扰体内的巯基、羧基、羟基、氨基等官能团
- 激活Ca<sup>2+</sup>介导的细胞生化反应，如磷脂酶A<sub>2</sub>激活、产生花生四烯酸、超氧阴离子，引起细胞损伤
- 免疫作用，Hg2+与蛋白结合后由半抗原成为抗原，引起变态反应，出现间质性肾炎或肾小球肾炎；大剂量Hg<sup>2+</sup>直接引起肾小球免疫性损伤
- 可疑致癌性，对DNA的攻击照成单链断裂

# 5.临床表现
- 急性毒性靶器官主要是（根据后果严重性）
	- **肾**（损害**肾近曲小管**，早期重吸收功能障碍）
	- 其次为肺、消化道、皮肤、脑等
- 慢性毒性靶器官主要是（根据后果严重性）
	- **脑**、肾脏及消化道

==**急性中毒**==：多由于短期内吸入高浓度汞蒸汽引起，潜伏期为几小时或1-4周，与吸入的汞蒸汽浓度有关
- **全身症状**：头痛、头晕、乏力、全身酸痛、发热、易兴奋等
- **呼吸系统表现**：出现支气管肺炎而有剧咳、胸痛、呼吸困难等
- **消化道症状**：口腔炎，如流涎、牙龈肿痛、溃疡、牙齿松动等，伴恶心、食欲不振、腹痛、腹泻，严重者出现肝大、肝功能异常
- **泌尿系统症状**：可引起急性肾小管坏死甚至急性肾功能衰竭，表现为蛋白尿、血尿、管型尿及肾小球滤过率下降等，严重者出现少尿、无尿、血肌酐升高等尿毒症表现
- **皮炎**：中毒后1-3天出现，为散在的红色斑丘疹，四肢及头面部较多，严重者可发生剥脱性皮炎
- 口服无机汞盐（氯化汞）主要表现为化学性坏死性胃肠炎，并有急性肾小管坏死，常进展为少尿型急性肾功能衰竭

**慢性中毒**：长期接触较低浓度的汞蒸汽引起，主要表现为：==**易兴奋征、震颤、口腔-牙龈炎（三个特征）**==
- 神经精神障碍：神经衰弱综合征，性格改变，如易激动、抑郁、注意力不集中，甚至出现幻觉。周围神经病表现为以疼痛为主的感觉异常
- 震颤：初见腱反射极度活跃，后手指震颤，可波及眼脸、唇、舌及四肢等，震颤为**意向性**，从事习惯性动作时不明显
- 口腔-牙龈炎：牙龈肿胀、易出血、牙齿松动，流涎多，甚至牙龈溃烂、牙齿脱落、“汞线”（蓝灰色硫化汞沉淀）
- 肾损伤：近端肾小管功能障碍，如低分子蛋白尿、氨基酸尿、糖尿等，高浓度的汞可引起肾小球基底膜免疫性损伤，出现白蛋白尿，甚至肾病综合征表现
- 其他：可见有肝大、汞中毒性晶状体炎和皮炎等
- 先天甲基汞中毒：如孕妇在孕期食用含有较多甲基汞污染的鱼类，可引起儿童先天中毒，严重者出现脑痉挛综合征

# 6.实验室检查
**尿汞测定**
- 正常参考值<35μg/g肌酐(20umol/mol肌酐)
- 尿汞反映近期接触汞的水平，停止接触后持续升高6-8个月
- 尿汞排泄的影响因素较多，慢性中毒难以反映体内的蓄积水平，尿汞水平不与临床症状平行，如尿汞不能反映慢性中毒最重要的靶器官——脑中的汞蓄积程度，必要时以驱汞实验后的尿汞水平反映体内的蓄积状况

**血汞**：半衰期2-4天，是早期汞吸收的良好指标，用于急性中毒程度和病情判断。我国尚无标准

**血汞**：反映近几天汞的接触水平
**尿汞**：反映近期汞的吸收状况
**唾液汞**：反映近期汞的吸收状况
**发汞**：反映常期汞的吸收状况

**三值：正常参考值、职业接触限值、诊断值**

| 检测指标                   | 正常参考值   | 职业接触限值 | 诊断值   |
| ---------------------- | ------- | ------ | ----- |
| 血汞 μmol/L(μg/L)        | 因方法而异   | 因方法而异  | 因方法而异 |
| 尿汞 μmol/mol肌酐(μg/g 肌酐) | 2.25(4) | 20(35) | 不适用   |

# 7.诊断
**急性中毒**：根据职业性汞中毒诊断标准（GBZ 89-2002）
- 有急性大量汞接触史
- 出现消化系统、泌尿系统、皮肤改变为主症状
- 血汞明显升高
- 尿汞早期不升高，如升高有参考意义
- 排除其他类似病因引起的疾病

==**急性汞中毒分级**==
- 轻度：短期内接触大量汞蒸气，血/尿汞增加，可出现发热、头晕、头痛、震颤等全身症状，并具备下列表现之一者，
	- 口腔-牙龈炎及胃肠炎
	- 急性支气管炎
- 中度：在轻度中毒基础上，并具备下列表现之一者，
	- 间质性肺炎 
	- 肾病综合征
- 重度：具备下列表现之一者，
	- 急性肾功能衰竭
	- 癫痫样发作
	- 精神障碍

**慢性中毒**：根据职业性汞中毒诊断标准（GBZ 89-2002）
- 根据接触汞的职业史
- 临床表现（**易兴奋征、震颤、口腔-牙龈炎**）
- 结合尿汞测定结果综合分析
- 参考现场劳动卫生检测结果
- 排除其他类似病因引起的疾病

==**慢性汞中毒分级**==
1. 轻度中毒 具备下列表现之三项者  
    a) 脑衰弱综合征  
    b) 口腔-牙龈炎  
    c) 眼睑、舌或手指震颤  
    d) 尿汞增加
2. 中度中毒 具备下列表现之二项者  
    a) 出现精神性格改变  
    b) 粗大震颤  
    c) 明显肾脏损害
3. 重度中毒 具备下列表现之一项者  
    a) 小脑共济失调  
    b) 精神障碍

**有关汞引起肾损害的问题**
- 文献报道汞引起的肾损害类型
	1. 肾病综合征
	2. 肾小管功能障碍
- 肾病综合征肾小球损害的病理类型
	1. 膜性肾病
	2. 微小病变肾病
	3. IgA肾病

# 8.鉴别诊断
**急性汞中毒**应与急性胃肠炎、急性泌尿系统感染等鉴别
**慢性汞中毒**应与脑血管硬化、震颤-麻痹综合征、甲亢、精神疾患等鉴别

# 9.治疗
- 急性汞中毒  
	- 迅速脱离现场，脱去污染衣物，静卧、补液等
    - 驱汞治疗
	    使用**巯基螯合剂，二巯丙磺酸钠 (2.5-5.0mg/kg, im) 或二巯丁二酸钠 (5.0mg/kg, im)**，急性期可每4~6小时用药一次，根据尿汞排出量决定用药天数，常用5-7天。驱汞治疗应在每日尿量400ml以上时使用。如出现急性肾衰表现，应在血液透析配合下用小剂量螯合剂驱汞
    - 对症治疗
	    - 出现化学性肺炎需吸氧、糖皮质激素等积极治疗
	    - 口服汞不需特殊处理，口服汞盐应尽快洗胃，灌入牛奶、活性炭等吸附，硫酸钠等导泻，并防治肾功能不全
- 慢性汞中毒
	- 驱汞治疗方法：二巯丙磺酸钠(125-250mg, im, qd)，二巯丁二酸钠（1g/次，qd, iv)，或口服二巯丁二酸(0.5g, bid)，3日为一个疗程，停药4天后开始下一个疗程，一般用药3-4个疗程或根据尿汞的排出情况决定是否进行下一疗程的治疗
	- 对症治疗：如镇静安眠等，口腔炎可用2%碳酸氢钠溶液及0.02%洗必泰漱口液交替含漱
	- 轻度汞中毒治愈后可继续从事原工作，中度以上汞中毒应脱离汞作业

# 10.预防
- 改善劳动条件, 提高工艺技术水平, 用无毒或低毒的物质代替汞
- 工作场所应当使用光滑不易被汞吸附的材料，有加水的汞收集槽。汞污染的场所可用碘 (1g/m³) 加酒精熏蒸
- 工人每天工作时应换工作服，工作后应沐浴，每年应健康体检一次（包括尿汞测定）。妊娠及哺乳期妇女应脱离汞作业

**职业接触限值**
- 汞为0.1mg/m<sup>3</sup>
- 有机汞为0.01mg/m<sup>3</sup>

**汞作业职业禁忌症**
- 慢性口腔炎
- 慢性肾疾病
- 中枢神经系统器质性疾病
- 各类精神病

**预防非职业性汞中毒**
- 环境干预：石油和煤炭工业；汞的生产和使用过程中的污染及其三废(废气、废水、废渣)排放
- 行为干预：不乱扔废弃的相机、手表电池，不乱扔荧光灯管(灯泡)
- 生活方式：不滥用化妆品，尽量食用动物链中低级的生物（小鱼小虾的污染物较大鱼少）
