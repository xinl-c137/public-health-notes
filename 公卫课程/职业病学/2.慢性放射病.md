# 1.概述
**辐射效应分类方式**
- 按其作用机理：随机性效应和确定性效应
- 按辐射作用产生损伤效应的客体：躯体效应和遗传效应
- 按照射单次剂量率高低：急性效应和慢性效应
- 按照射方式：外照射和内照射
- 按照机体受到照射的部位：局部照射和全身照射
- 按效应出现时间：早期效应和远期效应

==**慢性放射病 (chronic radiation sickness, CRS)**==：人体在较长时间内连续或间断受到超剂量限值的电离辐射作用，达到一定累积剂量后引起的多系统损害的全身性疾病，常以造血组织损伤为主
- 较长时间：数月-数年
- 超剂量限值：超限值，>20mSv/y
- 累积剂量阈值：>1.5Gy

==**病因**==
- 放射诊断：放射科、骨科、外科、血管造影的医生、技术人员每年接受的照射剂量0.5~1.0R
- 放射治疗：X、r射线、钴源、镭管
- 工业探伤：钴-60、铯-137
- 加速器：X、β射线、高能带电粒子
- 反应堆：核电站

**发生方式**
- 慢性外照射放射病：X、γ射线及中子流在体外照射引起
- 慢性内照射放射病：放射性核素进入体内引起

**发生条件**
- 战时，放射性沾染区内停留过久而又未有效防护的人员
- 平时，长期从事外照射工作未注意防护人员，放射性污染事故、核泄漏事故

# 2.临床表现
症状：神经衰弱症候群、出血倾向
体征：神经系统的变化，皮肤营养障碍，早老、眼晶体混浊

**主要症状**：无力型神经衰弱
数年后出现持续多年；头痛、头昏、食欲减退、睡眠障碍、记忆力减退

**次要症状**
- 慢性放射性皮炎(chronic radiation dermatitis)
	- 手部皮肤粗糙、皲裂、角化过度、指甲晦暗增厚，局部色素沉着
- 放射性皮肤癌
	- 皮肤萎缩、毛细血管扩张、溃疡形成、有多个赘生物
- 骨痛症候群
- 出血
	- 皮肤出血、鼻出血、牙龈出血

**慢性放射病和急性放射病（骨髓型）的比较

| 疾病   | 慢性  | 急性   |
| ---- | --- | ---- |
| 起始时间 | 数年后 | 照后数天 |
| 病程   | 多年  | 2个月  |

**体征**：较少发生，早衰体征、神经反射异常

**实验室检查**
- 外周血象
	- 白细胞数量变化：白细胞降低型、增高型、波动型
	- 血小板和红细胞变化：均减少
- 骨髓象
	- 骨髓增生低下，骨髓示造血细胞减少，脂肪细胞增多
- 染色体畸变分析
	- 外周血淋巴细胞染色体畸变分析——检测辐射损伤较为敏感的指标
	- 如果放射工作者染色体畸变明显增加又没有其他影响因素，应考虑受到一定剂量照射的可能，可作为慢性放射病的参考指标
- 其它功能异常
	- 肾上腺功能↓：血浆皮质醇↓、尿17-羟类固醇和17-酮类固醇↓
	- 甲状腺功能↓：吸碘率和基础代谢率↓；血清T3、T4↓；促甲状腺激素增高
	- 生殖功能↓：男性精子数量↓，畸形精子↑；女性雌激素↓，卵巢功能↓
	- 全身抵抗力下降

**临床特点**
- 起病慢，病程长
- 症状多，阳性体征少
- 出现时间：症状→外周血象→骨髓造血
- 效应与剂量、时间关系密切

# 3.诊断
- 无特异性诊断指标
- 根据照射史、个人剂量档案、受照累积剂量当量、临床表现和实验室检查、结合健康档案进行综合分析，排除其他因素和疾病方能作出诊断
- 进一步进行程度判断：Ⅰ度 / Ⅱ度

**诊断依据**：受照史＞1.5Gy、神经衰弱症候群、外周血细胞、骨髓象、其他功能异常、排除其他疾病

**分度诊断标准**


| 项目                     | Ⅰ度                     | Ⅱ度          |
| ---------------------- | ---------------------- | ----------- |
| 照射史                    | 长期受照史，＞1.5Gy           | 同Ⅰ度         |
| 症状                     | 明显                     | 较顽固，出血      |
| WBC（×10<sup>9</sup>/L） | ＜3.5                   | <3.0，兼PLT↓  |
| BM骨髓象                  | 正常，或↓                  | ↓           |
| 其他功能                   | 肾上腺、甲状腺、免疫、生殖、物质代谢一项异常 | 同Ⅰ度         |
| 预后                     | 可治愈                    | 恢复缓慢。不能返回工作 |

**Ⅰ度**：具备以下各项者可诊断
- 长期射线接触史和/或间断超剂量当量限值照射史，累计剂量当量1.5Gy以上
- 接触射线前健康，数年后出现明显无力型神经衰弱综合症，其症状的消失与脱离及接触射线有关
- 可有出血倾向
- 接触射线后血象经多次动态观察证明造血功能异常  
	- WBC持续在4.0×10⁹/L以下6个月
	- PLT低于8.0×10⁹/L
	- HGB男性：<110g/L，女性：<110g/L
	- 骨髓增生活跃或偏低下，或某一系列细胞生成不良或成熟障碍
- 伴有下列系统一项异常  
	- 肾上腺功能降低
	- 甲状腺功能降低
	- 生殖功能降低
	- 免疫功能降低
	- 物质代谢紊乱
- 脱离射线积极治疗后减轻或恢复

**鉴别诊断**

| 慢性放射病   | 其他疾病           |
| ------- | -------------- |
| 无力型神经衰弱 | 一般神经衰弱症、更年期综合症 |
| 造血组织的改变 | 慢性苯中毒、再生障碍性贫血  |

# 4.治疗
**对症治疗**：镇静安神、生白药、止血、替代疗法

**Ⅰ度**
- 脱离射线
- 中西医结合对症治疗，加强营养
- 前两年，每年检查一次；以后每两年全面检查一次，在此期间根据健康状况，可参加非放射性工作
- 恢复后，再继续观察一年临床确认治愈则不再按外照射慢性放射病Ⅰ度诊断

**Ⅱ度**
- 脱离射线
- 住院积极治疗，全休
- 必要时进行疗养
- 定期随访，1~2年全面复查一次
- 根据恢复情况可考虑参加力所能及的非放射性工作
