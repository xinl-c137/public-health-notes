# 1.概述
==**定义**==：是运用快速、简便的试验、检查或其他方法，将健康人群中那些可能有病或缺陷、但表面健康的个体，同那些可能无病者鉴别开来

**流程**
![[8.筛检-1.概述-20260401.png|322]]

**意义**：一级预防中健康促进、健康保护；二级预防中早期发现、诊断治疗

==**筛检试验与诊断试验的区别**==

|      | 筛检试验              | 诊断试验           |
| ---- | ----------------- | -------------- |
| 对象   | 表面健康的人或无症状的病人     | 病人或筛检阳性者       |
| 目的   | 区分可能患有某病的个体与可能无病者 | 区分病人与可疑有病者     |
| 要求   | 快速、简便，尽可能发现病人     | 复杂、准确，尽可能排除非病人 |
| 费用   | 经济、简单、廉价          | 一般费用较高         |
| 阳性处理 | 进一步确诊             | 及时治疗           |
**筛检例子**：宫颈癌-巴氏涂片、乳腺癌-钼靶X线、前列腺癌-前列腺特异性抗原PSA、肺结核-结核菌素皮内试验、心血管病-胆固醇和高血压、抑郁症-贝氏抑郁量表、听力问题-听力测试、PKU-苯丙氨酸Phe

==**筛检实施原则**==
为了在人群中成功筛查，必须具备以下条件
- 合适的疾病
	- 重大的公共卫生问题：在人群中现患率较高，能对人群健康和生命造成严重危害
	- 疾病的自然史清晰：有足够长的临床前期(preclinical phase)和可被识别的疾病标识
	- 有早期诊断的方法，且早期干预能显著降低死亡率
- 合适的筛检试验
	- 快速且简单：易学习和执行，监管快，结果获得迅速
	- 经济：高成本效益比
	- 安全：最小的不适，对个体几乎无害
	- 易被受检者接受
	- **高可靠性reliability和准确性validity**
- 合适的筛查程序

# 2.筛检试验的评价
## 2.1研究设计
**确定金标准(gold standard)**
**金标准**：当前临床医学界公认的诊断疾病的最准确可靠的方法。活检、手术发现、临床综合诊断……

**原则**：能代表筛检试验可能应用的目标人群
**病例**：不同病情程度、不同病程、典型不典型、有并发症无并发症、治疗未治疗……
**对照**：未患目标疾病，易产生混淆的疾病

## ==2.2筛检试验的评价==
### 2.2.1真实性
真实性/准确性/效度/validity/accuracy
真实值与测量值的符合程度
![[8.筛检-2.筛检试验的评价-20260401.png|345]]
**灵敏度sensitivity**
$\text{灵敏度}=\frac{A}{A+C}\times100\%$
筛检方法能将实际有病的人正确地判为患者的比例（总病人数中筛检检测阳性者占的比例）

**特异度specificity**
$\text{特异度}=\frac{D}{B+D}\times100\%$
筛检方法能将实际无病的人正确地判为非患者的比例（全部非病人数中筛检检测阴性者占的比例）

**假阴性率/漏诊率**
$\text{假阴性率}=\frac{C}{A+C}\times100\%$
灵敏度=1-假阴性率

**假阳性率/误诊率** 
$\text{假阳性率}=\frac{B}{B+D}\times100\%$
特异度=1-假阳性率

**约登指数(Youden's index)/正确指数**
- =（灵敏度+特异度）-1
- =1-（假阴性+假阳性）
### 2.2.2可靠性
精确度(precision)/信度(reliability)/可重复性(repeatability)
指在相同条件下针对同一事物，测量工具（筛查方法）重复测量其结果的稳定程度(stability)
**可靠性指标**
- 计量资料：标准差、变异系数
- 计数资料：一致率、Kappa值

![[8.筛检-2.筛检试验的评价-20260401-1.png|285]]
**一致率**
$\text{一致率}=\frac{A+D}{A+B+C+D}\times100\%$
一致率既可以评价真实性，也可以评价可靠性

**Kappa**
$Kappa=\frac{N(A+D)-(R_1C_1+R_2C_2)}{N^2-(R_1C_1+R_2C_2)}$
$$Kappa = \frac{ ( \text{Percent agreement observed} ) - ( \text{Percent agreement expected by chance alone} ) } { 100\% - ( \text{Percent agreement expected by chance alone} ) }$$
Kappa=（实际观察一致率-机遇一致率)/(1-机遇一致率）
反映扣除偶然因素外两次结果的一致性
取值范围-1~+1
### 2.2.3预测值
![[8.筛检-2.筛检试验的评价-20260401-2.png|346]]
**阳性预测值positive predictive value, PPV**
在所有检测结果呈阳性者中，实际患有该疾病者的比例
$PPV=\frac{A}{A+B}$


**阴性预测值negative predictive value, NPV**
在所有接受检测且结果为阴性的人群中，实际未感染疾病的比例
$NPV=\frac{D}{C+D}$

**阳性预测值、阴性预测值与患病率、 灵敏度和特异度的关系**

- 灵敏度越高，阴性预测值越高；特异度越高，阳性预测值越高
- 当灵敏度和特异度不变时，阳性预测值随患病率的上升而上升

**筛检试验阳性结果截断值的确定**
**假阴性（漏诊）**
以下情况下需要提高灵敏度：
- 早期干预可及
- 早期干预有效
- 疾病预后差，漏诊可能会带来严重后果

**假阳性（误诊）**
以下情况下需要提高特异度：
- 医疗负担：进一步诊断费用太高
- 心理负担
- 预后不严重
- 早期干预效果不理想

**ROC曲线**
ROC曲线/受试者工作特征曲线Receiver Operator Characteristic Curve
==**截断值的确定**==：将病人和非病人的测量值从小到大排序，即可划分一系列截断值。距离坐标轴左上角最近的坐标点可同时满足筛检试验的灵敏度和特异度相对最优，它所对应的取值即最佳截断值
**ROC曲线下面积AUC**：ROC曲线与对角线构成的曲面面积(area under the curve)也可反映检测方法的真实性。AUC越接近1，检测方法真实性越高；AUC=0.5时真实性最低，无应用价值。还可以用AUC直接比较2种以上筛检试验真实性的优劣

# 3.筛检效果评价
## 3.1收益yield
经筛检后能使多少原来未发现的病人（或临床前期患者、高危人群）得到诊断和治疗
**影响因素**
- 筛检试验的灵敏度
- 人群中某病的患病率
- 人群中参加筛检的人数
- 以前筛检的次数

**提高收益的措施**
- 提高试验的灵敏度
- 在高危人群中筛检
- 联合试验

==**联合试验**==
是指采用多个筛查方法去检测一种疾病，达到提高筛查的灵敏度或特异度的目的，以满足筛查对诊断准确性的需要
根据联合的方式，联合试验又分为串联（serial test）和并联（parallel test）两种
- 串联又称序列试验，是指先后采用几项筛查方法去检测疾病，只有全部检测结果皆为阳性者才判为阳性，凡有一项检测结果为阴性者即判为阴性，**提高特异度**
- 并联又称平行试验，是指同时采用几项筛查方法去检测疾病，只要有一项检测结果为阳性者就判为阳性，**提高灵敏度**

## 3.2生物学效果
干预后对结局的影响
- 病死率
- 死亡率
- 生存率

## 3.3卫生经济学评价
**成本效果分析(cost-effectiveness analysis)**
指筛查计划投入的**费用**与获得的**生物学效果**的关系评价，例如对检测出一个早期宫颈癌病例的平均成本与病人的治疗效果进行分析

**成本效益分析(cost-benefit analysis)**
指筛查计划投入的**费用**与获得的**经济效益**的关系评价，例如对检测出一个早期宫颈 癌病例并进行早期治疗的平均成本与晚期发现病人进行治疗的费用进行分析

**成本效用分析(cost-utility analysis)**
指筛查计划投入的**费用**与病人获得的**生活质量的改善**的关系的评价

# ==4.筛检的偏倚==
## 4.1领先时间偏倚lead time bias
**领先时间(lead time)定义**：指通过筛检试验，在慢性病自然史的早期阶段，如症状出现前，提前做出诊断，从而赢得提前治疗疾病的时间

**领先时间偏倚(lead time bias)定义**：指筛检诊断的时间和临床诊断时间之差被解释为因筛检延长的生存时间。这种表面上延长的生存时间，实际是筛检导致诊断时间提前所致的偏倚

## 4.2病程长短偏倚length bias
**定义**：一些恶性程度低的癌症病人常有较长的临床前期，而恶性程度高的同类癌症病人的临床前期较短，因此前者被筛检到的机会较后者大，而前者的生存期又比后者长，从而产生一种筛检者要比未筛检者生存时间长的假象
