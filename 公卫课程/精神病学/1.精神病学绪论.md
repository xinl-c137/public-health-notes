# ==1.精神病学的任务==
- 精神病学是临床医学的一个分支
- 研究各种精神疾病的病因、发病机制、临床特点、疾病发展规律，以及治疗、预防及康复为目的的一门科学

# 2.精神科与其他学科的关系（心-身关系）
- 心理因素是病因——心身疾病
- 心理症状是结果——脑外伤性精神病、患癌症后的抑郁反应
- 躯体症状是表现——躯体形式障碍、分离转换障碍

# 3.精神卫生在生物-心理-社会模式中的地位
- 精神卫生理念应当贯穿于各项医疗服务
- 临床沟通技巧是精神病学教学与培训的重要内容
	- 规范化培训影响医生的日常行为
	- 医疗纠纷中的沟通问题

# 4.精神障碍和精神病的定义
 **精神障碍**：一类具有诊断意义的精神方面的问题，症状须达到一定的严重程度和持续时间，伴有痛苦体验和/或功能损害，表现为认知、情绪、行为方面的改变
 **精神疾病**：精神障碍同义词，不推荐使用
 **精神病**：一类严重的精神障碍。无论属于何种精神障碍分类，只要出现持续的幻觉、妄想、严重精神运动兴奋或抑制表现者。典型如
- 脑和/或躯体疾病所致的谵妄状态
- 酒/药所致的幻觉、妄想状态
- 精神分裂症、其他妄想性障碍等
- 严重的情感障碍

# 5.精神障碍的病因学
**素质因素(predisposing factor)**：早年的、与发病时间远隔的有害因素，形成易感素质
**促发因素(precipitating factor)**：后来的、与发病时间接近的有害因素，与发病直接有关
**附加因素(perpetuating factor)**：疾病发生后附加于个体，使疾病加剧或使病程持续下去的事件
**常见的致病因素**
- 遗传因素
- 素质因素
- 理化、生物性因素
- 机体的功能状态
- 年龄因素
- 性别因素

**精神障碍的发病机制**：生物学因素和社会心理因素综合作用的发病机制

# 6.精神障碍的分类
- 器质性与功能性疾病
	- 存在肯定的结构或组织上病理改变的疾病，为器质性疾病
	- 至今无法确定其是否有病理改变的情况，为功能性疾病
- 精神病(psychosis)与神经症(neurosis)
- 多轴诊断
- 等级诊断
- 共病

# 7.心理健康
**健康**：不仅仅是没有疾病和身体虚弱，而是身体、心理和社会适应的完满状态
**心理健康**：没有精神障碍、良好履行精神功能（对自身和环境的清醒认识）的一种状态

# 8.精神障碍分类的基本原则
**症状学分类的基本原则**
- 不论病因是否相同，根据**共同的主要症状**命名，建立诊断
- 是**状态性诊断**，诊断名称只说明疾病当时所处的状态
- 同一病因诊断的疾病，因不同的症状群出现，可以分解成为几个互不相关的症状学诊断
- 有利于对症治疗

==**精神病性障碍和非精神病性障碍的概念与判断的基本标准**==
- 精神病性障碍：精神分裂症、情感性精神障碍、偏执性精神障碍及各种器质性病变伴发精神障碍
- 非精神病性障碍：各类神经症、人格异常、心身疾病及病情较轻的精神发育迟滞与不够精神病程度的反应状态或情绪反应等
