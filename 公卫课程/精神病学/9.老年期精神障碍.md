# 1.老年期心理卫生
**老年精神病学的重要性与发展趋势**
- 随着我国经济的发展，人民生活水平不断提高，医疗保健事业的日臻完善，人们的寿命不断延长
- 中国60岁以上老年人口比例已接近20%
- 在老年人口中，精神疾病已构成常见多发疾病之一，如老年期抑郁、老年期痴呆等

**老年期心理特点**
- 智力改变
    - 感知觉，心理运动速度，注意
    - 记忆、智力、思维、专长与智慧
- 情绪改变
    - 失落感与孤独感
- 性格改变
    - 健康方面的担心与不安
- 行为改变
    - 兴趣狭窄，过分注意自己的身体

**常见老年期精神障碍**
- 脑器质性精神障碍
    - 谵妄（delirium）
    - 痴呆（dementia）
- 老年期精神病性障碍
- 老年期抑郁（depression）

# 2.老年期痴呆的诊治
## 2.1痴呆概述
**中国痴呆的现状**
- 65岁以上老年人AD总患病率约5.9%，推算我国现有AD患病人数达350多万
- 国际共识研究提示，中国及西太区发展中国家60岁及以上人群中痴呆共识患病率为4.0%，并推测至2040年年发病率将为0.8%
- 每7秒钟出现1个新的痴呆病例

**痴呆（dementia）**
- 痴呆是由慢性、进行性脑部疾病导致的一组综合征
- 以多种高级皮层功能障碍为主要特征，包括记忆、思维、定向、理解计算、学习能力、语言、判断障碍
- 表现为认知功能损害、精神行为症状和人格改变，严重影响工作、生活和社交能力

**常见类型**
- 阿尔茨海默病（Alzheimer’s Disease，AD）
- 血管性痴呆（Vascular Dementia，VaD）
- 其他原因引起的痴呆
    - 药物或酒精中毒
    - 颅内肿瘤
    - 颅脑外伤
    - 颅内感染
    - 营养障碍和代谢障碍等

**危险因素**
- 无法改变：年龄、ApoE-4基因型
- 可以改变：抑郁、高胆固醇血症、糖尿病、头部外伤、心理应激、高血压、吸烟

**痴呆病因**
- 变性：AD、额颞叶痴呆、帕金森及路易体痴呆
- 血管：VD
- 感染：梅毒
- 外伤：拳击者脑病
- 占位
- 中毒
- 代谢
- 缺氧
- 内分泌
- 营养缺乏

## 2.2阿尔茨海默病（Alzheimer’s Disease, AD）
**概述**
- 中枢神经系统原发性变性疾病
- 主要临床相为痴呆综合征
- 老年期痴呆最常见的类型，占总痴呆例数的50%以上
- 仅次于心脏病、恶性肿瘤和中风的老年人第四位死亡原因

**ABC：AD的核心症状**
- Activities of daily living：日常生活能力降低
- Behavioral and Psychological Symptoms：精神行为异常
- Cognition：认知功能障碍

### 2.2.1认知功能障碍（C）
**认知能力下降（C）**
- 记忆力减退
- 注意力不能集中
- 对时间、地点不能分辨
- 思维和判断能力下降，不符合常识

**记忆障碍一AD早期的突出症状**
- 尤其是刚刚发生的事情，不是忘记事件的细节，而是忘记整个事件，经提醒仍不能回忆
- 对事件没有时间线索
- 学习新东西的能力减退
- 花一整天的时间准备晚餐，却忘了拿出来
- 同一样东西买了三遍
- 1小时内同样的问题问3次，每次都忘记答案
- 忘记已经吃过午餐，还不停地吃
- 就算已经打字打了20年，还是忘了怎么打字
- 忘记既往生活经历，重大生活事件
- 对记忆力下降不自知

**视空间和定向障碍**
- 时间：不知道年月日，不知季节，不知上下午
- 地点：不熟悉的地方迷路→熟悉的地方迷路→家里找不到房间和厕所
- 人物：不认识朋友、邻居→不认识远房亲戚→不认识侄儿外女儿媳女婿→不认识女儿儿子→不认识老伴→不认识镜中的自己

**言语障碍**
- 找词困难
- 用词不当
- 说话啰嗦冗赘含糊不得要领——赘述
- 命名障碍
- 言语贫乏
- 能读报，但不理解
- 言语理解困难
- 无法正常说话，会大吼大叫或发出奇怪的声音
- 缄默不语

**失认、失用**
- 不能认识物体、地点和面容
- 不能正确使用刮胡刀，不会开锁，严重者不会使用任何物品和工具，甚至不能使用筷子和勺子吃饭

**全面智能减退**：理解、推理判断、抽象概括、计算等

### 2.2.2精神行为异常（B）
- 1996年IPA专家共识提出痴呆的精神与行为症状（Behavioral and Psychological Symptoms of Dementia，BPSD）
- 定义：痴呆患者经常出现的紊乱的知觉、思维内容、心境或行为症状
- 发生率：70－90%
- 影响患者与照料者生活质量
- 容易成为痴呆患者住院的主要原因

**精神症状**：妄想、幻觉、抑郁心境、淡漠、焦虑

**行为症状**：无目的漫游、激越、反复抱怨、脱抑制、违拗、食欲改变和睡眠节律紊乱

### 2.2.3日常生活能力降低（A）
- 完成日常生活和工作越来越困难
- 吃饭、穿衣、上厕所也需要帮助
- 简单的财务问题也不能处理
- 日常生活需要他人照顾，最后完全不能自理

**日常生活能力**
- 理财、购物、算账能力
- 与人交往能力
- 不能把握时事新闻
- 使用交通工具
- 做家务（准备食物、收拾屋子、洗熨衣服）
- 打电话的能力
- 定时定量服药
- 洗漱、穿衣、吃饭、如厕

**人格改变**
- 懒散、退缩、自我中心、敏感多疑、乖戾自私、不负责任、脾气暴躁
- 言语粗俗、行为不顾社会规范、不知羞耻、不修边幅、不讲卫生、藏匿物品、捡烟头拾破烂视若珍宝

**神经系统症状**
- 早期无明显异常
- 锥体外系症状
- 原始反射
- 癫痫样发作

### 2.2.4痴呆的临床进程
- 第一阶段：轻度痴呆期
    - 记忆减退（近事遗忘）
    - 判断能力下降
    - 不能独立处理事物
    - 对新事物接受难
    - 常有多疑
    - EEG结果正常，MRI显示海马萎缩
- 第二阶段：中度痴呆期
    - 远近记忆受损
    - 简单结构视空间能力差
    - 时间、地点定向障碍
    - 情感淡漠
    - 语言功能异常（失语）
    - 不能独立生活
    - EEG显示背景节律变慢，MRI显示脑室扩大、脑沟增宽
- 第三阶段：重度痴呆期
    - 记忆力严重缺失，片断记忆
    - 智力障碍衰退
    - 生活完全不能自理       
    - 大小便失禁
    - 运动障碍（肢体强直和屈曲体位）
    - EEG显示弥漫性慢波，MRI显示脑室扩大、脑沟增宽

### 2.2.5AD特征性神经病理学改变
- 神经炎性斑块（Neuritic plaque）
- 神经原纤维缠结（Neurofibrillary tangles）
- 选择性神经元及突触缺失（Neuron Loss）

**AD递质系统功能异常**
- 胆碱能缺陷
- 谷氨酸能系统功能亢进
- 其他递质系统

## 2.3血管性痴呆（Vascular Dementia，VaD）
**概况**
- 由脑血管病变导致的痴呆
- 多发梗塞性痴呆（Multi-infarct Dementia）
- 危险因素：与卒中的危险因素类似

**临床表现**
- 临床表现形式常与脑血管病损部位、大小及次数有关
- 主要分为两大类：痴呆症状、血管病脑损害的局灶性症状
- VaD起病急缓不一，有的病人在一次或多次卒中发作后起病，有些病人没有明显的卒中发作
- VaD多为阶梯式发展，一次一次叠加，直至出现全面痴呆
- 早期症状
    - 脑衰弱综合征
    - 轻度认知障碍
- 局限性神经系统的症状体征
- 痴呆症状：局限性

## 2.4痴呆的诊断
**认知功能检查**
- 方式：筛查、系统检查
- 内容：记忆、语言、视空间、注意、执行、计算、运用、定向
- 常用筛查工具
    - 简易精神状态检查（MMSE）
    - 蒙特利尔认知评估（MoCA）
    - 画钟测验
    - AD8

**规范化诊断程序**
- 认知功能下降的主诉
- 排除抑郁、急性疾病
- 病史采集，全面躯体、神经系统检查及恰当的认知测查
- ICD-10诊断标准（以确认痴呆诊断）
- 恰当的辅助检查：神经影像学检查等
- 痴呆亚型的诊断

**痴呆诊断标准**
- ICD-10
- DSM-IV
- NINCDS-ADRDA
- NINDS-AIREN
- CCMD-3

**AD与VaD的鉴别诊断要点**

|      | AD                                   | VaD                                           |
| ---- | ------------------------------------ | --------------------------------------------- |
| 起病   | 隐渐                                   | 较急，常有高血压史                                     |
| 病程   | 进行性缓慢发展                              | 波动或阶梯恶化                                       |
| 早期症状 | 近记忆障碍                                | 脑衰弱综合征                                        |
| 精神症状 | 全面性痴呆<br>判断力、自知力丧失<br>有人格改变<br>淡漠或欣快 | 以记忆障碍为主的限局性痴呆<br>判断力、自知力较好<br>人格改变不明显<br>情感脆弱 |
| 神经系统 | 早期多无限局性体征                            | 存在限局性症状和体征                                    |
| 脑影像学 | 弥漫性脑皮质萎缩，内颞叶萎缩明显                     | 多发梗塞、腔隙梗塞或软化灶                                 |

**Hachinski缺血指数评分表改良版**

| 特征        | 评分  |
| --------- | --- |
| 急性起病      | 2   |
| 脑血管障碍既往史  | 1   |
| 限局性神经系统症状 | 2   |
| 限局性神经系统体征 | 2   |
| CT上的低密度区  |     |

| 多发        | 3   |
AD：0-2；MD：3-4；VaD：5-9

## 2.5痴呆的治疗与照料
**诊断告知与保健方案**
- 痴呆的诊断
- 告知诊断（采取恰当的方式）
- 用朴实的语言解释诊断或病理学特点
- 积极乐观地探讨照料与保健方案
- 保健方案包括药物治疗、心理社会干预、为照料者提供支持
- 定期访视，修改保健方案
- 晚期关怀照料及为照料者提供居丧咨询

**AD药物治疗**
- 胆碱酯酶抑制剂
    - 多奈哌齐（安理申）5-10mg/d
    - 卡巴拉汀（艾斯能）1.5mg Bid－4.5mg Bid
    - 加兰他敏（力益临）4mg Bid－12mg Bid
- NMDA受体拮抗剂
    - 美金刚（易倍申）5mg Bid－10mg Bid

**VaD药物治疗**
- 对VaD危险因素的预防和治疗
- 控制血压和其他危险因素
- 目前尚无特效药治疗VaD
	- 脑血管扩张剂
	- 钙离子拮抗剂
	- 脑代谢赋活剂

**其他药物治疗与BPSD**
- 抗精神病药
    - 精神病性症状（幻觉、妄想）、敌意、攻击、激越、暴力行为、睡眠-觉醒节律紊乱
- 抗抑郁药
    - 抑郁症状、与抑郁相关的激越、易激惹、睡眠－觉醒节律紊乱
- 苯二氮卓类药物
    - 焦虑、激越、紧张、睡眠紊乱
- 抗痉挛药
    - 激越、攻击、敌意、睡眠-节律紊乱、类躁狂行为

**非药物性干预**
- 确定BPSD靶症状
- 收集有关BPSD的资料
- 确定某一特殊症状的关键点与后遗事件
- 确定现实的目标并制定计划
- 鼓励照料者及其他人的目标实现时奖赏他们自己
- 持续性地评估并调整计划

**照料者干预**
- 健康宣教
- 心理支持
- 交流与宣泄
- 缓解负性情绪
- 提高照护质量

# 3.老年期抑郁的诊治
**临床特点**
- 焦虑、抑郁和激越的混合状态
- 兴趣索然
- 精力下降
- 自我评价低
- 自杀观念和行为
- 心境昼夜节律改变
- 躯体或生物学症状
- 认知功能障碍

**老年抑郁症的转归**
- 三分之一会有改善
- 三分之一不变
- 三分之一越来越差

**痴呆与抑郁的鉴别**
 
| 抑郁（假性痴呆） | 真性痴呆         |
| -------- | ------------ |
| 急性起病     | 不知不觉和无明显界限发病 |
| 症状短期存在   | 症状长期存在       |
| 心境持续忧郁   | 心境和行为波动起伏    |
| 常回答“不知道” | 常回答“忘了”      |
| 患者夸大缺陷   | 患者掩盖缺陷       |
| 认知损害程度波动 | 认知损害程度相对稳定   |

**继发性抑郁**
- 在一些常见老年病中，伴发抑郁的情况并不少见
- 帕金森病患者中，45%的病人同时符合重症抑郁或心境恶劣的诊断标准，其中以僵直和运动迟缓为主要表现的病人合并抑郁症的比例比以震颤为主的病人多
- 卒中后会有23%的患者很快出现抑郁，6个月后符合抑郁症诊断标准者高达34%，另有20%有轻度抑郁表现
- 住院癌症病人中，有25%符合抑郁症的诊断标准

**治疗**
- 急性期治疗需考虑
    - 老年人药代学特点
    - 药物的相互作用
    - 老年人生理特点使其对药物的认知功能损害更为敏感
    - 老年人心理社会因素复杂
- 维持治疗

**关注老年患者药代动力学特点**
- 老年患者药物代谢能力下降，对药物的反应更敏感，可能导致药物副作用的增加，故宜减少总治疗剂量
- 老年人常见共患病和多种药物联合治疗，容易出现药物之间相互作用
- SSRIs与其他蛋白结合率高的药联用（如华法林、洋地黄)，出现置换作用，血浆中游离型药浓度升高，使该药作用增强
- CYP-P450酶诱导剂增加SSRIs类药物清除率，影响SSRIs疗效；而CYP-P450酶抑制剂，会降低SSRIs类药物的清除率，血药浓度升高，导致毒副反应

**注重个体化治疗的药物选择原则**
选择药物前，应全面特别注重老年患者的医疗状况的复杂性
- 老年生理改变
- 合并疾病
- 多重联合用药
- 药物不良反应

**基本原则**
- 充分评估与监测
- 确定药物治疗时机
- 个体化合理用药
- 单一用药
- 起始剂量及调整原则

**治疗药物选择**
- A级推荐：SSRIs、SNRI、NaSSA(米氮片)、安非他酮、阿戈美拉汀
- B级推荐：三环类（阿米替林、氯米帕明等）；四环类（马普替林、米安色林），曲唑酮、瑞波西汀
- C级推荐：吗氯贝胺
- 其他药物：黛力新、中草药
