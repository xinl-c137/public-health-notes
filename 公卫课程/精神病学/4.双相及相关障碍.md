	- 进食模式
	- 社交节律

# 2.双相情感障碍的临床识别及诊疗
## 2.1定义及流行病学特征
**定义**：双相及相关障碍(Bipolar or related disorders)，是**发作性**的心境障碍，由躁狂发作、抑郁发作、混合发作及轻躁狂发作以及相关症状所定义
在双相及相关障碍的病程中，躁狂、混合、轻躁狂发作通常与抑郁发作（或一段时期的抑郁症状）交替出现

**流行病学（与抑郁症对比）**

|          | 双相障碍   | 抑郁症             |
| -------- | ------ | --------------- |
| 终生患病率    | 大约1%   | 3-16.8% (3.59%) |
| 性别比(男：女) | 1:1    | 1:2             |
| 一级亲属     |        |                 |
| 双相障碍患病风险 | 10%    | 2%              |
| 抑郁症患病风险  | 20-30% | 20-30%          |
| 平均发病年龄   | 18岁    | 27岁             |
| 复发患者的比例  | 90%    | 80%             |
| 平均发作次数   | 10     | 4               |

## 2.2临床特点及发作类型
**临床特点**
- 双相障碍临床表现复杂，在情绪低落或高涨反复、交替、不规则呈现的同时，常见焦虑、强迫和物质滥用，也可出现幻觉、妄想等症状
- 病程多样：发作性、循环往复型、混合迁延性、潮起潮落式的病程不一而足。间歇期或长或短
- 本质：情绪的不稳定性

==**临床发作类型**==
- 躁狂发作：伴或不伴精神病性症状，社会功能严重受损
	- **持续一周**，每天大部分时间（或经治疗干预而缩短）
	- 核心表现
		- 情感高涨、易激惹
		- 思维加快、夸大
		- 活动增多/主观感受的精力充沛
	- 伴随症状
		- 言语增快、滔滔不绝
		- 睡眠需要的减少
		- 注意力不集中
		- 行为鲁莽、冲动
		- 心境状态的快速变化（即心境不稳）
	- 其他：精神病性症状
- 轻躁狂发作：不伴幻觉和妄想，社会功能受损程度不严重
	- 轻躁狂发作与躁狂发作的区别
		![[4.双相及相关障碍-2.双相情感障碍的临床识别及诊疗-20260517.png]]
- 抑郁发作
	- **持续两周**，每天大部分时间
	- 核心表现
		- 情绪低落
		- 兴趣减退/快感缺失
		- 没有原因的疲劳感
	- 伴随症状
		- 思维或注意的能力降低
		- 自我评价和自信降低
		- 自罪观念和无价值感
		- 精神运动性迟滞或激越
		- 自伤、自杀观念或行为
		- 睡眠障碍
		- 食欲改变/伴体重变化
	- 其他
		- 生物学症状
		- 焦虑症状
		- 精神病性症状
- 混合发作：伴或不伴精神病性症状，社会功能受损
- 其它类型（未特定、环性心境）

## 2.3诊断
**临床评估**
- 临床评估的目的：确认是否存在（轻）躁狂或抑郁症状，及症状严重程度
- 掌握发作及波动情况、持续时间、病程特点
- 探询病因及其他相关的社会、心理或躯体因素，为诊疗提供依据

**评估内容**
- 病史收集
- 体格检查及实验室检查
- 精神检查
- 结构化临床访谈进行诊断
- 急症状况（自杀、攻击风险识别）
- 量表评定与辅助检查

**评估中的注意事项**
- 要全面采集病史：对患者的直接观察多局限于晤谈本身，有必要增加来自于和患者相关的他人报告来丰富评定
- 询问既往史、起病诱因、病程特点、治疗经过、家族史、个人史等，应当综合考虑和情感发作相关的躯体和实验室检查结果

**量表评定与辅助检查**
- 诊断量表
- 筛查量表
	- 他评量表：Young躁狂量表；汉密尔顿抑郁量表；蒙哥马利-艾森贝格抑郁量表
	- 自评量表：32项轻躁狂症状清单；心境障碍问卷；抑郁自评量表；焦虑自评量表

**诊断系统**
- ICD-10
	- 躁狂发作
	- 双相情感障碍
	- 抑郁发作
	- 复发性抑郁障碍
	- 持续性心境障碍
	- 其他心境障碍
	- 未特定的心境障碍
- ICD-11
	- 双相及相关障碍
		- 双相障碍I型
		- 双相障碍II型
		- 环性心境障碍
		- 其他特指的心境障碍
		- 双相及相关障碍，未特指的
	- 抑郁障碍
		- 单次发作的抑郁障碍
		- 复发性抑郁障碍
		- 恶劣心境障碍
		- 混合性抑郁焦虑障碍
		- 其他特指的抑郁障碍
		- 抑郁障碍，未特指的

**诊断系统的变化**：ICD-11心境障碍概述部分相比ICD-10有以下改变
心境障碍（Mood disorders）下分为抑郁障碍和双相障碍。定义不同的心境障碍，是根据心境发作（mood episodes）的特定类型及随时间变化的模式。主要的心境发作类型有：抑郁发作、躁狂发作、混合发作以及轻躁狂发作。心境发作不是独立的、可以被诊断的实体，因此在ICD-11中没有诊断编码，而是作为抑郁障碍和双相障碍主要组成部分。

**双相障碍Ⅰ型与Ⅱ型**
- 双相障碍Ⅰ型（Bipolar type Ⅰ disorder）
	- 是一类发作性的心境障碍，定义为**出现至少1次**躁狂发作或混合发作
	- 躁狂发作是一种极端的心境状态，持续至少1周（或经治疗干预而缩短）
	- 混合发作定义为在至少1周的大多数时间内明显的躁狂症状和抑郁症状混合、或快速交替出现
- 双相障碍Ⅱ型（Bipolar type Ⅱ disorder）
	- 是一类发作性的心境障碍，定义为至少出现1次的**轻躁狂发作** ，**同时至少出现1次抑郁发作**
	- 轻躁狂发作是一种持续的心境状态，症状持续至少数日
	- 既往无躁狂或混合发作

**环形心境障碍**
- 环性心境障碍是一类表现为**持续性的（至少2年的）心境不稳定**，并且在多数时间有轻躁狂症状及抑郁症状
- 轻躁狂症状群可满足或不满足轻躁狂发作的定义性需求，但既往无躁狂或混合发作
- 抑郁症状群的严重程度和持续时间不足以满足抑郁发作的诊断需求
- 这些症状导致显著的痛苦，或重要功能领域的显著损害

**双相与单相抑郁的区别**

|  | 双相 | 单相 |
| --- | --- | --- |
| 躁狂或轻躁狂病史 | 有 | 无 |
| 气质 | 环性 | 心境恶劣 |
| 性别比例 | 相当 | 女性>男性 |
| 起病年龄 | 青少年、20~30岁 | 30~50岁 |
| 发作次数/持续时间 | 频繁、较短 | 较少，较长 |
| 产后发作 | 更常见 | 不常见 |
| 精神病性症状 | 更常见 | 不常见 |
| 精神运动抑制 | 迟滞>激越 | 激越>迟滞 |
| 睡眠 | 睡眠过度>失眠 | 失眠>睡眠过度 |
| 双相家族史 | 高 | 低 |
| 单相家族史 | 高 | 高 |

**误诊漏诊**
- 躁狂/轻躁狂较难识别是误诊漏诊的主要原因
- 对于抑郁发作期就诊的患者和那些没有明确的躁狂或轻度躁狂发作史的患者，双相障碍Ⅰ型和Ⅱ型容易被诊断为单相抑郁
- 双相障碍Ⅱ型定义为该患者从未经历过躁狂发作，由于轻躁狂症状较难识别，更最常被误诊为单相抑郁
- 双相障碍Ⅱ型定义为该患者从未经历过躁狂发作，由于轻躁狂症状较难识别，更最常被误诊为单相抑郁

## 2.4早期识别
早期识别的关键
- 对轻躁狂发作的重视和识别
	- 患者常常忽视自己“轻躁狂”的时候，**家人或朋友的观察**很重要
	- 重视**量表评估**，比如抑郁自评量表、（轻）躁狂评定量表
	- **人格特征**的描述：具有环性人格、情感旺盛型人格特征易患双相障碍
- 充分利用双相抑郁发作时的临床特征
	- 首次抑郁发作的年龄偏低（如＜25岁）
	- 抑郁发作次数多或频繁发作
	- 伴非典型症状概率高（饮食增多、睡眠增多）
	- 抑郁治疗效果不好
	- 自杀相对多见
	- 抑郁发作速度快
	- 双相障碍家族史
- 双相障碍与其他疾病的共病
	- 双相障碍经常与以下疾病相互重叠、相互影响、增加诊断困难：注意力缺陷与多动障碍（ADHD）、焦虑障碍、人格障碍、酒依赖
- 正确认识情感发作伴有的幻觉、妄想等症状
	- 有些患者出现了幻听、妄想，就被误诊为精神分裂症
	- 但需识别这些患者是否情绪症状在先，精神病性症状在后，而且以情绪症状为主要表现？
- 危险因素总结
	- 生物风险因素：双相障碍家族史、明显外向性格、精力旺盛型素质
	- 临床特征：发病早、心境波动不稳、活动增加、睡眠或昼夜节律紊乱、伴幻觉、妄想等、非典型抑郁、难治性抑郁、产后抑郁
	- 共病诊断：注意缺陷与多动障碍、酒滥用或酒依赖、人格障碍

## 2.5病程与预后
**起病**
- 首次躁狂发作多发生于青年期，躁狂和轻躁狂的起病通常较急。躁狂发作是精神科急诊常见情况之一，可在数日内迅速发展至疾病状态
- 抑郁发作起病相对较缓，通常在数天至数周内逐渐发生，前驱期持续数周到数月的焦虑症状

**病程**
- 躁狂发作和混合发作的自然病程大概是数周到数月，平均3个月左右
- 抑郁发作持续时间通常6个月至1年，平均为9个月
- 躁狂和抑郁的发作没有固定的顺序，可连续多次躁狂发作后有一次抑郁发作，或躁狂和抑郁交替发作
- 快速循环发作
	- 快速循环发作是一种特殊的病程形式，是指过去12个月中，至少有4次情感障碍发作，每次发作形式不同，但符合（轻）躁狂发作、抑郁发作或混合性发作标准。两相间常无明显的间歇期，常被看作是双相障碍中的恶性病程形式
	- 双相抑郁发作的患者使用抗抑郁药物治疗有诱发躁狂的风险，并可能增加发作的频度，甚至导致快速循环发作

**预后的影响因素**
- 长期、反复的情感发作，可导致患者人格改变和社会功能受损
- 1/3的双相Ⅰ型障碍患者有慢性症状和明显的社会功能缺损
- 对双相Ⅰ型障碍患者的4年追踪研究发现，病前职业状况不良、酒依赖、有精神病性特征、发作间歇期的抑郁特征、男性与不良预后有关；躁狂发作期短暂、发病晚、无自杀观念、无共病者预后较好

## 2.6治疗
**治疗原则**
- 充分评估、量化监测原则
- 综合治疗原则
- 全病程治疗原则
- 全面治疗原则
- 提高治疗依从性原则
- 优先原则
- 患方共同参与治疗原则
- 治疗共病原则

**全病程治疗原则**
- 急性期（6-8周）：控制症状、缩短病程；药物治疗为主；治疗应充分，并达到完全缓解，以免症状复燃或恶化
- 巩固期（4-6月）：防止症状复燃、促使社会功能的恢复。主要治疗药物剂量应维持急性期治疗水平不变；配合心理治疗十分必要
- 维持期（长期）：防止复发，维持良好社会功能，提高患者生活。密切观察下适当调整药物剂量；去除潜在社会心理不良因素及施以心理治疗

**维持治疗时长**
- 维持治疗的持续时间因人而异
	- 大多数患者需要持续多年的维持治疗
	- 有些患者需要终生维持治疗
- 考虑因素
	- 疾病病程、发作次数及严重程度、住院治疗的次数
	- 对于女性患者来说，决定受孕可能会中断治疗
	- 不可耐受的副作用可能缩短整个维持治疗，但临床医生应该首先尝试找到耐受性较好的备选药物

**药物治疗**
- 心境稳定剂：锂盐、抗惊厥药
- 抗精神病药：SGAs、FGAs
- 抗抑郁剂
- BZs

**双相障碍共病的治疗选择**

| 临床表现         | 一线治疗                           |
| ------------ | ------------------------------ |
| 惊恐障碍         | 丙戊酸盐、加巴喷丁                      |
| 强迫症          | 非典型抗精神病药物                      |
| 人格障碍         | 认知行为治疗或人际关系治疗                  |
| PTSD         | 非典型抗精神病药物、丙戊酸盐                 |
| 酒精滥用         | 认知行为治疗、动机增强治疗、自助组织、丙戊酸盐、托吡酯、锂盐 |
| 肥胖           | 行为治疗                           |
| 偏头痛          | 丙戊酸盐、卡马西平                      |
| ADHD         | 安非他酮、哌醋甲酯                      |
| 神经性贪食障碍、暴食障碍 | 托吡酯、认知行为治疗                     |
| 其他物质滥用       | 认知行为治疗、动机增强治疗、自助组织             |

**药物治疗安全性、监测及处理**
- 治疗依从性
	- 治疗依从性是维持疾病持续缓解的关键，需要引起足够的重视
	- 在临床实践中，尽管医生给患者开具了合理有效的治疗医嘱，但患者病情仍可能转归不良、康复差，患者依从性差可能是最主要的原因之一
- 监测过程及指标
	- 病史
		- 躯体疾病史、烟酒及其他精神活性物质使用史
		- 家族史：尤其注意心脑血管疾病、高血压、脂代谢紊乱及糖尿病
		- 孕育及避孕史
	- 体格检查
		- 腰围及体重指数、血压
	- 实验室检查
		- 全血细胞计数、肝肾功能、血糖血脂、凝血指标（必要时）
		- 心电图等
- 常见药物不良反应及处理
	- 中毒、肾毒性：锂盐
	- 多囊卵巢综合症：丙戊酸
	- 皮疹：拉莫三嗪、丙戊酸、卡马西平
	- 肝功异常：丙戊酸、卡马西平
	- 白细胞减少：卡马西平
	- 代谢综合征、锥体外系反应、QT间期延长：抗精神病药

**物理治疗**：改良电痉挛治疗（MECT）、光照治疗、重复经颅磁刺激（rTMS）、经颅直流电刺激（tDCS）

**心理治疗**：心理治疗、认知-行为治疗、婚姻和家庭治疗、人际社会节奏治疗

**患者康复治疗措施**
- 医院内康复
- 医院外康复
	- 日间住院
	- 家庭康复
	- 社会康复
- 个案管理
- 家庭干预
- 生活及社会技能训练
- 职业康复
