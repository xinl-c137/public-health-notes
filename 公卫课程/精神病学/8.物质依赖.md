# 1.基本概念
**精神活性物质（Psychoactive Substances）/成瘾物质**
- 能够影响人类情绪、行为、改变意识状态，并有致依赖作用的一类化学物质
- 人们使用这些物质的目的在于取得或保持某些特殊的心理、生理状态
- 毒品指具有很强成瘾性，非医疗使用，社会上禁止使用的成瘾物质
- 根据药理特性分类
	- 中枢神经系统抑制剂(depressants)：能抑制中枢神经系统，如巴比妥类、苯二氮䓬类、酒精等
	- 中枢神经系统兴奋剂(stimulants)：能兴奋中枢神经系统，如咖啡因、苯丙胺类药物、可卡因等
	- 大麻(cannabis，marijuana)：大麻是世界上最古老、最有名的致幻剂，适量吸入或食用可使人欣快，增加剂量可使人进入梦幻，陷入深沉而爽快的睡眠之中，主要成分为D9-四氢大麻酚
	- 致幻剂(hallucinogen)：能改变意识状态或感知觉，如麦角酸二乙酰胺(LSD)、仙人掌毒素(mescaline)、苯环利定(PCP)、氯胺酮(ketamine)
	- 阿片类(opioids)：包括天然、人工合成或半合成的阿片类物质，如海洛因、吗啡、鸦片、美沙酮、二氢埃托啡、哌替啶(杜冷丁)、丁丙诺啡等
	- 挥发性溶剂(solvents)：如丙酮、汽油、稀料、甲苯、嗅胶等
	- 烟草(tobacco)

**精神活性药物（Psychoactive Drugs）**
- 对机体精神神经功能具有明显影响，具有一定致成瘾潜能，用于疾病的预防和/或治疗的化学物质
- 包括阿片类镇痛药及其复方制剂、镇静催眠药及其复方制剂、中枢兴奋药及其复方制剂和某些静脉麻醉药等
- 都具有精神活性，本身均是药物，有合法的产、供、销和用的渠道

**毒品**
- 对机体精神神经功能具有明显影响，具有强大致成瘾潜能
- 且能在社会上引起广泛严重滥用，造成严重的社会、公共卫生和经济问题
- 且与所在国家现行法律和文化传统相抵触的精神活性物质

**滥用（abuse）**
- 滥用：在ICD-11分类系统中称为有害使用(harmful use)，是一种适应不良方式，由于反复使用药物导致了明显的不良后果，如不能完成重要的工作、学业，损害了躯体、心理健康，导致法律上的问题等。滥用强调的是不良后果，滥用者没有明显的耐受性增加或戒断症状，反之就是依赖状态
- 低风险饮酒限量：每天饮酒不超过20克纯酒精（2个标准杯），每周饮酒不超过5天（每周至少2天不饮酒）

**成瘾（addiction）**：与依赖基本同义。从行为角度看，主要表现为失控，具有以下特征
- 成瘾者有做某种行为的强烈欲望，但其结果有害
- 如果控制不做，则紧张、焦虑逐渐增加
- 一旦完成此行为，则紧张、焦虑迅速、暂时得以解脱
- 过一段时间后，如几小时、几天或几周又重新出现实施此行为的欲望
- 外部、内部环境刺激可条件反射性引起此欲望
- 成瘾者希望能控制此行为，但屡屡失败

**依赖(dependence)综合征**
- 依赖：是一组认知、行为和生理症状群，使用者尽管明白滥用成瘾物质会带来问题但仍然继续使用。自我用药导致了耐受性增加、戒断症状和强制性觅药行为
- ==传统上将依赖分为躯体依赖和心理依赖==
	- 躯体依赖(physical dependence)：也称生理依赖，它是由于反复用药所造成的一种病理性适应状态，主要表现为耐受性增加和戒断症状
	- 心理依赖(psychological dependence)：又称精神依赖，指物质使吸食者产生一种愉快满足的或欣快的感觉，驱使使用者为寻求这种感觉而反复用药，表现出所谓的渴求状态(craving)

**不同类别成瘾药物的依赖性特征各异**
- 大部分成瘾物质同时兼具身体依赖性和精神依赖性，例如阿片类药物、镇静催眠药物、酒精等
- 少数药物仅有精神依赖性，如致幻剂
- 也有药物精神依赖性明显而身体依赖性很轻微，如可卡因、大麻等

**强制性觅药行为(compulsive drug seeking behavior)**：是指使用者冲动性使用药物，不顾一切后果，是自我失去控制的表现，不一定是人们常常理解的意志薄弱、道德败坏的问题

**耐受性tolerance**
- 反复使用成瘾物质后，机体变得不敏感，使用原来剂量达不到所需效果，个体需要获得原有药理作用或心理体验，不得不增加剂量，有的表现为使用方式的改变
- 耐受性产生的速度和程度不同
- 耐受性的可逆

**戒断withdrawal综合征**
- 指停止使用药物或减少使用剂量或使用拮抗剂占据受体后所出现的特殊的、令人痛苦的心理生理症状群
- 再次使用可缓解症状
- 不同物质所致戒断反应不同
- 其机制是由于长期用药后，突然停药引起适应性的反跳
- 酒精戒断：心慌、血压升高、发热、出汗、手抖、无力、腹泻、癫痫发作、幻觉、震颤谵妄等

**复发**：指物质依赖者在脱毒治疗完成，保持了一段时间的戒断状态以后，又因为种种原因再次使用脱毒前所滥用的成瘾物质的过程

**物质依赖**
- 又称药物成瘾（Drug Addiction）：反复由于使用精神活性物质而出现依赖综合征
- 主要特点：强烈渴求、耐受性增加、停用后出现戒断症状、以成瘾物质为中心、不计后果、无法控制的使用

==**物质依赖与疾病和道德的关系**==
- 物质依赖是慢性、进行性和复发性的大脑疾病，常与其它躯体及精神疾病伴发
- “物质”可改变大脑的结构和功能，所以依赖是大脑的疾病，大脑的这些改变是持久的，可以导致有害行为的发生（失去控制、强制性使用和负性情绪）
- 是可以治疗的疾病！不是道德问题，不是习惯问题


**成瘾性物质的依赖性**

# 2.病因和发病机制
**生物因素**
- 代谢速度：代谢速度的不同，对精神活性物质的耐受性就不同，依赖的易感性也不同。如天生缺乏乙醛脱氢酶的个体，饮酒后乙醇代谢成乙醛，但乙醛不能继续转化为乙酸，乙醛堆积，导致出现严重的不良反应，从而阻止个体继续饮酒，也就不太可能成为酒依赖者
- 遗传学因素：大量有关酒精与药物依赖的遗传或家族性研究已证明，动物对某些药物依赖的形成具有显著的遗传性
	- 如不同品系的小鼠对吗啡依赖的形成具有显著差异，有些品系的鼠极易造成阿片类依赖的动物模型，而有些品系则很难
	- 家系、双生子及寄养子研究均发现，药物滥用的易感性因素是由基因所决定的。目前发现有两个途径将这一易感性从上一代传至下一代，一是直接遗传的酒精/药物依赖易感性，另一个是间接的方式，将反社会人格传给下一代
	- 家系研究表明，药物依赖或滥用家系成员中，药物滥用、酒精滥用、反社会人格、单相抑郁的相对危险性分别为对照家系的6.7、3.5、7.6和5.1倍

**心理因素**
- 吸毒者有明显的个性问题，如反社会性、情绪控制较差、易冲动、缺乏有效的防御机制、追求即刻满足等
- 行为理论
	- 正性强化（快感）
	- 负性强化（戒断时的折磨）
	- 社会性强化（团伙影响）
- 习惯性奖赏（中脑边缘通路，成瘾物质激活中脑边缘多巴胺系统）与强化导致依赖行为定型

**社会因素**
- 药物的可获得性
- 家庭环境，如家庭矛盾、单亲家庭、家庭成员间交流差、家庭成员犯罪吸毒是吸毒、特别是青少年吸毒的重要危险因素
- 整个社会或地区的稳定程度
- 同伴影响、同伴间压力等
- 社会文化背景的差异、公众对物质的态度

**大脑奖赏系统**
- 脑内主要的三条DA通路：黑质纹状体通路、中脑边缘通路和中脑皮质通路
- 食物、性、成瘾物质激活奖赏系统，产生愉悦感
- 神经生化基础-多巴胺（DA）
	- 多巴胺能够调节人类的动机和学习功能、情感、认知以及快乐感
	- 成瘾物质都有一个共同特性，就是服用后都能引起伏隔核区域DA神经元末梢释放DA增加，从而引起药物的奖赏效应
	- 成瘾物质激活中脑边缘多巴胺系统（MLDS）是奖赏、强化等共性效应的基础，也是形成条件信号的关联、诱导渴求或者觅药行为的神经基础
- 自然奖赏和药物奖赏

**成瘾者体验**
- 饮酒：酒能给我带来我需要的轻松、愉悦，这是别人难以想象的，这种感觉就像是一个陪伴了我二十年的恋人，至今依然爱至铭心刻骨。在我不快乐的时侯，我会像飞蛾投火一样扑向酒
- 吸毒：我吸后感觉前20年真是白活了，20年的生活都太不值得一提了

**对个人的危害**
- 身体（生理）方面：各种躯体疾病、感染、艾滋病、中毒死亡、寿命下降等
- 心理及行为方面：情绪问题、行为异常、人格改变等

**对家庭及社会的危害**
- 严重破坏家庭
- 职业功能损害
- 违法犯罪

# 3.常见精神活性物质所致的精神和行为障碍
**流行病学**
- WHO 2018饮酒和健康全球状况报告：中国酒精使用障碍12个月患病率为4.4%，酒依赖为2.3%
- 2019年发表的黄悦勤团队中国流行病学调查：酒精滥用12个月患病率为1.15%，酒精依赖为0.69%

**疾病负担**
- 《柳叶刀》公布2010年全球疾病总负担排行，从1990年至2010年的20年间，在所有疾病风险因素中，饮酒已由原先的第六位升至第三位，仅次于高血压和吸烟及吸二手烟
- 在全球15-40岁年龄组人群中，饮酒在疾病总负担排行中居第一位；在50岁以上人群的疾病总负担排行中，饮酒居第三位

**酒依赖的危害-躯体损害**
- 消化道疾病：肝硬化、溃疡病、肾硬化、胃炎、胆囊炎、胰腺炎、贫血、结核
- 心血管疾病：心肌病、心律失常、冠心病、高血压
- 神经系统疾病：末梢神经炎、癫痫、小脑变性、脑出血、痴呆、脑外伤致残疾
- 代谢及泌尿生殖系统：糖尿病、痛风、甲状腺功能障碍、前列腺肥大、胎儿畸形

**酒依赖的危害-心理损害**
- 人格改变：自私、撒谎，甚至偷窃和诈骗，丧失对家庭和社会的责任感
- 情绪问题：抑郁障碍、双相情感障碍、焦虑
- 自杀：慢性酒中毒中6%-20%的人有自杀行为

**酒依赖的危害-社会损害**
- 刑事犯罪：我国因打架斗殴被拘役教养或判刑的青少年中，63％的人与酗酒有直接关系
- 经济损失：我国有酗酒者3亿多人，每人每年酒耗量已达4.7/L，消耗粮食300亿公斤，一年喝掉的酒资就要消费近200亿元
- 交通事故：酒驾、醉驾
- 家庭问题：家庭不和、贫困、家庭暴力、离婚

**病因和发病机制**
- 酒依赖的病因及发病机制非常复杂
- 一般认为是生物因素（生化、代谢、遗传等）、心理社会因素（包括文化、环境）等多种因素相互作用的结果

**酒所致精神障碍的临床表现**
- 急性中毒
- 酒精依赖
- 戒断综合征
- 震颤谵妄
- 酒精所致精神障碍
- 认知功能损害
	- Wernick脑病
	- Korsakoff综合征
	- 酒精相关的痴呆

**饮酒后表现**
- 甜言蜜语-豪言壮语-胡言乱语（少言寡语）-不言不语
- 兴奋期：50mg/dl，健谈、情绪不稳定、自负、有粗鲁行为和攻击行为，或沉默、孤僻
- 共济失调期：150mg/dl，行动笨拙、步态不稳、言语含糊、眼球震颤、视力模糊、复视、共济失调
- 昏迷期：250mg/dl，昏迷、昏睡、瞳孔散大、体温降低
- 死亡：一般人酒精致死量为5-8g/kg。血中酒精浓度400mg/dl以上，呼吸、循环麻痹而危及生命

**急性中毒**
- 急性酒精中毒（急性醉酒）：多见。一次大量饮酒后出现的反常行为表现：行为脱抑制，冲动行为，判断力下降，共济失调，言语不清，水平眼震，记忆丧失
- 病理性醉酒：少量饮酒后出现中毒症状，常以深睡结束
- 酒醒后状态：头痛、头晕、恶心、乏力、震颤

**酒精依赖特点**
- 精神依赖：渴求，将饮酒看成生活中最重要或非常重要的事
- 躯体依赖：停酒或减量使出现戒断症状，以酒当药，用酒来解除情绪困扰
- 耐受性：饮酒量逐渐增加
- 多次戒断，多次复发

**戒断综合征**
- 肢体震颤、激越静坐不能、恶心、呕吐、大汗、易激惹、感知异常、焦虑、失眠、幻觉
- 严重者震颤谵妄，癫痫大发作，可致死，应积极治疗

**震颤谵妄**
- 震颤谵妄：又称撤酒性谵妄或戒酒性谵妄，为一种急性脑综合征
- 常见于临床戒酒过程中及精神科急诊
- 常发生于戒酒后48-96小时内
- 发生率5%
- 自然病程3-5天，一般不超过2周

**震颤谵妄的危险因素**
- 饮酒量大
- 依赖时间长
- 肝损害严重
- 既往有震颤谵妄史或癫痫史者
- 合并急性躯体疾患，如感染、低钾血症
- 老年人
- 同时使用其他成瘾物质
- 戒断症状重

**震颤谵妄的临床表现**
- 意识模糊，时间地点定向错误，近记忆困难
- 幻觉妄想(幻觉多为恐怖性)，恐惧不安
- 粗大震颤
- 植物神经功能障碍(出汗、心跳快、血压高、瞳孔扩大)
- 脱水，电解质紊乱，肝功异常

**震颤谵妄的转归**
- 若治疗及时没有并发症，可以完全恢复
- 若病人机体状况较差或合并严重并发症则病死率会明显增加
	- 可达25－50％
	- 死于肺炎、肝功能不全、肾功能不全或心力衰竭等

**酒精中毒所致精神障碍**
- 使用酒精期间或之后出现的精神现象
	- 典型在1月内至少部分缓解，6月内痊愈
- 表现
	- 精神分裂症样
	- 以妄想为主
	- 以幻觉为主
	- 以抑郁症状为主
	- 以躁狂症状为主
	- 科萨可夫综合征

**维尼克（ Wernicke）脑病**
- 少见
- 表现：意识障碍，共济失调、眼球震颤或动眼麻痹、情感淡漠，记忆障碍
- 维生素B1缺乏所致
- 死亡率较高应积极救治，im补充维生素B1，100-200mg/d

**柯萨科夫综合征(Korsakoff Syndrome)**
- 由柯萨科夫于1889年首次描述
- 表现为：定向障碍，近事遗忘，虚构、错构等
- 可由有震颤谵妄过渡而来

**酒相关性痴呆**
- 长期(一般5年以上)或大量饮酒引起的脑器质性障碍，急性或慢性进行性人格改变、智力低下、记忆力障碍的痴呆状态
- 表现为认知功能受损(记忆，思维，定向，理解，计算，学习，语言，判断，情绪控制，社交行为等）
- 慢性脑综合症

**酒依赖治疗和康复**
- 早期评估，了解危害，激发戒酒动机
- 处理戒断：苯二氮卓类药物替代治疗
- 大剂量B族维生素、对症支持
- 抗精神病药物治疗：处理焦虑抑郁、激惹等精神症状
- 戒酒治疗：戒酒硫，纳曲酮等
- 心理行为治疗
- 康复：必不可少的一步，如戒酒者匿名互助会（AA）

**震颤谵妄的治疗**
- 详细的躯体检查至关重要
- 大量的苯二氮卓类药物，以控制病人的兴奋和可能发生的癫痫
- 抗精神病药如氟哌啶醇控制兴奋，但抗精神病药物降低惊厥阈
- 躯体对症治疗：酸碱平衡、水电解质紊乱等

**其他成瘾物质**
- 阿片类
- 镇静催眠药
- 烟草
- 致幻性ATS类

**阿片的药理作用**：镇痛、抑制呼吸中枢、抑制咳嗽中枢、缩瞳作用、抑制胃肠蠕动、致欣快作用

**阿片类戒断综合征**
- 停药8-12小时出现，用药间隔短者出现早
- 最初表现：哈欠、流泪、流涕、出汗等类似感冒症状
- 随后陆续出现
	- 瞳孔扩大、打喷嚏、起鸡皮疙瘩、寒战
	- 厌食、恶心呕吐、腹绞痛、腹泻
	- 全身骨和肌肉酸痛，肌肉抽动
	- 软弱无力、失眠、易醒、心跳加快、血压升高
	- 情绪恶劣、易激惹、烦躁不安、抑郁、甚至出现攻击行为
- 情绪恶劣、易激惹、烦躁不安、抑郁、甚至出现攻击行为
- 36-72小时之间达高峰，大部分症状在7-10天内消失

**治疗和康复原则**
- 不能仅仅着眼于生理症状，而要基于生物-心理-社会医学模式进行全面考虑
- 其完整的治疗应包括脱瘾、康复、后续照管等相互结合的阶段与步骤

**脱瘾(脱毒)治疗**
- 是指采用药物或其他方法使依赖者顺利度过急性戒断反应期
- 常用的药物有
	- 阿片受体激动剂（以美沙酮为代表）
	- 阿片受体部分激动剂（以丁丙诺啡为代表）
	- 非阿片类药物（如可乐定、洛非西定）
	- 中药

**康复和后续照管**
- 康复是指采用心理社会措施矫正个体的行为及态度，使之能够重返社会，变成社会所接纳的人
- 康复性治疗组织：治疗集体（Therapeutic Community, TC）
- 在个体回归社会后，建立一套监督、扶持系统，对曾经吸过毒的人提供心理或专业辅导，并提供其他方面的支持和帮助

**预防**
- 限制依赖性药物的易获得性
- 预防性教育，将有关知识告诉公众。方式多种多样，涉及各阶层、各角落
	- 教育青少年从小树立抵抗毒品的意识，不吸烟、不酗酒
	- 提高医务人员的认识，合理使用麻醉药品和精神药品，避免医源性依赖；同时能够及早识别、及早干预
- 需要全社会多学科、多部门的配合，综合治理

**镇静安眠药依赖**
- 戒断综合征的表现
	- 自主神经功能亢进(如出汗、脉搏加快)
	- 双上肢震颤
	- 失眠，恶心或呕吐
	- 一过性视、听或触幻觉，一过性错觉
	- 精神运动性激越、焦虑
	- 癫痫发作。谵妄状态
- 治疗要点：长效替代短效，逐渐减量，先快后慢

**烟草**
- 尼古丁的药理作用
	- 尼古丁是烟草成瘾性成分
	- 尼古丁作用于中脑边缘系统，产生强化效应
- 吸烟的危害：多系统疾病相关
- 治疗
	- 原则：预防为主、个体化、系统化、综合治疗
	- 药物：尼古丁替代、安非他酮、伐尼克兰

**致幻性ATS类**
- 药理作用
	- 中枢神经兴奋作用
	- 致欣快作用
- 急性中毒
	- 中枢神经系统的兴奋
	- 交感神经系统的兴奋
	- 常见症状之一：恶性高热
- 急性中毒的处理
	- 对症处理，基础支持
