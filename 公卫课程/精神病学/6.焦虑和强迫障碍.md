# 1.焦虑障碍
**焦虑障碍的分类**
- 正常焦虑
- 病理焦虑
	- 器质性
	- 心因性
		- 与情境无关
			- 惊恐障碍
			- 广泛性焦虑障碍
		- 与情境有关
			- 社交恐惧症
			- 广场恐惧症
			- 特定恐惧症
			- 分离焦虑障碍
			- 选择性缄默症

==**正常焦虑与病理性焦虑**==

|      | 正常焦虑           | 病理性焦虑                                        |
| ---- | -------------- | -------------------------------------------- |
| 强度   | 与威胁程度相称        | 过度反应                                         |
| 持续时间 | 威胁解除后缓解或消失     | 相对持久                                         |
| 造成影响 | 不影响社会功能，甚至提高表现 | 自主神经紊乱/运动性不安症状<br>导致精神痛苦，继发酒精或物质滥用<br>自我效能下降 |

**几种焦虑障碍的主要特征**
- 特定恐惧症：对**特殊物体或处境**感到极度恐惧、焦虑及回避
- 惊恐障碍：反复出现的**惊恐发作**，发作间期伴有行为改变
- 场所恐惧症：害怕置身于无法逃离或难以获得帮助的情境，伴有行为改变
- 广泛性焦虑：对日常生活多方面的过度紧张或担心
- 社交恐惧症/社交焦虑障碍：对可能被他人审视的社交处境感到极度恐惧、焦虑及回避
- 选择性缄默：无论语言能力如何，在期望说话的情况下始终无法说话
- 分离性焦虑：恐惧与家庭/主要依恋对象分离，害怕他们发生不好的事情，失去依恋

**惊恐障碍(panic disorder)**
**定义**：反复、不可预料的急性焦虑发作(panic attack)，持续几分钟或更久，心悸、胸痛、窒息、头晕、濒死感

**广泛性焦虑障碍(generalized anxiety disorder)**
**定义**：广泛、浮游的焦虑(free-floating anxiety)，一种慢性的焦虑障碍，因生活中各种不同事情无法控制、过度的担心
**临床表现**
- 情绪：担心、紧张、不安、忧虑、害怕、恐惧、无助、感到威胁、易激惹等
- 思维：非现实性评估自身或他人遇到的危险；无法应对，反复思考
- 行为：运动性不安（无目的小动作），回避可能增强不安全感的处境
- 躯体：唤起增加，过分警觉，对声音、光线等反应增加；过度换气；肌肉紧张，疲劳、疼痛，僵硬，颤抖；自主神经活动增加；心动过速，发红发白发热，口干、腹泻、出汗、尿频等；失眠、疲劳等

**广场恐惧症(Agoraphobia)**：对空旷场所、拥挤的人群、公共交通的恐惧

**特定恐惧症(specific phobia)**：对动物、高处、注射、血液的恐惧

**社交焦虑障碍(social anxiety disorder)**：对社交场所和当众发言的恐惧，担心他人的负面评价（社交恐惧症social phobia）

# 2.强迫性障碍obsessive-compulsive disorder, OCD
**强迫症状**：反复或重复出现的想法或行为，明知不合理或过分但难以控制和摆脱，需要通过重复或反复来确认减轻痛苦

**特点**
- 反复出现的观念和行为
- 没有现实意义而又摆脱不掉
- 能意识到是自我的心理活动

**临床表现**
- 强迫思维：反复的、持久的、侵入的和不必要的想法（如与污染有关）、表象（如暴力场景）、冲动/渴望（如刺伤某人），通常伴明显焦虑
	- 强迫怀疑，即患者对自己言行的正确性反复产生怀疑；明知毫无必要，但又不能摆脱
	- 强迫性穷思竭虑，即患者对日常生活中的一些事情或自然现象寻根究底，反复思索，明知缺乏现实意义，不能控制
	- 强迫联想，即患者脑子里出现一个观念或看到一句话，便不由自主地联想起另一个观念或语句
	- 强迫表象，即患者在头脑里反复出现生动的视觉体验（表象），并常具有令人厌恶的性质，无法摆脱
	- 强迫冲动，即患者反复体验到想要做某种违背自己意愿的动作或行为的冲动，明知不可能的，努力控制，但无法摆脱
- 强迫行为：外显的重复行为（如洗手，排序，检查）或内隐的心理活动（如祈祷，计数，反复默念字句)，是患者感觉被强迫思维驱使的执行反应，或需要达到“完美”的感觉
	- 强迫检查，即患者为减轻强迫性怀疑引起的焦虑采取的措施。如出门时反复检查门窗是否关好
	- 强迫清洗，即患者为了消除受到脏物、毒物或细菌污染的担心，常反复洗手、洗澡或洗衣服
	- 强迫询问，即患者为了消除疑虑或穷思竭虑带来的焦虑，常反复要求他人不厌其详地给予解释或保证
	- 强迫性仪式动作，即一些重复出现的动作，他人看来是不合理，但却可减轻或防止患者强迫思维引起的紧张不安
	- 强迫性迟缓，即患者为了避免出错或者不安，需要反复在脑海中确认下一步需要怎么做而导致行为显得迟缓
- 个体常试图忽略或抵制强迫思维，或通过强迫行为来中和它们；强迫行为是为了应对强迫思维而被迫执行的，目的在于防止或减轻焦虑或痛苦，防止某些可怕的事件或情境发生

**鉴别诊断**
- 器质性疾病
	- 躯体疾病伴发焦虑
	- 躯体疾病的症状与焦虑重叠，比如甲亢、冠心病、嗜铬细胞瘤、低血糖、癫痫
- 物质依赖
	- 安非他明、摇头丸、可卡因和致幻剂的使用和停用会导致严重焦虑，包括惊恐发作
	- 阿片剂、酒精、巴比妥类药物、苯二氮卓类药物突然停用会导致焦虑、激越、震颤、失眠
- 精神分裂症
	- 深陷令人不快的幻觉和妄想而感觉到紧张和焦虑
- 抑郁症
	- 焦虑障碍/强迫障碍和抑郁症的共病非常常见
	- 症状重叠：注意力不集中、易激惹、睡眠障碍、疲劳感
- 其他精神障碍
	- 应激障碍、进食障碍、躯体不适障碍
	- 澄清个体焦虑和恐惧的焦点、全面评估个体的精神活动

==**生物-心理-社会模型/4P因素**==

|      | 生物            | 心理           | 社会                     |
| ---- | ------------- | ------------ | ---------------------- |
| 易感因素 | 阳性家族史         | 焦虑素质<br>人际敏感 | 寄予厚望<br>管教严格           |
| 诱发因素 |               | 害怕负面评价       | 初中被嘲笑<br>大学环境陌生、社交需求增加 |
| 维持因素 |               | 学业和人际压力      |                        |
| 保护因素 | 身体健康<br>学习能力好 |              | 家庭支持尚可<br>没有严重创伤       |
