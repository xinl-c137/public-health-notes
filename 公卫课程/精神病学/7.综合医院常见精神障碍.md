# 1.概述
**综合医院精神障碍：高发生与低识别**

**问题严重：恶性循环与处理不当**

**会诊-联络精神病学（C-L Psychiatry）**
- 联络（Liaison）
	- 指定人定期派出精神科医生驻在友科，提供督导、管理和诊治病人
- 会诊（Consultation）
	- 临时应他科邀请处理个案，针对具体患者进行评估和建议

**心身医学：四种并存情况**
- 心身疾病：心理社会因素作为躯体疾病的重要病因（如高血压、消化性溃疡）
- 躯体化障碍：精神障碍表现为躯体症状（如神经症性障碍中的躯体化）
- 躯体→精神：躯体疾病导致精神障碍（如脑外伤性谵妄）
- 精神障碍+躯体疾病并发：两者独立并存（如抑郁症伴发肝炎）

==**躯体疾病所致精神障碍的共同临床特点**==
- 病因确凿性：存在明确的躯体疾病、脑部病变或系统性代谢毒性物质的生物学证据
- 时间联系：精神症状的发生、波动及缓解与原发躯体疾病的病程呈严密的高度平行关系
- 非心理介导性：不是患者对躯体疾病的心理反应（如适应性抑郁），而是病理生理直接作用于中枢
- 日轻夜重：精神症状多数有日轻夜重特点，这是鉴别器质性vs功能性精神障碍的重要线索
- 同病异相/异病同相：同一躯体疾病可诱发不同精神症状；不同躯体疾病可引起极为相似的综合征
- 病程关联：精神障碍与躯体疾病平行发展，其病程、预后与原发躯体疾病的性质、严重程度密切相关

**病因与发病机制**
- 主要因素（必备）：原发疾病是发病的必备、主要因素
	- 脑外伤、感染、中毒
	- 代谢与内分泌紊乱
	- 心脑血管疾病
	- 自身免疫性疾病
	- 营养缺乏
- 素质因素
	- 精神病遗传史
	- 人格缺陷
	- 神经系统功能状态
	- 年龄（老年/儿童高危）
	- 酒药依赖史
	- 躯体状况差
- 诱发因素
	- 应激（手术、创伤、丧亲）
	- 睡眠剥夺
	- 感觉剥夺（ICU环境）
	- 环境改变（住院、转科）
	- 药物变更
	- 疼痛控制不良

**病理生理机制**：从脑组织到神经递质
- 免疫炎性瀑布效应
- 微循环与内皮功能障碍
- 神经递质失衡
- 脑实质直接浸润或受损

**诊断标准**
- 症状标准
	- 躯体、神经系统及实验室检查证据
	- 有脑病/脑损伤/可引起脑功能紊乱的躯体疾病 + 至少下列1项：
		- ①智能损害②遗忘综合征③人格改变④意识障碍⑤精神病性症状⑥情感障碍⑦解离(转换)综合征⑧神经症样综合征
- 严重标准：日常生活或社会功能受损
- 病程标准：精神障碍的发生、发展及病程与原发器质性疾病相关
- 排除标准：缺乏精神障碍由其它原因引起的足够证据

**综合医院精神症状综合征总览**
- 神经症样症状
	- 多见于躯体疾病初期、恢复期
	- 类似神经衰弱表现
	- 疲劳、失眠、注意力不集中
	- 易激惹、情绪不稳
- 谵妄
	- 急性、广泛性认知障碍
	- 意识障碍为主要特征
	- 日轻夜重、病情波动
- 精神病性症状
	- 发生于疾病过程中
	- 表现为幻觉、妄想状态
	- 需与功能性精神病鉴别
- 情感症状
	- 抑郁状态、焦虑状态
	- 类躁狂状态
	- 情感脆弱、强哭强笑
- 慢性脑病综合征
	- 多发生于严重疾病之后
	- 遗忘综合征
	- 人格改变、痴呆综合征
	- 进行性、不可逆倾向
- 局灶性损害表现
	- 额叶→人格改变
	- 颞叶→情绪不稳/癫痫
	- 顶叶→癔症样表现
	- 间脑→嗜睡/内分泌紊乱

# 2.谵妄——急性脑病综合征
**定义**：谵妄是一组表现为**急性、一过性、广泛性的认知障碍**，尤以**意识障碍**为主要特征。因**急性起病、病程短暂、病变发展迅速**，故又称急性脑病综合征

**核心特征（5A）**
- Acute 急性起病 数小时至数天
- Attention 注意障碍 集中/持续/转移受损
- Awareness 意识改变 混浊到昏迷
- Alteration 认知紊乱 记忆/定向/思维受损
- Alternating 病情波动 日轻夜重/快速转换

**病程特征**
- 多数病人在4周或更短时间内恢复，病程一般不超过6个月
- 可发生在任何年龄，但常见于60岁以上
- 常见于脑外伤、感染、急性中毒、代谢与内分泌紊乱等

**临床表现**
- 意识水平下降
	- 从意识混浊到昏迷的连续谱
	- 通常夜间较重（"日落综合征"），白天可能部分清醒
	- 是谵妄与功能性精神病最重要的鉴别点之一
- 注意障碍（核心特征）
	- 指向、集中、持续和转移能力全面受损
	- 患者表现为"走神"、答非所问、容易被外界刺激打断
	- 注意力评估是谵妄筛查工具的核心条目（4AT/CAM）
- 认知功能全面损害
	- 抽象思维和理解能力损害 · 言语不连贯 · 即刻回忆和近记忆受损
	- 定向力障碍（时间→地点→人物） · 执行功能下降
	- 认知损害通常是全面性的，而非局灶性
- 幻觉、错觉及短暂妄想
	- 视幻觉最常见（看到不存在的人/动物/昆虫）
	- 错觉（把输液管看成蛇）
	- 短暂、片段性被害妄想
- 精神运动紊乱（分型关键）
	- 高活动型：兴奋不安、躁动、拔管、攻击
	- 低活动型：反应迟钝、嗜睡、沉默
	- 可从一个极端转向另一个极端（混合型）
- 睡眠-觉醒周期紊乱
	- 失眠 · 日间多眠 · 夜间少眠
	- 噩梦较多
	- 白天-黑夜节律完全颠倒
- 情绪紊乱
	- 抑郁、焦虑、恐惧
	- 易激惹（对轻微刺激过度反应）
	- 情感淡漠

**谵妄的病因与高危场景**
谵妄不是诊断终点，而是病因警报器！找到并逆转病因是治疗的关键。

| 病因类别 | 具体因素                 | 高危场景        | 处理方向        |
| ---- | -------------------- | ----------- | ----------- |
| 感染   | 泌尿系/肺部/伤口感染、脓毒症      | 老年住院、ICU    | 抗感染、控制源头    |
| 缺氧   | 低氧血症、贫血、心力衰竭         | COPD急性加重、术后 | 氧疗、纠正贫血     |
| 代谢紊乱 | 电解质失衡、肝肾功能衰竭、高血糖/低血糖 | 透析患者、糖尿病患者  | 纠正代谢素乱      |
| 药物   | 抗胆碱药、镇静催眠药、阿片类、多重用药  | 老年多重用药者     | 精简用药、停药试验   |
| 术后   | 手术创伤、麻醉残留、疼痛         | 老年骨科/心脏术后   | 镇痛、早期活动     |
| 感觉剥夺 | 视力/听力障碍、环境陌生、ICU环境   | 未佩戴眼镜/助听器   | 佩戴矫正器具、定向线索 |
| 睡眠剥夺 | 频繁检查、噪音、光照           | ICU、急诊留观    | 保障睡眠周期、减少干扰 |
| 尿便潴留 | 尿潴留、便秘               | 前列腺肥大、卧床    | 导尿、通便       |

**谵妄评估工具**

| 工具      | 时间  | 适用场景     | 核心条目           | 判定标准                 |
| ------- | --- | -------- | -------------- | -------------------- |
| 4AT     | 2分钟 | 急诊、病房初筛  | 警觉性、注意力、定向、认知  | ≥4分提示可能谵妄；1-3分提示认知障碍 |
| CAM     | 5分钟 | 病房常规评估   | 急性起病+注意力+思维+意识 | ①+②+③或④=谵妄阳性         |
| CAM-ICU | 2分钟 | ICU、气管插管 | 适用插管/镇静患者      | RASS≠0时即可评估          |
| 3D-CAM  | 3分钟 | 简短替代方案   | 定向、注意、执行       | 结构化问卷，无需专科培训         |

**高活动型谵妄和低活动型谵妄**

| 特征   | 高活动型谵妄          | 低活动型谵妄          |
| ---- | --------------- | --------------- |
| 行为表现 | 躁动不安、拔管、攻击、喊叫   | 嗜睡、沉默、反应迟钝、活动减少 |
| 识别难度 | 容易（行为引人注目）      | 困难（被误认为疲劳/抑郁）   |
| 被误诊为 | 精神病、躁狂          | 抑郁、疲劳、正常老年状态    |
| 预后   | 相对较好（症状明显，易被处理） | 更差（漏诊导致延误治疗）    |
| 发生率  | 约25-50%         | 约50-75%（更常见！）   |
| 护理负担 | 高（安全风险大）        | 中高（容易被忽视）       |

**谵妄的处理**
- 先逆转病因，非药物优先
	- 环境与行为改良是唯一被证实能缩短ICU住院时间的手段
	- 定向力与感官
		- 提供时钟、日历
		- 窗户保持自然光线
		- 必须佩戴患者自身的眼镜和助听器
		- 反复温和的定向提醒
		- 家属在场陪伴
	- 睡眠与活动
		- 维持明暗sleep-wake周期
		- 夜间减少噪音和检查
		- 保障早期肢体活动
		- 白天鼓励坐起/下床
		- 避免白天长时间午睡
	- 营养与代谢
		- 充足补液与营养
		- 纠正电解质紊乱
		- 治疗尿潴留/便秘
		- 优化疼痛控制
		- 保障氧合
	- 药物管理
		- 精简用药避免多药共用
		- 停用非必要抗胆碱药
		- 停用非必要镇静药
		- 审慎使用阿片类
		- 药物逐一审查
- 药物管理——何时用、用什么
	- 用药指征（缺一不可）：严重精神运动性兴奋 + 即刻自伤或伤人风险 + 口头去唤醒无效 → 方可极小剂量、短期使用
	- 药物：氟哌啶醇、喹硫平、奥氮平、右美托咪定
	- 绝对禁忌：禁止用抗精神病药物来预防谵妄或企图加速谵妄消退

**谵妄的预防策略与出院前评估**
- 三级预防策略
	- 一级预防（术前/入院前）
		- 识别高危患者（老年、痴呆、多种合并症）
		- 优化基础状态（纠正贫血、电解质）
		- 评估用药清单，停用高风险药物
	- 二级预防（住院中）
		- 每日谵妄筛查（CAM/4AT）
		- 多组分非药物干预包
		- 早期识别并逆转病因
	- 三级预防（出院时）
		- 出院前标准化谵妄评估
		- 未恢复基线认知 → 不应出院
		- 家属教育和随访安排
- 出院前谵妄评估
	- 目的：防止患者在未恢复基线认知状态的情况下被误诊出院
	- 未恢复出院的风险：出院后跌倒增加、再入院率升高、认知功能加速衰退
	- 评估工具：CAM/4AT，与入院时基线对比

**谵妄与痴呆、抑郁的鉴别**

| 鉴别维度 | 谵妄 | 痴呆 | 抑郁性假性痴呆 |
| --- | --- | --- | --- |
| 起病 | 急性 （数小时-数天) | 慢性 (数月-数年) | 亚急性，与情绪事件相关 |
| 病程 | 波动（日轻夜重） | 缓慢进展，不可逆 | 波动，有可逆性 |
| 注意力 | 明显受损 | 早期相对保留 | 选择性注意差，但可集中 |
| 意识 | 混浊 | 清醒 | 清醒 |
| 记忆 | 即刻+近记忆受损 | 近记忆→远记忆进行性受损 | 主观记忆差，客观测试不一致 |
| 幻觉 | 视幻觉常见 | 晚期可出现 | 少见 |
| 情绪 | 焦虑、恐惧、易激惹 | 情感淡漠 | 抑郁核心症状 |

# 3.抑郁与焦虑
**诊断**
- 可见的躯体症状（"伪装"）
	- 慢性疼痛（头痛、背痛、腹痛）
	- 疲劳乏力，"没劲儿"
	- 睡眠障碍（入睡困难、早醒）
	- 食欲下降、体重减轻
- 隐藏的情绪核心
	- 情绪低落、无愉快感（兴趣丧失）
	- 悲观、自责、无价值感
	- 自杀意念或行为（必须主动询问！）
- 识别线索：以下情况需高度警惕
	- 反复就诊、多种检查无明显异常
	- 对治疗反应差、抱怨多
	- 社交退缩、依从性下降
	- 伴有显著睡眠/食欲改变
	- 患者说"我觉得好不了了""活着没意思"

**精神性焦虑与躯体性焦虑**
- 精神性焦虑
	- 过度担心、紧张不安、恐惧
	- 坐立不安、注意力难集中
	- 反复要求保证
	- "医生我会不会死？"
	- 灾难化思维（把轻微症状想成重病）
- 躯体性焦虑
	- 心悸、心动过速
	- 出汗、震颤
	- 头晕、头胀
	- 胸闷、呼吸困难
	- 腹部不适、腹泻

**焦虑鉴别诊断**
- 甲状腺功能亢进
	- 心率快、怕热、多汗
	- 体重下降、易饥
	- 手颤、甲状腺肿大
	- 先查甲功 → TSH、FT3、FT4
- 急性冠脉综合征
	- 胸痛与活动相关
	- 压榨感、放射痛
	- ECG异常、肌钙蛋白升高
	- 先排除心脏急症 → ECG、心肌酶
- 低氧血症/COPD急性加重
	- 呼吸困难、气促
	- SpO2下降、紫绀
	- 肺部啰音、CO2潴留
	- 先处理呼吸问题 → 血气分析、胸片

**焦虑状态与焦虑障碍**

|  维度  |      焦虑状态       |      焦虑障碍      |
| :--: | :-------------: | :------------: |
|  性质  |   躯体疾病的正常心理反应   |  过度的、持久的病理性焦虑  |
| 严重程度 |       症状轻       |    症状重、功能受损    |
| 持续时间 | 持续时间短 （随躯体疾病波动） | 过度/持久、与躯体病情不匹配 |
| 功能影响 |    不影响日常生活功能    |     显著功能损害     |
|  处理  |   支持性沟通、解释、安抚   |   药物/心理/转诊治疗   |

**抑郁焦虑的初始处理：三柱模型**
- 治疗原发病
	- 优化疼痛控制
	- 改善睡眠质量
	- 纠正营养不良/贫血
	- 减少不必要的药物
	- 治疗躯体疾病本身
- 心理支持
	- 非污名化解释疾病
	- 支持性沟通与共情
	- 认知行为治疗（CBT）
	- 放松训练、正念减压
	- 家属教育与支持
- 药物治疗
	- SSRI为一线（舍曲林、艾司西酞普兰等）
	- 注意躯体疾病药物相互作用
	- BZD短期辅助（2-4周后减停）
	- 告知起效时间（2-4周）
	- 定期随访评估

**自杀风险评估：必须主动询问**
核心原则：询问自杀想法不会诱发自杀，反而可能挽救生命
- 自杀意念识别线索
	- 直接表达"活着没意思""想死"
	- 安排后事、赠送个人物品
	- 病情突然"好转"（可能已决定）
	- 社会支持突然丧失
	- 既往自杀未遂史
	- 严重失眠、绝望感
- 如何询问（三步法）
	- "很多像您这样的情况会觉得生活没有意义，您有过这种感觉吗？"
	- "您有没有想过，如果一觉睡过去不再醒来就好了？"
	- "您有没有具体想过怎么去做？有没有为此做过准备？"
		- 第3步阳性 = 高危，立即防自杀措施+紧急会诊

**转诊指征与何时启动精神科会诊**
- 必须转诊（红色警报）
	- 自杀意念阳性（尤其有计划）
	- PHQ-9 ≥ 15分或GAD-7 ≥ 15分
	- 精神运动性激越/攻击行为
	- 严重拒食/拒水
	- 精神病性症状（幻觉妄想）
	- 谵妄伴严重行为紊乱
	- 诊断不明确
- 建议转诊（黄色警报）
	- PHQ-9 10-14分或GAD-7 10-14分
	- 功能严重损害（无法自理）
	- 治疗反应差
	- 合并物质依赖
	- 复杂的躯体-精神共病
	- 需要心理治疗
	- 家属强烈请求

# 4.痴呆与认知障碍
**慢性脑病综合征概述**
- 定义：多发生于严重疾病之后，以持续性认知功能损害为核心特征的一组综合征。与谵妄不同，慢性脑病综合征起病缓慢、病程迁延、通常不可逆
- 遗忘综合征
	- 近记忆严重受损
	- 远记忆相对保留
	- 时间定向明显受损
	- 虚构（想象经历填补）
	- 常见于慢性酒精中毒
- 人格改变
	- 性格与前不同
	- 情感反应不协调
	- 行为幼稚化
	- 社会行为脱抑制
	- 常见于额叶损伤
- 痴呆综合征
	- 全面智能衰退
	- 记忆+执行+语言+视空间
	- 日常生活能力下降
	- 慢性进行性病程
	- 最常见：阿尔茨海默病

**痴呆的病因与一般临床特征**
- 核心特征
	- 慢性进展性疾病，数月至数年逐渐恶化
	- 记忆/执行功能/定向力进行性受损
	- 日常生活和工作能力下降
	- 无明确意识障碍（区别于谵妄）
- 主要病因
	- 阿尔茨海默病（最常见，60-70%）
	- 血管性痴呆（脑卒中后）
	- 路易体痴呆
	- 额颞叶痴呆
	- 帕金森病痴呆
- 严重程度分级
	- 轻度：独立完成日常活动，但复杂任务困难，可能忘记近期事件
	- 中度：需要帮助完成日常生活，时间/地点定向障碍，行为改变
	- 重度：完全依赖照顾，不能认人，大小便失禁，行动障碍

**遗忘综合征（柯萨科夫综合征）**
- 近记忆严重受损：无法记住几分钟前发生的事情
- 远记忆相对保留：年轻时的事情记得清楚
- 时间定向明显受损：不知道现在是哪年哪月
- 虚构（Confabulation）：用想象的、不真实的经历来填补记忆空白
- 自知力缺乏：不认为自己有记忆问题

**谵妄、痴呆与抑郁性假性痴呆的鉴别**

| 维度  | 谵妄        | 痴呆       | 抑郁性假性痴呆 |
| --- | --------- | -------- | ------- |
| 起病  | 急性        | 慢性       | 亚急性     |
| 病程  | 波动，日轻夜重   | 缓慢进展，不可逆 | 波动，有可逆性 |
| 注意力 | 明显受损      | 早期相对保留   | 选择性注意差  |
| 意识  | 混浊        | 清醒       | 清醒      |
| 记忆  | 即刻+近记忆    | 近→远进行性损  | 主观记忆差   |
| 幻觉  | 视幻觉常见     | 晚期可出现    | 少见      |
| 情绪  | 焦虑/恐惧/易激惹 | 情感淡漠     | 抑郁核心    |

**三问法快速鉴别谵妄、痴呆与假性痴呆**
- Q1：起病速度？
	- 急性（数小时-数天） → 优先考虑谵妄
	- 慢性（数月-数年） → 优先考虑痴呆
- Q2：有无波动？
	- 明显波动+日轻夜重 → 强烈提示谵妄
	- 相对稳定/缓慢恶化 → 提示痴呆
- Q3：注意力如何？
	- 注意力差+意识混浊 → 谵妄
	- 注意力可集中 → 痴呆/假性痴呆
- 情绪线索：显著抑郁情绪 + 主观记忆差但客观测试不一致 → 提示假性痴呆（抑郁所致认知下降，可逆）；假性痴呆经抗抑郁治疗后认知功能可完全恢复

**脑局灶性损害的不同精神表现**

| 部位    | 精神症状                  |
| ----- | --------------------- |
| 额叶    | 人格改变、定向力差、自知力受损、脱抑制   |
| 颞叶    | 智能缺陷、情绪不稳、癫痫、精神分裂症样障碍 |
| 顶叶    | 癔症样表现、忽视综合征           |
| 枕叶    | 视觉认知功能障碍、视物变形         |
| 间脑    | 嗜睡、情绪控制不良、贪食、内分泌障碍    |
| 胼胝体前部 | 情感障碍、左右手失协调           |
| 胼胝体后部 | 智能缺损、记忆障碍             |

**血管性认知障碍（VCI）与痴呆处理**
- 处理原则
	- 控制血管危险因素（血压、血糖、血脂）
	- 抗血小板治疗（无禁忌时）
	- 预防再次卒中
	- 卒中后抑郁筛查（PHQ-9）
	- 必要时使用认知增强药物
- 卒中后抑郁
	- 发生率：15-60%
	- 影响康复训练依从性
	- 增加死亡风险
	- 应在卒中单元常规筛查
	- 治疗：SSRIs + 康复 + 心理支持

# 5.躯体疾病相关精神障碍
**躯体疾病所致精神障碍的共同特点**
- 与躯体疾病有时间关联：精神症状随躯体疾病出现、加重、缓解
- 病程相关：精神症状的严重程度与躯体疾病的严重程度平行
- 症状可相似：不同躯体疾病可引起极为相似的精神症状
- 同病可异相：同一躯体疾病在不同患者身上可表现为不同精神症状
- 先治原发病：病因治疗是第一位，精神症状治疗是第二位
- 警惕因素：急性起病+意识改变+发热+抽搐 → 必须排除器质性病因

**癫痫所致精神障碍：发作时间轴分型**
- 发作前（Pre-ictal）
	- 焦虑、紧张
	- 易激惹、烦躁
	- 自主神经症状
	- 面色潮红/苍白
	- 预感（"要发作了"）
- 发作时（Ictal）
	- 精神运动性发作
	- 自动症
	- 发作性情感障碍
	- 短暂精神分裂症样
	- 朦胧状态
- 发作后（Post-ictal）
	- 澄清期/朦胧期
	- 随后急性精神病
	- 被害妄想、幻觉
	- 极高自杀/伤人风险
	- 数小时-数天
- 发作间期（Inter-ictal）
	- 慢性分裂样障碍
	- 情感障碍（抑郁/焦虑）
	- 人格改变
	- 癫痫性痴呆
	- 与电活动无直接时间关联

**发作后精神病（Postictal Psychosis, PP）**
- 核心特征——"澄清期（Lucid Interval）"：癫痫大发作完全终止后，患者神志一度完全恢复清醒（数小时至1-2天），随后突发严重的精神病性症状
- 临床特征
	- 多灶性被害妄想
	- 宗教/夸大妄想
	- 幻听、幻视
	- 极高自杀/冲动伤人风险
	- 情感不稳定
	- 持续数天至数周
- 发病机制与治疗
	- 机制
		- 受体超敏感（频繁放电后）
		- 强制性正常化
	- 治疗
		- 不降低抽搐阈值的第二代抗精神病药
		- 奥氮平、阿立哌唑、喹硫平
		- 症状缓解后平稳停药

**功能性发作（FS/癔症样抽搐）与癫痫的重要鉴别**

| 鉴别维度 | 真性癫痛大发作 | 功能性发作 |
| --- | --- | --- |
| 诱发环境 | 任何时间、睡眠中可发生 | 常有人围观、情绪波动时 |
| 眼部征象 | 眼球上翻/偏斜、瞳孔散大 | 双眼紧闭、对抗性闭眼 |
| 肢体运动 | 同步、对称、刻板 | 非同步、非对称、剧烈乱动 |
| 发作时长 | 通常<2分钟 | 常>5分钟甚至数十分钟 |
| 发作后 | 意识混浊、嗜睡、肌肉酸痛 | 迅速恢复、无嗜睡 |
| VEEG | 癫痛样放电 | 全程正常脑电 |
**正确治疗**
- 确诊后逐步停用所有ASMs
- CBT-FS（功能性发作特异性认知行为治疗）
- ReACT（重塑反应治疗）
- 家庭系统支持
- 治疗共病抑郁焦虑

**癫痫用药安全：精神科药物相互作用**
- 避免（降低抽搐阈值）
	- **氯氮平（禁忌）**
	- 氯丙嗪
	- 三环类抗抑郁药
	- 碳酸锂（锂盐）
	- 安非他酮
- 相对安全
	- 喹硫平（首选）
	- 奥氮平
	- 阿立哌唑
	- SSRIs（舍曲林、西酞普兰）

**病毒性脑炎所致精神障碍**
- 临床特征
	- 起病较急、常无前驱症状
	- 头痛、呕吐、发热、意识障碍
	- 精神症状常为首发表现
	- 不协调兴奋或精神萎靡、情感淡漠
	- 日轻夜重特点
- 辅助检查线索
	- EEG：**弥漫性慢波（重要线索）**
	- CSF：白细胞增多、蛋白升高
	- MRI：颞叶/额叶异常信号
	- 癫痫发作常见
- 报警症状
	- 急性起病（数小时-数天）
	- 意识水平改变
	- 发热/感染前驱史
	- 癫痫发作
	- 神经系统定位体征
	- EEG异常（弥漫性慢波）
- 治疗原则
	- 病因治疗：抗病毒（阿昔洛韦/更昔洛韦）、控制颅压（甘露醇）、控制癫痫发作
	- 精神症状处理：避免降低抽搐阈值的药物、喹硫平相对安全、支持治疗（营养、水电解质）

**自身免疫性脑炎（AIE）的过度诊断警示**
- 抗体滴度很关键：仅低滴度阳性 → 不足以诊断AIE
- CSF证据是金标准：脑脊液白细胞升高、寡克隆区带阳性 → 支持AIE
- MRI很重要：边缘系统T2/FLAIR高信号 → 支持AIE
- 排除感染：CSF PCR病毒检测阴性、细菌培养阴性
- 误诊后果严重：将病毒性脑炎误判为AIE → 大剂量激素冲击 → 病毒感染扩散

**甲状腺功能亢进所致精神障碍**
- 精神症状
	- 感知觉过敏（光、声、触觉过敏）
	- 注意力不集中、多言多动
	- 紧张、焦虑、易激惹
	- 严重者躁狂状态
	- 甲状腺危象 → 意识障碍为主
- 鉴别与处理
	- 先查甲功：TSH↓、FT3↑、FT4↑
	- 与原发性焦虑/躁狂鉴别
	- 有躯体症状：怕热、多汗、心悸、体重下降
- 治疗
	- 控制甲亢（抗甲状腺药物）
	- 对症：地西泮、丙戊酸盐

**甲状腺功能减退所致精神障碍：黏液性水肿**
- 精神症状
	- 进行性痴呆样改变
	- 重度抑郁/木僵状态
	- 急性妄想性精神病
	- "黏液性水肿面容"
	- 低体温、心动过缓
- 实验室与处理
	- TSH 病理性飙升
	- FT4 极重度低下
- 治疗
	- 左甲状腺素（小剂量起始）
	- 氢化可的松（先补充再纠正甲减）
	- 喹硫平（控制精神病症状）

**桥本脑病（SREAT）：甲功正常的自身免疫性脑病**
- 临床三联征
	- 波动性认知衰退（痴呆样）
	- 癫痫发作（局灶或全面）
	- 卒中样发作（短暂神经功能缺损）
- 其他表现：肌阵挛、急性分裂样精神错乱、震颤、意识障碍
- 治疗
	- 大剂量糖皮质激素（甲泼尼龙冲击→口服维持）
	- 而非甲状腺素替代
	- 激素治疗反应良好

**系统性红斑狼疮所致精神障碍**
- 早期表现
	- 乏力、头晕
	- 注意力不集中
	- 记忆减退
	- 情绪不稳
	- 睡眠障碍
	- 头痛
	- 青年女性多见
- 严重期表现
	- 意识模糊/谵妄
	- 癫痫发作
	- 脑血管意外
	- 横贯性脊髓炎
	- 严重运动障碍
	- 需紧急处理
- 慢性期表现
	- 全面智能损害/痴呆
	- 焦虑
	- 躁狂
	- 精神分裂样障碍
	- 人格改变
	- 病程迁延
- 主要治疗
	- 皮质激素治疗为主
	- 泼尼松长期维持 ≤5mg/d
	- 贝利尤单抗（生物制剂）
	- 免疫抑制剂（CTX、MMF）
	- 甲泼尼龙冲击（严重时）
	- 精神症状随SLE好转
- 精神症状对症处理
	- 精神药物对症（最小有效剂量）
	- 考虑多脏器损害
	- 避免药物相互作用
	- 心理支持
	- 家属教育
	- 定期随访评估

**糖皮质激素所致精神障碍**
- 精神症状
	- 失眠（最早出现）
	- 情绪不稳定、易激惹
	- 烦躁、焦虑
	- 自卑、抑郁
	- 严重者：类躁狂状态、精神病性症状
	- 谵妄（大剂量时）
- 处理原则
	- 评估症状严重程度
	- 根据症状选择精神药物
	- 尽可能减量激素
	- 失眠 → 小剂量镇静药
	- 抑郁 → SSRIs
	- 躁狂/精神病 → 非典型抗精神病药

**脑外伤所致精神障碍**
- 急性期表现
	- 短暂意识障碍
	- 谵妄
	- 外伤后遗忘
	- 头痛、头晕
	- 情绪不稳定
- 慢性期表现与治疗
	- 焦虑/抑郁/易激惹
	- 认知功能下降
	- 人格改变
	- 治疗：根据症状形式给药
	- 心理治疗重点：重树信心

**性传播疾病中的精神障碍**
- 已感染者（治疗后）
	- 不安全生活史 → 下腹生殖器不适
	- 感染者治疗后精神症状依旧
	- 恐惧、焦虑、强迫
	- 躯体形式障碍
	- 严重者：绝望、抑郁、消极
- 未感染者（偏执信念）
	- 偏执信念可达妄想程度
	- 坚信自己被感染（尽管反复检查阴性）
	- 反复就医、反复检查
	- 合并性功能障碍
	- 心理治疗 + 保密承诺

# 6.治疗原则与临床沟通
==**主要治疗原则：五位一体**==
- 病因治疗
	- 准确诊断
	- 及时干预
	- 彻底治疗原发病
	- 所有治疗的基础
- 营养支持
	- 充足营养摄入
	- 纠正水电解质紊乱
	- 维生素补充
	- 改善整体状态
- 对症治疗
	- 精神药物控制症状
	- 小剂量起始
	- 短期使用为原则
	- 注意药物安全
- 安静环境与良好护理
	- 减少环境刺激
	- 定向力支持
	- 安全看护
- 全程心理治疗
	- 支持性心理治疗
	- 家庭治疗
	- 康复期社会功能重建

**综合医院处理精神障碍的注意事项**
- 坚持症状学诊断与对症治疗，不完全依赖病因诊断
- 不出具"精神疾病诊断证明"——这是精神专科医生的职责
- 必要时用综合征代替（如"焦虑抑郁状态"）
- 合理运用等级诊断原则：器质性 → 功能性
- 出院记录写精神疾病诊断需经会诊确认
- 躯体疾病诊断写前面，精神障碍诊断写后面
- 遇到重性精神病 → 应请精神科会诊

**抗抑郁药物在躯体疾病患者中的使用要点**
- 选择原则
	- SSRI为一线（舍曲林、艾司西酞普兰）
	- 注意躯体疾病药物相互作用
	- 合并心血管疾病 → 舍曲林最安全
	- 合并癫痫 → 避免安非他酮
	- 合并出血风险 → 加PPI
	- 老年 → 起始半量
- 使用要点
	- BZD短期辅助（2-4周）
	- 告知起效时间（2-4周）
	- 定期随访评估疗效
	- 足量足疗程（6-12个月）
	- 减药需缓慢
	- 监测副作用

**临床沟通技巧概述**
- 与患者及家属沟通
	- 建立信任关系
	- 非污名化解释疾病
	- 耐心倾听
	- 使用通俗语言
	- 承认症状的真实性
	- 共同制定治疗计划
- 解释病情与协商治疗
	- 了解患者已有看法
	- 双重肯定：症状真实+检查真实
	- 解释共同物质基础
	- 解释药物必要性
	- 告知起效规律
	- 达成治疗同盟

**解释诊断的要点**
- 了解患者看法
	- 先了解患者已有的看法
	- 顺着患者能接受的程度逐渐解释
- 双重肯定与共病思维
	- 双重肯定：躯体不适感的真实性与检查阴性的真实性两者不矛盾
	- 解释共同物质基础
		- NE（去甲肾上腺素）系统紊乱 → 疼痛敏感性增加
		- 5-HT（5-羟色胺）系统紊乱 → 抑郁、焦虑、睡眠障碍
