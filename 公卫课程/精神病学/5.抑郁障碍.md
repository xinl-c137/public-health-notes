# 1.概述、病因与发病机制
**概念**
- **抑郁障碍**是指由各种原因引起的以显著和持久的抑郁症状群为主要临床特征的一类心境障碍
- 抑郁症状群包括情感低落、兴趣减退、快感缺失、精力缺乏、精神运动性迟滞或激越、注意力不集中、制定决策困难、自卑、自责、无价值感、体重与睡眠的变化以及自杀意念或行为等
- 患者常常伴有焦虑或激越，甚至出现幻觉、妄想等精神病性症状

**流行病学特点**
- 据世界卫生组织统计，全球约有3.5亿抑郁障碍患者，平均每20人就有1人曾患或目前患有抑郁障碍
- 费立鹏等在2009年对中国4省市进行的流行病学调查资料显示，抑郁障碍的患病率为2.06%，恶劣心境为2.03%；北京大学第六医院黄悦勤等报道的最新流行病学调查研究结果显示，抑郁障碍的年患病率为3.59%
- 自杀是抑郁障碍患者最为严重的后果之一，在所有自杀者中约50%可能符合抑郁障碍的诊断
- 地区差异大：社会和文化因素可能在发病中起作用
- 救治率低：从全球看，救治率不足50%
- 起病年龄两个高峰
- 性别差异：男VS女约为1：2

**病因与发病机制**
- 遗传
- 神经生化
	- 去甲肾上腺素能、多巴胺能和5-HT能神经递质系统在抑郁障碍发病中扮演重要角色
	- 不仅与神经递质的水平异常有关，也与相应受体功能的改变有关，可能累及受体后信号转导功能，甚至影响基因转录过程
- 神经内分泌
	- 下丘脑-垂体-肾上腺轴功能异常，引起血中皮质醇水平增高、内分泌昼夜节律紊乱
	- 下丘脑-垂体-甲状腺轴可能也参与发病，临床中可以观察到甲减患者出现抑郁相关表现
- 神经影像学
	- 以杏仁核和内侧前额叶皮质为中心的内隐情绪调节环路
	- 以腹侧纹状体/伏隔核、内侧前额叶皮质为中心的奖赏神经环路
- 神经电生理
	- 抑郁障碍患者脑电图研究发现，抑郁严重程度与其左右脑半球平均整合振幅呈负相关
	- 抑郁发作时存在事件相关电位（event related potentials，ERP）P300和N400潜伏期延长
- 心理社会
	- 生活中的应激事件如亲人去世、失业、严重躯体疾病等是疾病发生的危险因素
	- 如果多个严重不良的生活事件同时存在，则可能协同影响抑郁障碍的发生

# 2.临床分型与临床表现
**抑郁障碍的分型**
- 抑郁发作（轻/中/重度）
	- 抑郁发作是最常见的抑郁障碍，表现为单次发作或反复发作，病程迁延，此病具有较高的复发风险，发作间歇期或可能存在不同程度的残留症状
- 复发性抑郁障碍
- 破坏性心境失调障碍
	- 主要见于儿童和青少年
	- 核心特征
		- 严重、反复的脾气爆发：言语或行为上的爆发（如尖叫、摔东西、攻击行为），与发育水平不符，且强度/持续时间超出情境的触发因素，平均每周发生3次或以上
		- 持续的易激惹或愤怒情绪：在爆发以外的多数时间中，情绪基调以易怒、愤怒为主，且能被他人观察到（如家庭、学校）
	- 病程要求：症状持续至少12个月，期间无超过3个月的缓解期
	- 首次诊断年龄需在 6-18岁之间，症状通常在10岁前出现
- 持续性心境（情感）障碍：恶劣心境
	- 恶劣心境：一种以持久的心境低落状态为主（至少2年）的轻度抑郁，正常心境很少持续几周，从不出现躁狂或轻躁狂发作。这种慢性的心境低落，无论从严重程度还是一次发作的持续时间，均不符合轻度或中度复发性抑郁障碍的标准
	- 注意与抑郁发作的鉴别和重叠
		- 双重抑郁——在心境恶劣的基础上叠加的抑郁发作
	- 抑郁周期内至少应具有以下三种症状：精力或活动减少/失眠/自信心丧失或感到自信心不足/集中注意困难/经常流泪/对性活动或其它乐事的兴趣及乐趣丧失/无望感或绝望/感到无能力承担日常生活中的常规责任/对前途悲观或沉缅于过去/社会退缩/言谈比平时减少

**临床表现**：以心境低落为主要核心症状。一般有以下症状群
- 核心症状群
	- “三低”症状：心境低落、兴趣和愉快感丧失、精力下降
		- 心境低落不能通过自我调节、他人安慰、改变环境等得到有效缓解
		- 兴趣丧失不仅体现在对任何事情不感兴趣，对以往的嗜好都可能不再关注。在勉强被动地参与活动时，也体会不到任何乐趣。患者躲避亲友，回避社交，“什么都不想干”
		- 精力丧失表现在无任何原因主观、休息也不能缓解的精力不足，患者感到特别容易疲劳，干不了家务，工作也难以胜任
- 心理症状群
	- 认知功能损害：注意力困难、记忆力减退、脑子迟钝、思维迟缓
	- 负性认知模式：“三无”—无望无助无能
	- “三自”—自责自罪自杀
	- 焦虑：常见，尤其是老年期抑郁，焦虑和激越需高度关注
	- 精神病性症状：幻觉、妄想
- 躯体症状群
	- 睡眠障碍：以早醒为特征
	- 与自主神经系统紊乱相关的症状：胃肠道功能紊乱（无器质性病变）
	- 进食紊乱：食欲下降
	- 精力下降：精神运动性迟滞
	- 性功能障碍：性欲下降
	- 晨重暮轻

**DSM-5关于抑郁障碍的8种亚型**
- 伴焦虑特征：自杀风险更高，更长病程，更易有不良反应
- 伴混合特征：双相障碍危险因素
- 伴忧郁特征：“内源性抑郁”自杀风险更高
- 伴非典型特征：注意鉴别双相障碍
- 伴紧张症特征
- 伴季节特征：冬季型＞夏季型；多伴有非典型特征
- 围产期抑郁：注意扩大自杀
- 伴精神病性症状：可与心境协调或不协调

# 3.评估、诊断与鉴别诊断
**评估**
- 评估的具体内容包括现病史、目前症状、是否有自杀意念（自杀风险评估），既往是否有过躁狂发作或精神病性症状发作，目前的治疗情况及疗效、过去的治疗史、躯体疾病病史、家族史等
- 对疑似抑郁障碍的患者，除了进行全面的躯体检查及神经系统检查外，还要注意辅助检查及实验室检查
- 自杀的预防
	- 告诉患者亲友有关该疾病自杀率高的特征
	- 避免危险物品和场合，对有消极倾向者需严防自杀

**抑郁障碍自杀的危险因素与风险评估**
- 阳性自杀家族史
- 酒依赖和药物滥用
- 年龄(45岁及以上)，尤其是老年男性
- 易激怒、冲动和暴力行为
- 既往有自杀行为
- 男性
- 拒绝接受帮助
- 抑郁障碍合并自责、后悔
- 重大负性生活事件，如丧失或分离、严重躯体疾病、失业或退休、单身、丧偶或离婚
- 严重焦虑或惊恐发作

**诊断：ICD-10症状学标准**
- 核心症状
	- 心境低落
	- 兴趣和愉快感丧失
	- 精力减低或过度疲劳
- 其他症状
	- 集中注意和注意的能力降低
	- 自我评价和自信降低
	- 自罪观念和无价值感
	- 认为前途暗淡悲观
	- 自伤、自杀观念或行为
	- 睡眠障碍
	- 食欲下降
- 症状学标准：至少2条核心症状，加上至少2条其他症状
- 病程标准：满足症状学标准的症状每天持续存在至少2周
- 严重程度标准：社会功能受损
- 排除标准：排除器质性精神障碍、精神活性物质所致、精神分裂症
- 抑郁发作的病情程度
	- 轻度——2条核心症状+2条其他症状
	- 中度——2条核心症状+4条其他症状
	- 重度——3条核心症状+4条其他症状

**鉴别诊断**
- 躯体疾病所致的抑郁障碍
	- 神经系统疾病
		- 最常见包括帕金森病、痴呆性疾病、癫痫、脑血管病和肿瘤
	- 其他躯体疾病相关抑郁
		- 常见包括内分泌系统疾病如甲状腺疾病、风湿免疫系统疾病等
		- 注意评估：躯体疾病所致抑郁、躯体疾病诱发抑郁、躯体疾病伴发抑郁、抑郁相关躯体症状
	- 药物及精神活性物质使用所致抑郁
		- 如利血平、激素类药物等
- 精神分裂症
- 双相情感障碍
- 焦虑障碍

# 4.治疗
**治疗目标**
- 提高有效率和痊愈率——最大限度减少病残率和自杀率
- 提高生存质量，恢复社会功能，预防复发（高复发率，>50%）

**治疗原则**
- 单一、足量、足疗程、个体化
	- 由于新型抗抑郁药物的副作用很小，理论上首次用药就可以全量给药，但临床上对于耐受力较差的患者如老年人、或既往对药物敏感者，建议半量使用几天至1周后再加量。对于某些可能引起撤药反应的抗抑郁药，也建议缓慢减药
	- 首发患者尽量采用单一抗抑郁药物系统治疗；一种药物无效可以换用另一种治疗；复发患者或者治疗困难者，可以考虑两种以上不同作用机制类型的药物合用
- 建立治疗联盟

**治疗阶段**
- 全病程：急性期、巩固期、维持期

**治疗方法**
- 药物治疗
	- **选择性五羟色胺再摄取抑制剂（Selective Serotonin Reuptake Inhibitors, SSRIs）**
	- 五羟色胺/去甲肾上腺素再摄取抑制剂（Serotonergic Noradrenergic Reuptake Inhibitors, SNRIs）
	- 5-HT和特异NE能抗抑郁剂（NaSSAs）
	- 5-HT受体拮抗/再摄取抑制剂（SARIs）
- 心理治疗——支持性心理治疗、认知行为治疗、精神动力学治疗、人际心理治疗、婚姻家庭治疗
	- 支持性心理治疗：倾听、安慰、解释、指导和鼓励等
	- 认知行为治疗CBT：认识并矫正自身的错误信念、缓解情感压力，达到减轻症状、改善患者应对能力
	- 精神动力学治疗：长程、短程
	- 人际心理治疗IPT：处理人际交往问题，通过适当的人际关系调整和改善来减轻抑郁，提高社会适应能力
	- 婚姻家庭治疗：促进良好的配偶关系，将焦点放在家庭成员的互动关系上，从家庭系统角度解释个人行为与问题
	- 其他：正念、DBT、人本等
- 物理治疗——电抽搐治疗、重复经颅磁刺激、迷走神经刺激、深部脑刺激
	- 电抽搐治疗
		- 电抽搐治疗(electroconvulsive therapy, ECT)是用短暂适量的电流刺激大脑，引起病人意识丧失，皮层广泛性脑电发放和全身抽搐，以达到控制精神症状的一种治疗方法。在电抽搐前加用静脉麻醉药和肌肉松弛剂，使患者抽搐明显减轻和无恐惧感，称为改良电抽搐治疗(modified ECT,MECT)
		- MECT可快速有效的治疗抑郁症，可明显降低患者自杀风险，起效快，有效率高，可高达70%~90%
		- 通常一周2-3次，疗程视病情而定，一般为6-12次，通常不超过20次
		- 改良电抽搐治疗的适应证
			- 抑郁障碍
			- 躁狂发作
			- 精神分裂症
			- 其他精神障碍
			- 其他

==**抗抑郁药物的不良反应**==
- 早期不良反应
	- 胃肠道反应如恶心、腹泻、便秘等
	- 失眠或嗜睡
	- 焦虑、激越等
	- 多出现在服药前1~2周，多为轻度，可随用药时间延长耐受
- 长期不良反应
	- 性功能障碍
	- 体重增加

==**常见不良反应处理原则**==
- 一般原则
	- 多数不良反应轻微，通常在7-10天后消失
	- 首次处方时应当向患者解释这些药物反应，并告知处理方法，可以提高患者的依从性
- 鉴别诊断
	- 消化系统症状与药物的胃肠道反应
	- 性欲减退与药物引起的性功能问题
	- 焦虑症状和药物引发的焦虑
	- 失眠症状与药物对于睡眠的干扰
- 早期处理
	- 对失眠可以临时使用短效药物；如唑吡坦、佐匹克隆等
	- 对焦虑可短时使用抗焦虑药物，如劳拉西泮、奥沙西泮等

**5-HT综合征&撤药综合征**
- 单用SSRIs类药物出现严重不良反应或致死极少见，但与其他药物联用可能产生严重不良反应。如SSRIs可增加中枢5-HT能的药理作用，引起5-HT系统兴奋，可能增加出现中枢5-HT综合征的可能性，其临床表现为消化系统、植物神经系统、锥体外系综合征等症状，以及肌张力增高、酸中毒、心衰、肾衰、心血管休克及死亡
- 短效SSRIs突然停药，在部分患者可出现停药反应（撤药综合征）：表现为激越症状并伴有躯体主诉。这些症状可逐渐消失，恢复使用SSRIs类药物也可消除，故应逐渐减药。SSRIs类药物不会出现依赖现象

# 5.预后与康复
**抑郁障碍复发的危险因素**
- 残留症状持续存在
- 之前多次抑郁发作史
- 首次发作及后续发作症状重
- 起病年龄高
- 共病其他精神障碍
- 慢性躯体疾病
- 精神障碍家族史，尤其是情感障碍
- 持续的心理社会应激或功能缺陷
- 消极的认知观
- 持续的睡眠障碍

**康复内容**
- 生活自理能力
- 家庭干预
- 社交技能
- 职业技能
