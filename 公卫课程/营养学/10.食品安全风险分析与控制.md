# 2.基本概念与认识
==**风险分析框架**==
- 风险评估——基于科学
- 风险管理——基于政策
- 风险交流——相关各方

**危害(hazard)**：食品中可能会产生不良健康影响的生物性、化学性或物理性因素或状况（污染物、致病菌、农兽药残留等）

**风险(risk)**：各种危害产生不良健康作用的可能性及其强度

**风险评估(risk assessment)**
- 用来识别和评估特定（亚）人群暴露于某危害因素后健康风险的科学过程，包括四个步骤：危害识别、危害特征描述、暴露评估、风险特征描述
- 定义辨析

| 法定的评估      | 卫生健康委组织开展风险评估                           |
| ---------- | --------------------------------------- |
| 食品安全风险评估   | 对食品中生物性、化学性和物理性危害对人体健康可能造成的不良影响所进行的科学评估 |
| 食品安全风险分析研判 | 考虑政治、经济、舆论等                             |
| 广义风险评估     | 评价某种因素或事件给人们生活、生命、财产等造成影响和损失的可能性及严重程度   |
- 风险评估类型
	- 风险评估是指对食品、食品添加剂、食品相关产品中的生物性、化学性和物理性危害对人体健康造成**不良影响的可能性及其程度**进行定性或定量估计的过程，包括危害识别、危害特征描述、暴露评估和风险特征描述等
	- 应急风险评估是指在受时间等因素限制的特殊情形下，开展的紧急风险评估
	- 风险研判是指在现有数据资料不能满足完成全部风险评估程序的情况下，就现有数据资料按照食品安全风险评估方法，对食品安全风险进行的综合描述

**食品安全**：指食品无毒、无害，符合应当有的营养要求，对人体健康不造成任何急性、亚急性或者慢性危害

# 3.风险评估步骤与方法
**食品安全风险评估**：是评估食品、食品添加剂、食品相关产品中生物性、化学性和物理性因素对人体健康造成不良影响的可能性及其程度的科学过程

**风险评估的基本步骤**
- 危害识别
- 危害特征描述
	- 定性：有没有害
	- 定量：吃多少会有害
	- 特点：国际通用
- 暴露评估
	- 定性：高？低？
	- 定量：暴露量
	- 特点：本国数据、个案处理、目的性强
- 风险特征描述
	- 直接定量（＜10<sup>-6</sup>）
	- 健康指导值比较：有阈值化学物
	- 暴露边界法（MOE）：无阈值化学物

**危害识别**
- 根据人群流行病学、动物毒性试验、体外试验、结构-效应关系等科学数据和文献信息，确定生物、系统或（亚）人群暴露于某一因素或状况后对健康造成不良影响的类型和属性的过程
- 目的
	- 识别潜在的有害作用
	- 确定产生有害作用所必需的暴露条件
	- 评估剂量反应关系，包括确定参考点（如最大无毒副作用剂量，NOAEL）
	- 提供作用机制、代谢和毒代动力学等数据

==**危害特征描述(hazard characterization)**==
- 是指对食品中生物、化学和物理因素所产生的不良健康影响进行定性和/或定量分析。可以利用动物实验、临床研究以及流行病学研究确定危害与各种不良健康作用之间的剂量-反应关系、作用机制等
- 利用危害识别收集到的信息
- 确定关键效应，即随着剂量增加首先观察到的不良效应
- 剂量-反应评估是危害特征描述的重要组成部分
- 确定**健康指导值**是危害特征描述的重要目标
	- 健康指导值(health-based guidance values, HBGY)是指人类在一定时期内（终生或24小时）摄入某种（或某些）物质，而不产生可检测到的对健康产生危害的安全限值，包括每日允许摄入量、耐受摄入量、急性参考剂量等
	- 类型
		- 每日容许摄入量(ADI)
		- 每日/周/月耐受摄入量(TDI/PTDI/PTWI/PTMI)
		- 急性参考剂量(ARfD)
		- 剂量-反应关系数据
	- 方法
		- 阈值法：NOAEL/UF
		- 基准剂量法：BMDL<sub>05</sub>/UF

==**暴露评估**==
- 是指对在食用过程中可能摄入的生物、化学、物理因素及其他来源的暴露进行定性和/或定量的评估。描述危害进入人体的途径，估算不同人群摄入危害的含量水平
- 相关术语
	- 急性暴露：一餐或24小时以内的暴露/摄入
	- 慢性（长期）暴露：每天暴露/摄入并持续终生
	- 危害因素：食品添加剂、污染物（包括生物毒素）、营养素、农药残留和兽药残留、食品接触材料
	- 膳食暴露：通过食物暴露/摄入化学物的量
	- 适用范围：所有类型食品化学物质、食药物质和新食品原料安全性评估中应用膳食暴露评估

**风险特征描述**
- “定量化”策略：定量风险
- “设门槛”策略：判定风险
- “比高低”策略：相对风险
- 风险特征描述方法
	- 直接定量（＜10<sup>-6</sup>）
	- 健康指导值比较：有阈值化学物
	- 暴露边界法（MOE）：无阈值化学物

**术语含义**
- 一般人群：调查中所有被抽样的调查对象，即含有所关注化学物食品的消费者和非消费者
- 消费者/食用者(Consumer Only)：消费含关注化学物食品，并因此可能暴露该化学物的亚人群
- 高暴霹消费者(P95/P97.5/P99)
	- 消费大量含关注化学物食物的亚人群
	- 膳食暴露水平处于暴露分布顶端：包括高食物量消费者、消费具有高浓度关注化学物食物的消费者，或以平均量消费很多不同类型食物但都含有关注的化学物的食物

==**暴露评估常用方法**==
- 确定性评估（点评估）：输入变量及输出结果都是单个数值
	- 筛选法：所需数据量最少、最保守
	- 粗略点估计：粗略的消费量（如生理需要量）和其他模型膳食的暴露计算方法，如兽药残留模型膳食
	- 基于真实消费数据和化学物质浓度数据：更加精确，例如总膳食研究（TDSs）、单个食品的选择性研究和双份饭研究
- 筛选法
	- 筛选法应该旨在反映出暴露的特殊性
	- 选择的筛选法应当简单实用
	- 要足够保守，宁可高估，不能低估；最坏情况
	- 不应当使用毫无依据或很不现实的膳食，例如超过摄食的生理限制
	- 产销量法
		- 优点：快速；时间、空间及化学物间比较
		- 缺点：波动大；与膳食消费模式和食物中化学物的真实浓度无关；仅获得平均水平，无法估计高端暴露量
	- 预算法
		- 用于计算一些食品添加剂的理论每日最大膳食摄入量，基于三个假设
			- 食品和饮料的消费量采用每日最大生理消费量
			- 浓度为最具代表性的食品类别中的最大值
			- 含有添加剂的食品和非乳饮料比例，通常为12.5%和25%
		- 优点：简单快捷；适用于儿童和成人；可估计高暴露；含量为真实数据
		- 缺点：可能含有某种物质的食物和饮料比例的假设比较主观；某些食品中最大含量可能不具有代表性，如口香糖
		- 适用：所有添加到食物中的化学物（添加剂、调味剂、加工助剂）
- 更精确的点评估
	- 数据更精准
		- 消费量数据：实际调查
		- 含量数据：监测、专项调查
	- 方法更精确
		- 校正系数：如加工因子等
		- 总膳食研究：食品经加工后检测，反映了加工过程对总量、形态的影响。适合于国家及区域性暴露评估
			- 本研究所采用的食物消费量数据反映的是人群的平均消费水平，因此所计算出的膳食摄入量是标准人的摄入量。标准人是指18岁成年男子，轻体力活动水平
		- 双份饭研究：每餐份称重+检测；非常精确；成本高，适于小范围特定人群研究
			- 评价人群膳食营养结构和了解污染物摄入，进而全面掌握膳食营养与安全状况的最直接、最科学、准确的方法
- 简单分布模型
	- 消费量数据：个体分布
	- 含量数据：平均值或中位数
	- 可获得个体暴露分布及变异性
	- 未经过迭代
	- 非参数经验分布
	- 考虑了食品消费量的变异，但保留了假设，食品中化学物浓度使用固定参数

**急性膳食暴露评估**
- 概念：评估在单餐或一天(24小时)大量摄入食品中化学污染物引起食用者的急性健康危害效应的风险。评估情形：①仅考虑一种食品；②1天或1餐内同时暴露于多种含有目标化学物的食品
- 点评估、简单分布评估、概率评估
- 用于急性膳食暴露评估的数据
	- 确定性方法
		- 农药残留
		- 兽药残留
		- 污染物
	- 概率/随机方法
		- 一般人群、相关亚人群的急性膳食暴露评估

**慢性膳食暴露评估**
- 概念：评估食品中化学污染物长期甚至持续终生的反复摄入对人群的慢性健康危害的风险
- 概率评估
	- 优点：精确模拟、个人差异、体现差异性
	- 缺点：成本高、起点高、多种食物时较复杂
	- 基本概念
		- 分布(Distribution)：正态分布(Normal Distribution)、均匀分布(Uniform Distribution)、三角形分布(Triangle Distribution)
		- 迭代(Iteration)：从数据分布函数中随机抽取数据，按照预设的计算模型，重复计算的过程
		- 模拟(Simulation)：按照预设的迭代次数和计算模型，对已输入的分布完成了一轮模拟(计算)
- 按暴露方式

| 方法 | 目标化学物 | 暴露途径 |
| --- | --- | --- |
| 膳食暴露 | 单一化学物 | 食物、饮水 |
| 联合暴露 | 单一化学物 | 食物、空气、手-口等多种媒介 |
| 累积膳食暴露 | 多种化学物 | 食物、饮水 |
| 累积暴露 | 多种化学物 | 多种媒介 |

**多物质联合暴露评估方法**
- 类别ADI/TDI(Group ADI/TDI)
	- 化学物在结构或代谢上相关且显示出相似毒性，可为这类化学物建立一个ADI/TDI
	- 适应于毒理学资料有限但化学结构非常相近物质的评估，但至少有一种化合物的毒理学资料足够充分
	- 类别ADI/TDl所依据的NOAEL通常为该类化学物的平均值，也可采用最低NOAEL值
	- 应用：单端孢霉烯族毒素T-2毒素
- 替代物(Surrogate)方法
	- 以混合物中某单一成份的浓度来衡量整个混合物的效应
	- 适应于化学结构相似物质组成的混合物，假设效应一致
	- 各成份的毒理学特征一致，且至少有一种成份（替代物）的毒理学资料充分
	- 应用：多环芳烃一苯并[α]芘
- 毒性当量因子(Relative Potency Factor, RPF)法
	- 从一组具有共同作用机制的化合物中确定一个“指示化合物”，以各化合物的效能与指示化合物的效能的比值作为校正因子，对各化合物的暴露量进行标化，计算相当于“指示化合物”浓度的总暴露
	- 依赖于指示化合物的选择和毒理学资料的质量
	- 类似方法：相对效能因子(RPF)，效能当量因子
	- 应用：二恶英一TCDD
- 危害指数(Hazard Index, Hl)法
	- 危害指数是各物质暴露量与其健康指导值的比值之和
	- 当HI小于1时，有害物质的累积风险是可接受的
	- 需要以共同毒性终点推算各化合物的健康指导值
	- 类似方法：累积风险指数(CRI)
	- 应用：内分泌干扰物

**遗传毒性致癌物评估方法**
-  线性外推法(linear approach)
- 暴露限值(margin of exposure)
- 毒理学关注阈值(TTC)：结构明确、数据不足、暴露很低的物质

**暴露评估的基本原则**
- 暴露评估时尽可能使用本国膳食消费数据和浓度数据
- 尽可能选择最适宜的数据（评估目的、国家水平）
- 建议进行分级评估（先筛选，再精确）
- 应当覆盖一般人群和易感或关键人群
- 必要时可进行合理假设和推断，任何假设都需要透明
- 对所有已知的食品中化学物，可使用相似的精确方法
