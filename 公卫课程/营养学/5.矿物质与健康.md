# 1.概述
**矿物质定义**：除碳、氢、氧、氮外的元素称为矿物质（无机盐/灰分），分为常量元素（宏量元素）和微量元素
**常量元素（宏量元素）**：含量＞体重0.01%，例如钙、磷、钠、钾、硫、氯、镁
**微量元素**：含量＜体重0.01%，例如铁、锌、铜、锰、碘、硒

**人体必需矿物质元素的三大分类**
根据1996年WHO/FAO联合讨论结果，21种必需矿物质元素被分为以下三类
- 必需微量元素：铁、铜、锌、硒、铬、碘、钴、钼
- 可能必需微量元素：锰、硅、镍、硼、钒
- 潜在毒性微量元素：氟、铅、镉、汞、砷、铝、锡、锂

**矿物质的特点**
- 无法体内合成，必须从外界补充
	- 人体无法自行合成矿物质，且每日通过排泄、皮肤代谢等途径流失，因此必须不断从饮食中补充
- 唯一可以通过天然水途径获取的营养素
	- 矿物质是唯一可通过天然水获取的营养素，但长期饮用矿物质超标水有中毒风险，如饮水型氟中毒
- 复杂的相互作用：协同与拮抗
	- 矿物质间存在相互影响，如过量铁或铜会抑制锌吸收，而铁可促进氟吸收
- 某些矿物质在体内生理剂量与中毒剂量范围较窄
	- 某些矿物质生理剂量与中毒剂量范围窄，如氟的适宜摄入量为1.5mg/d，可耐受最高摄入量仅为3.5mg/d
- 体内分布极不均匀
	- 矿物质在机体内的分布具有高度的组织和器官特异性，并非均匀分布。例如钙集中在骨骼和牙齿，铁主要在红细胞中，碘主要在甲状腺

**矿物质缺乏与过量的原因**
- 地球环境因素
	- 地壳中矿物质元素分布天然不均衡，导致某些地区土壤中特定元素含量过低或过高。人群长期摄入当地食物或水源，可能引发亚临床症状甚至地方病
- 食物成分及加工因素
	- 食物本身含天然拮抗物；加工过程会造成矿物质损失；加工设备或添加剂可能引入杂质，造成食品污染
- 摄入不足或消耗增加
	- 摄入不足（如厌食、挑食）或需求增加（如儿童、孕妇）会导致矿物质缺乏。而排泄功能障碍则可能导致矿物质在体内蓄积，引发毒性作用

**生理功能**
- 构建机体组织
	- 构成骨骼、牙齿及蛋白质等身体结构的“建筑材料”
- 调节体液平衡
	- 维持细胞内外渗透压、酸碱平衡和水分流动
- 维持神经兴奋
	- 确保神经信号传递和肌肉收缩的正常进行；如钾、钠、钙、镁等离子
- 激活酶的活性
	- 作为酶的“催化剂”，促进体内各种生化反应的顺利进行。如氯离子激活唾液淀粉酶，镁离子激活磷酸转移酶等
- 参与血液凝固
	- 保障机体在受伤后，血液能够正常凝固，防止过度失血，如钙离子

# 2.钙
## 2.1钙的理化特性
- 钙是自然界中分布最广泛的元素之一，约占地壳重量的3%。在人体的含量仅次于氧、碳、氢和氮，排第五位，是人体含量最多的矿物元素
- 化学性质活泼，能与水、酸反应
- 加热时，几乎能还原所有的金属氧化物，在自然界多以离子状态或化合物形式存在

## 2.3钙的消化吸收与代谢
**钙的消化吸收**
- 膳食中的钙大多以不可溶的复合物形式存在
- 吸收前提：钙的溶解
	- 膳食中的钙需经胃酸及酶作用，从复合物中游离出来，形成溶解状态才能被吸收
- 主要吸收部位与效率
	- 场所：小肠
	- 吸收率：约20%-60%（受多种因素影响波动）
- 两种主要吸收方式
	- 主动吸收（低摄入）：十二指肠/空肠上部，依赖维生素D主动转运
	- 被动吸收（高摄入）：空肠/回肠段，通过浓度梯度进行被动扩散
- 膳食钙经肠道吸收入血液的比例为钙的TFCA（True Fractional Calcium Absorption），简称“钙吸收率”
- 总体上，吸收率与骨骼增长速度呈正相关。各年龄段人群钙吸收率分别为
	- 婴儿早期为60%
	- 1~3岁幼儿约为45%
	- 4~8岁儿童为24%~36%
	- 9~17岁少儿为30%~50%
	- 成年人大多在26%~30%
	- 女性40岁以后钙吸收率随年龄增长而降低

==**有利钙吸收的膳食因素**==

| 影响因素      | 作用原理                                              |
| --------- | ------------------------------------------------- |
| 维生素D      | 促进肠粘膜对钙的吸收，促进溶骨过程使血钙升高；<br>促进肾小管对钙的重吸收            |
| 乳糖        | 乳糖与钙可形成可溶性低分子物质；<br>乳糖在肠道中可被乳糖酶分解为乳酸，降低肠内PH，利于钙吸收 |
| 低磷膳食      | 低磷膳食可降低血液磷的水平，刺激维生素D活化，促进钙吸收                      |
| 氨基酸（赖色精组） | 氨基酸可与钙结合形成可溶性盐，利于吸收，但高蛋白质饮食抑制                     |
| 酪蛋白磷酸肽    | 酪蛋白磷酸肽在小肠可与钙结合成可溶性钙盐                              |

**不利于钙吸收的膳食因素**

| 影响因素  | 作用原理                     |
| ----- | ------------------------ |
| 草酸、植酸 | 与钙形成不可溶性复合物              |
| 膳食纤维  | 膳食纤维中的醛糖酸残基与钙结合形成难吸收的复合物 |
| 脂肪酸   | 脂肪消化不良时，未被吸收的脂肪酸与钙结合形成钙皂 |
| 高钠饮食  | 高钠促进尿钙的排泄                |
| 乙醇    | 肠粘膜水肿及糜烂、肾功能损害，减少钙的重吸收   |

==**影响钙吸收的生理因素**==
- 增加
	- 维生素D营养状况适宜
	- 钙缺乏
	- 磷缺乏
	- 妊娠（40%；56%；60%）
	- 哺乳
	- 儿童（40%）
	- 运动
- 降低
	- 维生素D缺乏
	- 绝经
	- 老年（15%）
	- 胃酸降低
	- 吸收不良综合症
	- 肝硬化
	- 慢性肾功能衰竭等

**钙的分布与储存**
- 主要存在形式：羟磷灰石、无定形磷酸钙
- 结合钙：与柠檬酸螯合或与蛋白质结合
- 离子钙：以离子状态存在于软组织、细胞外液和血液中
- 99%骨骼和牙齿中（库）——1%混溶钙池(miscible calcium pool)

**钙的调节**
- 甲状腺释放降钙素，增加骨骼中钙的沉积、降低肠道钙的摄取、降低尿液中钙的重吸收，使钙水平下降
- 甲状旁腺释放甲状旁腺激素，增加骨骼中钙释放、增加肠道钙的摄取、增加尿液中钙的重吸收，使钙水平升高

**钙的排泄**
- 钙的排泄途径：粪便、尿液、汗液、乳汁
	- 粪便：约80%
	- 尿液：150~450mg/d（10%-20%）
	- 乳汁：150~300mg/d
- 影响钙排泄的因素
	- 机体因素
		- **血钙**浓度可调节尿钙排出量，当血钙浓度低时，钙重吸收率增加，尿钙显著减少，当严重低血钙时，甚至无尿钙排出。若血钙升高时，则尿钙排出增加
		- 婴儿尿钙很低，随**年龄**增加尿钙排出增多
		- **绝经期**尿钙排泄量增加，反映骨钙动员加快
		- **补液、酸中毒及甲状腺素和肾上腺皮质激素**等均可使钙排出增加
	- 膳食因素
		- **钙的摄入量**对尿钙的排泄量影响不大，主要影响**粪钙**的排泄
		- **钠**的摄入量影响尿钙的排泄，由于钠和钙在肾小管重吸收过程中存在竞争，当钠摄入增加，会相应减少钙的重吸收，而增加尿钙排泄
		- **咖啡因**的摄入会在一定程度上增加钙的排泄
		- 膳食**蛋白质**能够增加尿钙的排出，这与膳食蛋白质促进钙吸收相抵，蛋白质不会降低净钙潴留

## ==2.4钙的生理功能==
- 构成骨骼和牙齿的主要成分
- 维持神经和肌肉的活动
- 促进细胞信息传递
- 血液凝固
- 调节机体酶的活性
- 维持细胞膜的稳定性
- 其他功能

## 2.5钙的摄入水平与健康
==**钙摄入不足与健康**==
- 血钙过低正常生理状态下，机体不会出现体液和细胞内液钙的缺乏或过量。病理状态下可出现血钙过低，并导致神经的过度兴奋，引起腓肠肌痉挛和其他部位肌肉痉等
- 骨骼钙化不良与骨质疏松
- 钙摄入不足者易患龋齿，影响牙齿质量
- **骨质疏松症(Osteoporosis）**
	- 以骨量减少及骨组织微观结构退变为特征
	- 伴有骨脆性增加，易于发生骨折；为脊柱压缩性、前臂和髋骨骨折
	- 全身性骨骼疾病
		- 骨量减少：指骨矿物质（钙盐）和骨基质（蛋白质）等比例减少
		- 骨的显微结构退化：表现为骨小梁变细、变稀，乃至微骨折，导致骨关节痛
		- 骨的强度下降：骨的脆性增高，难以承载原来的负荷，易发生骨折
	- 影响骨质疏松的因素
		- 遗传、内分泌、年龄、性别、运动、机械负荷和营养因素
		- 营养因素：钙、维生素A、维生素D、蛋白质、膳食纤维、过量饮酒、过量饮用咖啡、碳酸饮料等
- 婴幼儿及儿童长期钙缺乏和维生素D不足可导致生长发育迟缓，骨软化、骨骼变形，严重缺乏者可导致佝偻病，出现“O” 形或“X”形腿、肋骨串珠、鸡胸等症状

**钙过量与健康**
- 高钙血症与高钙尿症
- 奶碱综合征

**钙摄入水平与其他健康影响**
- 骨骼健康
- 心脑血管健康
- 肾结石
- 死亡

## 2.6钙的营养学评价
- 膳食调查
- 生化指标
	- 血钙浓度变化范围很小，尿钙易受到体内PTH和近期膳食因素的影响，个体差异大因此，血钙和尿钙均不适合用于评价体内钙的营养状况
- 钙平衡
	- 当钙的摄入量与排出量（粪钙+尿钙+汗液）的差值为零时为钙平衡，负值则为负平衡，正值为正平衡
	- 缺点是试验操作烦琐，不易实施，很难获得精确数据
- 骨质的测量
	- 骨矿物质含量
	- 骨矿物质密度
- 钙相关骨病

## 2.7膳食钙参考摄入量与食物来源
适宜摄入量(AI) mg/d：成年人800mg/d，老年人（＞50）1000mg/d
可耐受最高摄入量(UL)mg/d：2000mg/d
**食物来源**：钙的主要食物来源为奶及其制品、豆腐、叶菜、花菜和豆荚及辣椒类蔬菜、贝壳类和鱼类、柑橘类水果等。但不同食物钙吸收率存在差异

# 3.铁
**理化特性**
- 铁(iron)是人体重要的必需微量元素，含量最多的必需微量元素
- 体内铁的水平随年龄、性别、营养状况和健康状况的不同而异
- 正常人体内含铁总量约30~40mg/(kg·bw），4~5g
- 人体内的铁有两种存在形式：功能性铁和储存铁
	- 功能性铁
		- 65%~70%的铁存在于血红蛋白
		- 3%在肌红蛋白
		- 1%在含铁酶类（如细胞色素、细胞色素氧化酶、过氧化物酶、过氧化氢酶等）、辅助因子及运铁载体中
	- 储存铁（25%~30%）
		- 主要以铁蛋白(femitin)和含铁血黄素(hemosiderin)形式存在于肝、脾和骨髓的网状内皮系统中

**消化吸收**
- 吸收：食物中的铁吸收主要在**十二指肠和空肠上端**，胃和小肠的其余部分也吸收少量的铁
	- 食物中的铁：**血红素铁和非血红素铁**
		- 血红素铁
			- 主要来自动物性食物
			- 以含铁卟啉复合物的形式被吸收进入小肠黏膜上皮细胞，在胞浆内血红素加氧酶的作用下血红素的卟啉环打开，释放出二价铁离子
			- 血红素铁的吸收率受膳食因素影响较小
		- 非血红素铁
			- 非血红素铁主要存在于植物性食物中，常常以三价铁离子的形式存在
			- 膳食中的三价铁离子需由细胞色素B还原为二价铁形式被吸收
			- 非血红素铁受膳食因素影响较大
- **==影响铁吸收的因素==**
	- 机体因素
		- 促进因素
			- 贫血、孕期、生长发育可使铁的需求增加
			- 月经过多，钩虫感染，痢疾，血吸虫病等因铁丢失增加，促进机体增加铁的吸收
			- 胃酸有利于食物中铁的游离，胃肠道分泌的黏蛋白及胆汁对铁有稳定和促进吸收的作用
		- 抑制因素
			- 某些疾病如萎缩性胃炎、胃酸缺乏或过多服用抗酸药物时，抑制铁离子释放
			- 碱性的胰腺分泌液中的碳酸氢盐与铁结合容易形成不易溶解的复合物，不利于铁的吸收
	- 膳食因素
		- 促进因素
			- 膳食铁存在形式：血红素铁生物利用高，有效吸收率为15%~35%；非血红素铁有效吸收率仅2%~20%
			- 蛋白质类食物能够刺激胃酸分泌，促进铁的吸收
			- 氨基酸，如组氨酸、赖氨酸、胱氨酸、蛋氨酸、酪氨酸与铁螯合成小分子的可溶性单体，可提高铁的吸收
			- 维生素C是铁吸收的有效促进因子，维生素A、叶酸、维生素B12、维生素B2等对铁吸收起到重要协助作用
			- 柠檬酸、乳酸、丙酮酸、琥珀酸以及酒石酸等可促进铁的吸收
		- 抑制因素
			- 铅、铬、锰等矿物质过多摄入阻碍机体对铁的吸收
			- 一些金属络合物如EDTA也有阻碍机体对铁吸收的作用
			- 非营养素成分，如植酸、丹宁、多酚物质与铁结合能力较强，是阻碍铁吸收的重要因素
	- 其他
		- 肠道维生素的某些分解产物可抑制铁的吸收

==**生理功能**==
- 参与体内氧的运送和组织呼吸过程
- 维持正常的造血功能
- 参与其他重要功能
	- 参与维持正常的免疫功能，缺铁可引起机体感染性增加，白细胞的杀菌能力降低，淋巴细胞功能受损
	- 过量铁可促进细菌的生长，对抵抗感染不利
	- 铁可催化B-胡萝卜素转化为维生素A、嘌呤与胶原蛋白的合成，脂类在血液中转运以及药物在肝脏解毒等方面均需铁的参与
	- 铁与抗脂质过氧化有关

**摄入不足与健康**
- 铁摄入不足或缺乏的原因
	- 食物铁摄入不足
	- 生物利用率较低
		- 粮谷类及蔬菜中的植酸盐与草酸盐
		- 茶、咖啡中的酚类化合物
		- 膳食纤维摄入过多
		- 胃酸缺乏或过多服用抗酸药
		- 蛋类卵黄高磷蛋白摄入多
	- 铁需要量增加
		- 婴儿期尤其是低出生体重儿童
	- 铁吸收减少
		- 胃肠道黏膜异常
		- 铁的存在形式
- 后果：小细胞低色素性贫血（缺铁性贫血）

**营养学评价**：当机体缺铁时，铁损耗及其危害是一个从轻到重的渐进过程，一般可分为三个阶段（血清铁蛋白浓度<20μg/L 诊断铁缺乏）
- 铁减少期：体内储存铁减少以致耗竭
- 红细胞生成缺铁期：红细胞内铁供给受限，无贫血的铁缺乏期
- 缺铁性贫血期(iron deficiency anemia, IDA)：血红蛋白下降

==**缺铁性贫血的临床表现**==
- 常见症状：疲乏无力、心慌、气短、头晕、眼花，严重者出现面色苍白、口唇粘膜和睑结膜苍白、肝脾轻度肿大等
- 影响生长发育：身体发育与智力发育不良，近、远期神经功能和心理行为障碍
- 皮肤毛发：毛发干枯脱落，指（趾）甲缺乏光泽、变薄、脆而易折
- 活动和劳动耐力降低
- 机体免疫功能和抗感染能力下降
- 消化道改变：萎缩性胃炎
- 神经精神系统异常
- 抗寒能力降低
- 其他：月经紊乱

**婴幼儿铁缺乏临床表现**
- 一般表现
	- 在轻度或中度缺铁性贫血时（血红蛋白60-100g/L），患儿可能有烦躁、喜食冰块、异食癖等异常表现
	- 当血红蛋白＜50g/L时，患儿烦躁和厌食的表现更为突出
- 生长发育异常
	- 因生长速度越快，所需的铁量越多，因此生长速度较快的超重肥胖儿童更容易出现铁缺乏和轻中度缺铁性贫血，但严重缺铁性贫血对儿童生长也有不良影响，重度缺铁性贫血患儿常伴有其他营养素缺乏的证据
- 血液循环系统表现
	- 贫血可引起骨骼外造血增加，长期重度缺铁性贫血患儿的肝、脾、淋巴结增大，贫血时间越长，程度越重，但一般不超过中度肿大
	- 重度缺铁性贫血患儿可出现心率增快、气急、心脏扩大，伴有收缩期杂音，如同时合并呼吸道感染时，易发生心力衰竭
- 神经系统表现
	- 铁缺乏在引起缺铁性贫血前，已经影响到婴幼儿的注意力、警觉性以及学习能力
	- 缺铁性贫血患儿在给予铁剂治疗后，行为改善常常发生在血液学明显改善以前，但补充铁剂不能完全逆转铁缺乏所照成的损害
	- 此外，铁缺乏或缺铁性贫血幼儿易出现屏气发作
- 其他
	- 在铅污染地区，铁缺乏或缺铁性贫血患儿的肠道铅吸收增加，更容易发生铅中毒
	- 缺铁性贫血患儿的细胞免疫功能下降，易发生各种感染，且迁延不愈，反复发生

**缺铁性贫血的治疗处理**

| 贫血程度       | 处理                                        |
| ---------- | ----------------------------------------- |
| 铁缺乏和轻、中度贫血 | 口服铁剂治疗为主，并改善饮食，进食富含铁的食物                   |
| 重度贫血       | 口服铁剂或注射铁剂治疗，还可以少量多次输注浓缩红细胞                |
| 极重度贫血      | 首选输注浓缩红细胞，待Hb达到70g/L、症状改善后，可改为口服铁剂或注射铁剂治疗 |
- **治疗**
	- 一般治疗：对重症小儿宜加强护理
	- 病因治疗：增加铁丰富、铁吸收率高、富含维生素C及蛋白质食物
	- 铁剂治疗：铁剂应与维生素C同服；总疗程2~3月，血红蛋白恢复正常后再服用1~2个月增加铁储存量
	- 隐性铁缺乏的早期治疗：小儿每日每公斤体重需要1mg，最大量不超过12mg
	- 铁剂与补铁方式：口服、肌注或静脉铁剂
- 预防
	- 三类人群的缺铁预防措施
		- 小儿缺铁预防：母乳喂养、蛋黄、瘦肉、肝泥等富铁辅食
		- 成人缺铁预防：补铁同时适当给予蛋白质和VC食物
		- 妊娠期缺铁预防：膳食补充、口服铁剂
	- 预防铁缺乏的方案
		- 健康教育
		- 铁强化食品
		- 铁补充
		- 提高食物铁的利用率：增加VC，降低植酸、酚类化合物

# 4.锌
**理化特性**
- 锌是一种强电子接受体，与硫醇和胺的电子供体的结合力很强，具有快速的配体交换作用，在金属酶的催化作用中有重要意义
- 细胞内锌的化学性质主要涉及硫醇和咪唑配体，锌与硫醇的结合是调节锌细胞水平的重要机制

**消化吸收**
- 成人体内锌（zinc）的含量男性约2.5g，女性1.5g，锌分布于人体所有的组织、器官、体液及分泌物。约60%存在于肌肉，30%存在于骨骼中。在细胞中，30%~40%的锌存在于细胞核中，50%存在于细胞质，其余的存在于细胞膜中
- 口服锌的吸收主要在十二指肠和近侧小肠处，锌与小分子肽构成复合物后，主要经主动转运机制被吸收，回肠也有部分吸收，吸收率为30%左右
- 影响锌吸收的因素
	- 机体因素
		- 机体锌的营养状态，体内锌浓度高时可诱导肝脏金属硫蛋白合成增加，并与之结合存积于肠粘膜细胞内，当锌水平下降时，再释放至肠腔，以此调节体内锌的平衡
		- 促进因素：特殊生理阶段，如孕期、哺乳期锌的吸收率增加
		- 抑制因素：疾病状态导致机体吸收利用减少，如吸收障碍、肠胃功能紊乱，慢性肝肾疾病，贫血，恶性肿瘤等疾病状态
	- 膳食因素
		- 促进因素
			- 组氨酸、甲硫氨酸、半胱氨酸、维生素D3，葡萄糖可促进锌的吸收
			- 动物性食物中锌的生物利用率较高
			- 某些药物如碘喹、苯妥英钠均能促进锌的吸收
			- 锌与低分子量配体（如某些氨基酸和有机酸）或螯合物（如乙二胺四乙酸（EDTA））形成复合物时，可促进锌的吸收
			- 食物中蛋白质含量增加可提高锌的吸收和生物利用率
		- 抑制因素
			- 铁锌比很高时，铁会抑制锌的吸收
			- 膳食纤维、植酸可减少锌的吸收
			- 铜、钙、亚铁离子可抑制锌的吸收

**生理功能**
- 锌具有催化功能、结构功能和调节功能。锌通过约2800种蛋白质和酶，在人体发育、认知行为、创伤愈合、味觉、免疫调节等方面发挥着重要作用
- 金属酶的组成成分或酶的激活剂
- 促进生长发育
- 促进机体免疫功能
- 维持细胞膜结构

**锌缺乏临床表现**
- 轻度锌缺乏：生长迟缓腹泻、反复呼吸道或胃肠道感染、食欲低下、味觉异常、脱发等，症状都没有特异性
- 由肠病性肢端皮炎引起的严重锌缺乏
	- 可在婴儿出生几个月时出现进行性、致死性的严重表现：皮肤水疱、湿疹、干燥、鳞屑或类似银屑病的皮损，对称地分布于口周、肢端、会阴区以及脸颊、膝盖和肘部
	- 头发往往呈奇特的红色，有一定程度的脱发
	- 眼部表现包括畏光、结膜炎、睑缘炎、以及通过裂隙灯检出的角膜营养不良
	- 可伴有慢性腹泻、口腔炎、舌炎、甲沟炎、指甲营养不良、生长发育迟缓、烦躁不安，伤口延迟愈合、并发细菌感染以及白色念珠菌感染等
	- 如果不进行治疗，病程缓慢呈间歇性，但往往不断进展
- 暂时性新生儿锌缺乏：见于纯母乳喂养婴儿，出生时无异常，皮肤表现与肠病性肢端皮炎相似，生长迟缓

**锌缺乏原因与风险因素**
- 摄入不足
	- 母乳初乳中锌含量高（＞3mg/L），但随哺乳期延长，母乳锌含量持续下降，至婴儿6月龄时母乳锌含量＜1mg/L，难以满足婴儿需求。如未及时加富含锌的肉类等食物，6-24月龄婴幼儿容易出现锌缺乏
- 储存不足
	- 胎儿锌储存主要在妊娠后期，早产儿易发生锌缺乏。早产/低出生体重儿的追赶性生长需要更多的锌
- 异常丢失
	- 人体锌可通过肠道排出，而腹泻可加速锌的排出。感染也可致体内锌的异常丢失。经常腹泻、反复呼吸道感染的婴幼儿，由于锌丢失增加也容易出现锌缺乏
- 遗传性疾病
	- 肠病性肢端皮炎

**过量与健康**
- 过量
	- 盲目过量补锌或食用因镀锌罐头污染的食物和饮料可引起锌过量或锌中毒
	- 过量的锌可干扰铜、铁和其他微量元素的吸收和利用，影响中性粒细胞和巨噬细胞活力，抑制细胞杀伤能力，损害免疫功能
	- 成人摄入4-8g以上锌可观察到毒性症状，引起发烧、腹泻、恶心、呕吐和嗜睡等临床症状
- 与慢性病及其他疾病的关系
	- 一些研究报道锌摄入有利于减少某些疾病风险，但膳食锌、锌补充剂与总死亡、心血管疾病、糖尿病、认知功能障碍的关系以及对孕妇和婴儿健康的有益作用还需更充足的循证医学证据

**营养状况评价**
- 膳食评估
	- 考虑食物锌受地域水土影响很大，食物成分表的应用要谨慎
	- 膳食锌摄入量用于发现处于缺锌危险的个体和群体，但锌的吸收受膳食植酸等因素的影响
- 临床症状
	- 生长缓慢、皮肤伤口愈合不良、味觉障碍、胃肠道疾患发病率增加、免疫功能减退等
- 生化指标
	- 血浆锌：机体血浆含量相对稳定，受多种生理病理状态影响，只有严重锌缺乏时才具有诊断意义；对于边缘性或轻、中度锌缺乏时不建议作为个体的诊断指标。而血浆锌可作为锌缺乏的生物标志物来评估群体锌营养状况
	- 发锌：虽然发锌与人体锌营养状况相关性较好，但由于发锌含量受头发生长速度、洗护发、烫染等的影响，世卫组织认为发锌不再作为判断个体锌营养状况的可靠指标
	- 尿锌：尿锌含量与人体锌营养状况相关性好，且灵敏度高于血浆锌。但受尿量和近期饮食的影响，且尚无测定的标准方法，因此只作为参考指标
	- 唾液锌：味觉敏感度降低是锌缺乏早期症状，唾液锌和味觉敏感度的相关性很好，且唾液采样方便，可作为判断个体锌营养状况的参考指标
- 功能指标
	- 通过含锌酶活性、味觉、暗适应能力等的变化对锌功能进行评价。如血浆碱性磷酸酶是评价锌营养状况常用指标，但由于其缺乏特异性而使应用受到限制。目前国内外越来越多的实验证明单核细胞金属硫蛋白mRNA可靠性较好，是反映边缘性锌缺乏的良好指标，被认为是评价锌营养状况的相对金标准

**锌的食物来源**
- 锌在食物中广泛存在，但食物中锌含量差别很大，吸收利用率也不相同
- 锌含量较高的食物依次是肉类、蛋类、豆类和水产类
- 谷类和乳制品也含有较丰富的锌
- 我国居民膳食锌的主要来源为谷类和肉类，占总摄入来源的70%以上。过细的加工过程可导致大量的锌丢失，如小麦加工成精面粉约丢失80%锌

# 5.硒
**理化特性**
- 单质硒有灰色金属光泽。硒具有多种化学价态，包括-2，0，+4，+6，其中，+4和+6最为常见
- 可构成各种特性的无机和有机硒化物，后者如硒半胱氨酸和硒蛋氨酸

**消化吸收**
- 吸收
	- 硒主要在**小肠**吸收，人体对食物中硒的吸收良好
	- 硒吸收与硒的化学结构和溶解度有关，硒蛋氨酸较无机形式硒更易吸收，溶解度大的硒化合物比溶解度小的更易吸收
	- 硒在体内的吸收、转运、贮存和分布、排出会受许多外界因素影响，主要是膳食中硒的化学形式和量；性别、年龄、健康状况以及食物中是否存在硫、重金属、维生素等化合物也影响硒在体内的吸收与代谢
	- 总体来讲，膳食中的硒吸收率都较高。吸收率受机体硒营养状态影响较小
- 代谢
	- 人体硒总量约为14~21mg
	- 硒存在于所有细胞与组织器官中，在**肝脏、肾脏胰腺**、心脏、脾脏、牙釉质和指甲中浓度较高，肌肉、骨骼和血液中次之，脂肪组织最低
	- 体内大部分硒主要以两种形式存在，一种是来自膳食的硒蛋氨酸，它在体内不能合成，作为一种非调节性储存形式存在，当膳食中硒供给中断时，硒蛋氨酸可向机体提供硒； 另一种形式是硒蛋白中的硒半胱氨酸为具有生物活性的化合物
	- 体内的硒经代谢后大部分经**尿**排出，少量从肠道排出，粪中排出的硒大多为未被吸收的硒。硒摄入量高时可在肝内甲基化生成挥发性二甲基硒化合物由肺部呼气排出。此外，少量硒也可从汗液、毛发排出

**硒的生理功能**
- 抗氧化功能
	- 硒是谷胱甘肽过氧化物酶(glutathione peroxidase, GSH-Px)的组成成分，GSH-Px具有抗氧化功能，可清除体内脂质过氧化物，阻断活性氧和其他自由基对机体的损伤作用
- 保护心血管和心肌的健康
	- 硒缺乏还可引起脂质过氧化反应增强，导致心肌纤维坏死、心肌小动脉和毛细血管损伤，同时高硒地区人群中的心血管病发病率较低
- 增强免疫功能
	- 通过上调白细胞介素-2(interleukin-2, IL-2)受体表达，使淋巴细胞、NK细胞、淋巴因子激活杀伤细胞(lymphokine-activated killer cells, LAK cel)的活性增加，提高免疫功能
- 有毒重金属的解毒作用
	- 硒与金属有较强的亲和力，能与体内重金属如汞、镉、铅等结合成金属-硒-蛋白质复合物而起到解毒作用，并促进有毒金属排出体外
- 调节甲状腺激素
	- 3种碘化甲腺原氨酸脱碘酶的必需组分，可通过调节甲状腺激素水平进而影响机体全身代谢
- 保障男性生殖健康
	- 谷胱甘肽过氧化物酶4(GPx4)在精子发生发展的过程中发挥重要作用
- 参与细胞铁死亡
	- 最近有研究表明，GPx4为细胞铁死亡(ferroptosis)发生机制的重要参与者
- 其他
	- 硒还具有促进生长、抗肿瘤的作用。人群流行病学调查发现硒缺乏地区的肿瘤发病率明显增高。研究发现，硒缺乏可引起生长迟缓及神经性视觉损害，经补硒可改善视觉功能障碍

**摄入不足或过量与健康**
- 缺乏
	- 我国科学家首先证实缺硒是发生**克山病**的重要原因
	- 另外，缺硒被认为是发生**大骨节病**（Kashin-Beck disease）的重要原因，该病主要是发生在青少年期。缺硒还可影响机体的免疫功能，包括细胞免疫和体液免疫。补硒可提高宿主抗体和补体的应答能力
	- 若每日膳食硒摄入量不足，机体各种硒蛋白无法充分表达，会影响机体诸多生理功能的正常发挥，如甲状腺激素代谢、免疫和生殖
- 过量
	- 过量的硒可引起中毒，20世纪60年代，中国恩施地区水土中含硒量高，以致生长的植物含有大量硒，当时居民从膳食中平均每天摄入硒达到300mg，而发生**慢性硒中毒**。其中毒症状为头发和指甲脱落，皮肤损伤及神经系统异常，肢端麻木、抽搐等，严重者可致死亡
	- 近十多年来，大量证据表明，硒的摄入量超过生理需要量可增加患T2DM的风险。系统综述和荟萃分析表明，硒的每日摄入量达到80μg以上就有可能出现T2DM的风险升高
	- 国内外早期的动物和人体试验研究表明，硒具有显著的抗癌活性

**硒的营养学评价**
- 生化检测
	- 通过测定全血、血浆、红细胞、发、尿、指（趾）甲等组织的硒含量，评价硒的营养状况
	- 杨光圻等通过对中国不同硒水平地区膳食硒摄入量、血浆硒和发硒等的测定，提出适用于中国以谷类为主食的膳食硒摄入量对数回归方程式
		- Log膳食硒摄入量（μg/d）=1.304Log全血硒(mgL)+2.931
		- Log膳食硒摄入量（μg/d）=1.624Log血浆硒(mgL)+3.389
		- Log膳食硒摄入量（μg/d）=1.141Log发硒(mg/kg)+1.968
	- 根据以上公式，可用全血或血浆或发硒测定值来推算膳食硒摄入量
	- 由于不同地区土壤中硒含量不同，使不同地区同品种食物中硒含量也不同，因而，食硒摄入量不宜使用食物成分表中的数值来计算，只能用当地各种食物硒含量实际测定值来计算
- 谷胱甘肽过氧化物酶(GSH-Px)活性测定
	- GSH-Px是含硒酶，代表硒在体内的活性形式，因此，通过测定红细胞中的GSH-Px活力，可直接反映硒营养状况
	- 随着硒含量增加，GSH-Px活性也增高，但当血硒达到1.27μmol/L(0.1mgL)时，GSH-Px活性达饱和而不再升高，因此，以GSH-Px活性作为评价指标时，仅适用于低于正常硒水平人群
- 其他
	- 血浆硒蛋白酶-P(Sel-P)，某些组织中的抗氧化酶(TR)活性和硒蛋白酶-W(Sel-W)可作为硒的营养评价指标

**食物来源**
- 海产品和动物内脏是硒的良好食物来源，如鱼子酱、海参、牡蛎、蛤蛎和猪肾等
- 食物中的含硒量随地域不同而异，特别是植物性食物的硒含量与地表土壤层中硒元索的水平有关
