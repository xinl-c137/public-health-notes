# 1.饮用水的卫生学意义
饮用水是指供人生活的饮水和生活用水。2021年中国水资源公报显示，我国城镇人均生活用水量176L/d（占全国用水量的15.4%）。发达国家城市居民用水量达500L/d以上
- 生理需要：2.5~3L
- 日常生活关系密切：个人卫生、改善生活居住环境
- 存在问题严重：重金属和有机污染物污染

# 2.饮用水与健康
80%疾病与水有关
- 介水传染病
	- 介水传染病：指通过饮用或接触病原体污染的水而传播的疾病
	- 病原体分为3类
		- 细菌：伤寒与副伤寒杆菌、霍乱与副霍乱弧菌、痢疾杆菌、致病性大肠杆菌
		- 病毒：甲肝病毒，脊髓灰质炎病毒，柯萨奇病毒、腺病毒、轮状病毒
		- 原虫：贾地鞭毛虫、隐孢子虫、溶组织阿米巴虫
	- 介水传染病流行特点
		- 水源污染后，可呈暴发流行，短期内出现大量病人
		- 病例分布范围与供水范围一致
		- 对污染源采取措施，疾病流行能迅速控制
	- 病因
		- 水源受病原体污染后，未经妥善处理和消毒即供居民饮用
		- 净化消毒后的饮用水在输配水和贮水过程中，重新被病原体污染
- 化学性污染中毒
- 其他
	- 消毒副产物
		- **氯化消毒副产物**系指在氯化消毒过程中氯与水中的有机物（如腐殖酸、富里酸、藻类及蛋白质等）反应所产生的卤代烃类化合物
			- 挥发性卤代有机物主要有**三卤甲烷**(trihalomethane, **THM**)，包括氯仿、一溴二氯甲烷、二溴一氯甲烷和溴仿
			- 非挥发性卤代有机物主要有**卤代乙酸**(haloaceticacids, **HAA**)，如氯乙酸、二氯乙酸、三氯乙酸、溴乙酸、二溴乙酸、三溴乙酸、溴氯乙酸、二溴一氯乙酸、二氯一溴乙酸等
			- THM+HAA构成80%的消毒副产物
		- 氯化消毒副产物与肿瘤
			- 动物实验中证明具有致突变性和/或致癌性，致畸性和神经毒性可引起肝、肾和肠道肿瘤
	- 藻类及其代谢产物
		- 富营养化过程中，大量藻类生长，产生藻毒素
	- 内分泌干扰物
	- **二次供水**
		- 指供水单位将来自集中式供水或自备水源的生活饮用水，通过多种形式输送至水站或用户的供水系统中
			- 传统方式：增压设备和高位水池联合供水
			- 新方式：变频调速供水及叠压供水
		- 二次供水污染原因
			- 管道腐蚀、结垢
			- 水箱未定期清洗、消毒
			- 上下管道配置不合理
			- 水箱设计不合理

# 3.生活饮用水标准及其用水量标准
**生活饮用水标准制定标准的原则**
- 水中不得含有病原微生物
- 所含化学物质及放射性物质对人体健康无害
- 水的感官性状良好，不改变被洗物的理化性状
- 在选择指标和确定标准限量值时要考虑经济技术上的可行性

**《生活饮用水卫生标准》**
规定的水质检测指标数从106项减少至97项(新标准将原标准中的“非常规指标”调整为“扩展指标”)，包括常规指标43项和扩展指标54项
- 微生物学指标由6项减至5项：考虑大肠埃希氏菌比耐热大肠菌群有更强的指示作用，将耐热大肠菌群这一指标彻底删除
- 增加感官性状：2-甲基异莰醇、土臭素（与藻类爆发有关）
- 关注消毒副产物：一氯二溴甲烷、二氯一溴甲烷、三溴甲烷、三卤甲烷、二氯乙酸、三氯乙酸6项消毒副产物指标从非常规指标调整到常规指标；考虑到氨（以N计）的浓度对消毒剂的投加有较大影响，将其从非常规指标调整到常规指标
- 关注风险变化：增加了乙草胺和高氯酸两项扩展指标：其中，乙草胺是目前我国使用量最大的除草剂之一，在我国的使用历史有20多年，具有明显的环境激素效应，能够造成动物和人的蛋白质、DNA损伤，脂质过氧化；高氯酸盐广泛应用于烟火、军工、燃料、航天、纺织、冶炼等行业，对甲状腺功能有较强的干扰作用，影响人体发育
- 水质参考指标由原标准的28项调整为55项
- 新标准调整了标准正文中8项指标的限值，包括硝酸盐（以N计）、浑浊度、高锰酸盐指数（以O2计）、游离氯、硼、氯乙烯、三氯乙烯、乐果、硼标准限值从0.5mg/L放宽至1mg/L（欧盟1mg/L，世界卫生组织2.4mg/L）
- 主要特点
	- 指标减少9项
	- 关注消毒副产物指标
	- 增加感官形状和暴露增加指标
	- 指标分为常规指标（43项）和扩展指标（54项）

# 4.集中式给水
**定义**：是指由水源集中取水，经统一净化处理和消毒后，通过输水管和配水管网送到用户的供水方式
其供水方式有两种，即城建部门建设的各级自来水厂和由各单位自建的集中式供水方式
**优点**
- 有利于水源的选择和防护
- 易于采取改善水质的措施，保证水质良好
- 用水方便
- 便于卫生监督和管理

**水源选择的原则**
- 水量充足
- 水质良好
- 便于防护
- 技术经济合理

**水的净化和消毒**
生活饮用水的净化处理有常规净化、深度净化、特殊净化三种
- 常规净化工艺过程包括混凝沉淀（或澄清）-过滤-消毒。目的是除去原水中的悬浮物质、胶体颗粒和细菌等
- 深度净化是指在市政供水原有常规净化的基础上，对水质再进行净化处理
- 若原水中含铁、锰、氟等，则需特殊处理

**常规净化**
- 混凝沉淀
	- 通过添加混凝剂，将难以自然沉淀的细小颗粒，特别是胶体微粒去除的过程称为混凝沉淀
	- 混凝原理
		- 压缩双电层作用
		- 电性中和作用
		- 吸附架桥作用
	- 用化学物质来澄清浑水称为混凝。所加入的物质称为混凝剂。有些混凝剂本身在澄清浑水中只起辅助作用，称为助凝剂
		- 混凝剂 ：铝盐 、铁盐 、聚合氯化铝 、聚丙烯酰胺
		- 助凝剂：调节或改善混凝条件，改善絮凝体结构
	- 影响混凝效果的因素主要有：
		- 水中微粒的性质和含量
		- 水温：水温低时，絮凝体形成慢且细小、松散
		- 水的pH
		- 水中有机物和溶解盐含量
		- 混凝剂的种类和用量
		- 混凝剂的投加方法、搅拌强度和反应时间等
- 过滤
	- 以具有孔隙的粒状滤料层，如石英砂等，截留水中的杂质和微生物等，从而使水获得澄清的工艺过程
	- 过滤的功效
		- 使滤后水的浊度达到生活饮用水水质标准的要求
		- 去除水中大部分病原体，如致病菌、病毒以及寄生原虫和蠕虫等。特别是阿米巴包囊和隐孢子虫卵囊，它们对消毒剂的抵抗力很强，主要靠过滤去除
		- 水经过滤后，残留的微生物失去了悬浮物的保护作用，为滤后消毒创造了条件
- 消毒
	- 是指杀灭水中病原微生物的措施
	- 目前我国用于饮用水消毒的方法主要有氯化消毒、二氧化氯消毒、紫外线消毒和臭氧消毒等
	- 氯化消毒
		- 是指用氯或氯制剂进行饮水消毒的一种方法
		- 供饮用水消毒的氯制剂主要有液氯、漂白粉、漂白粉精和有机氯制剂等
		- 有效氯：指含氯化合物中具有杀菌能力的有效成分，含氯化合物分子团中氯的价数大于-1者均为有效氯
		- 消毒机制
			- 杀菌机制
				- 次氯酸体积小，电荷中性，容易穿过细胞壁
				- 强氧化剂，损坏细胞膜，影响酶系统
			- 病毒作用
				- 损害核酸
		- **影响氯化消毒效果的因素**
			- 加氯量和接触时间
				- 为保证消毒效果，加氯量必须超过水的需氯量，使在氧化和杀菌后还能剩余一些有效氯，称为“**余氯**”
				- 余氯有两种，一种为游离氯如HOCl和OCl<sup>-</sup>，另一种为化合氯如 NH<sub>2</sub>Cl和NHCl<sub>2</sub>
				- 一般要求氯加入水中后，接触30分钟，有0.3～0.5mg/L的游离氯，而对化合性余氯则要求接触1～2h后有1～2mg/L余氯
			- 水的pH值不宜太高，影响次氯酸的解离。HOCL 的杀菌效率较高
			- 水温
				- 水温每提高10℃，病菌杀灭率约提高2～3倍
			- 水的浑浊度
			- 水中微生物的种类和数量
	- 二氧化氯消毒
		- 常温下为橙黄色气体，带有刺激性的辛辣味
		- 易溶于水，但不和水起化学反应
		- 在水中极易挥发
		- 当空气中ClO<sub>2</sub>浓度大于10%或水中浓度大于30%时，具有爆炸性
		- ClO<sub>2</sub>在水中的消毒有其独特的优点
			- 杀菌效果好，用量少，消毒作用时间长
			- 可减少水中三卤甲烷（THMs）等氯化副产物的形成
			- 当水中含氨时不与氨反应，其氧化和消毒作用不受影响
			- 消毒作用不受水质酸碱度的影响
			- 消毒后水中余氯稳定持久，防止再污染的能力强
			- 可除去水中的色和味，不与酚形成氯酚臭
			- 对铁、锰的除去效果较氯强
		- 缺点
			- ClO<sub>2</sub>具有爆炸性
			- 必须在现场制备，立即使用
			- 制备ClO<sub>2</sub>较复杂，其成本较其它消毒方法高
			- ClO<sub>2</sub>的歧化产物对动物可引起溶血性贫血和变性血红蛋白症等中毒反应
	- 臭氧消毒
		- O<sub>3</sub>是极强的氧化剂，在水中的溶解度大。O<sub>3</sub>极不稳定，需在临用时制备，并立即通入水中
		- O<sub>3</sub>加入水后即放出新生态氧〔O〕，其杀菌和除病毒以及氧化有机物作用均很强
		- O<sub>3</sub>用于饮水消毒的投加量一般不大于1mg/L，要求接触10~15min，剩余O<sub>3</sub>不低于0.4mg/L
		- 优点
			- 消毒效果较ClO<sub>2</sub>和Cl<sub>2</sub>好
			- 用量少
			- 接触时间短
			- pH在6～8.5内均有效
			- 有除臭、色、铁、锰、酚等多种作用
			- 不产生三卤甲烷；用于前处理时尚能促进絮凝和澄清，降低混凝剂用量
		- 缺点
			- 投资大，费用较氯化消毒高
			- 水中O<sub>3</sub>不稳定，控制和检测O<sub>3</sub>需一定的技术
			- 消毒后对管道有腐蚀作用，故出厂水无剩余O<sub>3</sub>，因此需要第二消毒剂
			- 与铁、锰、有机物等反应，可产生微絮凝，使水的浊度提高
	- 紫外线消毒
		- 波长200-295nm紫外线具有杀菌作用
		- 原理：当微生物被照射时，紫外线可透入微生物体内作用于核酸、原浆蛋白与酶，使其发生化学变化而造成微生物死亡
		- 紫外线用于消毒的设备有两种，即浸入式和水面式，浸入式消毒效率较高。不管何种形式，消毒时要求原水色度和浊度要低，水深最好不要超过12cm
		- 优点：接触时间短、杀菌效率高、不影响水的嗅味
		- 缺点：消毒后无持续杀菌作用，价格贵

**深度净化**
- 饮用水深度净化是指在市政供水原有常规净化的基础上，对水质再进行净化处理
- 深度净化的方式
	- 分散式，如：矿化、磁化和纯水等家用净化器
	- 集中式，由专用管道入户或居民在净水站汲取，供饮用和做饭用
- 处理方法
	- 活性炭吸附法
	- 生物滤塔预处理的活性炭深度处理
	- 膜过滤法

**水质的特殊处理**
- 除氟：活性氧化铝吸附法；骨钙法
- 除铁和除锰：曝气充氧+过滤
- 除藻和除臭 ：硫酸铝、活性炭等

# 5.分散式给水
井水、泉水、地表水、雨雪水、**包装饮用水**、**直饮水**、淡化水

**包装饮用水**
**定义**：包装饮用水是指密封于符合食品安全标准和相关规定的包装容器中，可直接饮用的水。包装饮用水的水源可直接来源于地表、地下或公共供水系统
根据《饮料通则》（GB10789-2015），可分为三类
- 饮用天然矿泉水
- 饮用纯净水
- 其他类饮用水

**瓶（桶）装水**
包装饮用水根据包装容器外形，分为桶装水和瓶装水
- 纯水：除水分子外，基本上没有其它化学成分。市场上商品名称称之为纯净水、太空水、蒸馏水等
- 净水：以市政自来水为原水通过吸附、超滤以去除水中有害物质而保留原水的化学特征，即保留原水中的溶解性矿物质
- 天然矿泉水：储存于地下深处自然涌出或人工采集的未受污染且含有多种微量元素的泉水，经过过滤等工艺而成
- 人工矿化水：在纯水中加入某些微量元素，使其某一微量元素达到天然矿泉水的限量值
- 桶装水的卫生状况
	- 生产罐装过程、水桶清洗不彻底
		- 细菌总数、大肠菌群超标现象时常出现
	- 与饮水机配套使用
		- 用户桶装式饮水机常温出水的大肠菌群的超标率高达20%～50%，随每桶水使用时间延长而增加
- 从健康考虑，理想的饮用水应该保留天然化学特性，即含有适量的矿物质和微量元素

**包装饮用水**
铜绿假单胞菌（绿脓杆菌），是一种常见的条件致病菌，在自然界分布广泛，为土壤中最常见的细菌之一，在水体、空气、正常人的皮肤、呼吸道和肠道等都有该菌存在。该菌存活的重要条件是需要潮湿的环境，环境湿度是影响包装饮用水成品安全的重要因素之一，因此对铜绿假单胞菌进行监测，有助于监控包装饮用水的成品安全

**直饮水**
- 是对自来水进行深度处理后，再将符合直接饮用标准的自来水通过优质输水管道送入用户，供居民直接饮用
- 直饮水的基本工艺流程为：自来水加压泵→多介质过滤器→活性炭过滤器→阳离子软水器→保安过滤器→反渗透机→臭氧发生器→不锈钢储水罐→变频恒压供水泵→优质供水管道
- 属于分质供水的范畴
	- 分质供水系统是指在一栋楼房、一个小区或一个城市内，除设有供生活用水的自来水供水系统外，还设有供人们直接饮用的净水系统
	- 分质供水的优点在于无需对所有的水进行深度净化，处理过程整体费用降低

# 6.涉及饮用水卫生安全产品
简称**涉水产品**，是指在饮用水生产和供水过程中与饮用水接触的连接止水材料、塑料及有机合成管材、管件、防护涂料、水处理剂、除垢剂、水质处理器及其他新材料和化学物质
- 水质处理器：如三碘树脂，五碘树脂和KDF等在起到杀菌作用的同时，往往使出水中的碘、锌等含量增加
- 生活饮用水输配水设备：供水用塑料管材、管件塑料本身的毒性；助剂的毒性；未聚合物及裂解产物的毒性
- 涂料：涂料的分子量越大越稳定，越不易溶出迁移
- 水处理剂：在使用过程中，一些有害物质如砷、镉、铅、汞、有机物，甚至放射性物质等溶入水中

# 7.饮用水卫生的调查、监测
## 7.1集中式供水
- 水源卫生调查：主要调查取水点及水源卫生防护的执行情况，必要时应检测水源水质
- 供水单位调查：①涉及饮用水卫生安全产品是否符合卫生安全和产品质量标准。②水处理剂和消毒剂的投加和贮存室。③各项管理制度。④水质净化消毒设施和必要的水质检验仪器、设备和人员，能否对水质进行日常性检验。⑤供、管水人员健康
- 水质监测：集中式供水单位，自建集中式供水及二次供水的水质检测

## 7.2农村供水
- 水源调查
- 水质监测：每个县采样点数一般不少于20个，但可酌情增减
- 水性疾病的监测：水性传染病、水性地方病、肿瘤及慢性非传染性疾病死因监测

# 8.饮水氯化消毒副产物
系指在氯化消毒过程中氯与水中的有机物（如腐殖酸、富里酸、藻类及蛋白质等）反应所产生的卤代烃类化合物
**种类：已鉴定清楚的仅有50%**
- 挥发性卤代有机物：主要有**三卤甲烷类物质（trihalomethanes, THMs）**，包括氯仿、一溴二氯甲烷、二溴一氯甲烷和溴仿
- 非挥发性卤代有机物：主要有卤代乙酸及卤代醛，如氯乙酸、二氯乙酸、三氯乙酸、溴乙酸、二溴乙酸、三溴乙酸、溴氯乙酸、二溴一氯乙酸、二氯一溴乙酸等

**影响氯化消毒副产物形成的因素：**
- 有机前体物的含量
- 加氯量、溴离子浓度以及pH等因素
	- 当有机前体物的含量一定时，投氯量越大，接触时间越长，生成的THMs越多
	- 水源水中溴化物浓度较高时，则会生成各种溴代THMs，含量往往高于氯仿
	- 与水的pH值有关，随着pH升高，三卤甲烷生成量增大，但卤乙酸生成量降低

**健康危害**
- 动物实验中证明具有致突变性和/或致癌性
	- 可引起肝、肾和肠道肿瘤
- 关于氯化饮用水与人群癌症关系
	- 在长期饮用氯化自来水的地区，人群中膀胱癌、胃癌、结肠癌、直肠癌的发病率较对照高
	- 亦有人持相反观点，认为饮用氯化水与癌症无关
- 对生殖有影响，可引起自然流产、死胎和早产以及出生缺陷

# 9.硬水危害
**健康危害**
- 出现腹泻和消化不良等胃肠道功能紊乱症状和体征
- 对皮肤敏感者，沐浴后还可有不舒适感
- 动物实验和现场调查结果揭示，硬水对泌尿系统结石的形成可能有促进作用

**日常生活影响**
- 如用硬水泡茶，会使茶水变味
- 用以烹调食物，会降低营养价值
- 用以洗衣，会增加肥皂消耗，并影响纺织物的色泽和柔软性
- 硬水煮沸，会在水壶和锅炉内沉积水垢，从而增加燃料消耗，并易使容器受损
- 用作住宅热水采暖时，因易积水垢使水管管腔变窄，甚至堵塞

# 10.高层建筑二次供水
又称高层建筑二次加压供水，它是指供水单位将来自集中式供水或自备水源的生活饮用水，贮存于水箱或贮水池中，再通过机械加压或凭借高层建筑形成的自然压差，二次输送至水站或用户的供水系统

**高层建筑二次供水污染**
- 污染的原因
	- 贮水箱(池)设计不合理
	- 贮水箱(池)容积过大，水箱储水量过多，超过用户正常需水量而滞留时间过长
	- 水箱、管道壁的腐蚀、结垢、沉积物沉积造成对水质污染
	- 管道内壁防腐涂料等不符合要求
	- 卫生管理不善，水箱无定期清洗消毒制度、无盖、无排水孔等
- 对健康影响
	- 直接结果是使饮用者感到恶心、呕吐、腹胀、腹泻，严重的甚至发病
	- 二次供水水质污染对健康的影响取决于污染的来源及污染物的性质

# 10.生物地球化学性疾病
**生物地球化学性疾病biogeochemical disease**
**定义**：系指在自然环境中，由于地壳中元素分布不均衡，有的地区土壤和水中某些元素含量过高或过少，超出了人类和其他生物的适应范围，致使当地动物、植物以及人群中发生某种特异性的疾病
本病往往明显局限于一定地区故亦称为**地方病**
**流行特征**
- 明显的地区性分布
- 与环境中元素水平相关
	- 疾病的流行强度与某种化学元素的环境水平有明确的剂量反应关系
	- 关联有普遍性和特异性

**主要分类**
- 地球化学性地方病
- 自然疫源性地方病
- 与特定生产生活方式相关的疾病
- 原因未明的地方病
- 新发地方病

**影响因素**
- 营养条件
- 生活习惯
- 多种元素的联合作用

**控制措施**
- 组织措施
- 技术措施

**种类**
- 碘缺乏病（iodine deficiency disorders, IDD）
- 地方性氟中毒（endemic fluorosis）
- 地方性砷中毒（endemic arseniasis）
- 地方性硒中毒（endemic selenosis）
- 克山病（Keshan disease）
- 大骨节病（Kaschin-Beck disease）

## 10.1碘缺乏病（iodine deficiency disorders）
**定义**：是指从胚胎发育到成人期由于摄入碘不足所引起的一系列病症，它对人的损害是连续性的。包括地方性甲状腺肿、地方性克汀病、地方性亚临床克汀病等
**碘的分布**
- 以碘化物形式存在，易溶于水
- 山区水碘低于平原，平原低于沿海

**碘的代谢**
![[碘的代谢|700]]
**甲状腺激素的生理作用**
 - 促进生长发育
 - 维持正常新陈代谢
 - 影响蛋白质、糖和脂类的代谢
 - 调节水和无机盐
 - 维持神经系统正常功能

碘最低生理需要量：75µｇ/天/人

**流行特征**
- 地区分布：山区高于丘陵、丘陵高于平原，平原高于沿海；内陆高于沿海。内陆河的上游高于下游，农业地区高于牧区
- 人群分布：任何年龄。发病年龄一般在青春期，女性早于男性；患病率女性高于男性。甲状腺肿大在儿童期开始出现，青春发育期发病率急剧增高，40岁以后逐渐下降。重病区年龄组有提前和后移现象。以15~20岁年龄组，两性差异最大

### 地方性甲状腺肿(Endemic goiter)
- 主要由于地区性环境缺碘引起的地方病，是碘缺乏病的主要表现形式之一，主要症状是甲状腺肿大
- 发病原因
	- 缺碘：饮水、食物及土壤
	- 致甲状腺肿物质（goitrogenic substance）：有机硫化物、某些有机物（生物类黄酮等）、某些无机物（水中钙镁，硝酸盐等）
		- 致甲状腺肿物质：除碘缺乏外，其他能影响或干扰甲状腺激素合成、释放、代谢，引起甲状腺肿大的物质
	- 其他原因：营养因素，过量碘摄入；某些药物；维生素的缺乏
- 发病机制：下丘脑-垂体-甲状腺轴
	- 碘↓👉甲状腺激素合成↓👉促使垂体前叶分泌TSH↑👉甲状腺组织出现代偿性增生👉异常甲状腺球蛋白堆积于滤泡👉胶质性甲状腺肿👉退行性变
- 临床表现
	- 甲状腺肿大
	- 伴随周围组织压迫症状
		- 压迫气管可引起呼吸不畅、呼吸困难
		- 压迫食管引起吞咽困难
		- 压迫喉返神经导致声音嘶哑
		- 颈交感神经受压可使同侧瞳孔扩大，严重者出现Horner综合征（眼球下陷，瞳孔变小，眼睑见下垂）
- 诊断
	- 标准：居住在病区；甲状腺肿大超过本人拇指末节，或小于拇指末节而有结节；排除甲亢、甲状腺炎、甲状腺癌等；尿碘低于50µg/g肌酐
	- 分型
		- 弥漫型：甲状腺均匀增大，摸不到结节
		- 结节型：在甲状腺上摸到一个或几个结节
		- 混合型：在弥漫肿大的甲状腺上，摸到一个或几个结节
	- 分度
		- 按肿大程度可分：正常； Ⅰ度 ；Ⅱ度 ；Ⅲ度 ；Ⅳ度

### 地方性克汀病(Endemic cretinism)
- 碘缺乏地区出现的一种比较严重碘缺乏病的表现形式。主要表现智力低下、体格矮小、聋哑、生长发育落后、神经运动障碍、甲状腺功能低下、甲状腺肿。概括为呆、小、聋、哑、瘫
- 多出现在严重的地方性甲状腺肿流行区，患病率可达1%～5％，甚至 5％～10％
- 临床表现
	- 神经型：表现为精神缺陷、聋哑、神经运动障碍，没有甲状腺功能低下的特点
	- 粘液水肿型：具有甲状腺功能低下的全部特点：皮肤干燥与肿胀，声音嘶哑、表情淡漠、智力缺陷；生长迟滞和侏儒
	- 混合型
- 诊断
	- 必备条件：出生、居住在碘缺乏区；有精神发育不全，主要表现在不同程度的智力障碍
	- 辅助条件：神经系统症状；甲状腺功能低下症状
	- 具有上述必备条件，再具有辅助条件之一或一项以上

### 亚克汀病
以轻度智力落后为主，伴有神经系统轻微受损和生长发育迟缓。此病因临床异常表现不明显，易被人们所忽视，实际其患病率远高于典型克汀病

**碘缺乏病的预防措施**：补碘（碘盐、碘油、其他）

**高碘甲状腺肿致病机理**
- 碘摄入量高，使甲状腺内碘过多，甲状腺激素和甲状腺球蛋白增多，潴留在滤泡腔内，使滤泡内胶质增多，引起腺体增大
- 高碘可抑制甲状腺素的合成和释放，从而反馈地促使甲状腺代偿性增生
- 过量的碘可以引起碘离子进入甲状腺上皮细胞受阻，发生碘阻断效应，致使甲状腺素合成减少，腺体增生

## 10.2地方性氟中毒（Endemic fluorosis）
**定义**：由于一定地区的环境中氟元素过多，而致生活在该环境中居民经饮水、食物和空气等途径长期摄入过量氟而发生的一种慢性全身性疾病，主要表现为氟斑牙(dental fluorosis)和氟骨症(skeletal fluorosis)
**氟在自然界中的分布**：
- 分布广泛、以化合物形式存在
- 水、空气、食物

**氟的代谢**
![[3.水体与健康-生物地球化学性疾病-20260401|700]]

**生理作用**
- 构成骨骼和牙齿的重要成分
- 促进生长发育和生殖功能
- 对神经肌肉的作用
	- 通过抑制胆碱酯酶、三磷酸腺苷酶活性起作用

**发病原因**：病区的地质地理和气候条件
- 环境介质含氟量高
- 居民饮用水含氟量超过相应国家卫生标准
- 室内空气氟浓度高
- 食品含氟量超过相应国家卫生标准

**病区类型与分布**
- 饮水型：即长期饮用高氟水而患病
	- 患病率与饮水氟含量呈正相关。分布广，流行史最长，90％的患者属此型
- 燃煤型：即在特定地区煤中含氟量过多，当地居民采用落后的燃煤方式（敞开燃煤、烘烤粮食），致使室内空气和烘干食物中氟含量增加而导致
- 饮茶型：长期饮用含氟过高的茶叶而引起

**流行病学特征**
- 年龄分布
	- 氟斑牙主要和饮用高氟水起始年龄有关，一般乳牙患病较少，恒牙多见，以7~15岁发病率最高
	- 氟骨症多见于成年人，一般在10或15岁以后发病，患病率和病情均随年龄增加而升高和加重
- 性别分布
	- 一般无明显的性别差异，但有些病区，女性患者的病情较重，可能与妇女生育、授乳有关
- 居住时间
- 一般与民族职业关系不大
	- 前提：生活习惯、生活条件不影响摄氟量
- 其他影响因素：
	- 营养条件、钙离子浓度、硬度、pH值、气候、个体差异

**发病机制**
- 对骨组织和钙磷代谢的影响
	- 抑制肾小管对磷的重吸收，磷代谢紊乱
	- 骨质脱钙或溶骨
- 对牙齿的影响
	- 骨质硬化、骨密度增加
	- 牙釉质钙化受损，出现弥漫性钙化不全或疏松多孔区
- 影响其他组织
	- 神经系统、肌肉、肾脏、血管和内分泌腺等

**临床表现**
- 氟斑牙
	- 釉面光泽度改变
	- 釉面着色
	- 釉面缺损
- 氟骨症
	- 症状：疼痛、神经症状、肢体变形
	- 体征：硬化型、混合型
	- X线表现：骨结构改变、骨周改变、关节改变
- 非骨相氟中毒
	- 神经系统损害：神经根损害症状为首发
	- 骨骼肌损害
	- 肾脏、肝脏及其他损害

**预防措施**
- 饮水型
	- 改换水源
	- 饮水除氟
- 燃煤型
	- 改良炉灶
	- 减少食物氟污染
	- 不用或少用高氟劣质煤
- 饮茶型

**治疗原则**
- 减少摄入和吸收
- 促进排泄
- 增强体质
- 平衡膳食等

## 10.3地方性砷中毒（endemic arseniasis）
**定义**：由于长期自饮用水、室内烟煤、食物等环境介质中摄入过量的砷而引起的以末梢神经炎、皮肤色素代谢异常、掌跖部皮肤角化、肢端缺血坏疽、皮肤癌变为主要表现，伴有多系统、多脏器受损的慢性、全身性疾病

**在自然界的分布**
- 多以砷化合物和硫砷化物形式存在
- 地表水中含量因地理地质条件不同而差别很大
- 不同地区煤炭含砷量多少不等

**体内代谢**
- 吸收：呼吸道、消化道、皮肤粘膜
- 体内分布：实质性脏器、骨组织、皮肤、指（趾）甲、毛发
	>发砷和指（趾）甲砷反映砷在体内的蓄积水平
- 排泄：肾脏为主要排泄器官；粪便
	>尿砷反映机体砷暴露水平

**毒作用机制**
- 抑制酶的活性
- 诱导细胞凋亡
- 致癌致突变

**流行特点**
- 地理、地质和气候条件
	- 饮水型：平原、盆地或高原；干旱、少雨、多风
	- 燃煤型：高寒地区
- 性别
	- 患病率男高于女
- 年龄
	- 各年龄段。患病率随在病区居住年限增加而升高

**临床表现**
- 特异表现
	- 末梢神经炎症状，呈四肢对称性、向心性感觉障碍
	- 皮肤色素异常：色素沉着与色素缺失同时出现
	- 皮肤角化、皲裂
	- 间歇性跛行、皮肤发黑、坏死
- 多系统多脏器损害
	- 肝、神经系统、肾脏、血液、生殖系统、心血管系统
- 致癌致畸作用
	- 皮肤癌、肾癌、肝癌居多

**预防措施**
- 改换水源
- 饮水除砷
- 限制高砷煤炭的开采使用
- 改良炉灶以减少室内空气砷污染

## 10.4与硒相关的生物地球化学性疾病
**大骨节病(Kaschin-Beck disease)**
- 一种地方性、慢性**骨关节变形**性疾病，以**四肢关节软骨和骺板变形、坏死、增生、修复**为主要病理改变，以**骨关节增粗、畸形、强直、肌肉萎缩、运动障碍**为主要临床表现。阿坝州位于四川省西北部，是全国罕见的大骨节病重病区
- 大骨节病在我国已经基本没有新发病例
- 病因假说
	- 环境硒水平低
	- 饮水中有机物中毒：黄腐酸
	- 真菌毒素中毒：镰刀菌可产生T-2毒素

**克山病(Keshandisease)**
- 一种以**心肌变性坏死**为主要病理改变的生物地球化学性疾病。1935年在我国黑龙江省克山县发现大批急性病例，主要表现为**心脏扩大，心力衰竭，心律失常**。其病因未明，因此称为克山病
- 病因假说
	- 环境硒水平过低
	- 生物感染因素
	- 膳食中营养失衡

**地方性硒中毒**
- 我国湖北省恩施县部分地区的一种地方病，症状以脱甲和脱发为主

## 10.5其它罕见的地方病
**云南不明原因猝死（云南猝死）**
- 多为恶性心律失常引起的心源性猝死，是一种严重威胁病区群众健康的特殊地方病，也是云南省重要的公共卫生问题之一
- 青壮年高发，民族、家庭和村庄聚集性明显。该病病因至今尚未完全明确，主要病因研究包括毒沟褶菌（小白菌）中毒、肠道病毒感染、遗传性心脏病、克山病病因假说等
- 在推行以禁食小白菌为主的健康教育，改善病区生产生活条件，加强病区人群健康监测等综合性干预措施后，近年发病率呈明显下降趋势
