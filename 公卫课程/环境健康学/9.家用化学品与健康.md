# 1.家用化学品的健康危害
## 1.1日用化学品
- 日用化学品是指用于家庭日常生活和居住环境的化工产品，包括用于办公室和公共场所的化学制品
- 广义的日用化学品是指除职业环境以外用于人们的日常生活、学习、办公、交通等活动过程的化学产品，包括化妆品、洗涤剂、化学消毒剂、黏合剂、涂料、家用杀（驱）虫剂和生活中使用的化学纤维制品、汽车护理产品等
- 日用化学品已广泛渗透到人们生活的方方面面。因此，日用化学品具有**使用分散、需求量大、暴露人群广泛（包括各年龄段）和暴露时间长**等特点
- 化妆品是指以涂擦、喷洒或其他类似方法，散布于人体表面（皮肤、毛发、指甲、口唇等）、牙齿及口腔黏膜，以清洁、保护、美化、修饰以及保持其处于良好状态为目的的产品

**化妆品的种类**
- 护肤类化妆品：洗面奶、浴液、护肤霜
- 益发类化妆品：洗发露、护发素、发油.…
- 美容修饰类化妆品：香粉、胭脂类、口红、美白霜、睫毛膏、眼影膏、指甲油
- 芳香类化妆品：香水、花露水、爽身水、化妆水.…
- 口腔卫生用品：牙膏、牙粉、牙线
- 特殊用途化妆品：用于染发、烫发、脱毛、健美、除臭、祛斑和防晒等的化妆品

## 1.2化妆品对健康的影响
- 刺激性接触性皮炎：化妆品引起皮肤损伤中最常见的病变
- 变应性接触性皮炎：T细胞介导的皮肤迟发型变态反应性组织损伤
- 化妆品光感性皮炎：化妆品中光感物质经过光照引起皮肤黏膜局部炎症性反应
- 化妆品痤疮：由化妆品引起的面部痤疮样皮疹
- 化妆品皮肤色素异常：化妆品引起的皮肤色素沉着或色素脱失
- 化妆品毛发损害：使用化妆品后引起的毛发损伤
- 化妆品甲损害：使用甲化妆品所致的指（趾）甲本身及甲周围组织的病变
- 化妆品眼损害：化妆品引起的眼部损害，包括眼部接触性皮炎
- 化妆品中微生物污染的危害：生产过程及使用过程中均可受到污染
- 化妆品所含化学物质的毒性作用：毒性化合物、致癌物、重金属

**刺激性接触性皮炎(irritant contact dermatitis, ICD)**
- 由化妆品理化性质刺激引起的皮肤局部表浅的**渗出性炎症反应**
- 是化妆品引起皮肤损伤中**最常见**的病变
- **临床表现**：主要发生在涂擦化妆品的部位，皮损**限于**接触部位，边界清楚；呈红斑、丘疹或疱疹，皮肤烧灼或痛感 
- **停用**致病化妆品后症状可消退
- ICD的发生与化妆品含有的原发刺激物、pH、脂溶性溶剂、产品污染变质及个体皮肤敏感性有关

**变应性接触性皮炎(allergic contact dermatitis, ACD)**
- 化妆品中含有的变应原物质引起的皮肤**迟发型变态反应**
- 一般在初次使用或多次使用含变应原物质的化妆品后，经一段**潜伏期再次接触**同样的化妆品时出现
- **临床表现**：主要为瘙痒，皮损形态多样；丘疹边界不清、红斑鳞屑、局部红肿等
- **再次接触**时出现症状的时间大为缩短，皮损更严重
- **过敏性体质**是发生ACD的主要原因。此外，也与年龄、接触部位、皮肤状态、个体健康状况、服用药物等相关
- 常引起ACD的化妆品：护肤霜膏类、染发剂、香水等
- 最常引起ACD的化妆品原料
	- 香料、防腐剂、乳化剂、羊毛脂等含变应原的化妆品原料
	- 氧化染发剂：苯胺类染料 对苯二胺、氨基苯酚、间苯二酚等
	- 含有生物活性成分的化妆品： 羊毛脂、蜂浆等含有异种蛋白，细胞生长因子等

**刺激性接触性皮炎与变应性接触性皮炎的鉴别**

| 鉴别点  | 刺激性接触性皮炎                         | 变应性接触性皮炎                       |
| ---- | -------------------------------- | ------------------------------ |
| 发病   | 急，使用后短期内出现                       | 慢，施用数天后缓慢出现                    |
| 病程   | 短，避免接触后皮损减轻                      | 长，停止接触后皮损可持续                   |
| 病因   | 化妆品含有的刺激物                        | 化妆品中含有的变应原                     |
| 多发人群 | 以常施用者为多见                         | 多为过敏体质                         |
| 临床表现 | 皮疹边界清；常局限于接触部位；呈红斑、丘疹或疱疹；皮肤烧灼或痛感 | 皮疹边界不清；可超出接触部位；呈湿疹样变，形态多样；瘙痒明显 |

**化妆品光感性皮炎(photosensitive dermatitis induced by cosmetics)**
使用化妆品后，其中的光感物质经过光照而引起的皮肤粘膜局部炎症性反应，又分为**光变应性反应**和**光毒性反应**
- 光变应性接触性皮炎(photoallergic contact dermatitis, PCD)
	- 指局部使用含有光变应原物质的化妆品后，在**接触日光**的部位出现皮肤炎症反应，而**不接触光**的皮肤则不出现此种反应
	- **T淋巴细胞**参与介导的湿疹样反应，大多数光变应性反应以**湿疹样变**为特征
	- 含有光变应原的化妆品常见的是**防晒剂、染料和香水类**。防晒剂中的对氨基甲酸及其酯类物质，香料中的葵子麝香、肉桂醛，以及煤焦油染料类物质
- 光毒性皮炎(phototoxic dermatitis)
	- 指化妆品中某些物质能增加皮肤对光的敏感性，而产生光毒性反应导致皮肤损伤
	- **临床特点**为皮肤的红斑反应及红斑消退后的**色素沉着**。限于直接暴露部位，常于光照后数小时发病    
	- **引起光毒性皮炎的主要化妆品**：防晒剂（氨基苯甲酸酯）；香水（香精油：柠檬油、檀香木油、薰衣草油）
- **光毒性反应与光变应性反应的主要区别**：前者只要接触了引起光毒性反应的物质，经一定波长和强度的紫外线（**UVB、UVA**）或可见光照射，**人人均可能发生**；后者由于接触者机体**自身具有的特应性（atopy）**

**光毒性皮炎与光变应性接触性皮炎的鉴别**

| 区别点  | 光毒性皮炎                 | 光变应性接触性皮炎            |
| ---- | --------------------- | -------------------- |
| 病因   | 化妆品中的化学物质增加皮肤对光的敏感性   | 化妆品中的光变应原物质光照后诱发过敏反应 |
| 病理   | 单纯炎症反应                | T细胞介导的湿疹样反应          |
| 部位   | 多发生在皮肤向光的部位，如手臂伸侧、面部等 | 施用后在接触日光的部位，可累及非接触部位 |
| 发病时间 | 首次接触光毒性化合物和受光照后出现     | 起病缓慢，再次接触时反应迅速       |
| 皮损特征 | 红斑反应和消退后的色素沉着         | 湿疹样反应（疱疹、脱屑、结痴）      |
| 患者特点 | 人人均可发生                | 过敏体质者                |

**化妆品痤疮(acne induced by cosmetics)**
- 由化妆品引起的面部痤疮样皮疹
- 是**仅次于**接触性皮炎的常见化妆品皮肤病    
- 易引起痤疮的化妆品包括：护肤类的面脂、面霜；美容修饰类的粉底、油彩，含粉质较多的增白霜等
- 主要是由于这类化妆品的**基质**，如凡士林、石蜡油等矿物油、羊毛脂
- 由于化妆品**堵塞皮脂腺汗腺毛囊口**，或在患者皮脂腺分泌旺盛的情况下不当使用，增加毛囊堵塞的机会，皮脂不能顺畅排出，积聚而形成
- 皮疹表现在接触部位出现与毛孔一致的黑头粉刺、炎性丘疹及脓疱，或在原有痤疮的基础上施用后症状加重

**化妆品皮肤色素异常(skin discolouration)**
- 指施用化妆品引起的皮肤色素沉着或色素脱失，以**色素沉着**为多见
- **发生原因**：① 化妆品直接染色；② 化妆品刺激皮肤色素增生；③ 继发于化妆品接触性皮炎或光感性皮炎
- 多局限于施用化妆品的部位
- 化妆品中所含的香料、颜料、防腐剂、表面活性剂等成份

## 1.3洗涤剂与健康
**定义**：洗涤剂是指能够去除物体表面污垢的一类专门配方制品的总称，主要通过洗涤过程来达到去污保洁目的

**洗涤剂的种类**
- 纤维织物洗涤剂：用于衣服、被褥、地毯、毛皮等的洗涤剂
- 硬表面洗涤剂：用于金属、玻璃、餐具、卫生器具等的洗涤剂
- 个人清洁洗涤剂：如香皂、洗发香波、沐浴露、洗手液、口腔卫生用品、除臭剂等
- 特殊用品洗涤剂：如全功能液体清洁剂、地板蜡清洁剂、墙壁清洁剂、酸性洗涤剂、家用长效除臭剂和空气清新剂等

**洗涤剂对健康的影响**
- 皮肤损害：包括原发性刺激、变应性反应、局部或全身皮疹、继发细菌或霉菌感染
- 呼吸系统损伤：经常使用人群发生哮喘和其他呼吸道症状的危险性增加
- 中毒：含有特定化学物质（如硫化硒）的洗涤剂可导致中毒反应，并引起其他慢性损害

## 1.4化学消毒剂与健康
**定义**：化学消毒剂是指用于杀灭病原微生物的化学药物。有些化学消毒剂杀灭微生物的能力强，可以达到灭菌效果，称之为灭菌剂。只能抑制微生物生长而不能将其杀灭的化学消毒剂称为抑菌剂。

**化学消毒剂的种类**
- 含氯消毒剂
- 过氧化物消毒剂
- 醛类消毒剂
- 杂环类气体消毒剂
- 醇类消毒剂
- 酚类消毒剂
- 季铵类消毒剂
- 其他类消毒剂

**化学消毒剂对健康的影响**
- 次氯酸钙
	- 又称漂白粉，为白色粉末，有氯的气味，对物品有漂白和腐蚀作用
	- 皮肤长期接触可引起中、重度皮肤损害，高浓度溶液可引起皮肤强烈刺激及腐蚀
	- 其粉末对眼结膜和呼吸道有刺激作用，过量吸入可引起肺水肿甚至死亡
	- 低剂量长期反复吸入可导致慢性支气管炎
	- 含氯消毒剂与含酸洗涤剂混合使用产生氯气，会引起氯气中毒
- 过氯乙酸
	- 无色透明液体，有刺激性酸味，很强的腐蚀性、挥发性和氧化性
	- 对皮肤、眼睛和上呼吸道黏膜有强烈刺激作用，可引起呼吸系统炎症、水肿
- 环氧乙烷
	- 无色气体或液体，具有乙醚气味，易燃易爆
	- 是一种致癌物和中枢神经抑制剂，也具有致敏作用，可损伤皮肤和眼睛
	- 可导致急性中毒，引起搏动性头痛、恶心呕吐，出现心肌损害和肝功能损害

# 2.家用化学品危害的防控
**化妆品卫生标准与规范**
- 1987年，原卫生部颁布《化妆品卫生标准》 
- 1989年，原卫生部颁布《化妆品卫生监督条例》
- 1996年，《化妆品生产企业卫生规范》
- 1999年，《化妆品卫生规范》
- 多次修订《化妆品卫生规范》2002版 / 2007版    
- 国家食品药品监督管理总局《化妆品安全技术规范》2015年版

**化妆品生产经营的卫生监督与管理**
- 1989年颁布《化妆品卫生监督条例》
- 1991年颁布《化妆品卫生监督条例实施细则》
- 1996年颁布《化妆品生产企业卫生规范》
- 2001年实施更新的版本
- 2014年启动《化妆品卫生监督条例》修订工作

**《化妆品安全技术规范》主要特点**
- 化妆品安全性保障进一步提高：根据科学合理、保障安全的原则，调整了化妆品中的禁限用组分要求，调整了部分准用组分的限量要求和限制条件
- 适应性与可操作性进一步提高：对规范中涉及的名词与术语提供了释义，细化明确了相关概念并重点增加了技术标准和要求

**化妆品生产经营的卫生监督与管理**
- 生产的卫生监督与管理
	- 化妆品原料和产品的安全性评价
		- 化妆品在生产或销售之前，对其原料和产品进行安全性评价是预防性卫生监督的重要内容之一
		- 使用化妆品新原料、特殊用途化妆品在投放市场前均须进行安全性评价
		- **首次进口**到我国的化妆品，应由国外厂商或代理商提供化妆品的名称、种类、产品成份、限用物质含量、质量标准及检验方法、出口国批准生产的证明文件，**此外还须提供卫生安全性评价资料或卫生质量检验报告等有关资料。**
	- 化妆品生产企业的卫生监督：应申请行政许可证/备案登记凭证；生产卫生条件监督；工人健康体检；原料、容器和包装材料应符合卫生规范要求
- 经营的卫生监督与管理
	- 卫生监督条例规定化妆品经营单位和个人不得销售无卫生许可证企业生产的产品及不符合其他规范的产品
	- 网络化妆品经营销售也是监督的重点

**其他家用化学品的监督与管理**
- 洗涤剂：洗涤剂安全标准；严格要求洗涤剂的生物降解率和排放标准
- 涂料：两项国家强制标准进行规定监督，以及一系列用于食品容器内壁涂料的卫生标准
- 消毒剂：《消毒管理办法》&《消毒技术规范》

# 3.新污染物
## 3.1新污染物概述
**定义**：新污染物是指在环境中产生或正在使用的具有难降解、长距离迁移、生物可累积、毒性与健康危害的化学品或微生物

**新污染物具有以下共性特征**
- 多数为目前正在生产和应用的产品，尚未有效控制其生产和排放
- 污染正在发生，环境介质中的存量较高，对其环境行为缺乏足够认知
- 缺乏毒理与健康风险的数据，缺乏全面的科学评估

**典型的新污染物**：持久性有机污染物（POPs）、环境内分泌干扰物（EEDs）、抗生素、微塑料

- 对心血管疾病的影响
