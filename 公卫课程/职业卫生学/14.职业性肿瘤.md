# 1.概述
## 1.1肿瘤
**肿瘤研究历史和认识**
- 20世纪建立**肿瘤登记制度**，癌症是全世界引起死亡的主要原因，癌症发生率随城市化、工业化增加
- **癌症部位随时间变化趋势**：胃癌逐年减少，结肠癌、直肠癌、乳腺癌、前列腺癌与经济发展相平行
- **环境因素与肿瘤发生关系密切**

**致癌因素**：致癌因素就是能诱发恶性肿瘤的化学物质或其他因素
- 物理因素
- 化学因素：约40种，恶性肿瘤80-90%与化学因素有关
- 生物因素：HBV、HPV

## 1.2职业性肿瘤
**职业性致癌因素(occupational carcinogen)**
- 定义：与**职业**较长时间接触有关的，在一定条件下能使正常细胞转化为肿瘤细胞，且经过较长潜隐期能发展为可以检出肿瘤的致病因素
- 职业性致癌因素：机械刺激、物理因素、化学因素、生物因素(EB、HBV)

==**职业性肿瘤(occupational tumor)**==
- 定义：在**工作环境**中长期接触**职业致癌因素**，经过较长的潜伏期而患某种特定肿瘤，也叫职业癌(occupational cancer)
- 研究意义
	- 可以肯定职业性致癌因素的作用，有助于探索人类肿瘤的病因
	- 致癌因素一经识别，就可以采取预防措施，使发病率下降

==**职业肿瘤诊断标准**==
- 肿瘤临床诊断明确
	- 原发肿瘤
	- 肿瘤发生部位与职业性致癌因素靶器官一致
	- 组织病理检查（临床影像、内镜检查）
- 职业性致癌物接触史明确
	- 接触时间
	- 接触状况
- **潜隐期**符合要求

# ==2.职业致癌因素的识别和确认==
## 2.1临床观察——发现职业性肿瘤
 - 病例分析对病因探索可提供线索，可成为进一步研究的起点
 - 临床病例分析与观察是识别职业致癌因素的第一线索
## 2.2实验研究
- 可疑致癌物——动物诱癌实验（首选）
- 观察**能否诱发与人类相似的肿瘤**
- 判断是否具有致突变性或诱导染色体损伤的能力
- 实验肿瘤学上推断其致癌性，是证明职业因素致癌的重要依据之一
- 动物实验、体外试验

**实验研究**
- 整体动物实验
	- 动物实验设计的基本要求
		- 要用两种动物（一般为大鼠和小鼠），各组雌雄各半
		- 每个实验组和对照组有足够的动物数，每种性别至少50只
		- 投药和观察时间必须能够超过该种动物的期望寿命的大部分（大鼠和小鼠为2年）
		- 在实验组中，应至少有两个剂量组，高剂量组应接近最大耐受剂量
		- 结果的确定要有足够数量的病理学检查
		- 用合适的方法对资料进行统计学分析
	- 转基因小鼠和基因敲除小鼠——致癌作用模型
		- 转基因小鼠：20世纪80年代发展的转基因技术，借助基因工程技术将确定的外源基因通过生殖细胞或早期胚胎导入动物个体的染色体上，在基因组内稳定组合，并遗传到下一代
		- 肿瘤病因、发病机制的研究：建立带有癌基因的转基因动物模型研究癌基因的活性和肿瘤发生的新方法。检测一些抑癌基因在肿瘤发生中起的作用，可以识别其他与肿瘤发生相关基因
	- 动物实验结果的解释应注意
		- 引起动物致癌的化学物质是否已被证明为人类致癌物
		- 动物实验剂量是否可外推到人
		- 即使动物实验结果与人有较强的相关性，但致癌的部位不同即靶器官不同
			- 联苯胺致大鼠、小鼠和仓鼠肝癌，人、狗膀胱癌
		- **用动物实验来预测人类致癌物要慎重**
- 体外实验
	- 短期内观察、检测某些化学物的致突变性或诱导染色体损伤的能力，从而判断致癌性
	- 依据：肿瘤的发生是由于DNA突变引起，用短期试验检测该化合物是否有致突变性，判断其是否有致癌的可能性
	- 确定实验的化合物是否致癌——整体动物实验加以验证
	- 常用的体外实验
		- Ames实验一一检测化学物诱导DNA基因突变
		- DNA修复实验一一证明DNA暴露于一种化合物时发生损伤(DNA加合物)
		- 姊妹染色单体互换试验一一判断化合物对遗传物质的影响
		- 染色结构畸形分析一一检测化学物对细胞染色体的损伤
		- 哺乳动物细胞恶性转化实验一一判断加入细胞培养液的化学物是否具有使培养的细胞向恶性转化的能力。这种恶性转化的细胞注射给动物——肿瘤发生
		- 癌基因和抑癌基因检测——一组基因表达的上调或下调
	- 注意事项
		- 目前主张用一组短期的体外试验来检测化学物的致癌性
			- 低等动物、高等动物
			- 体内、体外实验
			- 体细胞、生殖细胞
		- 体外试验结果的评价
			- 与整体动物实验的一致性为60%

**实验研究判断化学物是否致癌**
- 体外实验，如呈阳性则应作整体实验和接触人群中做进一步的研究
- 体外实验和整体动物实验为阳性——可疑致癌物
## 2.3流行病学调查
- 流行病学调查对于识别和判定某种因素对人的致癌性提供**最有力的证据**。可以作为职业性致癌因素的判断标准
- 病例报告和描述性流行病学研究对致癌性研究只能提供建议性的证据，分析性流行病学研究可对致癌的因果关系得出结论
- 作用
	- 从**群体**的角度研究职业肿瘤的致病原因
		- 患者→作业环境和条件→健康（因果关系、致癌条件）
		- 目前接触的有害因素→接触者健康
	- 提示有某种职业性致癌因素存在
		- 出现异常集群肿瘤病理(abnormal cluster cases of cancer)
		- 肿瘤高发年龄提前：40岁
		- 肿瘤病例性别比例异常：男女比例接近
		- 肿瘤发生与同一种因素有关
		- 剂量-反应关系
		- 出现罕见肿瘤高发现象：间皮瘤
- **化学物为致癌物在流行病学的判断标准**
	- 因果关系的强度：相对危险度（RR值）
	- 因果关系的一致性：多中心研究
	- 接触水平一反应关系
	- 生物学上的合理性：医学基础（接触毒物剂量、年龄）
	- 时间的依存性：在时间上，作为原因的“接触”必须在作为结果的“效应”之前
- 流行病学研究的局限性
	- 人口的流动性——失访率增加
	- 暴露环境改变——接触剂量
	- 单一因素——研究对象困难
	- 长潜伏期的需要长时间的随访，缺乏敏感的流行病学方法

## 2.4职业性致癌因素
- 某些生产过程，研究结果明确，参加这些生产过程的人群中，某种特定的肿瘤的发生与该生产过程有关，但研究结果又不能确定与该生产过程中的那种因素有关
- 国际癌症研究中心：生产过程界定为职业性致癌因素

**确认的对人致癌有关的生产过程及致癌部位**

| 生产过程    | 致癌部位       |
| ------- | ---------- |
| 铅生产     | 肺、膀胱       |
| 靴鞋制作及修理 | 白血病        |
| 煤气制造    | 肺、膀胱、皮肤、阴囊 |
| 家具制造    | 鼻腔         |
| 品红制造    | 膀胱         |
| 铸铁和铸钢   | 肺          |

# 3.职业性肿瘤的发病特点
**职业性肿瘤(occupational tumor)**：在工作环境中长期接触职业性致癌因素，经过较长的潜隐期而患某种特定肿瘤
**职业癌(occupational cancer)**

==**发病特点**==
- 病因明确：职业性致癌因素
- 好发部位：较固定（**接触部位**）
- 潜隐期——发病年龄
	- **潜隐期(latency)**：从接触已确认的致癌物到确诊该致癌物所致职业性肿瘤的间隔时间
	- 潜隐期较长，10年以上
	- 职业肿瘤的发病年龄较非职业性同类肿瘤病人年龄小
- 病理类型
	- 有特定的病理类型
	- **恶性度高**
		- 苯导致的白血病类型不同，无规律
- 发病条件
	- 接触物的理化性质
		- 形态：颗粒状金属镍致癌；块状金属镍无致癌性
		- 结构：β-萘胺为强致癌剂；α-萘胺为弱致癌剂
	- 接触途径：不溶性的镍盐，吸入时致癌，经口或经皮无致癌性
	- 接触剂量和接触时间：与职业接触史有关
	- 个人生活习惯：吸烟促进多种职业性肿瘤的发生
	- 易感性

# 4.常见的职业性肿瘤
## 4.1职业性呼吸道肿瘤
**呼吸道致癌物种类**
铬、镍、砷、石棉、焦油化合物、氯甲甲醚、芥子气、放射性物质、硬木屑、氯丁二烯

**引起职业性肺癌的工种**
石棉、无机砷、二氯甲醚接触工、焦炉工、铀矿工、煤气发生炉工

石棉导致肺癌、胸腹膜间皮瘤
## 4.2职业性皮肤癌
**沥青**：在引起职业性皮肤癌之前出现癌前病变，如变黑、痤疮、乳头状瘤等，最常见于面、颈、前臂和阴囊
**砷**：癌前病变为四肢、面部出现过度角化、皮肤色素沉着、溃疡形成，最后可发展为扁平细胞角化癌。腺瘤
**X-射线**：无合适防护者，潜伏期为4~17年，多见于手指。早期可见皮肤呈局灶性增厚、局部萎缩、指甲变脆等皮炎表现，皮肤癌
## 4.3职业性膀胱癌
- 职业性膀胱癌在职业肿瘤25%~27%，与患者所从事的职业有明显关系，染料、橡胶、化工、美容美发、皮革业、印刷、纺织品印染、焦油、油漆和铝工业等行业的工人
- 芳香胺类物质为主要致膀胱癌的物质
- 萘胺、联苯胺、4-氨基联苯
- 联苯胺是一种无色结晶状化合物，其盐酸盐和硫盐均为染料的中间体，用于合成染料
## 4.4其他
- **接触氯乙烯致肝血管肉瘤**
- 苯接触致白血病：多数出现在接触后数年~20年，短者仅4~6月，长者为40年。苯中毒引起的白血病常常继发于全血细胞减少或再生障碍性贫血

# 5.职业性肿瘤的预防原则
- 严格执行职业卫生标准、法律：职业病防治法
- 建立并完善职业肿瘤监测体系：早发现、诊断、治疗；健康档案、报告体系
- 加强对职业性致癌因素的控制和管理
	- 用非致癌物代替致癌物
	- 改革操作工艺、减少直接接触机会
	- 改善劳动条件，减少有害气体、粉尘在劳动场所的跑、冒、滴、漏，对于有致癌因素存在的生产环节，应采用密闭生产或有通风、回收装置，避免污染环境
	- 对致癌物采取严格管理措施，建立登记制度
- 健康教育，提高自我防护能力
- 生活方式，合理膳食——增加机体的抵抗力
	- 消除促癌因素：吸烟、饮酒
	- 体育锻炼：免疫力
- 建立预测制度
- 定期体检
	- 定期体检，早期发现癌前病变
	- 苯、放射性——血液系统
	- 石棉、铬酸盐——肺
	- 联苯胺——尿、膀胱B超
	- 人体血细胞、口腔上皮细胞、尿液膀胱上皮细胞的突变性——癌前病变
